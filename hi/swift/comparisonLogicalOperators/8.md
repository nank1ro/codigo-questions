# --description--

आइए **or** `||` लॉजिकल ऑपरेटर के साथ जारी रखें।
यह पहला ऑपरेंड लौटाता है जो *true* का मूल्यांकन करता है या अंतिम यदि सभी *false* हैं।
```swift
print(2 == 2 || 2 == 3) // true
print(1 == 2 || 1 == 3) // false
```
