# --description--

अंत में **not** `!` लॉजिकल ऑपरेटर देखें।
यह एक बूलियन लौटाता है जो किसी एक्सप्रेशन की लॉजिकल स्थिति के विपरीत होता है।
```swift
print(!true) // false
print(!false) // true
print(!(2 == 2)) // false
```
