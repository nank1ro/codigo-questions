# --description--

आइए **बराबर नहीं** `!=` तुलना ऑपरेटर के साथ जारी रखें।
यह एक **बूलियन** (`true` या `false`) लौटाता है जो बताता है कि दो एक्सप्रेशन **बराबर नहीं** हैं, उदाहरण के लिए:
```swift
print(2 != 2) // false
print(2 != 3) // true
```
यह *बराबर* ऑपरेटर के बिल्कुल विपरीत है
