# --description--

आइए **से बड़ा या बराबर** `>=` तुलना ऑपरेटर के साथ जारी रखें।
यह एक **बूलियन** (`true` या `false`) लौटाता है जो बताता है कि एक एक्सप्रेशन दूसरे से बड़ा या बराबर है या नहीं, उदाहरण के लिए:
```swift
print(2 >= 2) // true
print(3 >= 2) // true
print(3 >= 4) // false
```
