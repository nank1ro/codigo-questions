# --description--

अब आइए **लॉजिकल** ऑपरेटर देखें, पहले __AND__ `&&` से शुरू करते हैं।
यह पहला ऑपरेंड लौटाता है जो *false* का मूल्यांकन करता है या अंतिम यदि सभी *true* हैं।
```swift
print(2 == 2 && 2 == 3) // false
print(1 == 1 && 1 == 1.0) // true
```
