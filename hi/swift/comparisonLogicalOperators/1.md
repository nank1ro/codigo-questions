# --description--

आइए **बराबर** `==` तुलना ऑपरेटर से शुरू करें।
यह एक **बूलियन** (`true` या `false`) लौटाता है जो बताता है कि दो एक्सप्रेशन बराबर हैं या नहीं, उदाहरण के लिए:
```swift
print(2 == 2) // true
print(2 == 3) // false
```
