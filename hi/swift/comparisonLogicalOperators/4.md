# --description--

आइए **से छोटा** `<` तुलना ऑपरेटर के साथ जारी रखें।
यह एक **बूलियन** (`true` या `false`) लौटाता है जो बताता है कि एक एक्सप्रेशन दूसरे से छोटा है या नहीं, उदाहरण के लिए:
```swift
print(2 < 2) // false
print(2 < 3) // true
```
