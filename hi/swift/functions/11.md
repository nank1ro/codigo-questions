# --instructions--

इस कोड स्निपेट में __समस्या__ क्या है?
```swift
print(greet(name: "Mathew"))
/// This function greets to the person passed in as argument
func greet(name: String) -> String {
    return "Hello, \(name). Good morning!"
}
print(greet(name: "Alex"))
```

# --answers--

- हम greet() को परिभाषित करने से पहले इसका उपयोग नहीं कर सकते
- हम greet() को दो बार उपयोग नहीं कर सकते
- greet() कुछ भी वापस नहीं करता
- हम name को पैरामीटर के रूप में उपयोग नहीं कर सकते
