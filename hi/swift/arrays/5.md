# --description--

एक array की लंबाई निश्चित होना जरूरी नहीं है।
आप जब चाहें array के अंत में आइटम जोड़ सकते हैं!
Array में आइटम जोड़ने के लिए हम `append` फंक्शन का उपयोग करते हैं:
```swift
var letters = ["a", "b"]
letters.append("c")
print(letters)
// Prints ["a", "b", "c"]
```
