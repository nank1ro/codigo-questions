# --description--

Swift में हम `for..in` कीवर्ड्स का उपयोग करके बहुत सरल तरीके से array में लूप कर सकते हैं:
```swift
var numbers = [1, 2, 3]
for num in numbers {
    print(num)
}
// prints 1, 2, 3
```
`for` कीवर्ड के बाद एक वेरिएबल नाम आता है, इसे बारी-बारी से प्रत्येक array आइटम का मान असाइन किया जाएगा।
