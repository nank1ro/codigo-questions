# --instructions--

क्या यह कोड ब्लॉक कभी चलेगा?
```swift
var shouldRepeat = false

repeat {
    print("run code")
} while shouldRepeat
```

# --answers--

- हां क्योंकि repeat-while लूप की शर्त जांचने से पहले एक बार चलता है
- नहीं क्योंकि shouldRepeat false है
- यह कोड ब्लॉक दो बार चलेगा
- यह कोड ब्लॉक तीन बार चलेगा
