# --instructions--

हमें कैसे पता चलता है कि यह लूप नहीं रुकता?
```swift
var isLoading = true

while isLoading {
    print("Loading...")
    isLoading = true
}
```

# --answers--

- क्योंकि वेरिएबल कोड ब्लॉक के अंदर false में अपडेट नहीं होता
- क्योंकि सभी while लूप नहीं रुकते
- क्योंकि वेरिएबल false में अपडेट होता है
