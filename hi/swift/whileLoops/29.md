# --instructions--

इस कोड में क्या गलत है?
```swift
while isLoading {
    print("Loading...")
    isLoading = false
}
```

# --answers--

- isLoading एक वैध वेरिएबल नाम नहीं है
- while लूप में कोड ब्लॉक के अंदर एक print स्टेटमेंट है
- कोड ब्लॉक में सही इंडेंटेशन नहीं है
