# --instructions--

`counter += 1` क्या करता है?
```swift
var counter = 1

while counter < 5 {
    print(counter)
    counter += 1
}
```

# --answers--

- यह हर बार लूप के कोड ब्लॉक चलने पर counter को एक से बढ़ाता है
- यह counter वेरिएबल बनाता है
- यह हर बार counter को दो से बढ़ाता है
- यह हर बार counter को तीन से बढ़ाता है
