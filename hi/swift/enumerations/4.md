# --description--

कुछ enumerations के लिए, उस enumeration के सभी cases का संग्रह होना उपयोगी होता है।
आप enumeration के नाम के बाद `: CaseIterable` लिखकर इसे सक्षम करते हैं।
Swift enumeration प्रकार की `allCases` प्रॉपर्टी के रूप में सभी cases का संग्रह प्रदान करता है:
```swift
enum Colors: CaseIterable {
    case blue, red, green
}
for color in Colors.allCases {
    print(color)
}
// prints blue, red, green
```

# --instructions--

प्रत्येक case को प्रिंट करने के लिए enumeration पर इटरेट करें।
`$0` एक शॉर्टहैंड आर्गुमेंट नाम है जिसका उपयोग क्लोजर के आर्गुमेंट्स के मानों को संदर्भित करने के लिए किया जा सकता है।
`$0` क्लोजर के पहले आर्गुमेंट को संदर्भित करता है, `$1` दूसरे आर्गुमेंट को, और इसी तरह आगे
