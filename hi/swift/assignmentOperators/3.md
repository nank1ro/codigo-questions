# --description--

योग असाइनमेंट की तरह ही, हमारे पास **घटाव असाइनमेंट** `-=` है।
कार्यप्रणाली वही है, एकमात्र अंतर यह है कि यह घटाव करता है।
तो निम्नलिखित बिल्कुल समान हैं
```swift
var num = num - 5
// is equal to
num -= 5
```
