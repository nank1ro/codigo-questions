# --instructions--

वेरिएबल `number` को पूर्णांक में बदलने का सही तरीका इंगित करें।
```dart
final number = 3.14;
```

# --answers--

```dart
final integer = number.toInt();
```

```dart
final integer = int(number);
```

```dart
final int integer = number;
```

```dart
final integer = number as int;
```
