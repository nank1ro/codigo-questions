# --description--

"__Hello, World!__" किसी नई प्रोग्रामिंग भाषा में शुरुआत करने के लिए पारंपरिक पहला प्रोग्राम है।
हम स्टैंडर्ड आउटपुट डिवाइस (स्क्रीन) पर डेटा प्रदर्शित करने के लिए `print()` फ़ंक्शन का उपयोग करते हैं।
Dart में स्क्रीन पर `Hello, World!` प्रिंट करने के लिए हम लिख सकते हैं
```dart
print("Hello, World!")
```
