# --description--

एक बूलियन चर मान `false` भी रख सकता है।
`false` का उपयोग करें जब कोई चीज़ मामला न हो।

```dart
bool isLoggedIn = false;
print(isLoggedIn); // false
```

`true` की तरह, `false` को लोअरकेस में लिखा जाना चाहिए।
