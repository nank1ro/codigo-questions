# --description--

__निषेध ऑपरेटर__ `!` (जिसे "not" भी कहा जाता है) एक बूलियन मान को उलट देता है।
`!` को `true` पर लागू करने से `false` मिलता है, और `!` को `false` पर लागू करने से `true` मिलता है।

```dart
bool isActive = true;
print(!isActive); // false
```

यह तब उपयोगी है जब आप किसी स्थिति के विपरीत को जांचना चाहते हैं।

# --instructions--

दिया गया है कि चर `isBlocked` को `false` पर सेट किया गया है, `!` का उपयोग करके इसके निषेध मान को प्रिंट करें।
