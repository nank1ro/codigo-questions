एक __बूलियन__ एक डेटा प्रकार है जो केवल दो मानों में से एक को रख सकता है: `true` या `false`।
Dart में, बूलियन प्रकार को `bool` कीवर्ड के साथ घोषित किया जाता है।

```dart
bool isRaining = true;
```

चर `isRaining` मान `true` को संग्रहित करता है, जिसका अर्थ है कि वर्तमान में बारिश हो रही है।
बूलियन मानों को हमेशा लोअरकेस में लिखा जाता है: `true` और `false`।

---

किसी भी अन्य चर की तरह, आप `print()` के साथ एक बूलियन मान को प्रिंट कर सकते हैं।
जब आप एक बूलियन को प्रिंट करते हैं, तो Dart पाठ `true` या `false` को आउटपुट करता है।

```dart
bool isSunny = true;
print(isSunny); // true
```

---

एक बूलियन चर मान `false` भी रख सकता है।
`false` का उपयोग करें जब कोई चीज़ मामला न हो।

```dart
bool isLoggedIn = false;
print(isLoggedIn); // false
```

`true` की तरह, `false` को लोअरकेस में लिखा जाना चाहिए।

---

__निषेध ऑपरेटर__ `!` (जिसे "not" भी कहा जाता है) एक बूलियन मान को उलट देता है।
`!` को `true` पर लागू करने से `false` मिलता है, और `!` को `false` पर लागू करने से `true` मिलता है।

```dart
bool isActive = true;
print(!isActive); // false
```

यह तब उपयोगी है जब आप किसी स्थिति के विपरीत को जांचना चाहते हैं।

---

बूलियन का उपयोग पूरे प्रोग्रामिंग में शर्तों का प्रतिनिधित्व करने और निर्णयों को संचालित करने के लिए किया जाता है।
जब भी आपके प्रोग्राम को दो संभावनाओं के बीच निर्णय लेना होता है, तो एक बूलियन शामिल होता है।

उदाहरण:
- क्या उपयोगकर्ता लॉगिन है? (`isLoggedIn`)
- क्या दरवाज़ा खुला है? (`isDoorOpen`)
- क्या ऑर्डर भेजा गया है? (`isShipped`)
