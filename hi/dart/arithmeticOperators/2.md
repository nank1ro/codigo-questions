# --description--

**घटाव ऑपरेटर** `-` एक मान को दूसरे से घटाता है।

```dart
int a = 10;
int b = 4;
int diff = a - b; // 6
```

यदि दूसरा संकार्य पहले से बड़ा है तो परिणाम नकारात्मक हो सकता है।

# --instructions--

कोड को पूरा करें `20` से `9` को घटाने के लिए और परिणाम को `diff` में संग्रहीत करें।
