# --description--

**विभाजन ऑपरेटर** `/` एक मान को दूसरे से विभाजित करता है। Dart में, `/` **हमेशा एक `double` लौटाता है**, भले ही परिणाम एक पूर्ण संख्या हो।

```dart
int a = 10;
int b = 4;
double result = a / b; // 2.5

double exact = 10 / 2; // 5.0  (not 5!)
```

हमेशा एक `double` वेरिएबल का उपयोग करें `/` के परिणाम को संग्रहीत करने के लिए।

# --instructions--

कोड को पूरा करें `9` को `4` से विभाजित करने के लिए और परिणाम को `result` में संग्रहीत करें।
