# --description--

Dart मानक गणितीय **संक्रियाओं का क्रम** (प्राथमिकता) का पालन करता है: गुणन और विभाजन को जोड़ और घटाव से पहले मूल्यांकन किया जाता है।

```dart
int result = 2 + 3 * 4; // 14, not 20
```

क्रम को बदलने के लिए कोष्ठक का उपयोग करें:

```dart
int result = (2 + 3) * 4; // 20
```

# --instructions--
