**जोड़ ऑपरेटर** `+` दो मानों को एक साथ जोड़ता है।

```dart
int a = 3;
int b = 4;
int sum = a + b; // 7
```

आप `int` या `double` मानों के साथ `+` का उपयोग कर सकते हैं।

---

**घटाव ऑपरेटर** `-` एक मान को दूसरे से घटाता है।

```dart
int a = 10;
int b = 4;
int diff = a - b; // 6
```

यदि दूसरा संकार्य पहले से बड़ा है तो परिणाम नकारात्मक हो सकता है।

---

**गुणन ऑपरेटर** `*` दो मानों को एक साथ गुणा करता है।

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

एक `int` को दूसरे `int` से गुणा करने से एक `int` मिलता है। एक `double` के साथ गुणा करने से एक `double` मिलता है।

---

**विभाजन ऑपरेटर** `/` एक मान को दूसरे से विभाजित करता है। Dart में, `/` **हमेशा एक `double` लौटाता है**, भले ही परिणाम एक पूर्ण संख्या हो।

```dart
int a = 10;
int b = 4;
double result = a / b; // 2.5

double exact = 10 / 2; // 5.0  (not 5!)
```

हमेशा एक `double` वेरिएबल का उपयोग करें `/` के परिणाम को संग्रहीत करने के लिए।

---

**पूर्णांक विभाजन ऑपरेटर** `~/` दो संख्याओं को विभाजित करता है और परिणाम को एक पूर्णांक तक **काट** देता है (किसी भी दशमलव भाग को छोड़ देता है)।

```dart
int a = 10;
int b = 3;
int result = a ~/ b; // 3  (not 3.333...)
```

जब आपको शेष के बिना पूर्ण-संख्या भागफल की आवश्यकता हो तो `~/` का उपयोग करें।
