# --description--

एक **`if` कथन** एक कोड ब्लॉक को केवल तभी निष्पादित करता है जब कोई शर्त `true` हो।

```dart
int age = 20;
if (age >= 18) {
  print('Adult');
}
```

घुंघराले ब्रेसेज़ के अंदर का कोड केवल तभी चलता है जब शर्त `age >= 18` का मान `true` हो।
