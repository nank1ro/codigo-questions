# --solutions--

```dart
String classify(int score) {
  if (score >= 90) {
    return 'A';
  } else if (score >= 70) {
    return 'B';
  } else if (score >= 50) {
    return 'C';
  } else {
    return 'F';
  }
}

void main() {
  print(classify(95));
}
```
