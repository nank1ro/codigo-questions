# --description--

जब कोई शर्त पूरी होती है तो आप `if` ब्लॉक के अंदर `print()` का उपयोग करके संदेश प्रदर्शित कर सकते हैं।

```dart
bool isRaining = true;
if (isRaining) {
  print('Take an umbrella');
}
```

यदि `isRaining` का मान `false` है, तो कुछ भी नहीं छपता।
