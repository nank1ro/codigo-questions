# --description--

**`else if`** आपको क्रम में कई शर्तें जाँचने देता है। पहली शाखा जिसकी शर्त `true` है वह निष्पादित होती है, और बाकी छोड़ दी जाती हैं।

```dart
int score = 75;
if (score >= 90) {
  print('A');
} else if (score >= 70) {
  print('B');
} else {
  print('C');
}
// prints: B
```
