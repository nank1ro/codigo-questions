# --description--

एक **`if-else` कथन** शर्त `true` होने पर `if` ब्लॉक चलाता है और `false` होने पर `else` ब्लॉक चलाता है।

```dart
int temperature = 10;
if (temperature > 20) {
  print('Warm');
} else {
  print('Cold');
}
// prints: Cold
```

दो शाखाओं में से हमेशा ठीक एक ही निष्पादित होती है।
