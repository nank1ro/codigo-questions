# --description--

एक **मेथड** किसी क्लास के अंदर परिभाषित एक फंक्शन है। मेथड किसी ऑब्जेक्ट के व्यवहार का वर्णन करते हैं।

```dart
class Circle {
  double radius;
  Circle(this.radius);

  double area() {
    return 3.14 * radius * radius;
  }
}

void main() {
  final c = Circle(5);
  print(c.area()); // 78.5
}
```

मेथड ऑब्जेक्ट के अपने फ़ील्ड को सीधे नाम से (या `this.fieldName` के माध्यम से) एक्सेस कर सकते हैं।

# --instructions--

`Rectangle` क्लास को पूरा करें ताकि उसमें `int` फ़ील्ड `width` और `height`, एक कंस्ट्रक्टर और एक मेथड `perimeter()` हो जो `2 * (width + height)` लौटाए। फिर मेथड को कॉल करें और परिणाम प्रिंट करें।

class Rectangle {
  int width;
  int height;
  Rectangle(this.width, this.height);

  int perimeter() {
    return 2 * (width + height);
  }
}

void main() {

}
