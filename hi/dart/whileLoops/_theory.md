`while` लूप एक कोड ब्लॉक को तब तक दोहराता है जब तक कोई शर्त `true` हो।

```dart
int i = 0;
while (i < 3) {
  print(i);
  i++;
}
```

लूप प्रत्येक पुनरावृत्ति से **पहले** शर्त की जाँच करता है। जब `i < 3` `false` हो जाता है, तो लूप रुक जाता है।

---

एक काउंटर वेरिएबल नियंत्रित करता है कि `while` लूप कितनी बार चलता है।

```dart
int count = 0;
while (count < 5) {
  count++;
}
print(count); // 5
```

काउंटर `0` से शुरू होता है, और लूप प्रत्येक पुनरावृत्ति में इसे तब तक बढ़ाता है जब तक `count < 5` `false` न हो जाए।

---

`do-while` लूप `while` लूप के समान है, लेकिन यह **हमेशा कम से कम एक बार बॉडी को निष्पादित करता है** — शर्त की जाँच प्रत्येक पुनरावृत्ति के *बाद* होती है।

```dart
int i = 0;
do {
  print(i);
  i++;
} while (i < 3);
```

यदि शुरू से ही शर्त `false` हो, तो भी बॉडी एक बार चलती है।

---

`while` लूप की शर्त में कोई भी बूलियन एक्सप्रेशन हो सकता है। जैसे ही शर्त `false` हो जाती है, लूप समाप्त हो जाता है।

```dart
int n = 100;
while (n > 1) {
  n ~/= 2;
}
print(n); // 0
```

यहाँ, `n` को पूर्णांक विभाजन (`~/`) का उपयोग करके प्रत्येक पुनरावृत्ति में आधा किया जाता है।

---

`break` कीवर्ड लूप की शर्त की परवाह किए बिना लूप से तुरंत बाहर निकल जाता है।

```dart
int i = 0;
while (true) {
  if (i == 3) break;
  print(i);
  i++;
}
```

`break` के बिना, `while (true)` लूप हमेशा के लिए चलता रहता। यहाँ यह तब रुकता है जब `i` `3` तक पहुँचता है।
