# --description--

`while` लूप एक कोड ब्लॉक को तब तक दोहराता है जब तक कोई शर्त `true` हो।

```dart
int i = 0;
while (i < 3) {
  print(i);
  i++;
}
```

लूप प्रत्येक पुनरावृत्ति से **पहले** शर्त की जाँच करता है। जब `i < 3` `false` हो जाता है, तो लूप रुक जाता है।
