# --description--

`break` कीवर्ड लूप की शर्त की परवाह किए बिना लूप से तुरंत बाहर निकल जाता है।

```dart
int i = 0;
while (true) {
  if (i == 3) break;
  print(i);
  i++;
}
```

`break` के बिना, `while (true)` लूप हमेशा के लिए चलता रहता। यहाँ यह तब रुकता है जब `i` `3` तक पहुँचता है।

# --instructions--

उस लूप को पूरा करें जो `0` से नंबर प्रिंट करता है और `break` का उपयोग करके तब रुकता है जब `i` `4` के बराबर हो।
