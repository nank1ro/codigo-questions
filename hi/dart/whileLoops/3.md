# --description--

`do-while` लूप `while` लूप के समान है, लेकिन यह **हमेशा कम से कम एक बार बॉडी को निष्पादित करता है** — शर्त की जाँच प्रत्येक पुनरावृत्ति के *बाद* होती है।

```dart
int i = 0;
do {
  print(i);
  i++;
} while (i < 3);
```

यदि शुरू से ही शर्त `false` हो, तो भी बॉडी एक बार चलती है।

# --instructions--

`do-while` लूप को पूरा करें जो `0`, `1` और `2` प्रिंट करता है।
