# --description--

**AND ऑपरेटर** `&&` दो बूलियन अभिव्यक्तियों को जोड़ता है। यह `true` लौटाता है केवल जब **दोनों** पक्ष `true` हों। यदि कोई भी पक्ष `false` है, तो परिणाम `false` है।

```dart
bool a = true;
bool b = true;
bool result = a && b; // true

bool c = true;
bool d = false;
bool result2 = c && d; // false
```
