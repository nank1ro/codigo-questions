# --description--

**असमान ऑपरेटर** `!=` `true` लौटाता है जब दो मान **अलग** हों, और `false` जब वे समान हों।

```dart
int x = 3;
int y = 7;
bool result = x != y; // true
```

यह `==` का विपरीत है।
