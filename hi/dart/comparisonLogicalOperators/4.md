# --description--

**से कम ऑपरेटर** `<` `true` लौटाता है जब बाईं ओर का मान दाईं ओर के मान से कड़ाई से कम हो।

```dart
int a = 2;
int b = 9;
bool result = a < b; // true
```

यदि बाईं ओर का मान दाईं ओर के बराबर या अधिक है, तो यह `false` लौटाता है।
