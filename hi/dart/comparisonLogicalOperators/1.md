# --description--

**समान ऑपरेटर** `==` दो मानों की तुलना करता है और यदि वे बराबर हैं तो `true` लौटाता है, या यदि नहीं हैं तो `false`।

```dart
int a = 5;
int b = 5;
bool result = a == b; // true
```

एक तुलना का परिणाम हमेशा एक `bool` मान होता है।
