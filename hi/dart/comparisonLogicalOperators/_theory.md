**समान ऑपरेटर** `==` दो मानों की तुलना करता है और यदि वे बराबर हैं तो `true` लौटाता है, या यदि नहीं हैं तो `false`।

```dart
int a = 5;
int b = 5;
bool result = a == b; // true
```

एक तुलना का परिणाम हमेशा एक `bool` मान होता है।

---

**असमान ऑपरेटर** `!=` `true` लौटाता है जब दो मान **अलग** हों, और `false` जब वे समान हों।

```dart
int x = 3;
int y = 7;
bool result = x != y; // true
```

यह `==` का विपरीत है।

---

**से अधिक ऑपरेटर** `>` `true` लौटाता है जब बाईं ओर का मान दाईं ओर के मान से कड़ाई से बड़ा हो।

```dart
int a = 8;
int b = 3;
bool result = a > b; // true
```

यदि बाईं ओर का मान दाईं ओर के बराबर या कम है, तो यह `false` लौटाता है।

---

**से कम ऑपरेटर** `<` `true` लौटाता है जब बाईं ओर का मान दाईं ओर के मान से कड़ाई से कम हो।

```dart
int a = 2;
int b = 9;
bool result = a < b; // true
```

यदि बाईं ओर का मान दाईं ओर के बराबर या अधिक है, तो यह `false` लौटाता है।

---

**से अधिक या बराबर ऑपरेटर** `>=` `true` लौटाता है जब बाईं ओर का मान दाईं ओर के मान से अधिक है **या उसके बराबर**।

```dart
int score = 50;
bool passed = score >= 50; // true
```

`>` के विपरीत, यह ऑपरेटर भी `true` लौटाता है जब दोनों मान बराबर हों।
