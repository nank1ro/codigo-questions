# --description--

**जोड़ असाइनमेंट ऑपरेटर** `+=` एक मान को एक चर में जोड़ता है और परिणाम को वापस उस चर में संग्रहीत करता है।

```dart
int score = 10;
score += 5; // score अब 15 है
```

यह `score = score + 5` का संक्षिप्त रूप है।

# --instructions--

कोड को पूरा करें ताकि `+=` का उपयोग करके चर `total` में `8` जोड़ा जा सके।
