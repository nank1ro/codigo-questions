**असाइनमेंट ऑपरेटर** `=` एक मान को एक चर में संग्रहीत करता है।

```dart
int x = 5;
String name = 'Alice';
```

एक बार असाइन होने के बाद, चर उस मान को तब तक रखता है जब तक इसे फिर से असाइन नहीं किया जाता।

---

**जोड़ असाइनमेंट ऑपरेटर** `+=` एक मान को एक चर में जोड़ता है और परिणाम को वापस उस चर में संग्रहीत करता है।

```dart
int score = 10;
score += 5; // score अब 15 है
```

यह `score = score + 5` का संक्षिप्त रूप है।

---

**घटाव असाइनमेंट ऑपरेटर** `-=` एक मान को एक चर से घटाता है और परिणाम को वापस उस चर में संग्रहीत करता है।

```dart
int lives = 3;
lives -= 1; // lives अब 2 है
```

यह `lives = lives - 1` का संक्षिप्त रूप है।

---

**गुणन असाइनमेंट ऑपरेटर** `*=` एक चर को एक मान से गुणा करता है और परिणाम को वापस उस चर में संग्रहीत करता है।

```dart
int side = 4;
side *= 3; // side अब 12 है
```

यह `side = side * 3` का संक्षिप्त रूप है।
