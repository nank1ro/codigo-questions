# --description--

**गुणन असाइनमेंट ऑपरेटर** `*=` एक चर को एक मान से गुणा करता है और परिणाम को वापस उस चर में संग्रहीत करता है।

```dart
int side = 4;
side *= 3; // side अब 12 है
```

यह `side = side * 3` का संक्षिप्त रूप है।

# --instructions--

कोड को पूरा करें ताकि `*=` का उपयोग करके `price` को `3` से गुणा किया जा सके।
