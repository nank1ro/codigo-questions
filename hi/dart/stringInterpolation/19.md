# --instructions--

String interpolation का उपयोग करके खाली जगह में संख्या `500` डालने के लिए `${}` का उपयोग करें

# --seed--

```dart
void main() {
    print("There are  items");
}
```

# --solutions--

```dart
void main() {
    print("There are ${500} items");
}
```

# --output--

There are 500 items
