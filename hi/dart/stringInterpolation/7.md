# --description--

हम string interpolation का उपयोग करके curly brackets से जितनी बार चाहें उतनी बार मान डाल सकते हैं

# --instructions--

यहाँ `2` और फिर `8` के साथ देखें

# --seed--

```dart
print("[/] new friends and [/] old friends");
```

# --answers--

- $2
- $8
- ${2}
- ${8}

# --solutions--

```dart
print("${2} new friends and ${8} old friends");
```

# --output--

2 new friends and 8 old friends
