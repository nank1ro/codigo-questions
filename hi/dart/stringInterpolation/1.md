# --description--

String _interpolation_ एक प्रोग्रामेटिक तरीका है String बनाने का।
Dart में हम `+` चिह्न (concatenation) का उपयोग करके दो या अधिक strings को एक साथ प्रदर्शित कर सकते हैं, जैसे:
```dart
print("Hello " + "Dart!");
// prints "Hello Dart!"
```
