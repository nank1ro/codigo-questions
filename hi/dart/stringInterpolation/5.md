# --description--

यदि `$` चिह्न के बाद जो आता है वह प्रोग्राम identifier के रूप में पहचाना नहीं जा सकता, तो Dart एक संकलन त्रुटि उत्पन्न करेगा: स्ट्रिंग के अंदर `$` के बाद हमेशा एक identifier या कर्ली ब्रेस (`${}`) में expression आना चाहिए।

# --instructions--

जब आप इस कोड को चलाने का प्रयास करते हैं तो क्या होता है?

```dart
void main() {
    print("$5 friends");
}
```

# --answers--

- यह कंपाइल नहीं होगा क्योंकि `$5` अमान्य string interpolation है
- यह `$5 friends` प्रिंट करता है
- यह `5 friends` प्रिंट करता है
- यह कुछ भी प्रिंट नहीं करता
