# --description--
एक `for` लूप एक कोड ब्लॉक को निश्चित संख्या में दोहराता है। बुनियादी सिंटैक्स है:

```dart
for (initialization; condition; update) {
  // body
}
```

**initialization**: लूप शुरू होने से पहले एक बार चलता है (जैसे, `int i = 0`)

**condition**: हर पुनरावृत्ति से पहले जाँची जाती है; जब झूठी हो तो लूप रुक जाता है

**update**: हर पुनरावृत्ति के बाद चलता है (जैसे, `i++`)
