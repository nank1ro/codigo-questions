# --description--
आप वर्तमान गिनती को ट्रैक करने के लिए लूप वेरिएबल को बॉडी के अंदर उपयोग कर सकते हैं। उदाहरण के लिए, एक योग संचित करना:

```dart
int total = 0;
for (int i = 1; i <= 5; i++) {
  total += i;
}
```

लूप के बाद, `total` 15 के बराबर है।
