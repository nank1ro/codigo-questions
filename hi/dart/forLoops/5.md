# --description--
`break` स्टेटमेंट किसी शर्त के पूरा होने पर लूप से तुरंत बाहर निकल जाता है:

```dart
for (int i = 0; i < 10; i++) {
  if (i == 5) break; // stops at 5
  print(i);
}
```

यह तब उपयोगी होता है जब आप किसी मान को खोज रहे हों और मिलने पर रुकना चाहते हों।
