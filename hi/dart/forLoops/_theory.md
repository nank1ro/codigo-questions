एक `for` लूप एक कोड ब्लॉक को निश्चित संख्या में दोहराता है। बुनियादी सिंटैक्स है:

```dart
for (initialization; condition; update) {
  // body
}
```

**initialization**: लूप शुरू होने से पहले एक बार चलता है (जैसे, `int i = 0`)

**condition**: हर पुनरावृत्ति से पहले जाँची जाती है; जब झूठी हो तो लूप रुक जाता है

**update**: हर पुनरावृत्ति के बाद चलता है (जैसे, `i++`)

---

आप वर्तमान गिनती को ट्रैक करने के लिए लूप वेरिएबल को बॉडी के अंदर उपयोग कर सकते हैं। उदाहरण के लिए, एक योग संचित करना:

```dart
int total = 0;
for (int i = 1; i <= 5; i++) {
  total += i;
}
```

लूप के बाद, `total` 15 के बराबर है।

---

एक `for` लूप डिक्रीमेंट (`i--`) और `>=` कंडीशन का उपयोग करके **नीचे की ओर** गिन सकता है:

```dart
for (int i = 5; i >= 1; i--) {
  print(i); // 5, 4, 3, 2, 1
}
```

यह काउंटडाउन या उल्टे क्रम में पुनरावृत्ति के लिए उपयोगी है।

---

`for-in` लूप बिना किसी इंडेक्स की आवश्यकता के किसी कलेक्शन के हर तत्व पर पुनरावृत्ति करता है:

```dart
List<String> fruits = ['apple', 'banana', 'cherry'];
for (var fruit in fruits) {
  print(fruit);
}
```

प्रत्येक पुनरावृत्ति अगले तत्व को लूप वेरिएबल (`fruit` यहाँ) में असाइन करती है।

---

`break` स्टेटमेंट किसी शर्त के पूरा होने पर लूप से तुरंत बाहर निकल जाता है:

```dart
for (int i = 0; i < 10; i++) {
  if (i == 5) break; // stops at 5
  print(i);
}
```

यह तब उपयोगी होता है जब आप किसी मान को खोज रहे हों और मिलने पर रुकना चाहते हों।
