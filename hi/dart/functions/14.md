# --instructions--

एक **पुनरावर्ती** फंक्शन `factorial` लिखें जो `int` पैरामीटर `n` लेता है और `n!` (`n` का भाज्य) लौटाता है।

`n` का भाज्य इस प्रकार परिभाषित है:
- `factorial(0) = 1`
- `factorial(n) = n * factorial(n - 1)` के लिए `n > 0`

उदाहरण:
```dart
print(factorial(5)); // 120
print(factorial(3)); // 6
```

```dart
int factorial(int n) {
  if (n == 0) return 1;
  return n * factorial(n - 1);
}
```
