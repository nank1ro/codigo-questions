# --description--

फंक्शन **पैरामीटर** स्वीकार कर सकते हैं — फंक्शन को कॉल करते समय पास किए गए मान। पैरामीटर को उनके टाइप और नाम के साथ कोष्ठक के अंदर घोषित किया जाता है:

```dart
int square(int n) {
  return n * n;
}

void main() {
  print(square(4)); // prints 16
}
```

प्रत्येक पैरामीटर का एक टाइप (`int`) और एक नाम (`n`) होता है। एकाधिक पैरामीटर अल्पविराम से अलग किए जाते हैं।

# --instructions--
