# --instructions--

एक फंक्शन `range` लिखें जो `int` पैरामीटर `n` लेता है और `1` से `n` तक (समावेशी) के पूर्णांकों वाली `List<int>` लौटाता है।

उदाहरण:
```dart
print(range(3)); // [1, 2, 3]
print(range(5)); // [1, 2, 3, 4, 5]
```

# --solutions--

```dart
List<int> range(int n) {
  List<int> result = [];
  for (int i = 1; i <= n; i++) {
    result.add(i);
  }
  return result;
}
```
