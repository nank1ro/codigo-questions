# --instructions--

Dart फंक्शन में `void` रिटर्न टाइप का क्या अर्थ है?

```dart
void sayHi() {
  print("Hi!");
}
```

# --answers--

- फंक्शन शून्य संख्या लौटाता है
- फंक्शन कोई मान नहीं लौटाता
- फंक्शन एक खाली स्ट्रिंग लौटाता है
- फंक्शन किसी भी टाइप को लौटा सकता है
