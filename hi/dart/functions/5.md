# --description--

Dart **नामांकित पैरामीटर** को सपोर्ट करता है — पैरामीटर जो घुंघराले कोष्ठक `{}` में लिपटे होते हैं। नामांकित पैरामीटर को नाम से कॉल किया जाना चाहिए और उनके डिफ़ॉल्ट मान हो सकते हैं या उन्हें `required` के रूप में चिह्नित किया जा सकता है:

```dart
void printInfo({required String name, int age = 0}) {
  print("$name is $age years old");
}

void main() {
  printInfo(name: "Alice", age: 30);
  // prints Alice is 30 years old
}
```

नामांकित पैरामीटर पठनीयता में सुधार करते हैं, विशेष रूप से जब किसी फंक्शन में कई पैरामीटर हों।
