# --description--

Dart `=>` सिंटैक्स का उपयोग करके **ऐरो फंक्शन** को सपोर्ट करता है। जब फंक्शन का मुख्य भाग एक ही अभिव्यक्ति हो, तो आप `{ return ...; }` की जगह `=>` से इसे छोटा कर सकते हैं:

```dart
// नियमित फंक्शन
int double(int n) {
  return n * 2;
}

// ऐरो फंक्शन — समान परिणाम
int double(int n) => n * 2;
```

ऐरो फंक्शन कोड को अधिक संक्षिप्त बनाते हैं। `=>` घुंघराले कोष्ठक और `return` कीवर्ड दोनों की जगह लेता है।
