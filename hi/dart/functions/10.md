# --instructions--

एक **ऐरो फंक्शन** `square` लिखें जो `int` पैरामीटर `n` लेता है और `n * n` लौटाता है।

`=>` सिंटैक्स का उपयोग करें (कोई घुंघराले कोष्ठक नहीं, कोई `return` कीवर्ड नहीं)।

उदाहरण:
```dart
print(square(4));  // 16
print(square(5));  // 25
```
