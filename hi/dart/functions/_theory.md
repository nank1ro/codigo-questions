एक **फंक्शन** कोड का एक पुन: उपयोग योग्य ब्लॉक है जो एक विशिष्ट कार्य करता है। Dart में, आप एक फंक्शन को रिटर्न टाइप, एक नाम और `()` के साथ परिभाषित करते हैं।

जब कोई फंक्शन कोई मान नहीं लौटाता, तो उसका रिटर्न टाइप `void` होता है:

```dart
void sayHello() {
  print("Hello!");
}
```

आप फंक्शन का नाम और उसके बाद `()` लिखकर फंक्शन को कॉल करते हैं:

```dart
sayHello(); // prints Hello!
```

---

फंक्शन कॉलर को **एक मान वापस लौटा** सकते हैं। आप फंक्शन के नाम से पहले रिटर्न टाइप घोषित करते हैं और मान वापस भेजने के लिए `return` कीवर्ड का उपयोग करते हैं:

```dart
int getAge() {
  return 25;
}
```

फंक्शन नाम से पहले का टाइप (`int`) Dart को बताता है कि फंक्शन किस प्रकार का मान लौटाएगा। आप `String`, `int`, `double`, `bool` और अन्य का उपयोग कर सकते हैं।

---

फंक्शन **पैरामीटर** स्वीकार कर सकते हैं — फंक्शन को कॉल करते समय पास किए गए मान। पैरामीटर को उनके टाइप और नाम के साथ कोष्ठक के अंदर घोषित किया जाता है:

```dart
int square(int n) {
  return n * n;
}

void main() {
  print(square(4)); // prints 16
}
```

प्रत्येक पैरामीटर का एक टाइप (`int`) और एक नाम (`n`) होता है। एकाधिक पैरामीटर अल्पविराम से अलग किए जाते हैं।

---

Dart `=>` सिंटैक्स का उपयोग करके **ऐरो फंक्शन** को सपोर्ट करता है। जब फंक्शन का मुख्य भाग एक ही अभिव्यक्ति हो, तो आप `{ return ...; }` की जगह `=>` से इसे छोटा कर सकते हैं:

```dart
// नियमित फंक्शन
int double(int n) {
  return n * 2;
}

// ऐरो फंक्शन — समान परिणाम
int double(int n) => n * 2;
```

ऐरो फंक्शन कोड को अधिक संक्षिप्त बनाते हैं। `=>` घुंघराले कोष्ठक और `return` कीवर्ड दोनों की जगह लेता है।

---

Dart **नामांकित पैरामीटर** को सपोर्ट करता है — पैरामीटर जो घुंघराले कोष्ठक `{}` में लिपटे होते हैं। नामांकित पैरामीटर को नाम से कॉल किया जाना चाहिए और उनके डिफ़ॉल्ट मान हो सकते हैं या उन्हें `required` के रूप में चिह्नित किया जा सकता है:

```dart
void printInfo({required String name, int age = 0}) {
  print("$name is $age years old");
}

void main() {
  printInfo(name: "Alice", age: 30);
  // prints Alice is 30 years old
}
```

नामांकित पैरामीटर पठनीयता में सुधार करते हैं, विशेष रूप से जब किसी फंक्शन में कई पैरामीटर हों।
