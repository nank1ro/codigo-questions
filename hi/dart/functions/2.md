# --description--

फंक्शन कॉलर को **एक मान वापस लौटा** सकते हैं। आप फंक्शन के नाम से पहले रिटर्न टाइप घोषित करते हैं और मान वापस भेजने के लिए `return` कीवर्ड का उपयोग करते हैं:

```dart
int getAge() {
  return 25;
}
```

फंक्शन नाम से पहले का टाइप (`int`) Dart को बताता है कि फंक्शन किस प्रकार का मान लौटाएगा। आप `String`, `int`, `double`, `bool` और अन्य का उपयोग कर सकते हैं।
