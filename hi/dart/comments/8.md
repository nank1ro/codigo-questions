# --instructions--

_डॉक्यूमेंटेशन कमेंट_ के भीतर हम _classes_, _methods_, _fields_, _variables_, _functions_ या _parameters_ को कैसे संदर्भित करते हैं?

# --answers--

- गोल कोष्ठकों `()` का उपयोग करके
- घुंघराले कोष्ठकों `{}` का उपयोग करके
- वर्ग कोष्ठकों `[]` का उपयोग करके
- डॉलर चिह्न `$` का उपयोग करके
