# --description--

`.add()` मेथड सूची के **अंत** में एक तत्व जोड़ता है:

```dart
var nums = [1, 2, 3];
nums.add(4);
print(nums); // [1, 2, 3, 4]
```

ध्यान दें कि `.add()` सूची को **स्थान पर** संशोधित करता है और `void` लौटाता है।
