# --description--

`.length` प्रॉपर्टी सूची में तत्वों की संख्या लौटाती है:

```dart
var scores = [95, 87, 72, 100];
print(scores.length); // 4
```

एक खाली सूची की लंबाई `0` होती है:

```dart
var empty = [];
print(empty.length); // 0
```
