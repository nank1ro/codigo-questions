# --description--

_Classes_ सामान्य-उद्देश्य वाली, लचीली संरचनाएँ हैं जो आपके प्रोग्राम के कोड की निर्माण इकाइयाँ बनती हैं।
एक बुनियादी class में केवल `class` कीवर्ड और उसका नाम होता है, उदाहरण के लिए:
```javascript
class ClassName {
    // class definition
}
```
