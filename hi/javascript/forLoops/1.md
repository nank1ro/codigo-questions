# --description--

हम जानते हैं कि `while` लूप का उपयोग करके कोड को कैसे दोहराया जाता है।
जैसे यह प्रोग्राम `hello` प्रदर्शित करने के लिए स्टेटमेंट्स को दोहराता है
```javascript
var counter = 0;

while (counter < 5) {
    console.log("hello");
    counter++;
}
```
लेकिन हम `for` लूप के साथ भी ऐसा कर सकते हैं:
```javascript
for (let i = 0; i < 5; i++) {
    console.log("hello");
}
```
