# --description--

"__Hello, World!__" किसी नई प्रोग्रामिंग भाषा में प्रोग्रामिंग शुरू करने का पारंपरिक पहला प्रोग्राम है।
हम `console.log()` फ़ंक्शन का उपयोग मानक आउटपुट डिवाइस (स्क्रीन) पर डेटा प्रदर्शित करने के लिए करते हैं।
JavaScript के साथ स्क्रीन पर `Hello, World!` प्रिंट करने के लिए हम लिख सकते हैं
```javascript
console.log("Hello, World!")
```
