# --instructions--

इस कोड स्निपेट में __समस्या__ क्या है?
```javascript
console.log(greet("Mathew"));
// This function greets to the person passed in as argument
const greet = (name) => `Hello, ${name}. Good morning!`;
console.log(greet("Alex"));
```

# --answers--

- हम greet() को परिभाषित करने से पहले उपयोग नहीं कर सकते
- हम greet() को दो बार उपयोग नहीं कर सकते
- greet() कुछ भी return नहीं करता
- हम name को parameter के रूप में उपयोग नहीं कर सकते
