# --instructions--

क्या यह कोड ब्लॉक कभी चलाया जाएगा?
```javascript
var shouldRepeat = false;

do {
    console.log("run code");
} while (shouldRepeat);
```

# --answers--

- हां क्योंकि do-while लूप की कंडीशन पर विचार करने से पहले एक बार चलता है
- नहीं क्योंकि shouldRepeat false है
- यह कोड ब्लॉक दो बार चलाया जाएगा
- यह कोड ब्लॉक तीन बार चलाया जाएगा
