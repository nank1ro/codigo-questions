# --instructions--

`counter += 1` क्या करता है?
```javascript
var counter = 1;

while (counter < 5) {
    console.log(counter);
    counter += 1;
}
```

# --answers--

- यह हर बार लूप अपना कोड ब्लॉक चलाने पर counter को एक से बढ़ाता है
- यह counter वेरिएबल बनाता है
- यह हर बार counter को दो से बढ़ाता है
- यह हर बार counter को तीन से बढ़ाता है
