# --description--

आइए पिछले उदाहरण को जारी रखें।
```javascript
var niceWeather = true;
if (niceWeather) {
    // play outside
}
```
हमने देखा कि `if` स्टेटमेंट कोड ब्लॉक को केवल तभी चलाता है जब शर्त `true` हो।
एक और महत्वपूर्ण बात **कर्ली ब्रैकेट्स** `{}` हैं जो एक कोड ब्लॉक को दर्शाते हैं।
