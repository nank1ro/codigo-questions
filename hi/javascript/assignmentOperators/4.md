# --description--

आइए **मल्टीप्लिकेशन असाइनमेंट** `*=` ऑपरेटर देखें।
यह वेरिएबल को एक मान से गुणा करता है और परिणाम को उस वेरिएबल में असाइन करता है।
तो निम्नलिखित बिल्कुल समान हैं
```javascript
var num = num * 5;
// is equal to
num *= 5;
```
