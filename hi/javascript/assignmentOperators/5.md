# --description--

आइए **डिवीज़न असाइनमेंट** `/=` ऑपरेटर देखें।
यह वेरिएबल को एक मान से विभाजित करता है और परिणाम को उस वेरिएबल में असाइन करता है।
तो निम्नलिखित बिल्कुल समान हैं
```javascript
num = num / 5;
// is equal to
num /= 5;
```
