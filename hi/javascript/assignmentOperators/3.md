# --description--

एडिशन असाइनमेंट की तरह ही, हमारे पास **डिक्रीमेंट असाइनमेंट** `-=` है।
कार्यप्रणाली समान है, केवल अंतर यह है कि यह घटाव करता है।
तो निम्नलिखित बिल्कुल समान हैं
```javascript
var num = num - 5;
// is equal to
num -= 5;
```
