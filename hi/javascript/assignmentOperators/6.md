# --description--

आइए **रिमेंडर असाइनमेंट** `%=` ऑपरेटर देखें।
यह वेरिएबल और एक मान का शेषफल निकालता है और परिणाम को उस वेरिएबल में असाइन करता है।
तो निम्नलिखित बिल्कुल समान हैं
```javascript
num = num % 5;
// is equal to
num %= 5;
```
