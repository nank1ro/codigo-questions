# --description--

आइए **बराबर** `==` तुलना ऑपरेटर से शुरू करें।
यह एक **boolean** (`true` या `false`) लौटाता है जो बताता है कि दो अभिव्यक्तियाँ बराबर हैं या नहीं, उदाहरण के लिए:
```javascript
console.log(2 == 2); 
// prints true
console.log(2 == 3);
// prints false
```
