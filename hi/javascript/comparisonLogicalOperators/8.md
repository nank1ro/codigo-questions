# --description--

आइए **or** `||` लॉजिकल ऑपरेटर के साथ आगे बढ़ें।
यह पहला ऑपरेंड लौटाता है जो *true* का मूल्यांकन करता है या अंतिम ऑपरेंड अगर सभी *false* हैं।
```javascript
console.log(2 == 2 || 2 == 3);
// prints true
console.log(1 == 2 || 1 == 3);
// prints false
```
