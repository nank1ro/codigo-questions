# --description--

आइए **से बड़ा** `>` तुलना ऑपरेटर के साथ आगे बढ़ें।
यह एक **boolean** (`true` या `false`) लौटाता है जो बताता है कि एक अभिव्यक्ति दूसरी से बड़ी है या नहीं, उदाहरण के लिए:
```javascript
console.log(2 > 2);
// prints false
console.log(3 > 2);
// prints true
```
