# --description--

आइए **बराबर नहीं** `!=` तुलना ऑपरेटर के साथ आगे बढ़ें।
यह एक **boolean** (`true` या `false`) लौटाता है जो बताता है कि दो अभिव्यक्तियाँ **बराबर नहीं** हैं, उदाहरण के लिए:
```javascript
console.log(2 != 2);
// prints false
console.log(2 != 3); 
// prints true
```
यह *बराबर* ऑपरेटर का बिल्कुल विपरीत है
