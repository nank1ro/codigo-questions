# --description--

आइए **not** `!` लॉजिकल ऑपरेटर के साथ समाप्त करें।
यह एक boolean लौटाता है जो किसी अभिव्यक्ति की लॉजिकल स्थिति का उल्टा होता है।
```javascript
console.log(!true);
// prints false
console.log(!false);
// prints true
console.log(!(2 == 2));
// prints false
```
