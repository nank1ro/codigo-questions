# --description--

अब आइए **लॉजिकल** ऑपरेटर देखें, पहले __AND__ `&&` से शुरू करते हैं।
यह पहला ऑपरेंड लौटाता है जो *false* का मूल्यांकन करता है या अंतिम ऑपरेंड अगर सभी *true* हैं।
```javascript
console.log(2 == 2 && 2 == 3);
// prints false
console.log(1 == 1 && 1 == 1.0);
// prints true
```
