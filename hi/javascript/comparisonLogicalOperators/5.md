# --description--

आइए **से बड़ा या बराबर** `>=` तुलना ऑपरेटर के साथ आगे बढ़ें।
यह एक **boolean** (`true` या `false`) लौटाता है जो बताता है कि एक अभिव्यक्ति दूसरी से बड़ी या बराबर है, उदाहरण के लिए:
```javascript
console.log(2 >= 2); 
// prints true
console.log(3 >= 2);
// prints true
console.log(3 >= 4);
// prints false
```
