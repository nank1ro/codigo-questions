# --description--

JavaScript में हम `for..of` कीवर्ड का उपयोग करके बहुत ही सरल तरीके से ऐरे के माध्यम से लूप कर सकते हैं:
```javascript
var numbers = [1, 2, 3];
for (num of numbers) {
    console.log(num);
}
// prints 1, 2, 3
```
`for` कीवर्ड के बाद एक वेरिएबल नाम आता है, इसे बारी-बारी से प्रत्येक ऐरे आइटम का मान असाइन किया जाएगा।
