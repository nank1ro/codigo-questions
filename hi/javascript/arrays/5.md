# --description--

एक ऐरे की लंबाई निश्चित नहीं होती।
आप जब चाहें ऐरे के अंत में आइटम जोड़ सकते हैं!
ऐरे में आइटम जोड़ने के लिए हम `push` फंक्शन का उपयोग करते हैं:
```javascript
var letters = ["a", "b"];
letters.push("c");
console.log(letters);
// Prints ["a", "b", "c"]
```
