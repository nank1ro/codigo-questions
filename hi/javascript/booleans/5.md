# --instructions--

`true` एक string क्यों नहीं है?
```javascript
let a = true
```

# --answers--

- यह एक वेरिएबल के अंदर स्टोर है
- इसके चारों ओर कोई उद्धरण चिह्न (quotes) नहीं हैं
- यह अपरकेस अक्षर से शुरू होता है
