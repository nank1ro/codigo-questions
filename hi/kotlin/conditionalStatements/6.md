# --description--

हम एक कंडीशनल स्टेटमेंट (`if`, `else if` या `else`) को दूसरे कंडीशनल स्टेटमेंट के अंदर नेस्ट भी कर सकते हैं, एक अधिक जटिल संरचना बनाने के लिए।
```kotlin
var num = 4
if (num < 3) {
    println("the number is lower than 3")
} else {
    if (num == 3) {
        println("the number is 3")
    } else if (num == 4) {
        println("the number is 4")
    } else {
        println("the number is greather than 4")
    }
}
```
और इस कोड का आउटपुट `the number is 4` है।

# --instructions--

आइए दिन के समय के प्रोग्राम को बदलें, `else` स्टेटमेंट के अंदर एक कंडीशनल स्टेटमेंट नेस्ट करें।
नेस्टेड कंडीशनल स्टेटमेंट को जांचना चाहिए कि `time` `18` से *कम* है या नहीं और कोड ब्लॉक में `"It's afternoon"` प्रिंट करना चाहिए।
यदि पिछली शर्त `false` है, तो `"It's evening"` प्रिंट करें
