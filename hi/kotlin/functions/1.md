# --description--

एक फंक्शन कोड का एक ब्लॉक होता है जो एक विशिष्ट कार्य करता है और आपके प्रोग्राम में पुनः उपयोग किया जा सकता है।
Kotlin में, आप `fun` कीवर्ड का उपयोग करके एक फंक्शन परिभाषित करते हैं, उसके बाद फंक्शन का नाम और कोष्ठक:
```kotlin
fun greet() {
    println("Hello!")
}
```
किसी फंक्शन को कॉल (निष्पादित) करने के लिए, उसके नाम के बाद कोष्ठक का उपयोग करें:
```kotlin
greet() // prints Hello!
```
जो फंक्शन कोई मान नहीं लौटाता, वह अंतर्निहित रूप से `Unit` लौटाता है।

# --instructions--

`sayHello` नाम का एक फंक्शन परिभाषित करें जो `"Hello, Kotlin!"` स्ट्रिंग प्रिंट करे, फिर उसे `main` के अंदर कॉल करें।

# --solutions--

```kotlin
fun sayHello() {
    println("Hello, Kotlin!")
}
fun main() {
    sayHello()
}
```
