# --description--

जब फंक्शन का बॉडी एक ही एक्सप्रेशन से बना होता है, तो आप `=` के साथ सिंगल-एक्सप्रेशन सिंटैक्स का उपयोग कर सकते हैं:
```kotlin
fun square(n: Int): Int = n * n
```
यह ब्लॉक बॉडी फॉर्म से अधिक संक्षिप्त है। Kotlin अक्सर रिटर्न टाइप का अनुमान लगा सकता है, इसलिए आप यह भी लिख सकते हैं:
```kotlin
fun square(n: Int) = n * n
```
सिंगल-एक्सप्रेशन फंक्शन सरल गणनाओं के लिए एक सामान्य Kotlin मुहावरा है।
