एक फंक्शन कोड का एक ब्लॉक होता है जो एक विशिष्ट कार्य करता है और आपके प्रोग्राम में पुनः उपयोग किया जा सकता है।
Kotlin में, आप `fun` कीवर्ड का उपयोग करके एक फंक्शन परिभाषित करते हैं, उसके बाद फंक्शन का नाम और कोष्ठक:
```kotlin
fun greet() {
    println("Hello!")
}
```
किसी फंक्शन को कॉल (निष्पादित) करने के लिए, उसके नाम के बाद कोष्ठक का उपयोग करें:
```kotlin
greet() // prints Hello!
```
जो फंक्शन कोई मान नहीं लौटाता, वह अंतर्निहित रूप से `Unit` लौटाता है।

---

फंक्शन एक मान लौटा सकते हैं। आप कोलन का उपयोग करके कोष्ठक के बाद रिटर्न टाइप निर्दिष्ट करते हैं:
```kotlin
fun getNumber(): Int {
    return 42
}
```
`return` कीवर्ड कॉलर को एक मान वापस भेजता है:
```kotlin
var result = getNumber()
println(result) // prints 42
```
रिटर्न टाइप उस मान के टाइप से मेल खाना चाहिए जो आप लौटाते हैं।

---

फंक्शन पैरामीटर स्वीकार कर सकते हैं, जो वे इनपुट हैं जो आप फंक्शन को कॉल करते समय पास करते हैं।
पैरामीटर को नाम और टाइप के साथ कोष्ठक के अंदर घोषित किया जाता है:
```kotlin
fun greet(name: String) {
    println("Hello, $name!")
}
```
आप फंक्शन को कॉल करते समय आर्गुमेंट पास करते हैं:
```kotlin
greet("Alice") // prints Hello, Alice!
```
पैरामीटर आपको पुनः उपयोगी कोड लिखने देते हैं जो विभिन्न मानों के साथ काम करता है।

---

Kotlin डिफ़ॉल्ट पैरामीटर मानों का समर्थन करता है। यदि कॉलर कोई आर्गुमेंट नहीं देता, तो डिफ़ॉल्ट मान उपयोग किया जाता है:
```kotlin
fun greet(name: String = "World") {
    println("Hello, $name!")
}
greet()         // prints Hello, World!
greet("Alice")  // prints Hello, Alice!
```
डिफ़ॉल्ट मान पैरामीटर को वैकल्पिक बनाते हैं, जिससे ओवरलोडेड फंक्शन की आवश्यकता कम होती है।

---

जब फंक्शन का बॉडी एक ही एक्सप्रेशन से बना होता है, तो आप `=` के साथ सिंगल-एक्सप्रेशन सिंटैक्स का उपयोग कर सकते हैं:
```kotlin
fun square(n: Int): Int = n * n
```
यह ब्लॉक बॉडी फॉर्म से अधिक संक्षिप्त है। Kotlin अक्सर रिटर्न टाइप का अनुमान लगा सकता है, इसलिए आप यह भी लिख सकते हैं:
```kotlin
fun square(n: Int) = n * n
```
सिंगल-एक्सप्रेशन फंक्शन सरल गणनाओं के लिए एक सामान्य Kotlin मुहावरा है।

---

फंक्शन `Boolean` मान लौटा सकते हैं, जो स्थितियों की जांच करने के लिए उपयोगी है:
```kotlin
fun isEven(n: Int): Boolean {
    return n % 2 == 0
}
println(isEven(4)) // prints true
println(isEven(7)) // prints false
```
एक `Boolean` फंक्शन `true` या `false` लौटाता है।

---

फंक्शन में विभिन्न टाइप के कई पैरामीटर हो सकते हैं:
```kotlin
fun introduce(name: String, age: Int): String {
    return "My name is $name and I am $age years old."
}
println(introduce("Bob", 30))
// prints My name is Bob and I am 30 years old.
```
नामांकित आर्गुमेंट आपको पैरामीटर नाम का उपयोग करके किसी भी क्रम में मान पास करने देते हैं:
```kotlin
println(introduce(age = 30, name = "Bob"))
```

---

Kotlin में एक बेसिक फंक्शन `fun` कीवर्ड के बाद एक नाम और कोष्ठक का उपयोग करता है:
```kotlin
fun sayBye() {
    println("Bye!")
}
```

---

किसी फंक्शन के लिए रिटर्न टाइप घोषित करने के लिए, कोष्ठक के बाद कोलन और टाइप जोड़ें:
```kotlin
fun getName(): String {
    return "Alice"
}
```

---

एक फंक्शन पैरामीटर को नाम, कोलन और टाइप के साथ घोषित किया जाता है:
```kotlin
fun greet(name: String) {
    println("Hi, $name!")
}
```

---

एक सिंगल-एक्सप्रेशन फंक्शन ब्लॉक बॉडी के बजाय `=` का उपयोग करता है:
```kotlin
fun triple(n: Int) = n * 3
```
यह इसका संक्षिप्त रूप है:
```kotlin
fun triple(n: Int): Int {
    return n * 3
}
```
