# --description--

Kotlin डिफ़ॉल्ट पैरामीटर मानों का समर्थन करता है। यदि कॉलर कोई आर्गुमेंट नहीं देता, तो डिफ़ॉल्ट मान उपयोग किया जाता है:
```kotlin
fun greet(name: String = "World") {
    println("Hello, $name!")
}
greet()         // prints Hello, World!
greet("Alice")  // prints Hello, Alice!
```
डिफ़ॉल्ट मान पैरामीटर को वैकल्पिक बनाते हैं, जिससे ओवरलोडेड फंक्शन की आवश्यकता कम होती है।

# --instructions--

`multiply` नाम का एक फंक्शन परिभाषित करें जो दो `Int` पैरामीटर लेता है: `a` और `b` जिसका डिफ़ॉल्ट मान `2` है। `a * b` लौटाएं। केवल आर्गुमेंट `5` के साथ `multiply` को कॉल करें और परिणाम को `result` नाम के एक वेरिएबल में संग्रहीत करें।
