# --description--

फंक्शन `Boolean` मान लौटा सकते हैं, जो स्थितियों की जांच करने के लिए उपयोगी है:
```kotlin
fun isEven(n: Int): Boolean {
    return n % 2 == 0
}
println(isEven(4)) // prints true
println(isEven(7)) // prints false
```
एक `Boolean` फंक्शन `true` या `false` लौटाता है।
