# --description--

फंक्शन पैरामीटर स्वीकार कर सकते हैं, जो वे इनपुट हैं जो आप फंक्शन को कॉल करते समय पास करते हैं।
पैरामीटर को नाम और टाइप के साथ कोष्ठक के अंदर घोषित किया जाता है:
```kotlin
fun greet(name: String) {
    println("Hello, $name!")
}
```
आप फंक्शन को कॉल करते समय आर्गुमेंट पास करते हैं:
```kotlin
greet("Alice") // prints Hello, Alice!
```
पैरामीटर आपको पुनः उपयोगी कोड लिखने देते हैं जो विभिन्न मानों के साथ काम करता है।

# --instructions--

`double` नाम का एक फंक्शन परिभाषित करें जो `n` नाम का एक `Int` पैरामीटर लेता है और `n * 2` लौटाता है। इसे आर्गुमेंट `7` के साथ कॉल करें और परिणाम को `result` नाम के एक वेरिएबल में संग्रहीत करें।
