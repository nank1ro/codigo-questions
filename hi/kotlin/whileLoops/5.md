# --description--

यहाँ _while_ लूप के अंदर `break` के उपयोग का एक उदाहरण है:
```kotlin
var i = 0
while (i < 3) {
  i++
  if (i == 2) break // [1]
  println(i)
}
// prints 1
```

जैसा कि आप __[1]__ पर देख सकते हैं जब `i` _2_ के बराबर है, हम लूप को _break_ करते हैं। वास्तव में संख्या 2 और 3 कभी प्रिंट नहीं होती।
