# --instructions--

क्या यह कोड ब्लॉक कभी निष्पादित होगा?
```kotlin
var shouldRepeat = false

do {
    print("run code")
} while (shouldRepeat)
```

# --answers--

- हाँ क्योंकि do-while लूप की कंडीशन जांचने से पहले एक बार चलता है
- नहीं क्योंकि shouldRepeat false है
- यह कोड ब्लॉक दो बार निष्पादित होगा
- यह कोड ब्लॉक तीन बार निष्पादित होगा
