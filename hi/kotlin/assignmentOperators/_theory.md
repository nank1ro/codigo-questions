असाइनमेंट ऑपरेटरों का उपयोग वेरिएबल को मान असाइन करने के लिए किया जाता है।
सबसे बुनियादी असाइनमेंट ऑपरेटर `=` है, जो दाईं ओर के मान को बाईं ओर के वेरिएबल को असाइन करता है:
```kotlin
var x = 10
```
यहाँ, `10` वेरिएबल `x` को असाइन किया गया है।
आप किसी मौजूदा वेरिएबल को नया मान भी पुनः असाइन कर सकते हैं:
```kotlin
x = 20
```
अब `x` में मान `20` है।

---

`+=` ऑपरेटर एक शॉर्टहैंड है जो किसी वेरिएबल में एक मान जोड़ता है और परिणाम को वापस उसमें असाइन करता है।
इसके बजाय लिखने के:
```kotlin
x = x + 5
```
आप लिख सकते हैं:
```kotlin
x += 5
```
दोनों व्यंजन एक ही काम करते हैं: `x` के मान को `5` से बढ़ाते हैं।

---

`-=` ऑपरेटर किसी वेरिएबल से एक मान घटाता है और परिणाम को वापस उसमें असाइन करता है।
इसके बजाय लिखने के:
```kotlin
x = x - 3
```
आप लिख सकते हैं:
```kotlin
x -= 3
```
यह `x` के मान को `3` से कम करता है।

---

`*=` ऑपरेटर किसी वेरिएबल को एक मान से गुणा करता है और परिणाम को वापस उसमें असाइन करता है।
इसके बजाय लिखने के:
```kotlin
x = x * 4
```
आप लिख सकते हैं:
```kotlin
x *= 4
```
यह `x` को `4` से गुणा करता है और परिणाम को वापस `x` में संग्रहीत करता है।

---

`/=` ऑपरेटर किसी वेरिएबल को एक मान से विभाजित करता है और परिणाम को वापस उसमें असाइन करता है।
इसके बजाय लिखने के:
```kotlin
x = x / 2
```
आप लिख सकते हैं:
```kotlin
x /= 2
```
ध्यान दें: जब दोनों ऑपरेंड `Int` हों, तो Kotlin **पूर्णांक विभाजन** करता है (भिन्नात्मक भाग हटा दिया जाता है):
```kotlin
var x = 7
x /= 2
// x अब 3 है (3.5 नहीं)
```
