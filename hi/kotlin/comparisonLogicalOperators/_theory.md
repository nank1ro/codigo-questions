आइए **बराबर** `==` तुलना ऑपरेटर से शुरू करें।
यह एक **boolean** (`true` या `false`) लौटाता है जो बताता है कि दो एक्सप्रेशन बराबर हैं या नहीं, उदाहरण के लिए:
```kotlin
println(2 == 2) // true
println(2 == 3) // false
```

---

आइए **बराबर नहीं** `!=` तुलना ऑपरेटर के साथ आगे बढ़ें।
यह एक **boolean** (`true` या `false`) लौटाता है जो बताता है कि दो एक्सप्रेशन **बराबर नहीं** हैं, उदाहरण के लिए:
```kotlin
println(2 != 2) // false
println(2 != 3) // true
```
यह *बराबर* ऑपरेटर का बिल्कुल उल्टा है

---

आइए **से बड़ा** `>` तुलना ऑपरेटर के साथ आगे बढ़ें।
यह एक **boolean** (`true` या `false`) लौटाता है जो बताता है कि एक एक्सप्रेशन दूसरे से बड़ा है या नहीं, उदाहरण के लिए:
```kotlin
println(2 > 2) // false
println(3 > 2) // true
```

---

आइए **से छोटा** `<` तुलना ऑपरेटर के साथ आगे बढ़ें।
यह एक **boolean** (`true` या `false`) लौटाता है जो बताता है कि एक एक्सप्रेशन दूसरे से छोटा है या नहीं, उदाहरण के लिए:
```kotlin
println(2 < 2) // false
println(2 < 3) // true
```

---

आइए **से बड़ा या बराबर** `>=` तुलना ऑपरेटर के साथ आगे बढ़ें।
यह एक **boolean** (`true` या `false`) लौटाता है जो बताता है कि एक एक्सप्रेशन दूसरे से बड़ा या बराबर है या नहीं, उदाहरण के लिए:
```kotlin
println(2 >= 2) // true
println(3 >= 2) // true
println(3 >= 4) // false
```

---

आइए **से छोटा या बराबर** `<=` तुलना ऑपरेटर के साथ आगे बढ़ें।
यह एक **boolean** (`true` या `false`) लौटाता है जो बताता है कि एक एक्सप्रेशन दूसरे से छोटा या बराबर है या नहीं, उदाहरण के लिए:
```kotlin
println(2 <= 2) // true
println(3 <= 2) // false
println(3 <= 4) // true
```

---

अब आइए **लॉजिकल** ऑपरेटर देखें, पहले __AND__ `&&` से शुरू करते हैं।
यह पहला ऑपरेंड लौटाता है जो *false* के रूप में मूल्यांकित होता है या अंतिम ऑपरेंड अगर सभी *true* हैं।
```kotlin
println(2 == 2 && 2 == 3) // false
println(1 == 1 && 1.0 == 1.0) // true
```

---

आइए **or** `||` लॉजिकल ऑपरेटर के साथ आगे बढ़ें।
यह पहला ऑपरेंड लौटाता है जो *true* के रूप में मूल्यांकित होता है या अंतिम ऑपरेंड अगर सभी *false* हैं।
```kotlin
println(2 == 2 || 2 == 3) // true
println(1 == 2 || 1 == 3) // false
```

---

अंत में **not** `!` लॉजिकल ऑपरेटर देखते हैं।
यह एक boolean लौटाता है जो किसी एक्सप्रेशन की लॉजिकल स्थिति का उल्टा होता है।
```kotlin
println(!true) // false
println(!false) // true
println(!(2 == 2)) // false
```
