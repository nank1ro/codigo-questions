# --description--

आइए **से छोटा या बराबर** `<=` तुलना ऑपरेटर के साथ आगे बढ़ें।
यह एक **boolean** (`true` या `false`) लौटाता है जो बताता है कि एक एक्सप्रेशन दूसरे से छोटा या बराबर है या नहीं, उदाहरण के लिए:
```kotlin
println(2 <= 2) // true
println(3 <= 2) // false
println(3 <= 4) // true
```
