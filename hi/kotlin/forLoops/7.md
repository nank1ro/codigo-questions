# --description--

यदि आपको बस एक कोड ब्लॉक को `n` बार दोहराना है, तो आप `repeat(times: Int)` फ़ंक्शन का उपयोग कर सकते हैं।

```kotlin
repeat(3) {
    println("repeat")
}
// prints repeat 3 times
```

आप index को भी एक्सेस कर सकते हैं
```kotlin
repeat(3) { index ->
    println(index)
}
// prints 0, 1, 2
```
