# --description--

Kotlin में हम `for-in` का उपयोग iterable collections के लिए भी कर सकते हैं, जो प्रत्येक तत्व पर दिए गए closure को कॉल करता है:
```kotlin
// this is a list, we'll see about that soon
val numbers = listOf(2, 4, 6, 8, 10)
for (num in numbers) {
    println(num)
}
// prints (2, 4, 6, 8, 10)
```
