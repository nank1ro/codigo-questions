# --description--

क्लासेस में उनके **प्राथमिक कंस्ट्रक्टर** में कई प्रॉपर्टी हो सकती हैं। प्रत्येक प्रॉपर्टी को कंस्ट्रक्टर पैरामीटर सूची में सीधे `val` (केवल-पठनीय) या `var` (परिवर्तनीय) के साथ घोषित किया जाता है:

```kotlin
class Person(val name: String, var age: Int)
```

`val` कीवर्ड `name` को निर्माण के बाद केवल-पठनीय बनाता है, जबकि `var` `age` को परिवर्तनीय बनाता है — आप इसे बाद में बदल सकते हैं:

```kotlin
val p = Person("Alice", 30)
p.age = 31  // अनुमत है क्योंकि age var है
```

# --instructions--

एक क्लास `Person` बनाएं जिसमें `String` प्रकार की एक `val` प्रॉपर्टी `name` और `Int` प्रकार की एक `var` प्रॉपर्टी `age` हो।

# --asserts--

`Person("Alice", 30)` में `name` का मान `"Alice"` होना चाहिए

```kotlin
    val p = Person("Alice", 30)
    tryCatch(p.name == "Alice")
```

`Person("Alice", 30)` में `age` का मान `30` होना चाहिए

```kotlin
    tryCatch(p.age == 30)
```

`age` को `31` में बदलने से प्रॉपर्टी अपडेट होनी चाहिए

```kotlin

    tryCatch(p.age == 31)
```
