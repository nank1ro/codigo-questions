# --instructions--

एक **companion object** एक क्लास से जुड़ा एक सिंगलटन होता है। यह आपको ऐसी प्रॉपर्टी और मेथड परिभाषित करने देता है जो इंस्टेंस के बजाय क्लास से संबंधित हों — अन्य भाषाओं में `static` सदस्यों के समान:

```kotlin
class MathHelper {
    companion object {
        fun square(n: Int): Int = n * n
    }
}
println(MathHelper.square(4)) // 16
```

एक क्लास `Counter` बनाएं जिसमें एक companion object हो जिसमें `Int` प्रकार की एक `var` प्रॉपर्टी `total` (`0` से इनिशियलाइज़) और एक मेथड `increment()` हो जो `total` में `1` जोड़े।
