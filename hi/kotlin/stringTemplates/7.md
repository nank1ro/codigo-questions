# --description--

हम string templates के अंदर जितनी बार चाहें उतनी बार मान डालने के लिए curly brackets का उपयोग कर सकते हैं

# --instructions--

इसे यहाँ `2` के साथ और फिर `8` के साथ देखें

# --seed--

```kotlin
println("[/] new friends and [/] old friends")
```

# --answers--

- $2
- $8
- ${2}
- ${8}

# --solutions--

```kotlin
println("${2} new friends and ${8} old friends")
```

# --output--

2 new friends and 8 old friends
