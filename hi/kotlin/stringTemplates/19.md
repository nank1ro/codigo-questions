# --instructions--

String template की खाली जगह में संख्या `500` डालने के लिए `${}` का उपयोग करें

# --seed--

```kotlin
fun main() {
    println("There are  items")
}
```

# --solutions--

```kotlin
fun main() {
    println("There are ${500} items")
}
```
