# --description--

String _template_ एक String उत्पन्न करने का प्रोग्रामेटिक तरीका है।
Kotlin में हम दो या अधिक strings को एक साथ प्रदर्शित करने के लिए `+` चिह्न (concatenation) का उपयोग कर सकते हैं, जैसे:
```kotlin
println("Hello " + "Kotlin!")
// prints "Hello Kotlin!"
```
