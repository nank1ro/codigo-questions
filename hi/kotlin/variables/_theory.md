वेरिएबल डेटा मानों को संग्रहीत करने के कंटेनर हैं।
Kotlin में प्रत्येक वेरिएबल एक ऑब्जेक्ट है।
एक वेरिएबल बनाने के लिए, हमें इसे एक __नाम__ देना होगा और यह ध्यान रखना होगा कि इसमें स्पेस नहीं होने चाहिए।
एक वेरिएबल उस समय बनता है जब आप पहली बार इसे कोई मान असाइन करते हैं।
Kotlin में आप `val` (_value_ का संक्षिप्त रूप) कीवर्ड से कॉन्स्टेंट और `var` (_variable_ का संक्षिप्त रूप) कीवर्ड से वेरिएबल घोषित करते हैं।
एक कॉन्स्टेंट का मान एक बार सेट होने के बाद बदला नहीं जा सकता, जबकि एक वेरिएबल को भविष्य में एक अलग मान दिया जा सकता है।
`x` नाम के वेरिएबल बनाने का एक उदाहरण है:
```kotlin
var x = 1
```
इस तरह हमने `x` नाम के वेरिएबल को मान `1` असाइन किया है।
यदि हम वेरिएबल `x` को प्रिंट करें तो हमें संख्या `1` मिलती है:
```kotlin
println(x) // prints 1
```

---

वेरिएबल को इसलिए ऐसा कहा जाता है क्योंकि उनमें संग्रहीत मान बदल सकता है।
हम `=` का उपयोग करके और एक नया मान देकर `x` को अपडेट कर सकते हैं।
```kotlin
var x = 1
println(x) // prints 1
x = 2
println(x) // prints 2
```

---

हम वेरिएबल को अन्य वेरिएबल के मान भी दे सकते हैं। यहाँ, हम `y` वेरिएबल को `x` का मान दे सकते हैं
```kotlin
var x = 5
var y = x
println(y) // prints 5
```

---

जब हम एक वेरिएबल को अपडेट करते हैं, तो यह अपना पिछला मान भूल जाता है। यहाँ हम `x` वेरिएबल को दो बार प्रदर्शित कर सकते हैं और देख सकते हैं कि इसका मान कैसे अपडेट होता है।
```kotlin
var x = 5
println(x) // prints 5
x = 10
println(x) // prints 10
```

---

Kotlin में स्ट्रिंग वेरिएबल केवल डबल कोट्स का उपयोग करके घोषित किए जा सकते हैं:
```kotlin
val x = "May"
```

---

यदि हम कई शब्दों वाला वेरिएबल नाम चाहते हैं, तो हम **camelCase** का उपयोग करते हैं।
यह वाक्यांशों को इस तरह लिखने की प्रथा है कि वाक्यांश के बीच में प्रत्येक शब्द बड़े अक्षर से शुरू होता है
