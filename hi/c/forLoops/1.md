# --description--

हम जानते हैं कि `while` लूप का उपयोग करके कोड को कैसे दोहराया जाता है।
जैसे यह प्रोग्राम `hello` प्रदर्शित करने के लिए स्टेटमेंट दोहराता है
```c
int counter = 0;

while (counter < 5) {
    printf("Hello\n");
    counter++;
}
```
लेकिन हम `for` लूप से भी यही कर सकते हैं:
```c
for (int i = 0; i < 5; i++) {
    printf("Hello\n");
}
```
