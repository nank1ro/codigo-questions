# --description--

आइए `floor()` फ़ंक्शन का उपयोग करके **फ़्लोर डिवीज़न** देखें।
यह फ़ंक्शन भागफल का पूर्णांक भाग लौटाता है, उदाहरण के लिए:
```
>>> 5.0 / 2
2.5
>>> floor(2.5)
2.0
```
इसे पूर्णांक भाग भी कहा जाता है। परिणामी मान एक पूर्ण पूर्णांक होता है, हालांकि परिणाम का *प्रकार* आवश्यक रूप से int नहीं होता।
