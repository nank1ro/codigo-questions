# --description--

ऑपरेटर्स का उपयोग वेरिएबल्स और वैल्यूज़ पर ऑपरेशन करने के लिए किया जाता है।
आइए अंकगणितीय ऑपरेटर्स से शुरू करें, विशेष रूप से **जोड़** `+` ऑपरेटर से।
इसका उपयोग दो संख्याओं को जोड़ने के लिए किया जाता है, जैसे:
```
>>> 5 + 3
8
```
