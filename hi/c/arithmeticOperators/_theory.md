ऑपरेटर्स का उपयोग वेरिएबल्स और वैल्यूज़ पर ऑपरेशन करने के लिए किया जाता है।
आइए अंकगणितीय ऑपरेटर्स से शुरू करें, विशेष रूप से **जोड़** `+` ऑपरेटर से।
इसका उपयोग दो संख्याओं को जोड़ने के लिए किया जाता है, जैसे:
```
>>> 5 + 3
8
```

---

आइए **घटाव** `-` ऑपरेटर के साथ आगे बढ़ें।
इसका उपयोग एक संख्या को दूसरी से घटाने के लिए किया जाता है, जैसे:
```
>>> 5 - 3
2
```

---

आइए **गुणा** `*` ऑपरेटर देखें।
इसका उपयोग दो संख्याओं को आपस में गुणा करने के लिए किया जाता है, जैसे:
```
>>> 5 * 3
15
```

---

आइए **भाग** `/` ऑपरेटर देखें।
इसका उपयोग दो संख्याओं को विभाजित करने के लिए किया जाता है, जैसे:
```c
>>> 10 / 5
2
```

---

आइए **मॉड्यूलस** `%` ऑपरेटर देखें।
इसका उपयोग दो संख्याओं के भाग के बाद शेषफल ज्ञात करने के लिए किया जाता है, जैसे:
```
>>> 5 % 2
1
```
यह 1 का मूल्यांकन करता है क्योंकि 5 को 2 से भाग करने पर भागफल 2 और शेषफल 1 आता है
```
>>> 9 % 3
0
```
यह दूसरा 0 का मूल्यांकन करता है क्योंकि 9 को 3 से भाग करने पर भागफल 3 और शेषफल 0 आता है

---

C में **घातांक** ऑपरेटर नहीं होता, इसलिए हमें `math.h` लाइब्रेरी में शामिल `pow()` फ़ंक्शन का उपयोग करना होता है।
घातांक का अर्थ है आधार का बार-बार गुणा: अर्थात, **b** घातांक *n* के साथ *n* आधारों के गुणनफल के बराबर है:
![exponentiation](https://bit.ly/3zcz6Lt)
```
>>> pow(5, 2);
25
```

---

आइए `floor()` फ़ंक्शन का उपयोग करके **फ़्लोर डिवीज़न** देखें।
यह फ़ंक्शन भागफल का पूर्णांक भाग लौटाता है, उदाहरण के लिए:
```
>>> 5.0 / 2
2.5
>>> floor(2.5)
2.0
```
इसे पूर्णांक भाग भी कहा जाता है। परिणामी मान एक पूर्ण पूर्णांक होता है, हालांकि परिणाम का *प्रकार* आवश्यक रूप से int नहीं होता।
