# --description--

आइए **भाग असाइनमेंट** `/=` ऑपरेटर देखें।
यह वेरिएबल को एक मान से भाग करता है और परिणाम को उसी वेरिएबल में असाइन करता है।
तो निम्नलिखित बिल्कुल समान हैं
```c
num = num / 5;
// is equal to
num /= 5;
```
