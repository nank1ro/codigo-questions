आइए **बराबर** `==` संबंधात्मक ऑपरेटर से शुरू करें।
यह एक **boolean** लौटाता है, true `1` या false `0`, जो बताता है कि दो व्यंजक बराबर हैं या नहीं, उदाहरण के लिए:
```c
>>> 2 == 2
1
>>> 2 == 3
0
```

---

आइए **असमान** `!=` संबंधात्मक ऑपरेटर के साथ जारी रखें।
यह एक **boolean** लौटाता है, true `1` या false `0`, जो बताता है कि दो व्यंजक **बराबर नहीं** हैं, उदाहरण के लिए:
```c
>>> 2 != 2
0
>>> 2 != 3
1
```
यह *बराबर* ऑपरेटर के बिल्कुल विपरीत है

---

आइए **से बड़ा** `>` संबंधात्मक ऑपरेटर के साथ जारी रखें।
यह एक **boolean** लौटाता है, true `1` या false `0`, जो बताता है कि एक व्यंजक दूसरे से बड़ा है या नहीं, उदाहरण के लिए:
```c
>>> 2 > 2
0
>>> 3 > 2
1
```

---

आइए **से छोटा** `<` संबंधात्मक ऑपरेटर के साथ जारी रखें।
यह एक **boolean** लौटाता है, true `1` या false `0`, जो बताता है कि एक व्यंजक दूसरे से छोटा है या नहीं, उदाहरण के लिए:
```c
>>> 2 < 2
0
>>> 2 < 3
1
```

---

आइए **से बड़ा या बराबर** `>=` संबंधात्मक ऑपरेटर के साथ जारी रखें।
यह एक **boolean** लौटाता है, true `1` या false `0`, जो बताता है कि एक व्यंजक दूसरे से बड़ा या बराबर है या नहीं, उदाहरण के लिए:
```c
>>> 2 >= 2
1
>>> 3 >= 2
1
>>> 3 >= 4
0
```

---

आइए **से छोटा या बराबर** `<=` संबंधात्मक ऑपरेटर के साथ जारी रखें।
यह एक **boolean** लौटाता है, true `1` या false `0`, जो बताता है कि एक व्यंजक दूसरे से छोटा या बराबर है या नहीं, उदाहरण के लिए:
```c
>>> 2 <= 2
1
>>> 3 <= 2
0
>>> 3 <= 4
1
```

---

अब आइए **boolean** ऑपरेटरों को देखें, पहले __and__ `&&` से शुरू करते हैं।
यह पहला ऑपरेंड लौटाता है जो *false* का मूल्यांकन करता है या अंतिम यदि सभी *true* हैं।
```c
>>> 2 == 2 && 2 == 3
0
>>> 1 == 1 && 1 == 1.0
1
```

---

आइए **or** `||` boolean ऑपरेटर के साथ जारी रखें।
यह पहला ऑपरेंड लौटाता है जो *true* का मूल्यांकन करता है या अंतिम यदि सभी *false* हैं।
```c
>>> 2 == 2 || 2 == 3
1
>>> 1 == 2 || 1 == 3
0
```

---

आइए **not** `!` boolean ऑपरेटर के साथ समाप्त करें।
यह एक boolean लौटाता है जो किसी व्यंजक की तार्किक स्थिति के विपरीत होता है।
```c
>>> !true
0
>>> !false
1
>>> !(2 == 2)
0
```
