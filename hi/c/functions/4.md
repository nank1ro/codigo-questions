# --description--

फ़ंक्शन में कई इनपुट पैरामीटर हो सकते हैं, जो फ़ंक्शन के कोष्ठक के भीतर अल्पविराम द्वारा अलग किए जाते हैं।
```c
void say_hello(char *name, bool new_user) {
  char greet[40] = "Hello ";
  strcat(greet, name);
  if (new_user) {
    strcat(greet, "! Welcome on board :)");
  }
  printf("%s\n", greet);
}

int main() {
    // prints "Hello Tom"
    say_hello("Tom", true);
    return 0;
};
```

# --instructions--

दूसरे पैरामीटर में `true` पास करके `say_hello` नामक एक वैध फ़ंक्शन बनाने और परिणाम प्रिंट करने के लिए कोड को पूरा करें
