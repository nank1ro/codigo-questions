# --description--

C में हम दो या अधिक मानों को एक साथ प्रदर्शित करने के लिए `printf` फ़ंक्शन का उपयोग एक एक्सप्रेशन के साथ करते हैं, जैसे:
```c
>>> char c = 'C';
>>> printf("Hello %c!\n", c);
Hello C!
```
`%c` का उपयोग एक अकेला कैरेक्टर प्रदर्शित करने के लिए किया जाता है
