# --description--

हम एक सशर्त स्टेटमेंट (`if`, `else if` या `else`) को दूसरे सशर्त स्टेटमेंट के अंदर नेस्ट भी कर सकते हैं, ताकि एक अधिक जटिल संरचना बनाई जा सके।
```c
int num = 4;
if (num < 3) {
    printf("the number is lower than 3\n");
} else {
    if (num == 3) {
        printf("the number is 3\n");
  } else if (num == 4) {
        printf("the number is 4\n");
  } else {
        printf("the number is greather than 4\n");
  }
}
```
और इस कोड का आउटपुट `the number is 4` है।

# --instructions--

आइए दिन के समय के प्रोग्राम को बदलें, `else` स्टेटमेंट के अंदर एक सशर्त स्टेटमेंट नेस्ट करें।
नेस्टेड सशर्त स्टेटमेंट को जाँचना चाहिए कि `time` `18` से *कम* है या नहीं और कोड ब्लॉक में `"It's afternoon"` प्रिंट करना चाहिए।
यदि पिछली शर्त `false` है, तो `"It's evening"` प्रिंट करें
