# --description--

"__Hello, World!__" किसी नई भाषा में प्रोग्रामिंग शुरू करने के लिए पारंपरिक पहला प्रोग्राम है।
हम मानक आउटपुट डिवाइस (स्क्रीन) पर डेटा प्रदर्शित करने के लिए `printf()` फ़ंक्शन का उपयोग करते हैं।
हर C प्रोग्राम लाइब्रेरीज़ का उपयोग करता है, जो आवश्यक फ़ंक्शन चलाने की क्षमता देती हैं, उदाहरण के लिए स्क्रीन पर प्रिंट करने का फ़ंक्शन `stdio.h` हेडर फ़ाइल में परिभाषित है।
पहला कोड जो चलेगा वह हमेशा main फ़ंक्शन में होगा।
C में स्क्रीन पर `Hello, World!` प्रिंट करने के लिए हम लिख सकते हैं
```c
printf("Hello, World!");
```
अंत में हम `0` रिटर्न करते हैं जो दर्शाता है कि हमारा प्रोग्राम सफल रहा
