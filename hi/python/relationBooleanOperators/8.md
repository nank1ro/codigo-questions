# --description--

आइए **or** बूलियन ऑपरेटर के साथ आगे बढ़ते हैं।
यह पहला ऑपरेंड लौटाता है जो *True* का मूल्यांकन करता है या यदि सभी *False* हैं तो अंतिम ऑपरेंड लौटाता है।
```python
>>> 2 == 2 or 2 == 3
True
>>> 1 == 2 or 1 == 3
False
```
