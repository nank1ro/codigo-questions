# --description--

अब आइए **बूलियन** ऑपरेटर देखें, पहले `and` नामक ऑपरेटर से शुरू करते हैं।
यह पहला ऑपरेंड लौटाता है जो *False* का मूल्यांकन करता है या यदि सभी *True* हैं तो अंतिम ऑपरेंड लौटाता है।
```python
>>> 2 == 2 and 2 == 3
False
>>> 1 == 1 and 1 == 1.0
True
```
