# --description--

आइए **से छोटा** `<` संबंधपरक ऑपरेटर के साथ आगे बढ़ते हैं।
यह एक **बूलियन** (`True` या `False`) लौटाता है जो बताता है कि एक व्यंजक दूसरे से छोटा है या नहीं, उदाहरण के लिए:
```python
>>> 2 < 2
False
>>> 2 < 3
True
```
