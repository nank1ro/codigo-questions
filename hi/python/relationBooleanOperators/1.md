# --description--

आइए **बराबर** `==` संबंधपरक ऑपरेटर से शुरू करते हैं।
यह एक **बूलियन** (`True` या `False`) लौटाता है जो बताता है कि दो व्यंजक बराबर हैं या नहीं, उदाहरण के लिए:
```python
>>> 2 == 2
True
>>> 2 == 3
False
```
