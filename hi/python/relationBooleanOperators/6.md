# --description--

आइए **से छोटा या बराबर** `<=` संबंधपरक ऑपरेटर के साथ आगे बढ़ते हैं।
यह एक **बूलियन** (`True` या `False`) लौटाता है जो बताता है कि एक व्यंजक दूसरे से छोटा या बराबर है या नहीं, उदाहरण के लिए:
```python
>>> 2 <= 2
True
>>> 3 <= 2
False
>>> 3 <= 4
True
```
