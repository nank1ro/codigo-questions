# --description--

आइए **not** बूलियन ऑपरेटर के साथ समाप्त करते हैं।
यह एक बूलियन लौटाता है जो किसी व्यंजक की तार्किक स्थिति का उल्टा होता है।
```python
>>> not True
False
>>> not False
True
>>> not 2 == 2
False
```
