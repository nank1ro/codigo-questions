# --description--

आइए **से बड़ा या बराबर** `>=` संबंधपरक ऑपरेटर के साथ आगे बढ़ते हैं।
यह एक **बूलियन** (`True` या `False`) लौटाता है जो बताता है कि एक व्यंजक दूसरे से बड़ा या बराबर है या नहीं, उदाहरण के लिए:
```python
>>> 2 >= 2
True
>>> 3 >= 2
True
>>> 3 >= 4
False
```
