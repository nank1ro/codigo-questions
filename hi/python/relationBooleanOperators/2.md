# --description--

आइए **बराबर नहीं** `!=` संबंधपरक ऑपरेटर के साथ आगे बढ़ते हैं।
यह एक **बूलियन** (`True` या `False`) लौटाता है जो बताता है कि दो व्यंजक **बराबर नहीं** हैं, उदाहरण के लिए:
```python
>>> 2 != 2
False
>>> 2 != 3
True
```
यह *बराबर* ऑपरेटर का बिल्कुल विपरीत है
