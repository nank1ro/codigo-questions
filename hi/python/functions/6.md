# --instructions--

इस कोड स्निपेट में __समस्या__ क्या है?
```python
greet('Mathew')

def greet(name):
    """This function greets to the person passed in as a parameter"""
    print(f"Hello, {name}. Good morning!")

greet('Alex')
```

# --answers--

- हम greet() को परिभाषित करने से पहले उपयोग नहीं कर सकते
- हम greet() को दो बार उपयोग नहीं कर सकते
- greet() कुछ भी लौटाता नहीं है
- हम name को पैरामीटर के रूप में उपयोग नहीं कर सकते
