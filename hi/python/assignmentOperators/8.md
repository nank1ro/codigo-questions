# --description--

आइए **फ्लोर असाइनमेंट** `//=` ऑपरेटर देखें।
यह वेरिएबल को एक वैल्यू से फ्लोर डिवीज़न करता है और परिणाम को उस वेरिएबल में असाइन करता है।
तो निम्नलिखित बिल्कुल समान हैं
```python
num = num // 5
# is equal to
num //= 5
```

# --instructions--
