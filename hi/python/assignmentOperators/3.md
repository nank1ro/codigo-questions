# --description--

एडिशन असाइनमेंट की तरह ही, हमारे पास **डिक्रीमेंट असाइनमेंट** `-=` है।
कार्यप्रणाली वही है, बस अंतर यह है कि यह घटाव करता है।
तो निम्नलिखित बिल्कुल समान हैं
```python
num = num - 5
# is equal to
num -= 5
```
