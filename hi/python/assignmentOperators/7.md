# --description--

आइए **मॉड्यूलस असाइनमेंट** `%=` ऑपरेटर देखें।
यह वेरिएबल और एक वैल्यू का मॉड्यूलस निकालता है और परिणाम को उस वेरिएबल में असाइन करता है।
तो निम्नलिखित बिल्कुल समान हैं
```python
num = num % 5
# is equal to
num %= 5
```
