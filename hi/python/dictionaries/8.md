# --description--

लिस्ट की तरह, हम `for..in` कीवर्ड का उपयोग करके dictionary के तत्वों पर लूप कर सकते हैं।
लूप में key और value दोनों प्राप्त करने के लिए हम `items()` मेथड का उपयोग कर सकते हैं:
```python
for key, value in dict_name:
    print(key, value)
```
