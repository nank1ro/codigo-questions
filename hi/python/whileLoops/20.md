# --instructions--

हमें कैसे पता चलता है कि `while` लूप अपना कोड ब्लॉक छोड़ देता है?
```python
is_loading = False

while is_loading:
    print("Loading...")
```

# --answers--

- शर्त में वेरिएबल की True से तुलना की जाती है
- शर्त में वेरिएबल की False से तुलना की जाती है
