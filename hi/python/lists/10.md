# --description--

Python में हम `for..in` कीवर्ड का उपयोग करके बहुत सरल तरीके से लिस्ट में लूप कर सकते हैं:
```python
>>> numbers = [1, 2, 3]
>>> for num in numbers:
>>>     print(num)
1
2
3
```
`for` कीवर्ड के बाद एक वेरिएबल नाम आता है, इसे बारी-बारी से प्रत्येक लिस्ट आइटम का मान असाइन किया जाएगा।
