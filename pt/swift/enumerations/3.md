# --description--

Podemos combinar valores individuais de enumeração com uma declaração `switch`:
```swift
let color = Colors.red
switch color {
    case .blue:
        print("Blue")
    case .red:
        print("Red")
    case .green:
        print("Green")
}
// prints "Red"
```
Lembre-se de que se você não precisar fornecer um `case` para cada caso de enumeração, você pode fornecer um caso `default` para cobrir quaisquer casos que não sejam abordados explicitamente
