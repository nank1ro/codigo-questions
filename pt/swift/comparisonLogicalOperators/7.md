# --description--

Agora vamos ver os operadores **lógicos**, começando pelo primeiro chamado __AND__ `&&`.
Ele retorna o primeiro operando que é avaliado como *false* ou o último se todos forem *true*.
```swift
print(2 == 2 && 2 == 3) // false
print(1 == 1 && 1 == 1.0) // true
```
