# --description--

Vamos continuar com o operador de comparação **diferente** `!=`.
Ele retorna um **booleano** (`true` ou `false`) indicando se duas expressões **NÃO** são iguais, por exemplo:
```swift
print(2 != 2) // false
print(2 != 3) // true
```
É exatamente o oposto do operador *igual*
