# --description--

Vamos continuar com o operador de comparação **menor ou igual a** `<=`.
Ele retorna um **booleano** (`true` ou `false`) indicando se uma expressão é menor ou igual à outra, por exemplo:
```swift
print(2 <= 2) // true
print(3 <= 2) // false
print(3 <= 4) // true
```
