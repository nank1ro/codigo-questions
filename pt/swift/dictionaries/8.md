# --description--

Assim como para arrays, podemos percorrer os elementos do dicionário usando as palavras-chave `for..in`.
Para obter tanto a chave quanto o valor no loop não precisamos especificar nenhuma propriedade:
```swift
for (key, value) in dictName {
    print("\(key): \(value)")
}
```
