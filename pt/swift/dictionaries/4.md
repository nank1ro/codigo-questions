# --description--

O tamanho `count` de um dicionário é o número de _pares chave-valor_ que ele tem.
Cada par conta apenas uma vez, mesmo que o valor seja um array. (Isso mesmo: você também pode colocar arrays dentro de dicionários!)

# --instructions--

Imprima o tamanho do dicionário
