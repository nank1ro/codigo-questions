# --description--

Como os dicionários são mutáveis, eles podem ser alterados de muitas maneiras. Itens podem ser removidos de um dicionário com o método `removeValue(forKey:)`:
```swift
if let removedValue = dictName.removeValue(forKey: "keyName") {
    print("The removed value is \(removedValue).") // imprime o valor removido, se a chave existir
}
```
removerá a chave `keyName` e seu valor associado do dicionário.
