# --description--

Vamos ver o operador de **atribuição de multiplicação** `*=`.
Multiplica a variável por um valor e atribui o resultado a essa variável.
Portanto, os seguintes são exatamente iguais
```swift
var num = num * 5
// is equal to
num *= 5
```
