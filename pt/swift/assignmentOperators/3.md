# --description--

Assim como na atribuição de adição, temos a **atribuição de decremento** `-=`.
A funcionalidade é a mesma, a única diferença é que realiza a subtração.
Portanto, os seguintes são exatamente iguais
```swift
var num = num - 5
// is equal to
num -= 5
```
