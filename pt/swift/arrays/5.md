# --description--

Um array não precisa ter um tamanho fixo.
Você pode adicionar itens ao final de um array a qualquer hora!
Para adicionar um item a um array usamos a função `append`:
```swift
var letters = ["a", "b"]
letters.append("c")
print(letters)
// Prints ["a", "b", "c"]
```
