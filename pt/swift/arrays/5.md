# --description--

Um array nao precisa ter um tamanho fixo.
Voce pode adicionar itens ao final de um array quando quiser!
Para adicionar um item a um array, usamos a funcao `append`:
```swift
var letters = ["a", "b"]
letters.append("c")
print(letters)
// Prints ["a", "b", "c"]
```
