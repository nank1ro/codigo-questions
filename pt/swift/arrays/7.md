# --description--

Em Swift podemos fatiar um array como quisermos!
```swift
// Pega os primeiros dois itens
listName[..<2]
// Pega do quarto ao último item
listName[3...]
```
Se o seu slice de array incluir o primeiro ou último item de um array, o índice desse item não precisa ser incluído
