# --description--

Você pode acessar um item individual do array pelo seu índice.
Um índice é como um endereço que identifica o lugar do item no array.
O índice aparece diretamente após o nome do array, entre colchetes, assim:
```swift
arrayName[index]
```

Os índices de array começam com `0`, **não** `1`! Você acessa o primeiro item de um array assim: `arrayName[0]`.
O segundo item de um array está no índice 1: `arrayName[1]`.
