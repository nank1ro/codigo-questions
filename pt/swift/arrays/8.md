# --description--

Os elementos do array podem ser de qualquer tipo, se especificarmos o tipo `Any`:
```swift
var arrayName: [Any] = ["one", 2, true]
```
Na verdade, acima temos, em ordem, uma string, um inteiro e um booleano.
Mas também podemos ter arrays com arrays!
