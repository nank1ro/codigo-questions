# --description--

Em Swift podemos percorrer um array de uma forma muito simples usando as palavras-chave `for..in`:
```swift
var numbers = [1, 2, 3]
for num in numbers {
    print(num)
}
// imprime 1, 2, 3 
```
Um nome de variável segue a palavra-chave `for`, ele será atribuído ao valor de cada item do array por vez.
