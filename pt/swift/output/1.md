# --description--

"__Hello, World!__" é o primeiro programa tradicional para quem está começando a programar em uma nova linguagem.
Usamos a função `print()` para exibir dados no dispositivo de saída padrão (tela).
Para imprimir `Hello, World!` na tela com Swift, podemos escrever
```swift
print("Hello, World!")
```
