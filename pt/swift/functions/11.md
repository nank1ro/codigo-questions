# --instructions--

Qual é o __problema__ com este trecho de código?
```swift
print(greet(name: "Mathew"))
/// This function greets to the person passed in as argument
func greet(name: String) -> String {
    return "Hello, \(name). Good morning!"
}
print(greet(name: "Alex"))
```

# --answers--

- Não podemos usar greet() antes de defini-la
- Não podemos usar greet() duas vezes
- greet() não retorna nada
- Não podemos usar name como parâmetro
