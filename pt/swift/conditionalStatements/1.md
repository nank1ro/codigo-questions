# --description--

Vamos assumir que temos outra condição para verificar, como neste exemplo:
```swift
var num = 3
if num == 2 {
    print("the number is 2")
} else if num == 3 {
    print("the number is 3")
} else {
    print("do something else")
}
```
e a saída deste código é `the number is 3`.
Primeiro, vamos verificar se o número é igual a 2, isso é falso.
Então vamos passar para a segunda instrução e verificar se `num` é igual a 3, sendo verdadeiro executamos o seguinte bloco de código imprimindo `the number is 3`
