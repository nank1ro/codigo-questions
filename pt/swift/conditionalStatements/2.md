# --description--

A tomada de decisões é necessária quando queremos executar código apenas se uma determinada condição for satisfeita.
Vamos assumir que queremos brincar fora apenas se o tempo estiver bom.
Na programação, podemos salvar uma variável booleana `niceWeather` e realizar a ação de brincar fora `if` essa variável for `true`, como:
```swift
var niceWeather = true
if niceWeather {
    // play outside
}
```
