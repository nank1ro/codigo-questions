# --description--

O operador condicional ternário é um operador especial com três partes, que tem a forma `question ? answer1 : answer2`.
É um atalho para avaliar uma de duas expressões com base se `question` é verdadeira ou falsa.
Se `question` for verdadeira, ele avalia `answer1` e retorna seu valor; caso contrário, avalia `answer2` e retorna seu valor.
```swift
let a = 10, b = 20, c: Int
if (a < b) {
    c = a
} else {
    c = b
}
print(c)
```
O código abreviado para o código acima é:
```swift
let a = 10, b = 20, c: Int
c = a < b ? a : b
print(c)
```
`c` é definido como igual a `a`, porque a condição `a < b` era verdadeira
