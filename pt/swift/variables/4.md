# --description--

Quando atualizamos uma variavel, ela esquece seu valor anterior. Aqui podemos exibir a variavel `x` duas vezes e ver como seu valor atualiza.
```swift
var x = 5
print(x) // imprime 5
x = 10
print(x) // imprime 10
```
