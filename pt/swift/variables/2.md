# --description--

Variáveis são chamadas assim porque o valor que armazenam pode mudar.
Podemos atualizar `x` usando `=` e atribuindo um novo valor.
```swift
var x = 1
print(x) // prints 1
x = 2
print(x) // prints 2
```
