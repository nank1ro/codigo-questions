# --description--

Se quisermos um nome de variavel com varias palavras, usamos **camelCase**.
E a pratica de escrever frases de forma que cada palavra no meio da frase comeca com uma letra maiuscula

# --instructions--

Preencha os espacos em branco para criar uma variavel *camelCase*.
