# --description--

Vamos ver o operador de **resto** `%`.
É usado para encontrar o resto após uma divisão entre dois números, como:
```swift
print(5 % 2) // 1
```
Isso avalia para 1 porque 5 dividido por 2 tem um quociente de 2  e um resto de 1
```swift
print(9 % 3) // 0
```
Este outro avalia para 0 porque 9 dividido por 3 tem um quociente de 3  e deixa um resto de 0
