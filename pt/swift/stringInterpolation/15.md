# --instructions--

O que ha de errado com este codigo?
```swift
print("\(10} friends")
```

# --answers--

- A interpolacao de string deveria estar dentro de \()
- O valor dentro de \() nao e um numero
- A string esta sem as aspas " "
- O valor dentro de \() deve ser um nome de variavel
