# --description--

Em seguida, adicionamos o tipo diferente de valor entre chaves para que ele seja exibido como uma unica instrucao de impressao. Como aqui, com `\(5)`

# --instructions--

Preencha o espaco vazio para obter esta saida
```swift
5 friends
```

# --seed--

```swift
print(" friends")
```

# --solutions--

```swift
print("\(5) friends")
```

# --output--

5 friends
