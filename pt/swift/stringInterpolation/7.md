# --description--

Podemos usar parenteses para inserir valores quantas vezes quisermos dentro da interpolacao de string

# --instructions--

Veja aqui com `2` e depois com `8`

# --seed--

```swift
print("\([/]) new friends and \([/]) old friends")
```

# --answers--

- 2
- 8

# --solutions--

```swift
print("\(2) new friends and \(8) old friends")
```

# --output--

2 new friends and 8 old friends
