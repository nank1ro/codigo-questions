# --instructions--

Como sabemos que este é um loop infinito?
```swift
var isLogged = false

while true {
    print("Please login")
}
```

# --answers--

- Este loop não será executado pois a condição é false
- Não há variável na condição
- Este loop será executado uma vez
- Este loop será executado duas vezes
