# --instructions--

O que `counter += 1` faz?
```swift
var counter = 1

while counter < 5 {
    print(counter)
    counter += 1
}
```

# --answers--

- Ele incrementa counter em um cada vez que o loop executa seu bloco de código
- Ele cria a variável counter
- Ele incrementa counter em dois cada vez
- Ele incrementa counter em três cada vez
