# --instructions--

O que acontece quando a condição do loop `while` permanece `true`?

# --answers--

- O loop se repete para sempre
- O loop executa seu bloco de código uma vez
- O loop executa seu bloco de código duas vezes
- O loop executa seu bloco de código três vezes
