# --instructions--

O que ha de errado com este codigo?
```swift
while isLoading {
    print("Loading...")
    isLoading = false
}
```

# --answers--

- isLoading nao e um nome de variavel valido
- O laco while tem uma instrucao print dentro do bloco de codigo
- O bloco de codigo nao tem a indentacao correta
