# --instructions--

O que há de errado com este código?
```swift
while isLoading {
    print("Loading...")
    isLoading = false
}
```

# --answers--

- isLoading não é um nome de variável válido
- O loop while tem uma instrução print dentro do bloco de código
- O bloco de código não tem a indentação correta
