# --description--

Sabemos como repetir codigo usando um loop `while`.
Como neste programa que repete instrucoes para exibir `hello`
```swift
var counter = 0

while counter < 5 {
    print("hello")
    counter += 1;
}
```
Mas podemos fazer o mesmo com loops `for`:
```swift
for i in 0..<5 {
    print("hello")
}
```
