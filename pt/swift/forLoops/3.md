# --description--

Podemos usar `..<` para iterar ate o proximo numero excluido, ou `...` para iterar ate o proximo numero incluido

# --instructions--

Complete o loop `for` no codigo a seguir para executar o bloco de codigo `3` vezes

    print(i)
