# --description--

Vamos continuar com o operador relacional **menor ou igual** `<=`.
Ele retorna um **booleano**, verdadeiro `1` ou falso `0`, indicando se uma expressao e menor ou igual a outra, por exemplo:
```c
>>> 2 <= 2
1
>>> 3 <= 2
0
>>> 3 <= 4
1
```
