# --description--

Vamos continuar com o operador relacional **maior ou igual** `>=`.
Ele retorna um **booleano**, verdadeiro `1` ou falso `0`, indicando se uma expressao e maior ou igual a outra, por exemplo:
```c
>>> 2 >= 2
1
>>> 3 >= 2
1
>>> 3 >= 4
0
```
