# --description--

Agora vamos ver os operadores **booleanos**, comecando pelo primeiro chamado __and__ `&&`.
Ele retorna o primeiro operando que e avaliado como *false* ou o ultimo se todos forem *true*.
```c
>>> 2 == 2 && 2 == 3
0
>>> 1 == 1 && 1 == 1.0
1
```
