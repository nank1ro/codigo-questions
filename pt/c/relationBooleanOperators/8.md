# --description--

Vamos continuar com o operador booleano **or** `||`.
Ele retorna o primeiro operando que e avaliado como *true* ou o ultimo se todos forem *false*.
```c
>>> 2 == 2 || 2 == 3
1
>>> 1 == 2 || 1 == 3
0
```
