Em C, usamos a função `printf` com uma expressão para exibir dois ou mais valores juntos, como:
```c
>>> char c = 'C';
>>> printf("Hello %c!\n", c);
Hello C!
```
`%c` é usado para exibir um único caractere

---

Strings formatadas nos permitem exibir expressões como adicionar uma string a um número, sem nenhum erro.
Para exibir um inteiro usamos a expressão `%i`

---

Toda instrução de string formatada consiste em duas partes: o texto que queremos incluir e a expressão que queremos formatar dentro do texto

---
