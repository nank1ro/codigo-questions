# --description--

Em C, usamos a função `printf` com uma expressão para exibir dois ou mais valores juntos, como:
```c
>>> char c = 'C';
>>> printf("Hello %c!\n", c);
Hello C!
```
`%c` é usado para exibir um único caractere

# --instructions--

Imprima o nome usando a expressão `%s` (usada para exibir um _array de caracteres_ ou uma _string_)
