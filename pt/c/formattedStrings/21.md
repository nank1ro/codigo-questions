# --instructions--

O que há de errado com este código?
```c
int pieces = 5;
char label[] = "we have %i pieces", 5;
```

# --answers--

- Não podemos salvar strings formatadas em variáveis de string
- O caractere f está faltando
- Não podemos usar variáveis dentro de %
- Não há nada de errado com o código
