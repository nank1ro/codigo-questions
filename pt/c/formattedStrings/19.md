# --instructions--

O que há de errado com este código?
```c
int manchester_united = 0;
int barcelona = 1;
printf("The score is %i : %i\n", manchester_united, barcelona);
```

# --answers--

- Não há nada de errado com o código
- manchester_united e barcelona não têm valores
- Não podemos usar variáveis dentro da string formatada
- Não podemos usar %i duas vezes na mesma string
