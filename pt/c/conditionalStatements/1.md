# --description--

A tomada de decisao e necessaria quando queremos executar codigo apenas se uma certa condicao for satisfeita.
Vamos supor que queremos brincar fora apenas se o tempo estiver bom.
Na programacao, podemos salvar uma variavel booleana `nice_weather` e realizar a acao de brincar fora `if` essa variavel for `true`, como:
```c
bool nice_weather = true;
if (nice_weather) {
    // play outside
}
```
