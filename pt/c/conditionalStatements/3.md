# --description--

Acabamos de ver como executar um bloco de codigo se uma condicao ocorrer, agora vejamos como executar outro bloco de codigo se a primeira condicao falhar.
Nos vamos brincar fora se o tempo estiver bom; caso contrario, ficamos em casa.
Em C podemos usar a instrucao `else`, como:
```c
bool nice_weather = false;
if (nice_weather) {
    // play outside
} else {
    // stay home
}
```
