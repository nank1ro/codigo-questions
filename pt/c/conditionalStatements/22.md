# --instructions--

Como sabemos que `Hello World` sempre será impresso?
```c
bool online = false;
if (online) {
    printf("User is online\n");
}
printf("Hello World\n");
```

# --answers--

- Porque não está dentro da instrução condicional
- Porque a instrução if não pode ter mais de uma linha
- Porque não há uma instrução else
- Porque não há uma instrução else if
