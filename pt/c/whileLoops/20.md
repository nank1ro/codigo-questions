# --instructions--

Como sabemos que o laco `while` pula seu bloco de codigo?
```c
bool is_loading = false;

while (is_loading) {
    printf("Loading...\n");
}
```

# --answers--

- A variavel na condicao e comparada com true
- A variavel na condicao e comparada com false
