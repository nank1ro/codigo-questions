# --instructions--

O que `counter += 1` faz?
```c
int counter = 1;

while (counter < 5) {
    printf("%d\n", counter);
    counter += 1;
}
```

# --answers--

- Cria a variavel counter
- Aumenta counter em dois a cada vez
