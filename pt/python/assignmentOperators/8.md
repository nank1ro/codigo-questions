# --description--

Vamos ver o operador de **atribuição de módulo** `%=`.
Ele calcula o módulo da variável e um valor e atribui o resultado a essa variável.
Portanto, os seguintes são exatamente iguais
```python
num = num % 5
# é igual a
num %= 5
```

# --instructions--
