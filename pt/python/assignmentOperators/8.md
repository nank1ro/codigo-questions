# --description--

Vamos ver o operador de **atribuição com divisão inteira** `//=`.
Ele realiza a divisão inteira da variável por um valor e atribui o resultado a essa variável.
Portanto, as seguintes expressões são exatamente iguais
```python
num = num // 5
# é igual a
num //= 5
```

# --instructions--

Calcule a divisão inteira de `result` com o valor `3` usando o operador de *atribuição com divisão inteira*, depois imprima seu valor
