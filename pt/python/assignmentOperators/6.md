# --description--

Vamos ver o operador de **atribuição com exponenciação** `**=`.
Ele eleva a variável a uma potência especificada e atribui o resultado à variável.
Portanto, as seguintes expressões são exatamente iguais
```python
num = num ** 5
# é igual a
num **= 5
```
