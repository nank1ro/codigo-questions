# --description--

Vamos ver o operador de **atribuição de potência** `**=`.
Ele eleva a variável a uma potência especificada e atribui o resultado à variável.
Portanto, os seguintes são exatamente iguais
```python
num = num ** 5
# é igual a
num **= 5
```
