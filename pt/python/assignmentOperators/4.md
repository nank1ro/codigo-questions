# --description--

Vamos ver o operador de **atribuição com multiplicação** `*=`.
Ele multiplica a variável por um valor e atribui o resultado a essa variável.
Portanto, as seguintes expressões são exatamente iguais
```python
num = num * 5
# é igual a
num *= 5
```
