# --description--

Vamos ver o operador de **atribuição de divisão inteira** `//=`.
Ele divide a variável por um valor usando divisão inteira e atribui o resultado a essa variável.
Portanto, os seguintes são exatamente iguais
```python
num = num // 5
# é igual a
num //= 5
```

# --instructions--
