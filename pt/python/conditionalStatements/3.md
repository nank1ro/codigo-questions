Acabamos de ver como executar um bloco de codigo se uma condicao ocorrer, agora vamos ver como executar outro bloco de codigo se a primeira condicao falhar.

Em Python podemos usar a instrucao `else`, assim:
```python

if (nice_weather):
    # play outside
else:
    # stay home
```
