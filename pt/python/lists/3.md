# --description--

Uma lista nao precisa ter um tamanho fixo.
Voce pode adicionar itens ao final de uma lista quando quiser!
Para adicionar um item a uma lista usamos a palavra-chave `append`:
```python
>>> letters = ["a", "b"]
>>> letters.append("c")
>>> print(letters)
['a', 'b', 'c']
```
