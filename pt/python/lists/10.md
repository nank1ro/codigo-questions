# --description--

Em Python podemos percorrer uma lista de forma muito simples usando as palavras-chave `for..in`:
```python
>>> numbers = [1, 2, 3]
>>> for num in numbers:
>>>     print(num)
1
2
3
```
Um nome de variavel segue a palavra-chave `for`, ele recebera o valor de cada item da lista por vez.
