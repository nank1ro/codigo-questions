# --description--

Sabemos como repetir código usando um loop `while`.
Como este programa repetindo instruções para exibir `hello`
```python
counter = 0

while counter < 5:
    print("hello")
    counter += 1
```
Mas podemos fazer o mesmo com loops `for`:
```python
for i in range(5):
    print("hello")
```
