# --instructions--

Qual e o __problema__ com este trecho de codigo?
```python
greet('Mathew')

def greet(name):
    """This function greets to the person passed in as a parameter"""
    print(f"Hello, {name}. Good morning!")

greet('Alex')
```

# --answers--

- Nao podemos usar greet() antes de defini-la
- Nao podemos usar greet() duas vezes
- greet() nao retorna nada
- Nao podemos usar name como parametro
