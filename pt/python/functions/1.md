# --instructions--

Qual é o __problema__ com este trecho de código?
```python
greet('Mathew')

def greet(name):
    """This function greets to the person passed in as a parameter"""
    print(f"Hello, {name}. Good morning!")

greet('Alex')
```

# --answers--

- Não podemos usar greet() antes de defini-lo
- Não podemos usar greet() duas vezes
- greet() não retorna nada
- Não podemos usar name como parâmetro
