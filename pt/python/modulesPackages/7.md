# --instructions--

Pacotes sao nada mais do que pastas que contem um arquivo `__init__.py`.
O que pode acontecer se esquecermos de adicionar esse arquivo?

# --answers--

- O Python nao reconhecera a pasta como um pacote
- Ainda podemos importar pacotes, mas nao subpacotes
- Ainda podemos importar funcoes
