# --instructions--

Pacotes nada mais são do que pastas que contêm um arquivo `__init__.py`.
O que pode acontecer se esquecermos de adicionar tal arquivo?

# --answers--

- Python won't recognize the folder as a package
- We can still import packages but not subpackages
- We can still import functions
