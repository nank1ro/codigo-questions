# --description--

Vamos ver o operador de **divisão inteira** `//`.
Este operador retorna a parte inteira do quociente, por exemplo:
```python
>>> 5.0 / 2
2.5
>>> 5.0 // 2
2.0
```
Também conhecido como divisão inteira. O valor resultante é um número inteiro, embora o *tipo* do resultado não seja necessariamente int.

# --instructions--
