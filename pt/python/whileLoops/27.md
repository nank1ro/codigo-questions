# --instructions--

O que `counter += 1` faz?
```python
counter = 1

while counter < 5:
    print(counter)
    counter += 1
```

# --answers--

- Ela aumenta counter em um cada vez que o loop executa seu bloco de código
- Ela cria a variável counter
- Ela aumenta counter em dois cada vez
- Ela aumenta counter em três cada vez
