# --description--

Quando atualizamos uma variável, ela esquece seu valor anterior. Aqui podemos exibir a variável `x` duas vezes e ver como seu valor é atualizado.
```python
>>> x = 5
>>> print(x)
5
>>> x = 10
>>> print(x)
10
```
