# --description--

Variaveis sao chamadas assim porque o valor que elas armazenam pode mudar.
Podemos atualizar `x` usando `=` e dando a ela um novo valor.
```python
>>> x = 1
>>> print(x)
1
>>> x = 2
>>> print(x)
2
```
