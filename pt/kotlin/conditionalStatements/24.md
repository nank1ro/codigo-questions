# --instructions--

Como sabemos que `Hello World` sempre será impresso?
```kotlin
var online = false;
if (online) {
    println("User is online")
}
println("Hello World")
```

# --answers--

- Porque não está dentro de um bloco de código
- Porque a declaração if não pode ter mais de uma linha
- Porque não há uma declaração else
- Porque não há uma declaração else if
