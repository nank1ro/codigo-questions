# --description--

Uma função de expressão única usa `=` em vez de um corpo em bloco:
```kotlin
fun triple(n: Int) = n * 3
```
Esta é uma abreviação para:
```kotlin
fun triple(n: Int): Int {
    return n * 3
}
```
