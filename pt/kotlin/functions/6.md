# --description--

Funções podem retornar valores `Boolean`, o que é útil para verificar condições:
```kotlin
fun isEven(n: Int): Boolean {
    return n % 2 == 0
}
println(isEven(4)) // prints true
println(isEven(7)) // prints false
```
Uma função `Boolean` retorna `true` ou `false`.
