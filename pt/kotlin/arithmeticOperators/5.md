# --description--

Vamos ver o operador de **resto** `%`.
Ele é usado para encontrar o resto de uma divisão entre dois números, como:
```kotlin
println(5 % 2) // 1
```
O resultado é 1 porque 5 dividido por 2 tem um quociente de 2 e um resto de 1
```kotlin
println(9 % 3) // 0
```
Este outro resulta em 0 porque 9 dividido por 3 tem um quociente de 3 e deixa um resto de 0
