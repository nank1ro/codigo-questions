# --description--

Um _template_ de String é uma forma programática de gerar uma String.
Em Kotlin podemos usar o sinal `+` (concatenação) para exibir duas ou mais strings juntas, assim:
```kotlin
println("Hello " + "Kotlin!")
// imprime "Hello Kotlin!"
```
