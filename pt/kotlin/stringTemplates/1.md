# --description--

Um _template_ de String  uma forma programtica de gerar uma String.
Em Kotlin podemos usar o sinal `+` (concatenao) para exibir duas ou mais strings juntas, como:
```kotlin
println("Hello " + "Kotlin!")
// prints "Hello Kotlin!"
```
