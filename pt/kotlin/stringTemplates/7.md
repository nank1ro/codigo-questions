# --description--

Podemos usar chaves para inserir valores quantas vezes quisermos dentro dos templates de string

# --instructions--

Veja aqui com `2` e depois com `8`

# --seed--

```kotlin
println("[/] new friends and [/] old friends")
```

# --answers--

- $2
- $8
- ${2}
- ${8}

# --solutions--

```kotlin
println("${2} new friends and ${8} old friends")
```

# --output--

2 new friends and 8 old friends
