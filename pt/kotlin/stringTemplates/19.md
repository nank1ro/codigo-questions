# --instructions--

Use `${}` para inserir o número `500` no espaço vazio do template de string

# --seed--

```kotlin
fun main() {
    println("There are  items")
}
```

# --solutions--

```kotlin
fun main() {
    println("There are ${500} items")
}
```

# --output--
