# --description--

Caso você simplesmente precise repetir um bloco de código `n` vezes, você pode usar a função `repeat(times: Int)`.

```kotlin
repeat(3) {
    println("repeat")
}
// imprime repeat 3 vezes
```

Você pode até acessar o índice com
```kotlin
repeat(3) { index ->
    println(index)
}
// imprime 0, 1, 2
```
