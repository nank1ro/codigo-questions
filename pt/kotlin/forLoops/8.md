# --description--

Em Kotlin podemos usar o `for-in` também para coleções iteráveis chamando o fechamento dado em cada elemento:
```kotlin
// isso é uma lista, veremos sobre isso em breve
val numbers = listOf(2, 4, 6, 8, 10)
for (num in numbers) {
    println(num)
}
// imprime (2, 4, 6, 8, 10)
```
