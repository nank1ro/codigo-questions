# --instructions--

Um **objeto companion** é um singleton associado a uma classe. Ele permite definir propriedades e métodos que pertencem à própria classe, e não às instâncias — semelhante aos membros `static` em outras linguagens:

```kotlin
class MathHelper {
    companion object {
        fun square(n: Int): Int = n * n
    }
}
println(MathHelper.square(4)) // 16
```

Crie uma classe `Counter` com um objeto companion que contenha uma propriedade `var` chamada `total` do tipo `Int` (inicializada com `0`) e um método `increment()` que adiciona `1` ao `total`.

# --asserts--

`Counter.total` deve começar com `0`

```kotlin
    tryCatch(Counter.total == 0)
```

Após um `increment()`, `total` deve ser `1`

```kotlin
    Counter.increment()
    tryCatch(Counter.total == 1)
```

Após mais dois `increment()`, `total` deve ser `3`

```kotlin
    Counter.increment()
    Counter.increment()
    tryCatch(Counter.total == 3)
```
