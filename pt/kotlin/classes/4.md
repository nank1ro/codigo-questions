# --description--

Classes podem conter **métodos** — funções que pertencem à classe e operam sobre suas propriedades:

```kotlin
class Counter(var count: Int) {
    fun increment() {
        count++
    }
    fun value(): Int {
        return count
    }
}
```

Os métodos são chamados usando a notação de ponto em uma instância:

```kotlin
val c = Counter(0)
c.increment()
println(c.value()) // 1
```
