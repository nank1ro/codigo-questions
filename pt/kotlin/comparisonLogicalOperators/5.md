# --description--

Vamos continuar com o operador de comparação **maior ou igual a** `>=`.
Ele retorna um **booleano** (`true` ou `false`) indicando se uma expressão é maior ou igual à outra, por exemplo:
```kotlin
println(2 >= 2) // true
println(3 >= 2) // true
println(3 >= 4) // false
```
