# --instructions--

Quais são as principais diferenças entre uma `List` e um `Set`?

# --answers--

- Um `Set` não pode conter duplicatas
- Uma `List` não pode conter duplicatas
- Não há diferenças
- Um `Set` pode realizar operações de _diagrama de Venn_
