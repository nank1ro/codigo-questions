# --description--

A ordem dos elementos em um `Set` não é importante.
Na verdade, se você tentar comparar dois `Set`s com os mesmos elementos mas em ordem diferente, verá que eles são iguais.

# --instructions--

- Verifique se o primeiro e o segundo `Set`s são iguais
- Verifique se o primeiro e o terceiro `Set`s são iguais

# --seed--

```kotlin
val names1 = setOf("Charlotte", "Mia")
val names2 = setOf("Mia", "Charlotte")
val names3 = setOf("Luke", "Ariana")
println([/][/][/])
println([/][/][/][/])
```
