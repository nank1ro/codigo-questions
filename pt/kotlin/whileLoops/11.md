# --instructions--

O que acontece quando a condicao do loop `while` permanece `true`?

# --answers--

- O loop se repete para sempre
- O loop executa seu bloco de codigo uma vez
- O loop executa seu bloco de codigo duas vezes
- O loop executa seu bloco de codigo tres vezes
