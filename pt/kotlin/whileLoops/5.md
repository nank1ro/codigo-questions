# --description--

Aqui esta um exemplo do uso de `break` dentro de um loop _while_:
```kotlin
var i = 0
while (i < 3) {
  i++
  if (i == 2) break // [1]
  println(i)
}
// prints 1
```

Como voce pode ver em __[1]__ quando `i` e igual a _2_, _interrompemos_ o loop. Na verdade, os numeros 2 e 3 nunca sao impressos.
