# --description--

Variaveis sao recipientes para armazenar valores de dados.
Toda variavel em Kotlin e um objeto.
Para criar uma variavel, precisamos dar-lhe um __nome__, tendo em mente que nao deve conter espacos.
Uma variavel e criada no momento em que voce atribui um valor a ela pela primeira vez.
Em Kotlin, voce declara constantes com a palavra-chave `val` (abreviacao de _value_) e variaveis com a palavra-chave `var` (abreviacao de _variable_).
O valor de uma constante nao pode ser alterado depois de definido, enquanto uma variavel pode receber um valor diferente no futuro.
Um exemplo de criacao de uma variavel chamada `x` e:
```kotlin
var x = 1
```
Desta forma, atribuimos o valor `1` a variavel chamada `x`.
Se imprimirmos a variavel `x`, obtemos de volta o numero `1`:
```kotlin
println(x) // imprime 1
```
