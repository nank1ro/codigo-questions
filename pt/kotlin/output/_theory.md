"__Hello, World!__" e o tradicional primeiro programa para iniciar a programacao em uma nova linguagem.
Usamos a funcao `println()` para exibir dados no dispositivo de saida padrao (tela).
Para imprimir `Hello, World!` na tela com Kotlin, podemos escrever
```kotlin
println("Hello, World!")
```

---

`println()` move o cursor para uma nova linha apos exibir seu parametro, de modo que a saida subsequente apareca na proxima linha.
Voce pode usar `print()` em vez disso, que deixa o cursor na mesma linha.
