"__Hello, World!__"  o tradicional primeiro programa para comear a programar em uma nova linguagem.
Ns usamos a funo `println()` para exibir dados no dispositivo de sada padro (tela).
Para imprimir `Hello, World!` na tela com Kotlin podemos escrever
```kotlin
println("Hello, World!")
```

---

`println()` moves the cursor to a new line after displaying its parameter, so subsequent output appears on the next line.
You can use `print()` instead, which leaves the cursor on the same line.
