# --instructions--
O que o laço `for-in` faz em Dart?

# --answers--

- Itera sobre cada elemento de uma coleção
- Itera um número fixo de vezes usando um índice
- Repete até que uma condição booleana se torne verdadeira
- Executa o código somente se a coleção não estiver vazia
