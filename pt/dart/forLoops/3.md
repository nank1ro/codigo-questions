# --description--
Um laço `for` pode contar **para baixo** usando um decremento (`i--`) e uma condição `>=`:

```dart
for (int i = 5; i >= 1; i--) {
  print(i); // 5, 4, 3, 2, 1
}
```

Isso é útil para contagens regressivas ou iteração reversa.
