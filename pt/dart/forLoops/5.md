# --description--
A instrução `break` sai de um laço imediatamente quando uma condição é satisfeita:

```dart
for (int i = 0; i < 10; i++) {
  if (i == 5) break; // stops at 5
  print(i);
}
```

Isso é útil quando você está procurando um valor e deseja parar assim que ele for encontrado.
