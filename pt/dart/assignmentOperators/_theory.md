O **operador de atribuição** `=` armazena um valor em uma variável.

```dart
int x = 5;
String name = 'Alice';
```

Uma vez atribuído, a variável mantém esse valor até ser reatribuída.

---

O **operador de atribuição de adição** `+=` adiciona um valor a uma variável e armazena o resultado de volta nessa variável.

```dart
int score = 10;
score += 5; // score agora é 15
```

Esta é uma forma abreviada de `score = score + 5`.

---

O **operador de atribuição de subtração** `-=` subtrai um valor de uma variável e armazena o resultado de volta nessa variável.

```dart
int lives = 3;
lives -= 1; // lives agora é 2
```

Esta é uma forma abreviada de `lives = lives - 1`.

---

O **operador de atribuição de multiplicação** `*=` multiplica uma variável por um valor e armazena o resultado de volta nessa variável.

```dart
int side = 4;
side *= 3; // side agora é 12
```

Esta é uma forma abreviada de `side = side * 3`.
