# --description--

O **operador de atribuição** `=` armazena um valor em uma variável.

```dart
int x = 5;
String name = 'Alice';
```

Uma vez atribuído, a variável mantém esse valor até ser reatribuída.

# --instructions--

Complete o código para atribuir o valor `42` à variável `answer`.

# --seed--

```dart
void main() {
  int answer [/] 42;
  print(answer);
}
```
