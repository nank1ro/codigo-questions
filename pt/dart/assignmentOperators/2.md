# --description--

O **operador de atribuição de adição** `+=` adiciona um valor a uma variável e armazena o resultado de volta nessa variável.

```dart
int score = 10;
score += 5; // score agora é 15
```

Esta é uma forma abreviada de `score = score + 5`.

# --instructions--

Complete o código para adicionar `8` à variável `total` usando `+=`.
