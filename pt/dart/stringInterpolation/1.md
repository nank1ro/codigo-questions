# --description--

Uma _interpolacao_ de String e uma forma programatica de gerar uma String.
Em Dart, podemos usar o sinal `+` (concatenacao) para exibir duas ou mais strings juntas, como:
```dart
print("Hello " + "Dart!");
// imprime "Hello Dart!"
```
