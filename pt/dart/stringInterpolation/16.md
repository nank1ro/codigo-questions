# --instructions--

O que ha de errado com este codigo?
```dart
print("$(10} friends");
```

# --answers--

- A expressao deve estar dentro de ${}
- O valor dentro de ${} nao e um numero
- A string esta sem suas aspas " "
- O valor dentro de ${} deve ser um nome de variavel
