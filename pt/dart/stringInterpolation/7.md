# --description--

Podemos usar chaves para inserir valores quantas vezes quisermos usando a interpolacao de string

# --instructions--

Veja aqui com `2` e depois com `8`

# --seed--

```dart
print("[/] new friends and [/] old friends");
```

# --answers--

- $2
- $8
- ${2}
- ${8}

# --solutions--

```dart
print("${2} new friends and ${8} old friends");
```

# --output--

2 new friends and 8 old friends
