# --description--

O **operador menor que** `<` retorna `true` quando o valor à esquerda é estritamente menor que o valor à direita.

```dart
int a = 2;
int b = 9;
bool result = a < b; // true
```

Se o valor à esquerda for igual ou maior que o da direita, retorna `false`.
