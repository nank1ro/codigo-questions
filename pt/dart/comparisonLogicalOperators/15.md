# --description--

O **operador OR** `||` combina duas expressões booleanas. Retorna `true` quando **pelo menos um** lado é `true`. Retorna `false` apenas quando **ambos** os lados são `false`.

```dart
bool a = false;
bool b = true;
bool result = a || b; // true

bool c = false;
bool d = false;
bool result2 = c || d; // false
```
