# --description--

O **operador de igualdade** `==` compara dois valores e retorna `true` se forem iguais, ou `false` se não forem.

```dart
int a = 5;
int b = 5;
bool result = a == b; // true
```

O resultado de uma comparação é sempre um valor `bool`.
