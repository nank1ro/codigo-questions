# --instructions--

O que um laço `while` faz?

# --answers--

- Executa um bloco de código repetidamente enquanto uma condição for `true`
- Executa um bloco de código exatamente uma vez
- Executa um bloco de código um número fixo de vezes
- Executa um bloco de código e depois verifica a condição
