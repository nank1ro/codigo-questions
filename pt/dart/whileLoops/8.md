# --instructions--

O que um laço `do-while` garante em comparação com um laço `while`?

# --answers--

- O corpo é executado pelo menos uma vez, mesmo que a condição seja inicialmente `false`
- O corpo é executado no máximo uma vez
- A condição é verificada antes do corpo ser executado
- O laço é executado para sempre a menos que `break` seja usado
