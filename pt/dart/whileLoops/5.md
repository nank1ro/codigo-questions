# --description--

A palavra-chave `break` sai de um laço imediatamente, independentemente da condição do laço.

```dart
int i = 0;
while (true) {
  if (i == 3) break;
  print(i);
  i++;
}
```

Sem `break`, o laço `while (true)` seria executado para sempre. Aqui ele para quando `i` chega a `3`.
