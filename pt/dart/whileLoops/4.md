# --description--

A condição de um laço `while` pode envolver qualquer expressão booleana. Assim que a condição for avaliada como `false`, o laço termina.

```dart
int n = 100;
while (n > 1) {
  n ~/= 2;
}
print(n); // 0
```

Aqui, `n` é reduzido à metade a cada iteração usando divisão inteira (`~/`).
