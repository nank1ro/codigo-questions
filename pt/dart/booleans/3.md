# --description--

Uma variável booleana também pode conter o valor `false`.
Use `false` quando algo não é o caso.

```dart
bool isLoggedIn = false;
print(isLoggedIn); // false
```

Assim como `true`, `false` deve ser escrito em minúsculas.
