# --instructions--

O que é uma **classe** em Dart?

# --answers--

- Uma função que retorna um valor
- Um modelo para criar objetos que agrupa dados e comportamentos
- Um tipo de dado embutido como `int` ou `String`
- Um laço que repete um bloco de código
