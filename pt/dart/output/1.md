# --description--

"__Hello, World!__" é o tradicional primeiro programa para quem está começando a programar em uma nova linguagem.
Usamos a função `print()` para exibir dados no dispositivo de saída padrão (tela).
Para imprimir `Hello, World!` na tela com Dart, podemos escrever
```dart
print("Hello, World!")
```
