# --instructions--

O que significa o tipo de retorno `void` em uma função Dart?

```dart
void sayHi() {
  print("Hi!");
}
```

# --answers--

- A função retorna o número zero
- A função não retorna nenhum valor
- A função retorna uma string vazia
- A função pode retornar qualquer tipo
