# --instructions--

Escreva uma função **recursiva** `factorial` que receba um parâmetro `int` `n` e retorne `n!` (o fatorial de `n`).

O fatorial de `n` é definido como:
- `factorial(0) = 1`
- `factorial(n) = n * factorial(n - 1)` para `n > 0`

Exemplo:
```dart
print(factorial(5)); // 120
print(factorial(3)); // 6
```

# --solutions--

```dart
int factorial(int n) {
  if (n == 0) return 1;
  return n * factorial(n - 1);
}
```
