# --description--

O **operador de multiplicação** `*` multiplica dois valores juntos.

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

Multiplicar um `int` por outro `int` produz um `int`. Multiplicar com um `double` produz um `double`.
