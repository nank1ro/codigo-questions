# --description--

Dart segue a **ordem de operações** matemática padrão (precedência): multiplicação e divisão são avaliadas antes de adição e subtração.

```dart
int result = 2 + 3 * 4; // 14, not 20
```

Use parênteses para alterar a ordem:

```dart
int result = (2 + 3) * 4; // 20
```
