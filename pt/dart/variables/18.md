# --instructions--

E sempre necessario especificar o tipo de uma variavel em Dart?

# --answers--

- O tipo de uma variavel deve sempre ser especificado
- Dart suporta __inferencia de tipo__, entao ele e capaz de inferir por si so
- Dart nao suporta tipos
