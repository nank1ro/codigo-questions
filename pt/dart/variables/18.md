# --instructions--

É sempre necessário especificar o tipo de uma variável em Dart?

# --answers--

- O tipo de uma variável deve sempre ser especificado
- Dart suporta __inferência de tipos__, então é capaz de inferir por conta própria
- Dart não suporta tipos
