# --instructions--

Qual é a diferença entre uma constante de __tempo de compilação__ e uma de __tempo de execução__?

# --answers--

- As constantes de __tempo de compilação__ contêm valores fixos conhecidos antes de executar o programa, enquanto as constantes de __tempo de execução__ obtêm um valor fixo apenas durante a execução do programa
- As constantes de __tempo de compilação__ contêm valores fixos conhecidos após executar o programa, enquanto as constantes de __tempo de execução__ obtêm um valor fixo antes de executar o programa
- Não há diferença entre as duas
