# --description--

Uma **instrução `if`** executa um bloco de código somente quando uma condição é `true`.

```dart
int age = 20;
if (age >= 18) {
  print('Adult');
}
```

O código dentro das chaves é executado apenas se a condição `age >= 18` avaliar como `true`.
