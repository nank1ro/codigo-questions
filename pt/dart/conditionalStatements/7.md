# --instructions--

O que uma instrução `if` faz em Dart?

# --answers--

- Ela sempre executa o código dentro do seu bloco
- Ela executa seu bloco somente quando a condição é `true`
- Ela repete seu bloco enquanto a condição é `true`
- Ela define uma nova função
