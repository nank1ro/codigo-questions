# --description--

Uma **instrução `if-else`** executa o bloco `if` quando a condição é `true`, e o bloco `else` quando ela é `false`.

```dart
int temperature = 10;
if (temperature > 20) {
  print('Warm');
} else {
  print('Cold');
}
// prints: Cold
```

Exatamente um dos dois ramos sempre é executado.
