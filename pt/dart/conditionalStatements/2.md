# --description--

Você pode usar `print()` dentro de um bloco `if` para exibir uma mensagem quando uma condição é satisfeita.

```dart
bool isRaining = true;
if (isRaining) {
  print('Take an umbrella');
}
```

Se `isRaining` for `false`, nada é impresso.
