# --instructions--

Indique a forma correta de converter a variável `number` para um inteiro.
```dart
final number = 3.14;
```

# --answers--

```dart
final integer = number.toInt();
```

```dart
final integer = int(number);
```

```dart
final int integer = number;
```

```dart
final integer = number as int;
```
