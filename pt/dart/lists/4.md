# --description--

A propriedade `.length` retorna o número de elementos em uma lista:

```dart
var scores = [95, 87, 72, 100];
print(scores.length); // 4
```

Uma lista vazia tem comprimento `0`:

```dart
var empty = [];
print(empty.length); // 0
```
