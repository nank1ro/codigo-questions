# --description--

O método `.add()` adiciona um único elemento ao **final** de uma lista:

```dart
var nums = [1, 2, 3];
nums.add(4);
print(nums); // [1, 2, 3, 4]
```

Note que `.add()` modifica a lista **no local** e retorna `void`.
