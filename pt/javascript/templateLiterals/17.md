# --instructions--

Use `${}` para inserir o número `500` no espaço vazio do template literal

# --seed--

```javascript
console.log(`There are  items`);
```

# --solutions--

```javascript
console.log(`There are ${500} items`);
```

# --output--

There are 500 items
