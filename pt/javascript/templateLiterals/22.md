# --instructions--

O que há de errado com este código?
```javascript
var pieces = 5;
var label = `we have ${pieces} pieces`;
```

# --answers--

- Não podemos salvar template literals em variáveis
- Os caracteres ${} estão faltando
- Não podemos usar variáveis dentro de ${}
- Não há nada de errado com o código
