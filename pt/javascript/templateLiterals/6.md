# --description--

Podemos usar `${}` para inserir valores quantas vezes quisermos dentro dos template literals

# --instructions--

Veja aqui com `2` e depois com `8`

# --seed--

```javascript
console.log(`${[/]} new friends and ${[/]} old friends`);
```

# --answers--

# --solutions--

```javascript
console.log(`${2} new friends and ${8} old friends`);
```
