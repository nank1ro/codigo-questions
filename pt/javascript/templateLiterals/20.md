# --instructions--

O que há de errado com este código?
```javascript
var manchesterUnited = 2;
var barcelona = 2;
console.log(`The final score is ${manchesterUnited} : ${barcelona}`);
```

# --answers--

- Não há nada de errado com o código
- manchesterUnited e barcelona não têm valores
- Não podemos usar variáveis dentro das chaves
- Não podemos usar ${} duas vezes na mesma string
