Em JavaScript, podemos usar o sinal `+` para exibir duas ou mais strings juntas, assim:
```javascript
console.log("Hello " + "JavaScript!") // imprime "Hello JavaScript!"
```

---

Template literals nos permitem exibir expressões, como adicionar uma string a um número, sem usar o sinal `+`.
Para criar um template literal válido, você precisa usar crases \`\` em vez de aspas duplas ou simples.
Você pode inserir variáveis dentro dos template literals usando o cifrão e chaves `${expression}`

---

Toda instrução de template literals consiste em duas partes: o `${}` onde inserimos o número ou variável, e a string normal

---

Em seguida, adicionamos o tipo diferente de valor entre chaves para que seja exibido como uma única instrução console.log. Como aqui, com `${5}`

---

Inserir variáveis como `friends` entre os parênteses também exibe seus valores

---

Podemos usar `${}` para inserir valores quantas vezes quisermos dentro dos template literals

---

Template literals são mais usados em instruções de impressão, mas também podemos armazená-los em variáveis como strings normais.
