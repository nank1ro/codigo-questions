# --instructions--

O que há de errado com este código?
```javascript
console.log(`$(10} friends`);
```

# --answers--

- As expressões devem estar dentro de ${}
- O valor dentro de ${} não é um número
- A string está sem suas crases ``
- O valor dentro de ${} deve ser um nome de variável
