# --description--

Em seguida, adicionamos o tipo diferente de valor entre chaves para que seja exibido como uma única instrução console.log. Como aqui, com `${5}`

# --instructions--

Preencha o espaço vazio para obter esta saída
```javascript
5 friends
```

# --seed--

```javascript
console.log(` friends`);
```

# --solutions--

```javascript
console.log(`${5} friends`);
```
