# --description--

Template literals nos permitem exibir expressões, como adicionar uma string a um número, sem usar o sinal `+`.
Para criar um template literal válido, você precisa usar crases \`\` em vez de aspas duplas ou simples.
Você pode inserir variáveis dentro dos template literals usando o cifrão e chaves `${expression}`
