# --description--

Se queremos um nome de variavel com varias palavras, usamos **camelCase**.
E a pratica de escrever frases de tal forma que cada palavra no meio da frase comeca com uma letra maiuscula

# --instructions--

Preencha os espacos em branco para criar um nome de variavel *camelCase*.

```javascript
first[/]ame
```
