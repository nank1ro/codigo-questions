JavaScript tem um tipo booleano basico, chamado `boolean`.
Valores booleanos sao referidos como logicos, porque so podem ser true ou false.
Voce pode avaliar qualquer expressao em JavaScript e obter uma de duas respostas, `true` ou `false`.

---

Podemos armazenar o valor booleano `true` em uma variavel assim como um numero ou uma string.
