# --description--

JavaScript tem um tipo Booleano básico, chamado `boolean`.
Valores booleanos são referidos como lógicos, porque só podem ser verdadeiros ou falsos.
Você pode avaliar qualquer expressão em JavaScript e obter uma de duas respostas, `true` ou `false`.
