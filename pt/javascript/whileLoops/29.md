# --instructions--

O que ha de errado com este codigo?
```javascript
while (isLoading) {
    console.log("Loading...");
    isLoading = false;
}
```

# --answers--

- isLoading nao e um nome de variavel valido
- O laco while tem uma instrucao de impressao dentro do bloco de codigo
- O bloco de codigo nao tem a indentacao correta
