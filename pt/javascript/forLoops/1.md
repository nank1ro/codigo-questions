# --description--

Sabemos como repetir código usando um loop `while`.
Como este programa que repete instruções para exibir `hello`
```javascript
var counter = 0;

while (counter < 5) {
    console.log("hello");
    counter++;
}
```
Mas podemos fazer o mesmo com loops `for`:
```javascript
for (let i = 0; i < 5; i++) {
    console.log("hello");
}
```
