# --description--

O operador ternário condicional é um operador especial com três partes, que tem a forma `questão ? resposta1 : resposta2`.
É um atalho para avaliar uma de duas expressões com base se a `questão` é verdadeira ou falsa.
Se a `questão` é verdadeira, ele avalia `resposta1` e retorna seu valor; caso contrário, ele avalia `resposta2` e retorna seu valor.
```javascript
let a = 10, b = 20, c = 0;
if (a < b) {
    c = a;
} else {
    c = b;
}
console.log(c);
// imprime 10
```
O código abreviado para o código acima é:
```javascript
let a = 10, b = 20, c = 0;
c = a < b ? a : b;
console.log(c);
// imprime 10
```
`c` é definido como igual a `a`, porque a condição `a < b` era verdadeira
