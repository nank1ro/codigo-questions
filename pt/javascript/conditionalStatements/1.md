# --description--

A tomada de decisão é necessária quando queremos executar código apenas se uma determinada condição for satisfeita.
Vamos supor que queremos brincar fora apenas se o clima estiver bom.
Em programação, podemos salvar uma variável booleana `niceWeather` e realizar a ação de brincar fora `if` (se) esta variável for `true` (verdadeira), como:
```javascript
var niceWeather = true;
if (niceWeather) {
    // brincar fora
}
```
