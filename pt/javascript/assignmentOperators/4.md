# --description--

Vamos ver o operador de **atribuicao com multiplicacao** `*=`.
Ele multiplica a variavel por um valor e atribui o resultado a essa variavel.
Portanto, as expressoes a seguir sao exatamente iguais
```javascript
var num = num * 5;
// e igual a
num *= 5;
```
