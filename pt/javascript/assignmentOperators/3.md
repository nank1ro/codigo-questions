# --description--

Assim como na atribuicao com adicao, temos a **atribuicao com subtracao** `-=`.
O funcionamento e o mesmo, a unica diferenca e que ela realiza a subtracao.
Portanto, as expressoes a seguir sao exatamente iguais
```javascript
var num = num - 5;
// e igual a
num -= 5;
```
