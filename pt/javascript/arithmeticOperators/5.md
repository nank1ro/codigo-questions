# --description--

Vamos ver o operador de **resto** `%`.
E usado para encontrar o resto apos uma divisao entre dois numeros, como:
```javascript
console.log(5 % 2);
// prints 1
```
Isto avalia para 1 porque 5 dividido por 2 tem um quociente de 2 e um resto de 1
```javascript
console.log(9 % 3);
// prints 0
```
Este outro avalia para 0 porque 9 dividido por 3 tem um quociente de 3 e deixa um resto de 0
