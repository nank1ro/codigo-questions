# --description--

Em JavaScript podemos percorrer um array de uma forma muito simples usando as palavras-chave `for..of`:
```javascript
var numbers = [1, 2, 3];
for (num of numbers) {
    console.log(num);
}
// prints 1, 2, 3
```
Um nome de variável segue a palavra-chave `for`, ele receberá o valor de cada item do array por vez.
