# --description--

Um array não precisa ter um comprimento fixo.
Você pode adicionar itens ao final de um array quando quiser!
Para adicionar um item a um array usamos a função `push`:
```javascript
var letters = ["a", "b"];
letters.push("c");
console.log(letters);
// Prints ["a", "b", "c"]
```
