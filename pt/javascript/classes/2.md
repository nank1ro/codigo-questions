# --description--

_Classes_ são construções flexíveis e de uso geral que se tornam os blocos de construção do código do seu programa.
Uma classe básica consiste apenas na palavra-chave `class` e seu nome, por exemplo:
```javascript
class ClassName {
    // class definition
}
```
