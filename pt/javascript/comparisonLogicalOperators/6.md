# --description--

Vamos continuar com o operador de comparacao **menor ou igual a** `<=`.
Ele retorna um **booleano** (`true` ou `false`) indicando se uma expressao e menor ou igual a outra, por exemplo:
```javascript
console.log(2 <= 2);
// imprime true
console.log(3 <= 2);
// imprime false
console.log(3 <= 4);
// imprime true
```
