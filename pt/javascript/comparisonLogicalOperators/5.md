# --description--

Vamos continuar com o operador de comparacao **maior ou igual a** `>=`.
Ele retorna um **booleano** (`true` ou `false`) indicando se uma expressao e maior ou igual a outra, por exemplo:
```javascript
console.log(2 >= 2);
// imprime true
console.log(3 >= 2);
// imprime true
console.log(3 >= 4);
// imprime false
```
