# --description--

Agora vamos ver os operadores **logicos**, comecando pelo primeiro chamado __AND__ `&&`.
Ele retorna o primeiro operando que resulta em *false* ou o ultimo se todos forem *true*.
```javascript
console.log(2 == 2 && 2 == 3);
// imprime false
console.log(1 == 1 && 1 == 1.0);
// imprime true
```
