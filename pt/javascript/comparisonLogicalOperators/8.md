# --description--

Vamos continuar com o operador logico **ou** `||`.
Ele retorna o primeiro operando que resulta em *true* ou o ultimo se todos forem *false*.
```javascript
console.log(2 == 2 || 2 == 3);
// imprime true
console.log(1 == 2 || 1 == 3);
// imprime false
```
