# --description--

Vamos comecar com o operador de comparacao **igual** `==`.
Ele retorna um **booleano** (`true` ou `false`) indicando se duas expressoes sao iguais, por exemplo:
```javascript
console.log(2 == 2);
// imprime true
console.log(2 == 3);
// imprime false
```
