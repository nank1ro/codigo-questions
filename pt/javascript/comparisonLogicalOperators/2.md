# --description--

Vamos continuar com o operador de comparacao **diferente** `!=`.
Ele retorna um **booleano** (`true` ou `false`) indicando se duas expressoes **NAO** sao iguais, por exemplo:
```javascript
console.log(2 != 2);
// imprime false
console.log(2 != 3);
// imprime true
```
E exatamente o oposto do operador *igual*
