# --description--

"__Hello, World!__" é o primeiro programa tradicional para quem está começando a programar em uma nova linguagem.
Usamos a função `console.log()` para exibir dados no dispositivo de saída padrão (tela).
Para imprimir `Hello, World!` na tela com JavaScript, podemos escrever
```javascript
console.log("Hello, World!")
```
