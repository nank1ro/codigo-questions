# --description--

Мы можем использовать фигурные скобки для вставки значений сколько угодно раз внутри строковых шаблонов

# --instructions--

Посмотрите здесь с `2` и затем с `8`

# --seed--

```kotlin
println("[/] new friends and [/] old friends")
```

# --answers--

- $2
- $8
- ${2}
- ${8}

# --solutions--

```kotlin
println("${2} new friends and ${8} old friends")
```

# --output--

2 new friends and 8 old friends
