# --instructions--

Что не так с этим кодом?
```kotlin
val pieces = 5

```

# --answers--

- Нельзя сохранять строковые шаблоны в переменных
- Отсутствуют символы $
- Нельзя использовать переменные внутри ${}
- С кодом всё в порядке
