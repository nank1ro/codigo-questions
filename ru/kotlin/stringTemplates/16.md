# --instructions--

Что не так с этим кодом?
```kotlin
println("$(10} friends")
```

# --answers--

- Идентификатор должен быть внутри ${}
- Значение внутри ${} не является числом
- В строке отсутствуют кавычки " "
- Значение внутри ${} должно быть именем переменной
