# --instructions--

Что не так с этим кодом?
```kotlin
val manchesterUnited = 2
val barcelona = 2
println("The final score is $manchesterUnited : $barcelona")
```

# --answers--

- С кодом всё в порядке
- manchesterUnited и barcelona не имеют значений
- Нельзя использовать переменные внутри фигурных скобок
- Нельзя использовать $ дважды в одной строке
