# --instructions--

Как отобразить числовые значения с помощью строковых шаблонов?

# --answers--

- Мы помещаем их между ${}
- Мы не можем отображать числовые значения с помощью строковых шаблонов
- Мы помещаем их между $[]
- Мы помещаем их между $
