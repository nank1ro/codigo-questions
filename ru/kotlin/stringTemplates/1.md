# --description--

Строковый _шаблон_ — это программный способ генерации строки.
В Kotlin мы можем использовать знак `+` (конкатенация) для отображения двух или более строк вместе, например:
```kotlin
println("Hello " + "Kotlin!")
// выводит "Hello Kotlin!"
```
