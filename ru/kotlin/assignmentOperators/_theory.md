Операторы присваивания используются для присваивания значений переменным.
Самым базовым оператором присваивания является `=`, который присваивает значение справа переменной слева:
```kotlin
var x = 10
```
Здесь `10` присваивается переменной `x`.
Вы также можете присвоить новое значение существующей переменной:
```kotlin
x = 20
```
Теперь `x` хранит значение `20`.

---

Оператор `+=` — это сокращение, которое прибавляет значение к переменной и присваивает результат обратно ей.
Вместо того чтобы писать:
```kotlin
x = x + 5
```
Можно написать:
```kotlin
x += 5
```
Оба выражения делают одно и то же: увеличивают значение `x` на `5`.

---

Оператор `-=` вычитает значение из переменной и присваивает результат обратно ей.
Вместо того чтобы писать:
```kotlin
x = x - 3
```
Можно написать:
```kotlin
x -= 3
```
Это уменьшает значение `x` на `3`.

---

Оператор `*=` умножает переменную на значение и присваивает результат обратно ей.
Вместо того чтобы писать:
```kotlin
x = x * 4
```
Можно написать:
```kotlin
x *= 4
```
Это умножает `x` на `4` и сохраняет результат обратно в `x`.

---

Оператор `/=` делит переменную на значение и присваивает результат обратно ей.
Вместо того чтобы писать:
```kotlin
x = x / 2
```
Можно написать:
```kotlin
x /= 2
```
Примечание: когда оба операнда имеют тип `Int`, Kotlin выполняет **целочисленное деление** (дробная часть отбрасывается):
```kotlin
var x = 7
x /= 2
// x теперь равно 3 (не 3.5)
```
