# --description--

Принятие решений необходимо, когда мы хотим выполнить код только при выполнении определённого условия.
Предположим, мы хотим играть на улице только при хорошей погоде.
В программировании мы можем сохранить булеву переменную `niceWeather` и выполнить действие — играть на улице — `if` (если) эта переменная равна `true`, например:
```kotlin
var niceWeather = true
if (niceWeather) {
    // play outside
}
```
