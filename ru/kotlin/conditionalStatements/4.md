# --description--

Предположим, у нас есть ещё одно условие для проверки, как в этом примере:
```kotlin
var num = 3
if (num == 2) {
    println("the number is 2")
} else if (num == 3) {
    println("the number is 3")
} else {
    println("do something else")
}
```
и вывод этого кода — `the number is 3`.
Сначала проверяем, равно ли число 2 — это ложь.
Переходим ко второму оператору и проверяем, равно ли `num` 3. Это истина, поэтому мы выполняем следующий блок кода, выводя `the number is 3`
