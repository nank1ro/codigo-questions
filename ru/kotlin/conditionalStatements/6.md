# --description--

Мы также можем вложить условный оператор (`if`, `else if` или `else`) внутрь другого условного оператора, создавая более сложную структуру.
```kotlin
var num = 4
if (num < 3) {
    println("the number is lower than 3")
} else {
    if (num == 3) {
        println("the number is 3")
    } else if (num == 4) {
        println("the number is 4")
    } else {
        println("the number is greather than 4")
    }
}
```
и вывод этого кода — `the number is 4`.
