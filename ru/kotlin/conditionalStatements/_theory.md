Принятие решений необходимо, когда мы хотим выполнить код только при выполнении определённого условия.
Предположим, мы хотим играть на улице только при хорошей погоде.
В программировании мы можем сохранить булеву переменную `niceWeather` и выполнить действие — играть на улице — `if` (если) эта переменная равна `true`, например:
```kotlin
var niceWeather = true
if (niceWeather) {
    // play outside
}
```

---

Продолжим с предыдущим примером.
```kotlin
var niceWeather = true
if (niceWeather) {
    // play outside
}
```
Мы видели, что оператор `if` выполняет блок кода только если условие равно `true`.
Ещё один важный момент — это **фигурные скобки** `{}`, которые обозначают блок кода.

---

Мы только что увидели, как выполнить блок кода при выполнении условия. Теперь давайте посмотрим, как выполнить другой блок кода, если первое условие не выполняется.
Мы идём играть на улицу, если погода хорошая; в противном случае остаёмся дома.
В Kotlin мы можем использовать оператор `else`, например:
```kotlin
var niceWeather = true
if (niceWeather) {
    // play outside
} else {
    // stay home
}
```

---

Предположим, у нас есть ещё одно условие для проверки, как в этом примере:
```kotlin
var num = 3
if (num == 2) {
    println("the number is 2")
} else if (num == 3) {
    println("the number is 3")
} else {
    println("do something else")
}
```
и вывод этого кода — `the number is 3`.
Сначала проверяем, равно ли число 2 — это ложь.
Переходим ко второму оператору и проверяем, равно ли `num` 3. Это истина, поэтому мы выполняем следующий блок кода, выводя `the number is 3`

---

Мы можем добавлять сколько угодно операторов `else if`, ограничений нет
```kotlin
var num = 4
if (num == 2) {
    println("the number is 2")
} else if (num == 3) {
    println("the number is 3")
} else if (num == 4) {
    println("the number is 4")
} else if (num == 5) {
    println("the number is 5")
} else if (num == 6) {
    println("the number is 6")
}
```
и вывод этого кода — `the number is 4`.

---

Мы также можем вложить условный оператор (`if`, `else if` или `else`) внутрь другого условного оператора, создавая более сложную структуру.
```kotlin
var num = 4
if (num < 3) {
    println("the number is lower than 3")
} else {
    if (num == 3) {
        println("the number is 3")
    } else if (num == 4) {
        println("the number is 4")
    } else {
        println("the number is greather than 4")
    }
}
```
и вывод этого кода — `the number is 4`.

_Элвис-оператор_ `a ?: b` разворачивает опциональное значение `a`, если оно содержит значение, или возвращает значение по умолчанию `b`, если `a` равно `null`.
Выражение `a` всегда имеет опциональный тип.
Выражение `b` должно соответствовать типу, хранящемуся внутри a.
Элвис-оператор является сокращением для следующего кода:
```kotlin
if (a != null) a else b
```
