Переменные — это контейнеры для хранения значений данных.
Каждая переменная в Kotlin является объектом.
Чтобы создать переменную, нужно дать ей __имя__, помня, что оно не должно содержать пробелов.
Переменная создаётся в момент, когда вы впервые присваиваете ей значение.
В Kotlin вы объявляете константы с помощью ключевого слова `val` (сокращение от _value_), а переменные — с помощью ключевого слова `var` (сокращение от _variable_).
Значение константы нельзя изменить после установки, тогда как переменной можно присвоить другое значение в будущем.
Пример создания переменной с именем `x`:
```kotlin
var x = 1
```
Таким образом мы присвоили значение `1` переменной с именем `x`.
Если мы выведем переменную `x`, мы получим число `1`:
```kotlin
println(x) // выводит 1
```

---

Переменные называются так потому, что хранимое в них значение может меняться.
Мы можем обновить `x`, используя `=` и присвоив новое значение.
```kotlin
var x = 1
println(x) // выводит 1
x = 2
println(x) // выводит 2
```

---

Мы также можем присвоить переменным значения других переменных. Здесь мы можем присвоить переменной `y` значение `x`
```kotlin
var x = 5
var y = x
println(y) // выводит 5
```

---

Когда мы обновляем переменную, она забывает своё предыдущее значение. Здесь мы можем вывести переменную `x` дважды и увидеть, как её значение обновляется.
```kotlin
var x = 5
println(x) // выводит 5
x = 10
println(x) // выводит 10
```

---

В Kotlin строковые переменные могут быть объявлены только с использованием двойных кавычек:
```kotlin
val x = "May"
```

---

Если мы хотим создать имя переменной из нескольких слов, мы используем **camelCase**.
Это практика написания фраз, при которой каждое слово в середине фразы начинается с заглавной буквы
