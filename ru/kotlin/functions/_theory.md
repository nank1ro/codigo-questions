Функция — это блок кода, который выполняет определённую задачу и может многократно использоваться в программе.
В Kotlin функция определяется с помощью ключевого слова `fun`, за которым следует имя функции и круглые скобки:
```kotlin
fun greet() {
    println("Hello!")
}
```
Чтобы вызвать (выполнить) функцию, используйте её имя с круглыми скобками:
```kotlin
greet() // prints Hello!
```
Функция, которая не возвращает значение, неявно возвращает `Unit`.

---

Функции могут возвращать значение. Тип возвращаемого значения указывается после скобок через двоеточие:
```kotlin
fun getNumber(): Int {
    return 42
}
```
Ключевое слово `return` передаёт значение обратно вызывающему коду:
```kotlin
var result = getNumber()
println(result) // prints 42
```
Тип возвращаемого значения должен соответствовать типу возвращаемого значения функции.

---

Функции могут принимать параметры — входные значения, передаваемые при вызове.
Параметры объявляются внутри скобок с именем и типом:
```kotlin
fun greet(name: String) {
    println("Hello, $name!")
}
```
Аргументы передаются при вызове функции:
```kotlin
greet("Alice") // prints Hello, Alice!
```
Параметры позволяют писать многократно используемый код, работающий с разными значениями.

---

Kotlin поддерживает значения параметров по умолчанию. Если вызывающий код не передаёт аргумент, используется значение по умолчанию:
```kotlin
fun greet(name: String = "World") {
    println("Hello, $name!")
}
greet()         // prints Hello, World!
greet("Alice")  // prints Hello, Alice!
```
Значения по умолчанию делают параметры необязательными, сокращая необходимость в перегрузке функций.

---

Когда тело функции состоит из одного выражения, можно использовать синтаксис одного выражения с `=`:
```kotlin
fun square(n: Int): Int = n * n
```
Это более лаконично, чем форма с блоком тела. Kotlin часто может вывести тип возвращаемого значения, поэтому можно также написать:
```kotlin
fun square(n: Int) = n * n
```
Функции с одним выражением — распространённая идиома Kotlin для простых вычислений.

---

Функции могут возвращать значения `Boolean`, что полезно для проверки условий:
```kotlin
fun isEven(n: Int): Boolean {
    return n % 2 == 0
}
println(isEven(4)) // prints true
println(isEven(7)) // prints false
```
Функция `Boolean` возвращает `true` или `false`.

---

Функции могут иметь несколько параметров разных типов:
```kotlin
fun introduce(name: String, age: Int): String {
    return "My name is $name and I am $age years old."
}
println(introduce("Bob", 30))
// prints My name is Bob and I am 30 years old.
```
Именованные аргументы позволяют передавать значения в любом порядке, используя имя параметра:
```kotlin
println(introduce(age = 30, name = "Bob"))
```

---

Базовая функция в Kotlin использует ключевое слово `fun`, за которым следует имя и круглые скобки:
```kotlin
fun sayBye() {
    println("Bye!")
}
```

---

Чтобы объявить тип возвращаемого значения функции, добавьте двоеточие и тип после скобок:
```kotlin
fun getName(): String {
    return "Alice"
}
```

---

Параметр функции объявляется с именем, двоеточием и типом:
```kotlin
fun greet(name: String) {
    println("Hi, $name!")
}
```

---

Функция с одним выражением использует `=` вместо блочного тела:
```kotlin
fun triple(n: Int) = n * 3
```
Это сокращение для:
```kotlin
fun triple(n: Int): Int {
    return n * 3
}
```
