# --description--

Функция — это блок кода, который выполняет определённую задачу и может многократно использоваться в программе.
В Kotlin функция определяется с помощью ключевого слова `fun`, за которым следует имя функции и круглые скобки:
```kotlin
fun greet() {
    println("Hello!")
}
```
Чтобы вызвать (выполнить) функцию, используйте её имя с круглыми скобками:
```kotlin
greet() // prints Hello!
```
Функция, которая не возвращает значение, неявно возвращает `Unit`.
