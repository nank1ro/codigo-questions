# --description--

Kotlin поддерживает значения параметров по умолчанию. Если вызывающий код не передаёт аргумент, используется значение по умолчанию:
```kotlin
fun greet(name: String = "World") {
    println("Hello, $name!")
}
greet()         // prints Hello, World!
greet("Alice")  // prints Hello, Alice!
```
Значения по умолчанию делают параметры необязательными, сокращая необходимость в перегрузке функций.
