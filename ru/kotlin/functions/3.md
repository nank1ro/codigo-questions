# --description--

Функции могут принимать параметры — входные значения, передаваемые при вызове.
Параметры объявляются внутри скобок с именем и типом:
```kotlin
fun greet(name: String) {
    println("Hello, $name!")
}
```
Аргументы передаются при вызове функции:
```kotlin
greet("Alice") // prints Hello, Alice!
```
Параметры позволяют писать многократно используемый код, работающий с разными значениями.
