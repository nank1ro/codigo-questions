# --description--

Когда тело функции состоит из одного выражения, можно использовать синтаксис одного выражения с `=`:
```kotlin
fun square(n: Int): Int = n * n
```
Это более лаконично, чем форма с блоком тела. Kotlin часто может вывести тип возвращаемого значения, поэтому можно также написать:
```kotlin
fun square(n: Int) = n * n
```
Функции с одним выражением — распространённая идиома Kotlin для простых вычислений.
