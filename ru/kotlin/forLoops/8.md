# --description--

В Kotlin мы также можем использовать `for-in` для итерируемых коллекций, вызывая данное замыкание для каждого элемента:
```kotlin
// this is a list, we'll see about that soon
val numbers = listOf(2, 4, 6, 8, 10)
for (num in numbers) {
    println(num)
}
// prints (2, 4, 6, 8, 10)
```
