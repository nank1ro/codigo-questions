# --description--

Если вам просто нужно повторить блок кода `n` раз, вы можете использовать функцию `repeat(times: Int)`.

```kotlin
repeat(3) {
    println("repeat")
}
// prints repeat 3 times
```

Вы даже можете получить доступ к индексу с помощью
```kotlin
repeat(3) { index ->
    println(index)
}
// prints 0, 1, 2
```
