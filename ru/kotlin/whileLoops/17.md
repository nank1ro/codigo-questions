Как мы узнаём, что это бесконечный цикл?
```kotlin

while (true) {
    println("Please login")
}
```

# --answers--

- Этот цикл не выполнится, так как условие ложно
- В условии нет переменной
- Этот цикл выполнится один раз
- Этот цикл выполнится дважды
