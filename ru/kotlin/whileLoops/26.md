# --instructions--

Что вызывает сбой этой программы?
```kotlin
while (true) {
    println("repeat")
}
```

# --answers--

- В коде есть бесконечный цикл
- Ничего не вызывает сбой этого кода
- У цикла while неправильный отступ
- У блока кода неправильный отступ
