# --instructions--

Что делает `counter += 1`?
```kotlin
var counter = 1

while (counter < 5) {
    println(counter)
    counter += 1
}
```

# --answers--

- Увеличивает counter на единицу при каждом выполнении блока кода цикла
- Создаёт переменную counter
- Увеличивает counter на два каждый раз
- Увеличивает counter на три каждый раз
