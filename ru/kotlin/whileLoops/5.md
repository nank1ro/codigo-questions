# --description--

Вот пример использования `break` в цикле _while_:
```kotlin
var i = 0
while (i < 3) {
  i++
  if (i == 2) break // [1]
  println(i)
}
// prints 1
```

Как видите, в __[1]__, когда `i` равно _2_, мы _прерываем_ цикл. Поэтому числа 2 и 3 никогда не выводятся.
