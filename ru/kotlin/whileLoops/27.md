# --instructions--

Что не так с этим кодом?
```kotlin
while (isLoading) {
    println("Loading...")
    isLoading = false
}
```

# --answers--

- isLoading не является допустимым именем переменной
- В цикле while есть оператор вывода внутри блока кода
- У блока кода неправильный отступ
