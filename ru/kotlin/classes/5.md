# --description--

**`data class`** — это специальный класс, предназначенный для хранения данных. Kotlin автоматически генерирует для вас `equals()`, `hashCode()`, `toString()` и `copy()`:

```kotlin
data class Point(val x: Int, val y: Int)

val p = Point(1, 2)
println(p)           // Point(x=1, y=2)
println(p.toString()) // Point(x=1, y=2)
```

Функция `copy()` создаёт новый экземпляр с изменёнными некоторыми свойствами:

```kotlin
val p2 = p.copy(y = 10)
println(p2) // Point(x=1, y=10)
```

`Point(3, 5)` должен иметь `x` равным `3`

```kotlin
    val p = Point(3, 5)
    tryCatch(p.x == 3)
```

`Point(3, 5)` должен иметь `y` равным `5`

```kotlin
    tryCatch(p.y == 5)
```

`toString()` должен быть равен `"Point(x=3, y=5)"`

```kotlin
    tryCatch(p.toString() == "Point(x=3, y=5)")
```

`copy(y = 10)` должен вернуть точку с `y` равным `10`

```kotlin

    tryCatch(p2.y == 10)
```
