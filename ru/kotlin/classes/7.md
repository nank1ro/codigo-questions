# --instructions--

**Объект-компаньон** (`companion object`) — это одиночный объект, связанный с классом. Он позволяет определять свойства и методы, принадлежащие самому классу, а не его экземплярам — аналогично членам `static` в других языках:

```kotlin
class MathHelper {
    companion object {
        fun square(n: Int): Int = n * n
    }
}
println(MathHelper.square(4)) // 16
```

Создайте класс `Counter` с объектом-компаньоном, который содержит свойство `var` с именем `total` типа `Int` (инициализированное значением `0`) и метод `increment()`, прибавляющий `1` к `total`.

# --asserts--

`Counter.total` должен начинаться с `0`

```kotlin
    tryCatch(Counter.total == 0)
```

После одного вызова `increment()`, `total` должен быть равен `1`

```kotlin
    Counter.increment()
    tryCatch(Counter.total == 1)
```

После ещё двух вызовов `increment()`, `total` должен быть равен `3`

```kotlin

    Counter.increment()
    tryCatch(Counter.total == 3)
```
