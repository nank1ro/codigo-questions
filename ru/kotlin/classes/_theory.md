**Класс** — это шаблон для создания объектов. В Kotlin класс объявляется с помощью ключевого слова `class`, за которым следует имя класса.

Класс может содержать **свойства** (данные) и **методы** (поведение). Чтобы объявить свойство непосредственно в конструкторе, добавьте перед ним `val` или `var`:

```kotlin
class Dog(val name: String)
```

Это создаёт класс `Dog` со свойством `name`. Экземпляр создаётся вызовом класса как функции:

```kotlin
val dog = Dog("Rex")
println(dog.name) // Rex
```

---

Классы могут иметь несколько свойств в своём **основном конструкторе**. Каждое свойство объявляется с помощью `val` (только для чтения) или `var` (изменяемое) непосредственно в списке параметров конструктора:

```kotlin
class Person(val name: String, var age: Int)
```

Ключевое слово `val` делает `name` доступным только для чтения после создания, тогда как `var` делает `age` изменяемым — его можно изменить позже:

```kotlin
val p = Person("Alice", 30)
p.age = 31  // разрешено, так как age является var
```

---

**Блок `init`** выполняется сразу после основного конструктора. Он используется для выполнения проверки или логики инициализации при создании объекта:

```kotlin
class Circle(val radius: Double) {
    init {
        require(radius > 0) { "Радиус должен быть положительным" }
    }
}
```

Внутри `init` можно также инициализировать дополнительные свойства:

```kotlin
class Square(val side: Int) {
    val area: Int
    init {
        area = side * side
    }
}
```

---

Классы могут содержать **методы** — функции, принадлежащие классу и работающие с его свойствами:

```kotlin
class Counter(var count: Int) {
    fun increment() {
        count++
    }
    fun value(): Int {
        return count
    }
}
```

Методы вызываются с использованием точечной нотации на экземпляре:

```kotlin
val c = Counter(0)
c.increment()
println(c.value()) // 1
```

---

**`data class`** — это специальный класс, предназначенный для хранения данных. Kotlin автоматически генерирует для вас `equals()`, `hashCode()`, `toString()` и `copy()`:

```kotlin
data class Point(val x: Int, val y: Int)

val p = Point(1, 2)
println(p)           // Point(x=1, y=2)
println(p.toString()) // Point(x=1, y=2)
```

Функция `copy()` создаёт новый экземпляр с изменёнными некоторыми свойствами:

```kotlin
val p2 = p.copy(y = 10)
println(p2) // Point(x=1, y=10)
```
