# --description--

Классы могут содержать **методы** — функции, принадлежащие классу и работающие с его свойствами:

```kotlin
class Counter(var count: Int) {
    fun increment() {
        count++
    }
    fun value(): Int {
        return count
    }
}
```

Методы вызываются с использованием точечной нотации на экземпляре:

```kotlin
val c = Counter(0)
c.increment()
println(c.value()) // 1
```

# --asserts--

После внесения `100.0` на счёт с балансом `0.0`, баланс должен быть `100.0`

```kotlin
    val account = BankAccount(0.0)
    account.deposit(100.0)
    tryCatch(account.balance == 100.0)
```

После снятия `30.0`, баланс должен быть `70.0`

```kotlin
    account.withdraw(30.0)
    tryCatch(account.balance == 70.0)
```

После ещё одного пополнения на `50.0`, баланс должен быть `120.0`

```kotlin
    account.deposit(50.0)
    tryCatch(account.balance == 120.0)
```

class BankAccount(var balance: Double) {

    }
    fun withdraw(amount: Double) {
        balance -= amount
