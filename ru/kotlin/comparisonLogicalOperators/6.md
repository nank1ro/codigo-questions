# --description--

Продолжим с оператором сравнения **меньше или равно** `<=`.
Он возвращает **логическое значение** (`true` или `false`), указывающее, меньше ли одно выражение другого или равно ему, например:
```kotlin
println(2 <= 2) // true
println(3 <= 2) // false
println(3 <= 4) // true
```
