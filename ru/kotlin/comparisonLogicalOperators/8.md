# --description--

Продолжим с логическим оператором **или** `||`.
Он возвращает первый операнд, который вычисляется как *true*, или последний, если все *false*.
```kotlin
println(2 == 2 || 2 == 3) // true
println(1 == 2 || 1 == 3) // false
```
