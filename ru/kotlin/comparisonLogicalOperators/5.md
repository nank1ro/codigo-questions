# --description--

Продолжим с оператором сравнения **больше или равно** `>=`.
Он возвращает **логическое значение** (`true` или `false`), указывающее, больше ли одно выражение другого или равно ему, например:
```kotlin
println(2 >= 2) // true
println(3 >= 2) // true
println(3 >= 4) // false
```
