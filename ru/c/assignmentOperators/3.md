# --description--

Так же, как и присваивание сложения, существует **присваивание вычитания** `-=`.
Функциональность та же, единственное отличие в том, что выполняется вычитание.
Таким образом, следующие записи абсолютно одинаковы
```c
num = num - 5;
// равно
num -= 5;
```
