# --description--

У нас уже есть инициализированная переменная `total`
```c
int total = 5;
```
Допустим, мы хотим прибавить число `2` к переменной `total`, мы можем написать
```c
total = total + 2;
```
Хорошо, это работает! Но есть более короткая запись для того же действия:
```c
total += 2;
```
Знак `+=` называется **присваивание сложения**.
Он прибавляет значение к переменной и присваивает результат этой переменной.
