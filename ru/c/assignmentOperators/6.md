# --description--

Рассмотрим оператор **присваивания остатка от деления** `%=`.
Он вычисляет остаток от деления переменной на значение и присваивает результат этой переменной.
Таким образом, следующие записи абсолютно одинаковы
```c
num = num % 5;
// равно
num %= 5;
```
