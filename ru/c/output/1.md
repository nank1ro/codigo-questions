# --description--

"__Hello, World!__" — это традиционная первая программа при изучении нового языка программирования.
Мы используем функцию `printf()` для вывода данных на стандартное устройство вывода (экран).
Каждая программа на C использует библиотеки, которые предоставляют возможность выполнять необходимые функции, например, функция вывода на экран определена в заголовочном файле `stdio.h`.
Первый код, который будет выполнен, всегда находится в функции main.
Чтобы вывести `Hello, World!` на экран на языке C, мы можем написать
```c
printf("Hello, World!");
```
В конце мы возвращаем `0`, чтобы указать, что наша программа завершилась успешно
