# --description--

Продолжим с оператором сравнения **меньше или равно** `<=`.
Он возвращает **булево значение**, истина `1` или ложь `0`, указывая, меньше или равно ли одно выражение другому, например:
```c
>>> 2 <= 2
1
>>> 3 <= 2
0
>>> 3 <= 4
1
```
