# --description--

Продолжим с логическим оператором **или** `||`.
Он возвращает первый операнд, который вычисляется как *истина*, или последний, если все *ложны*.
```c
>>> 2 == 2 || 2 == 3
1
>>> 1 == 2 || 1 == 3
0
```
