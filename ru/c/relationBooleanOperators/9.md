# --description--

Завершим логическим оператором **не** `!`.
Он возвращает булево значение, противоположное логическому состоянию выражения.
```c
>>> !true
0
>>> !false
1
>>> !(2 == 2)
0
```

# --instructions--

Выведите булево значение, проверяя, **НЕ** равно ли `num` числу `5`
