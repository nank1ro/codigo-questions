# --description--

Теперь рассмотрим **логические** операторы, начнём с первого — __и__ `&&`.
Он возвращает первый операнд, который вычисляется как *ложь*, или последний, если все *истинны*.
```c
>>> 2 == 2 && 2 == 3
0
>>> 1 == 1 && 1 == 1.0
1
```
