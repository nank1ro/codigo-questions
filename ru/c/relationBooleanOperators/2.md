# --description--

Продолжим с оператором сравнения **не равно** `!=`.
Он возвращает **булево значение**, истина `1` или ложь `0`, указывая, **НЕ** равны ли два выражения, например:
```c
>>> 2 != 2
0
>>> 2 != 3
1
```
Он является полной противоположностью оператора *равно*
