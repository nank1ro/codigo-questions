# --description--

Продолжим с оператором сравнения **больше или равно** `>=`.
Он возвращает **булево значение**, истина `1` или ложь `0`, указывая, больше или равно ли одно выражение другому, например:
```c
>>> 2 >= 2
1
>>> 3 >= 2
1
>>> 3 >= 4
0
```
