# --instructions--

Как мы узнаем, что `Hello World` всегда будет выведено?
```c
bool online = false;
if (online) {
    printf("User is online\n");
}
printf("Hello World\n");
```

# --answers--

- Потому что оно не находится внутри условного оператора
- Потому что оператор if не может содержать более одной строки
- Потому что нет оператора else
- Потому что нет оператора else if
