Массивы — это тип данных, который можно использовать для хранения набора различных элементов информации в виде последовательности под одним именем переменной.
Массив хранит несколько значений одного типа и использует **индексы** для различения этих значений.
Вы можете присвоить элементы массиву с помощью выражения следующего вида:
```
data_type array_name[array_size] = {item1, item2};
```
`data_type` — это тип данных, который вы будете использовать для массива, например `int`, `double` и т.д.
`array_name` — это имя переменной, в которой хранятся элементы.
`array_size` — это максимальный размер массива.
Наконец, `item1` и `item2` — это значения, которые мы хотим сохранить в массиве

---

Вы можете получить доступ к отдельному элементу массива по его индексу.
Индекс — это как адрес, который определяет место элемента в массиве.
Индекс указывается сразу после имени массива, в квадратных скобках, вот так:
```
list_name[index];
```

Индексы массива начинаются с `0`, а **не** с `1`! Доступ к первому элементу массива осуществляется так: `list_name[0]`.
Второй элемент массива находится по индексу 1: `list_name[1]`.

---

Индекс массива ведёт себя как любое другое имя переменной! Его можно использовать как для доступа, так и для присвоения значений.
Вы видели, как получить доступ к элементу массива по индексу:
```c
int numbers[4] = {5, 6, 7, 8};
numbers[0]; // Gets the value 5
```
Вот как работает присвоение:
```c
int numbers[4] = {5, 6, 7, 8};
numbers[0] = 1;
printf("%d\n", numbers[0]); // prints the new value 1
```

---

Вы можете вычислить длину массива в байтах, получив `sizeof` массива, а затем разделив его на размер одного элемента.
Это работает, потому что каждый элемент массива имеет одинаковый тип и, соответственно, одинаковый размер.
Полученная *длина* — это количество содержащихся в нём элементов

---

Массив в C должен иметь фиксированную длину.
Вы не можете добавлять элементы в конец массива после объявления его размера.

---

В языке программирования C можно создать массив массивов.
Такие массивы называются многомерными массивами, например:
```c
int numbers[2][3] = {{1, 2, 3}, {4, 5, 6}};
```
