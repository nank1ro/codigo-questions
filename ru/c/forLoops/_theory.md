Мы знаем, как повторять код с помощью цикла `while`.
Например, эта программа повторяет инструкции для вывода `hello`
```c
int counter = 0;

while (counter < 5) {
    printf("Hello\n");
    counter++;
}
```
Но мы можем сделать то же самое с помощью циклов `for`:
```c
for (int i = 0; i < 5; i++) {
    printf("Hello\n");
}
```

---

В цикле `for` мы можем указать, сколько раз мы хотим, чтобы цикл выполнился

---

Мы можем использовать `<`, чтобы выполнять цикл до следующего числа, не включая его, или `<=`, чтобы выполнять цикл до следующего числа, включая его

---

Переменная с именем `i` является переменной-счётчиком.
Мы можем дать ей любое имя.
Она считает, на какой итерации цикла мы сейчас находимся
