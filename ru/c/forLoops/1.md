# --description--

Мы знаем, как повторять код с помощью цикла `while`.
Например, эта программа повторяет инструкции для вывода `hello`
```c
int counter = 0;

while (counter < 5) {
    printf("Hello\n");
    counter++;
}
```
Но мы можем сделать то же самое с помощью циклов `for`:
```c
for (int i = 0; i < 5; i++) {
    printf("Hello\n");
}
```
