# --instructions--

Что не так с этим кодом?
```c
while (is_loading) {
    printf("Loading...\n");
    is_loading = false;
}
```

# --answers--

- is_loading не является допустимым именем переменной
- В цикле while есть оператор print внутри блока кода
- У блока кода неправильный отступ
