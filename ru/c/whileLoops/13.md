# --instructions--

Что происходит, когда условие цикла `while` остаётся `true`?

# --answers--

- Цикл повторяется бесконечно
- Цикл выполняет свой блок кода один раз
- Цикл выполняет свой блок кода два раза
- Цикл выполняет свой блок кода три раза
