# --instructions--

Как мы узнаем, что этот цикл останавливается?
```c
bool is_loading = true;

while (is_loading) {
    printf("Loading...\n");
    is_loading = false;
}
```

# --answers--

- Переменная в цикле обновляется до false
- Цикл никогда не остановится
