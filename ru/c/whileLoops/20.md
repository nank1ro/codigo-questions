# --instructions--

Как мы узнаем, что цикл `while` пропускает свой блок кода?
```c
bool is_loading = false;

while (is_loading) {
    printf("Loading...\n");
}
```

# --answers--

- Переменная в условии сравнивается с true
- Переменная в условии сравнивается с false
