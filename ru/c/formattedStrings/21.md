# --instructions--

Что не так с этим кодом?
```c
int pieces = 5;
char label[] = "we have %i pieces", 5;
```

# --answers--

- Нельзя сохранять форматированные строки в строковых переменных
- Отсутствует символ f
- Нельзя использовать переменные внутри %
- С кодом всё в порядке
