# --instructions--

Как отобразить целочисленные значения с помощью форматированных строк?

# --answers--

- Мы размещаем их с помощью выражения %i
- Мы не можем отображать числовые значения с помощью форматированных строк
- Мы размещаем их в квадратных скобках []
- Мы размещаем их в круглых скобках ()
