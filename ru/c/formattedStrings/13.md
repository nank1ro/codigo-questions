# --instructions--

Что не так с этим кодом?
```c
printf(0, "%i friends");
```

# --answers--

- Значение выражения должно идти после строки
- Выражение не является числом
- У строки отсутствуют кавычки " "
- Выражение должно содержать имя переменной
