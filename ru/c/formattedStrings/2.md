# --description--

Форматированные строки позволяют отображать выражения, например, добавлять строку к числу, без ошибок.
Для отображения целого числа мы используем выражение `%i`

# --seed--

```c
printf("[/] friends\n", [/]);
```

# --answers--

- %i
- ?i
- %0
- 5

# --solutions--

```c
printf("%i friends\n", 5);
```

# --output--

5 friends
