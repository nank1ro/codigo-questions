# --instructions--

Что не так с этим кодом?
```c
int manchester_united = 0;
int barcelona = 1;
printf("The score is %i : %i\n", manchester_united, barcelona);
```

# --answers--

- С кодом всё в порядке
- У manchester_united и barcelona нет значений
- Нельзя использовать переменные внутри форматированной строки
- Нельзя использовать %i дважды в одной строке
