# --description--

Массив не обязательно должен иметь фиксированную длину.
Вы можете добавлять элементы в конец массива в любое время!
Для добавления элемента в массив используется функция `push`:
```javascript
var letters = ["a", "b"];
letters.push("c");
console.log(letters);
// Выводит ["a", "b", "c"]
```
