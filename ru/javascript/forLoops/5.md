# --description--

В JavaScript также есть цикл `forEach`.
`forEach` вызывает переданную функцию для каждого элемента последовательности в том же порядке, что и цикл `for`:
```javascript
// это массив, мы скоро узнаем о нём подробнее
let numbers = [1, 3, 5, 7, 9];
numbers.forEach((num) => console.log(num));}
```
Использование метода `forEach` отличается от цикла `for` двумя важными особенностями:
1. Операторы `break` или `continue` нельзя использовать для выхода из текущего вызова функции обратного вызова или для пропуска последующих вызовов.
2. Использование оператора `return` в функции обратного вызова приведёт только к выходу из неё, а не из внешней области видимости, и не пропустит последующие вызовы.
ПРИМЕЧАНИЕ: `=>` — это так называемая _стрелочная функция_, сокращённый синтаксис функции ES6, который заменяет фигурные скобки {} и возвращает значение (при необходимости)
