# --description--

Мы можем использовать `<` для цикла до следующего числа, не включая его, или `<=` для цикла до следующего числа, включая его

# --instructions--

Дополните цикл `for` в следующем коде, чтобы блок кода выполнился `3` раза

# --seed--

```javascript
for (let i = 1; i[/]3; i++) {
    console.log(i);
}
```
