# --description--

Начнём с оператора сравнения **равно** `==`.
Он возвращает **логическое значение** (`true` или `false`), указывающее, равны ли два выражения, например:
```javascript
console.log(2 == 2);
// выводит true
console.log(2 == 3);
// выводит false
```
