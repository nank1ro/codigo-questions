# --description--

Теперь рассмотрим **логические** операторы, начнём с первого — __И__ `&&`.
Он возвращает первый операнд, который вычисляется как *false*, или последний, если все равны *true*.
```javascript
console.log(2 == 2 && 2 == 3);
// выводит false
console.log(1 == 1 && 1 == 1.0);
// выводит true
```
