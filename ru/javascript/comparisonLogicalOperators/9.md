# --description--

Завершим логическим оператором **НЕ** `!`.
Он возвращает логическое значение, противоположное логическому состоянию выражения.
```javascript
console.log(!true);
// выводит false
console.log(!false);
// выводит true
console.log(!(2 == 2));
// выводит false
```
