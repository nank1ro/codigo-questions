# --description--

Продолжим с логическим оператором **или** `||`.
Он возвращает первый операнд, который вычисляется как *true*, или последний, если все равны *false*.
```javascript
console.log(2 == 2 || 2 == 3);
// выводит true
console.log(1 == 2 || 1 == 3);
// выводит false
```
