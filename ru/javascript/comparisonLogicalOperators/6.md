# --description--

Продолжим с оператором сравнения **меньше или равно** `<=`.
Он возвращает **логическое значение** (`true` или `false`), указывающее, меньше ли или равно одно выражение другому, например:
```javascript
console.log(2 <= 2);
// выводит true
console.log(3 <= 2);
// выводит false
console.log(3 <= 4);
// выводит true
```
