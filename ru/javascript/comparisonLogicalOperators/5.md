# --description--

Продолжим с оператором сравнения **больше или равно** `>=`.
Он возвращает **логическое значение** (`true` или `false`), указывающее, больше ли или равно одно выражение другому, например:
```javascript
console.log(2 >= 2);
// выводит true
console.log(3 >= 2);
// выводит true
console.log(3 >= 4);
// выводит false
```
