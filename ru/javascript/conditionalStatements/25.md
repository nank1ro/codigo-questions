# --instructions--

Почему мы знаем, что `Hello World` всегда будет выведено?
```javascript
var online = False;
if (online) {
    console.log("User is online");
}
console.log("Hello World");
```

# --answers--

- Потому что оно не находится внутри блока кода
- Потому что оператор if не может содержать более одной строки
- Потому что нет оператора else
- Потому что нет оператора else if
