Переменные — это контейнеры для хранения данных.
Каждая переменная в JavaScript является объектом.
Чтобы создать переменную, нужно дать ей **имя**, помня, что оно не должно содержать пробелов.
Переменная создаётся в момент первого присвоения ей значения.
В JavaScript константы объявляются с помощью ключевых слов `let` или `const`, а переменные — с помощью ключевого слова `var`.
Значение константы нельзя изменить после установки, тогда как переменной можно присвоить другое значение в будущем.
Пример создания переменной с именем `x`:
```javascript
var x = 1;
```
Таким образом мы присвоили значение `1` переменной с именем `x`.
Если мы выведем переменную `x`, то получим число `1`:
```javascript
console.log(x);
// prints 1
```

---

Переменные называются так, потому что хранимое в них значение может изменяться.
Мы можем обновить `x`, используя `=` и задав новое значение.
```javascript
var x = 1;
console.log(x); // prints 1
x = 2;
console.log(x); // prints 2
```

---

Мы также можем присваивать переменным значения других переменных.
Здесь мы можем присвоить переменной `y` значение `x`
```javascript
var x = 5;
var y = x;
console.log(y); // prints 5
```

---

Когда мы обновляем переменную, она забывает своё предыдущее значение.
Здесь мы можем вывести переменную `x` дважды и увидеть, как её значение обновляется.
```javascript
var x = 5;
console.log(x); // prints 5
x = 10;
console.log(x); // prints 10
```

---

В JavaScript строковые переменные можно объявлять как с двойными, так и с одинарными кавычками:
```javascript
let x = "May";
// both are the same string
let y = 'May';
console.log(x === y);
// prints true
```

---

Если мы хотим дать переменной имя из нескольких слов, мы используем **camelCase**.
Это практика написания фраз, при которой каждое слово в середине фразы начинается с заглавной буквы
