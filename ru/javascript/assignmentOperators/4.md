# --description--

Рассмотрим оператор **присваивания умножения** `*=`.
Он умножает переменную на значение и присваивает результат этой переменной.
Таким образом, следующие записи абсолютно одинаковы
```javascript
var num = num * 5;
// равносильно
num *= 5;
```
