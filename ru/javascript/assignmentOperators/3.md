# --description--

Как и в присваивании сложения, существует **присваивание вычитания** `-=`.
Функциональность та же, единственная разница в том, что выполняется вычитание.
Таким образом, следующие записи абсолютно одинаковы
```javascript
var num = num - 5;
// равносильно
num -= 5;
```
