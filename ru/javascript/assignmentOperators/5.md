# --description--

Рассмотрим оператор **присваивания деления** `/=`.
Он делит переменную на значение и присваивает результат этой переменной.
Таким образом, следующие записи абсолютно одинаковы
```javascript
num = num / 5;
// равносильно
num /= 5;
```
