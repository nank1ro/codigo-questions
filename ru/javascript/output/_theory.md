"__Привет, мир!__" — это традиционная первая программа при изучении нового языка программирования.
Мы используем функцию `console.log()` для вывода данных на стандартное устройство вывода (экран).
Чтобы вывести `Hello, World!` на экран с помощью JavaScript, можно написать
```javascript
console.log("Hello, World!")
```
