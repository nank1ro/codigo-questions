# --instructions--

Что не так с этим кодом?
```javascript
console.log(`$(10} friends`);
```

# --answers--

- Выражения должны быть внутри ${}
- Значение внутри ${} не является числом
- В строке отсутствуют обратные кавычки ``
- Значение внутри ${} должно быть именем переменной
