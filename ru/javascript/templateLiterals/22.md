# --instructions--

Что не так с этим кодом?
```javascript
var pieces = 5;
var label = `we have ${pieces} pieces`;
```

# --answers--

- Мы не можем сохранять шаблонные литералы в переменных
- Символы ${} отсутствуют
- Мы не можем использовать переменные внутри ${}
- С кодом всё в порядке
