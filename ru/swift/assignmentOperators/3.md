# --description--

Так же как и присваивание сложения, у нас есть **присваивание вычитания** `-=`.
Функциональность такая же, единственное отличие — он выполняет вычитание.
Следующие записи полностью эквивалентны
```swift
var num = num - 5
// is equal to
num -= 5
```
