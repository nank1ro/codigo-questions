# --description--

Мы знаем, как повторять код с помощью цикла `while`.
Например, эта программа повторяет инструкции для вывода `hello`
```swift
var counter = 0

while counter < 5 {
    print("hello")
    counter += 1;
}
```
Но то же самое можно сделать с помощью циклов `for`:
```swift
for i in 0..<5 {
    print("hello")
}
```
