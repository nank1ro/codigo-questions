# --description--

Мы можем использовать `..<` для цикла до следующего числа (не включая его) или `...` для цикла до следующего числа (включая его)

# --instructions--

Дополните цикл `for` в следующем коде, чтобы выполнить блок кода `3` раза

# --seed--

```swift
for i in 1[/]3 {
    print(i)
}
```
