# --description--

Для некоторых перечислений полезно иметь коллекцию всех вариантов.
Вы включаете это, написав `: CaseIterable` после имени перечисления.
Swift предоставляет коллекцию всех вариантов в виде свойства `allCases` типа перечисления:
```swift
enum Colors: CaseIterable {
    case blue, red, green
}
for color in Colors.allCases {
    print(color)
}
// prints blue, red, green
```

# --instructions--

Переберите перечисление, чтобы вывести каждый вариант.
`$0` -- это сокращённое имя аргумента, которое можно использовать для обращения к значениям аргументов замыкания.
`$0` ссылается на первый аргумент замыкания, `$1` на второй и так далее
