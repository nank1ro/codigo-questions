# --description--

Продолжим с оператором сравнения **больше или равно** `>=`.
Он возвращает **логическое значение** (`true` или `false`), указывающее, больше ли одно выражение или равно другому, например:
```swift
print(2 >= 2) // true
print(3 >= 2) // true
print(3 >= 4) // false
```
