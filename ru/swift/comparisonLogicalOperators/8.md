# --description--

Продолжим с логическим оператором **или** `||`.
Он возвращает первый операнд, который оценивается как *true*, или последний, если все *false*.
```swift
print(2 == 2 || 2 == 3) // true
print(1 == 2 || 1 == 3) // false
```
