# --description--

Продолжим с оператором сравнения **меньше или равно** `<=`.
Он возвращает **логическое значение** (`true` или `false`), указывающее, меньше ли одно выражение или равно другому, например:
```swift
print(2 <= 2) // true
print(3 <= 2) // false
print(3 <= 4) // true
```
