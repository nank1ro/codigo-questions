# --description--

Теперь рассмотрим **логические** операторы, начнём с первого, который называется __И__ `&&`.
Он возвращает первый операнд, который оценивается как *false*, или последний, если все *true*.
```swift
print(2 == 2 && 2 == 3) // false
print(1 == 1 && 1 == 1.0) // true
```
