# --description--

Как и для массивов, мы можем перебирать элементы словаря с помощью ключевых слов `for..in`.
Чтобы получить и ключ, и значение в цикле, не нужно указывать никаких свойств:
```swift
for (key, value) in dictName {
    print("\(key): \(value)")
}
```
