# --description--

Массив не обязан иметь фиксированную длину.
Вы можете добавлять элементы в конец массива в любое время!
Для добавления элемента в массив мы используем функцию `append`:
```swift
var letters = ["a", "b"]
letters.append("c")
print(letters)
// Prints ["a", "b", "c"]
```
