# --instructions--

Что произойдёт, если условие цикла `while` останется `true`?

# --answers--

- Цикл будет повторяться бесконечно
- Цикл выполнит свой блок кода один раз
- Цикл выполнит свой блок кода два раза
- Цикл выполнит свой блок кода три раза
