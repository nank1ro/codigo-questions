# --instructions--

Что вызывает сбой этой программы?
```swift
while true {
    print("repeat")
}
```

# --answers--

- В коде бесконечный цикл
- Ничто не вызывает сбой этого кода
- У цикла while неправильный отступ
- У блока кода неправильный отступ
