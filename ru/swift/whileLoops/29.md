# --instructions--

Что не так с этим кодом?
```swift
while isLoading {
    print("Loading...")
    isLoading = false
}
```

# --answers--

- isLoading не является допустимым именем переменной
- В цикле while есть оператор print внутри блока кода
- У блока кода неправильный отступ
