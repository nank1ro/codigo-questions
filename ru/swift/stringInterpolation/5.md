# --description--

Затем мы добавляем значение другого типа в круглые скобки, чтобы оно отображалось как одна инструкция print. Например, с `\(5)`

# --instructions--

Заполните пропуск, чтобы получить следующий вывод
```swift
5 friends
```

# --seed--

```swift
print(" friends")
```

# --solutions--

```swift
print("\(5) friends")
```

# --output--

5 friends
