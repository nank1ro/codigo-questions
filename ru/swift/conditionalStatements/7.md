# --description--

Тернарный условный оператор -- это специальный оператор с тремя частями, который имеет форму `вопрос ? ответ1 : ответ2`.
Это сокращённый способ вычисления одного из двух выражений в зависимости от того, истинен `вопрос` или ложен.
Если `вопрос` истинен, он вычисляет `ответ1` и возвращает его значение; в противном случае он вычисляет `ответ2` и возвращает его значение.
```swift
let a = 10, b = 20, c: Int
if (a < b) {
    c = a
} else {
    c = b
}
print(c)
```
Сокращённый код для приведённого выше:
```swift
let a = 10, b = 20, c: Int
c = a < b ? a : b
print(c)
```
`c` присваивается значение `a`, потому что условие `a < b` было истинным
