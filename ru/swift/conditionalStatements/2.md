# --description--

Продолжим с предыдущим примером.
```swift
var niceWeather = true
if niceWeather {
    // play outside
}
```
Мы видели, что оператор `if` выполняет блок кода только если условие равно `true`.
Ещё одна важная вещь -- это **фигурные скобки** `{}`, которые обозначают блок кода.

# --instructions--

Исправьте оператор `if`, добавив *фигурные скобки* `{}`
