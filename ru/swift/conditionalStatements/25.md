# --instructions--

Как мы можем быть уверены, что `Hello World` всегда будет выведено?
```swift
var online = False;
if online {
    print("User is online")
}
print("Hello World")
```

# --answers--

- Потому что оно не находится внутри блока кода
- Потому что оператор if не может содержать больше одной строки
- Потому что нет оператора else
- Потому что нет оператора else if
