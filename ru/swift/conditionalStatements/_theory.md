Принятие решений необходимо, когда мы хотим выполнить код только при соблюдении определённого условия.
Допустим, мы хотим играть на улице, только если погода хорошая.
В программировании мы можем сохранить логическую переменную `niceWeather` и выполнить действие "играть на улице" `if` (если) эта переменная равна `true`, например:
```swift
var niceWeather = true
if niceWeather {
    // play outside
}
```

---

Продолжим с предыдущим примером.
```swift
var niceWeather = true
if niceWeather {
    // play outside
}
```
Мы видели, что оператор `if` выполняет блок кода только если условие равно `true`.
Ещё одна важная вещь -- это **фигурные скобки** `{}`, которые обозначают блок кода.

---

Мы только что увидели, как выполнить блок кода при наступлении условия, теперь давайте посмотрим, как выполнить другой блок кода, если первое условие не выполняется.
Мы идём играть на улицу, если погода хорошая; в противном случае остаёмся дома.
В Swift мы можем использовать оператор `else`, например:
```swift
var niceWeather = true
if niceWeather {
    // play outside
} else {
    // stay home
}
```

---

Допустим, у нас есть ещё одно условие для проверки, как в этом примере:
```swift
var num = 3
if num == 2 {
    print("the number is 2")
} else if num == 3 {
    print("the number is 3")
} else {
    print("do something else")
}
```
и результат выполнения этого кода -- `the number is 3`.
Сначала проверяем, равно ли число 2 -- это ложь.
Поэтому переходим ко второму оператору и проверяем, равна ли `num` 3 -- это истина, и мы выполняем следующий блок кода, выводя `the number is 3`

---

Мы можем добавить сколько угодно операторов `else if`, ограничений нет
```swift
var num = 4
if (num == 2) {
    print("the number is 2")
} else if num == 3 {
    print("the number is 3")
} else if (num == 4) {
    print("the number is 4")
} else if (num == 5) {
    print("the number is 5")
} else if (num == 6) {
    print("the number is 6")
}
```
и результат выполнения этого кода -- `the number is 4`.

---

Мы также можем вложить условный оператор (`if`, `else if` или `else`) внутрь другого условного оператора, чтобы создать более сложную структуру.
```swift
var num = 4
if num < 3 {
    print("the number is lower than 3")
} else {
    if num == 3 {
        print("the number is 3")
    } else if num == 4 {
        print("the number is 4")
    } else {
        print("the number is greather than 4")
    }
}
```
и результат выполнения этого кода -- `the number is 4`.

---

Тернарный условный оператор -- это специальный оператор с тремя частями, который имеет форму `вопрос ? ответ1 : ответ2`.
Это сокращённый способ вычисления одного из двух выражений в зависимости от того, истинен `вопрос` или ложен.
Если `вопрос` истинен, он вычисляет `ответ1` и возвращает его значение; в противном случае он вычисляет `ответ2` и возвращает его значение.
```swift
let a = 10, b = 20, c: Int
if (a < b) {
    c = a
} else {
    c = b
}
print(c)
```
Сокращённый код для приведённого выше:
```swift
let a = 10, b = 20, c: Int
c = a < b ? a : b
print(c)
```
`c` присваивается значение `a`, потому что условие `a < b` было истинным

---

_Оператор объединения с nil_ `a ?? b` разворачивает опционал `a`, если он содержит значение, или возвращает значение по умолчанию `b`, если `a` равно `nil`.
Выражение `a` всегда имеет опциональный тип.
Выражение `b` должно соответствовать типу, хранящемуся внутри `a`.
Оператор объединения с nil -- это сокращение для следующего кода:
```swift
a != nil ? a! : b
```
