# --description--

_Оператор объединения с nil_ `a ?? b` разворачивает опционал `a`, если он содержит значение, или возвращает значение по умолчанию `b`, если `a` равно `nil`.
Выражение `a` всегда имеет опциональный тип.
Выражение `b` должно соответствовать типу, хранящемуся внутри `a`.
Оператор объединения с nil -- это сокращение для следующего кода:
```swift
a != nil ? a! : b
```
