# --instructions--

В чём __проблема__ этого фрагмента кода?
```swift
print(greet(name: "Mathew"))
/// This function greets to the person passed in as argument
func greet(name: String) -> String {
    return "Hello, \(name). Good morning!"
}
print(greet(name: "Alex"))
```

# --answers--

- Нельзя использовать greet() до её определения
- Нельзя использовать greet() дважды
- greet() ничего не возвращает
- Нельзя использовать name как параметр
