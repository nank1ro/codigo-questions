Переменные — это контейнеры для хранения значений данных.
Каждая переменная в Swift является объектом.
Чтобы создать переменную, нужно дать ей **имя**, помня, что оно не должно содержать пробелов.
Переменная создаётся в момент, когда вы впервые присваиваете ей значение.
В Swift константы объявляются с помощью ключевого слова `let`, а переменные — с помощью ключевого слова `var`.
Значение константы нельзя изменить после установки, тогда как переменной можно присвоить другое значение в будущем.
Пример создания переменной с именем `x`:
```swift
var x = 1
```
Таким образом мы присвоили значение `1` переменной с именем `x`.
Если мы выведем переменную `x`, мы получим число `1`:
```swift
print(x) // выводит 1
```

---

Переменные называются так потому, что хранимое в них значение может изменяться.
Мы можем обновить `x`, используя `=` и присвоив новое значение.
```swift
var x = 1
print(x) // выводит 1
x = 2
print(x) // выводит 2
```

---

Мы также можем присваивать переменным значения других переменных. Здесь мы можем присвоить переменной `y` значение `x`
```swift
var x = 5
var y = x
print(y) // выводит 5
```

---

Когда мы обновляем переменную, она забывает своё предыдущее значение. Здесь мы можем вывести переменную `x` дважды и увидеть, как её значение обновляется.
```swift
var x = 5
print(x) // выводит 5
x = 10
print(x) // выводит 10
```

---

В Swift строковые переменные можно объявлять только с использованием двойных кавычек:
```swift
let x = "May"
```

---

Если мы хотим использовать имя переменной из нескольких слов, мы используем **camelCase**.
Это практика написания фраз, в которых каждое слово в середине фразы начинается с заглавной буквы
