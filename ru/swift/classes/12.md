`legs` должно быть внутри класса `Animal`, а `breed` — внутри класса `Dog`

# --solutions--

```swift
class Animal {
    var legs = 4
}
class Dog: Animal {
    var breed: String?
}
let max = Dog()
max.breed = "Chihuahua"
print(max.breed!)
```

# --output--

Chihuahua
