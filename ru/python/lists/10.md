# --description--

В Python можно перебирать список очень простым способом, используя ключевые слова `for..in`:
```python
>>> numbers = [1, 2, 3]
>>> for num in numbers:
>>>     print(num)
1
2
3
```
Имя переменной после ключевого слова `for` будет последовательно принимать значение каждого элемента списка.
