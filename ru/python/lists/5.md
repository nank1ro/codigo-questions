# --description--

Список не обязан иметь фиксированную длину.
Вы можете добавлять элементы в конец списка в любой момент!
Для добавления элемента в список используется ключевое слово `append`:
```python
>>> letters = ["a", "b"]
>>> letters.append("c")
>>> print(letters)
['a', 'b', 'c']
```
