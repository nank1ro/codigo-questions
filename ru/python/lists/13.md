# --description--

Аналогично можно преобразовать список в кортеж

# --instructions--

Используйте функцию `tuple()`, чтобы преобразовать список в кортеж

# --seed--

```python
list1 = ["dog", "bear", "cat"]
tuple1 = [/]
print(tuple1)
```

# --answers--

- tuple[list1]
- tuple(list1)

# --solutions--

```python
list1 = ["dog", "bear", "cat"]
tuple1 = tuple(list1)
print(tuple1)
```

# --output--

('dog', 'bear', 'cat')
