В чём __проблема__ с этим фрагментом кода?
```python

def greet(name):
    """This function greets to the person passed in as a parameter"""
    print(f"Hello, {name}. Good morning!")

greet('Alex')
```

# --answers--

- Мы не можем использовать greet() до её определения
- Мы не можем использовать greet() дважды
- greet() ничего не возвращает
- Мы не можем использовать name как параметр
