"__Hello, World!__" — это традиционная первая программа для начала программирования на новом языке.
Мы используем функцию `print()` для вывода данных на стандартное устройство вывода (экран).
Чтобы вывести `Hello, World!` на экран с помощью Python, можно написать
```python
print("Hello, World!")
```
