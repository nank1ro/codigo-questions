# --description--

Продолжим с оператором сравнения **больше или равно** `>=`.
Он возвращает **логическое значение** (`True` или `False`), указывающее, больше или равно ли одно выражение другому, например:
```python
>>> 2 >= 2
True
>>> 3 >= 2
True
>>> 3 >= 4
False
```
