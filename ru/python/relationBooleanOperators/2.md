# --description--

Продолжим с оператором сравнения **не равно** `!=`.
Он возвращает **логическое значение** (`True` или `False`), указывающее, **НЕ** равны ли два выражения, например:
```python
>>> 2 != 2
False
>>> 2 != 3
True
```
Он является полной противоположностью оператора *равно*
