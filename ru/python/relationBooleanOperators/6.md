# --description--

Продолжим с оператором сравнения **меньше или равно** `<=`.
Он возвращает **логическое значение** (`True` или `False`), указывающее, меньше или равно ли одно выражение другому, например:
```python
>>> 2 <= 2
True
>>> 3 <= 2
False
>>> 3 <= 4
True
```
