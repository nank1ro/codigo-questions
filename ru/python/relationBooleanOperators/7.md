# --description--

Теперь рассмотрим **логические** операторы, начнём с первого — `and`.
Он возвращает первый операнд, который вычисляется как *False*, или последний, если все равны *True*.
```python
>>> 2 == 2 and 2 == 3
False
>>> 1 == 1 and 1 == 1.0
True
```
