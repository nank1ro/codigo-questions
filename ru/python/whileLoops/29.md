# --instructions--

Что не так с этим кодом?
```python
while is_loading:
    print("Loading...")
    is_loading = False
```

# --answers--

- is_loading не является допустимым именем переменной
- Цикл while содержит оператор print внутри блока кода
- Блок кода имеет неправильный отступ
