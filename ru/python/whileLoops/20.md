# --instructions--

Как мы узнаём, что цикл `while` пропускает свой блок кода?
```python
is_loading = False

while is_loading:
    print("Loading...")
```

# --answers--

- Переменная в условии сравнивается с True
- Переменная в условии сравнивается с False

# --solutions--

- Переменная в условии сравнивается с True
