# --instructions--

Что вызывает сбой этой программы?
```python
while True:
    print("repeat")
```

# --answers--

- В коде есть бесконечный цикл
- Ничто не вызывает сбой этого кода
- Цикл while имеет неправильный отступ
- Блок кода имеет неправильный отступ
