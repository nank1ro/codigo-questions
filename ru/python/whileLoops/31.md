# --instructions--

Что делает `counter += 1`?
```python
counter = 1

while counter < 5:
    print(counter)
    counter += 1
```

# --answers--

- Увеличивает counter на один при каждом выполнении блока кода цикла
- Создаёт переменную counter
- Увеличивает counter на два каждый раз
- Увеличивает counter на три каждый раз
