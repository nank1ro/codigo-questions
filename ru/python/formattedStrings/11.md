# --instructions--

Как отобразить числовые значения с помощью форматированных строк?

# --answers--

- Мы помещаем их между фигурными скобками {}
- Мы не можем отображать числовые значения с форматированными строками
- Мы помещаем их между квадратными скобками []
- Мы помещаем их между круглыми скобками ()
