# --instructions--

Что не так с этим кодом?
```python
pieces = 5
label = f"we have {pieces} pieces"
```

# --answers--

- Мы не можем сохранять форматированные строки в переменных
- Отсутствует символ f
- Мы не можем использовать переменные внутри {}
- С кодом всё в порядке
