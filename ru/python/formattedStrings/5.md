# --description--

Далее мы добавляем значение другого типа в фигурные скобки, чтобы оно отобразилось в одном операторе print. Как здесь, с `{5}`

# --instructions--

Заполните пустое место, чтобы получить такой вывод
```python
5 friends
```

# --seed--

```python
print(f" friends")
```

# --solutions--

```python
print(f"{5} friends")
```

# --output--

5 friends
