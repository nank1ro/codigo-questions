# --instructions--

Что не так с этим кодом?
```python
print("{10} friends"f)
```

# --answers--

- Символ f должен стоять перед строкой
- Значение между фигурными скобками не является числом
- В строке отсутствуют кавычки " "
- Значение внутри фигурных скобок должно быть именем переменной
