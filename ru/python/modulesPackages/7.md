# --instructions--

Пакеты — это просто папки, содержащие файл `__init__.py`.
Что произойдёт, если мы забудем добавить такой файл?

# --answers--

- Python не распознает папку как пакет
- Мы всё ещё сможем импортировать пакеты, но не подпакеты
- Мы всё ещё сможем импортировать функции
