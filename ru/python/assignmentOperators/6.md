# --description--

Рассмотрим оператор **присваивания возведения в степень** `**=`.
Он возводит переменную в указанную степень и присваивает результат переменной.
Таким образом, следующие записи полностью эквивалентны
```python
num = num ** 5
# is equal to
num **= 5
```
