# --description--

Так же, как и присваивание сложения, существует **присваивание вычитания** `-=`.
Функциональность та же, единственное отличие — выполняется вычитание.
Таким образом, следующие записи полностью эквивалентны
```python
num = num - 5
# is equal to
num -= 5
```
