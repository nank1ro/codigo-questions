# --description--

Когда конструктору подкласса нужно вызвать конструктор суперкласса, используется ключевое слово `super` в **списке инициализаторов**:

```dart
class Vehicle {
  String brand;
  Vehicle(this.brand);
}

class Car extends Vehicle {
  int doors;
  Car(String brand, this.doors) : super(brand);
}
```

`super(brand)` передаёт аргумент `brand` конструктору класса `Vehicle`.
