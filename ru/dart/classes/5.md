# --description--

**Метод** — это функция, определённая внутри класса. Методы описывают поведение объекта:

```dart
class Animal {
  String name;

  Animal(this.name);

  void speak() {
    print('$name издаёт звук.');
  }
}
```

Метод вызывается на объекте с помощью точечной нотации: `dog.speak()`.
