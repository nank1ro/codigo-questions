# --description--

**Абстрактный класс** — это класс, который нельзя инстанцировать напрямую. Он используется как основа, определяющая контракт — методы, которые должны быть реализованы подклассами. Абстрактные методы обозначаются отсутствием тела:

```dart
abstract class Shape {
  double area(); // абстрактный метод — без тела
}

class Circle extends Shape {
  double radius;
  Circle(this.radius);

  @override
  double area() => 3.14159 * radius * radius;
}
```

Попытка напрямую инстанцировать `Shape()` приводит к ошибке.
