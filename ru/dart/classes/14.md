# --description--

**Статические члены** принадлежат самому классу, а не конкретному экземпляру. Они объявляются с помощью ключевого слова `static` и доступны непосредственно через имя класса:

```dart
class MathHelper {
  static const double pi = 3.14159;

  static double circleArea(double r) => pi * r * r;
}

// доступ без создания объекта:
print(MathHelper.pi);
print(MathHelper.circleArea(5));
```

Статические поля и методы разделяются всеми экземплярами.
