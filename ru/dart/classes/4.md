# --description--

Dart предоставляет сокращённый синтаксис конструкторов, который автоматически присваивает параметры переменным экземпляра. Вместо:

```dart
class Point {
  int x;
  int y;

  Point(int x, int y) {
    this.x = x;
    this.y = y;
  }
}
```

Можно написать:

```dart
class Point {
  int x;
  int y;

  Point(this.x, this.y);
}
```

Эта более короткая форма называется **инициализирующими формальными параметрами**.

# --instructions--

Какой из вариантов правильно использует сокращённый синтаксис инициализирующих формальных параметров?

# --answers--

- `Point(this.x, this.y);`
- `Point(x, y) { self.x = x; self.y = y; }`
- `Point(x, y) => x + y;`
- `Point.init(x, y);`
