# --description--

**Наследование** позволяет классу (**подклассу**) расширить другой класс (**суперкласс**) и повторно использовать его поля и методы. Для этого используется ключевое слово `extends`:

```dart
class Animal {
  String name;
  Animal(this.name);

  void speak() {
    print('...');
  }
}

class Dog extends Animal {
  Dog(String name) : super(name);

  @override
  void speak() {
    print('Woof!');
  }
}
```

`Dog` наследует `name` от `Animal` и переопределяет метод `speak`.
