# --description--

**Переопределение методов** позволяет подклассу предоставить собственную реализацию метода, уже существующего в суперклассе. Для этого используется аннотация `@override`:

```dart
class Shape {
  double area() => 0;
}

class Circle extends Shape {
  double radius;
  Circle(this.radius);

  @override
  double area() => 3.14159 * radius * radius;
}
```

Аннотация `@override` сообщает Dart (и другим разработчикам), что метод суперкласса намеренно заменяется.
