# --description--

**Геттер** — это специальный метод, который читает вычисляемое или приватное значение и выглядит как обращение к свойству. Он определяется с помощью ключевого слова `get`:

```dart
class Circle {
  double radius;

  Circle(this.radius);

  double get area => 3.14159 * radius * radius;
}
```

К геттеру обращаются как к полю: `circle.area` (без скобок).
