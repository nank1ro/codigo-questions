# --description--

Dart поддерживает **именованные конструкторы**, позволяющие определить дополнительные способы создания объекта. Именованные конструкторы записываются как `ИмяКласса.имяКонструктора`:

```dart
class Point {
  double x;
  double y;

  Point(this.x, this.y);

  Point.origin()
      : x = 0,
        y = 0;
}
```

Объект в начале координат создаётся следующим образом: `var p = Point.origin();`
