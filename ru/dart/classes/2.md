# --description--

Класс может иметь **переменные экземпляра** (также называемые полями), которые хранят данные каждого объекта. Они объявляются внутри тела класса:

```dart
class Animal {
  String name;
  int age;
}
```

Каждый объект, созданный из этого класса, будет иметь собственные поля `name` и `age`.
