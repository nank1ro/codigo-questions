# --description--

**Конструктор** — это специальный метод, который вызывается при создании (инстанцировании) объекта класса. Конструктор имеет то же имя, что и класс:

```dart
class Animal {
  String name;

  Animal(String name) {
    this.name = name;
  }
}
```

Объект создаётся с помощью ключевого слова `new` (необязательного в Dart) или непосредственно по имени класса:

```dart
var dog = Animal('Rex');
```
