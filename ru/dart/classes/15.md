# --description--

**Фабричный конструктор** использует ключевое слово `factory` и позволяет управлять созданием объектов — например, возвращать кэшированный экземпляр или подтип:

```dart
class Logger {
  static final Logger _instance = Logger._internal();

  factory Logger() => _instance;

  Logger._internal();
}
```

Каждый вызов `Logger()` возвращает один и тот же экземпляр (паттерн синглтон).

# --instructions--

Создайте класс `Counter` с приватным полем `_count`, инициализированным значением `0`. Добавьте метод экземпляра `increment()`, увеличивающий `_count` на `1`, и геттер `count`, возвращающий `_count`. Используйте фабричный конструктор `Counter.start()`, который всегда возвращает новый объект `Counter`, инициализированный с `_count = 0` (через приватный именованный конструктор `Counter._internal()`).

```dart

class Counter {
  int _count = 0;

  factory Counter.start() => Counter._internal();

  void increment() {
    _count++;
  }

  int get count => _count;
}
```
