# --description--

**Класс** — это шаблон для создания объектов. В Dart класс определяется с помощью ключевого слова `class`, после которого указывается имя класса и пара фигурных скобок:

```dart
class Animal {
  // поля и методы
}
```

По соглашению имена классов используют нотацию **PascalCase** (каждое слово начинается с заглавной буквы).
