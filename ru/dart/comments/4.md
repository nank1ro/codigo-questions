# --description--

Вы также можете создавать блоки комментариев, или _многострочные_ комментарии.
_Многострочные_ комментарии начинаются с `/*` и заканчиваются `*/`.
```dart
/*
This is a multi-line
comment
*/
```

# --instructions--

Дополните код, чтобы добавить _многострочный_ комментарий.
