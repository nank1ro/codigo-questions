Цикл `while` повторяет блок кода до тех пор, пока условие равно `true`.

```dart
int i = 0;
while (i < 3) {
  print(i);
  i++;
}
```

Цикл проверяет условие **перед** каждой итерацией. Когда `i < 3` становится `false`, цикл останавливается.

---

Переменная-счётчик управляет количеством выполнений цикла `while`.

```dart
int count = 0;
while (count < 5) {
  count++;
}
print(count); // 5
```

Счётчик начинается с `0`, и цикл увеличивает его при каждой итерации, пока `count < 5` не станет `false`.

---

Цикл `do-while` похож на цикл `while`, но он **всегда выполняет тело хотя бы один раз** — условие проверяется *после* каждой итерации.

```dart
int i = 0;
do {
  print(i);
  i++;
} while (i < 3);
```

Даже если условие `false` с самого начала, тело выполнится один раз.

---

Условие цикла `while` может содержать любое булево выражение. Как только условие принимает значение `false`, цикл завершается.

```dart
int n = 100;
while (n > 1) {
  n ~/= 2;
}
print(n); // 0
```

Здесь `n` делится пополам при каждой итерации с помощью целочисленного деления (`~/`).

---

Ключевое слово `break` немедленно завершает цикл, независимо от его условия.

```dart
int i = 0;
while (true) {
  if (i == 3) break;
  print(i);
  i++;
}
```

Без `break` цикл `while (true)` выполнялся бы бесконечно. Здесь он останавливается, когда `i` достигает `3`.
