# --description--

Цикл `do-while` похож на цикл `while`, но он **всегда выполняет тело хотя бы один раз** — условие проверяется *после* каждой итерации.

```dart
int i = 0;
do {
  print(i);
  i++;
} while (i < 3);
```

Даже если условие `false` с самого начала, тело выполнится один раз.
