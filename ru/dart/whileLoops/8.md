# --instructions--

Что гарантирует цикл `do-while` по сравнению с циклом `while`?

# --answers--

- Тело выполняется хотя бы один раз, даже если условие изначально `false`
- Тело выполняется не более одного раза
- Условие проверяется до выполнения тела
- Цикл выполняется бесконечно, если не использовать `break`
