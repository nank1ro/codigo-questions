# --description--

Оператор **присваивания** `=` сохраняет значение в переменной.

```dart
int x = 5;
String name = 'Alice';
```

После присваивания переменная хранит это значение до тех пор, пока оно не будет переопределено.

# --instructions--

Заполните код, чтобы присвоить значение `42` переменной `answer`.

# --seed--

```dart
void main() {
  int answer [/] 42;
  print(answer);
}
```
