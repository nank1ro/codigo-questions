Оператор **присваивания** `=` сохраняет значение в переменной.

```dart
int x = 5;
String name = 'Alice';
```

После присваивания переменная хранит это значение до тех пор, пока оно не будет переопределено.

---

Оператор **сложения с присваиванием** `+=` добавляет значение к переменной и сохраняет результат обратно в переменной.

```dart
int score = 10;
score += 5; // score теперь равна 15
```

Это сокращение для `score = score + 5`.

---

Оператор **вычитания с присваиванием** `-=` вычитает значение из переменной и сохраняет результат обратно в переменной.

```dart
int lives = 3;
lives -= 1; // lives теперь равна 2
```

Это сокращение для `lives = lives - 1`.

---

Оператор **умножения с присваиванием** `*=` умножает переменную на значение и сохраняет результат обратно в переменной.

```dart
int side = 4;
side *= 3; // side теперь равна 12
```

Это сокращение для `side = side * 3`.

---

Операторы присваивания объединяют арифметическую операцию с присваиванием в одном шаге.

Например, `x += 3` — это сокращение для `x = x + 3`.
