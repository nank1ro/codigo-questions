# --description--

Операторы присваивания объединяют арифметическую операцию с присваиванием в одном шаге.

Например, `x += 3` — это сокращение для `x = x + 3`.

# --instructions--

Что делает `x += 5`, когда `x` в настоящее время равна `10`?

# --answers--

- Устанавливает `x` на `15`
- Устанавливает `x` на `5`
- Устанавливает `x` на `10`
- Устанавливает `x` на `50`
