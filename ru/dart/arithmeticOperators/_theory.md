Оператор **сложения** `+` складывает два значения вместе.

```dart
int a = 3;
int b = 4;
int sum = a + b; // 7
```

Вы можете использовать `+` со значениями `int` или `double`.

---

Оператор **вычитания** `-` вычитает одно значение из другого.

```dart
int a = 10;
int b = 4;
int diff = a - b; // 6
```

Результат может быть отрицательным, если второй операнд больше первого.

---

Оператор **умножения** `*` умножает два значения вместе.

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

Умножение `int` на другой `int` дает `int`. Умножение с `double` дает `double`.

---

Оператор **деления** `/` делит одно значение на другое. В Dart `/` **всегда возвращает `double`**, даже когда результат — целое число.

```dart
int a = 10;
int b = 4;
double result = a / b; // 2.5

double exact = 10 / 2; // 5.0  (не 5!)
```

Всегда используйте переменную `double` для сохранения результата `/`.

---

Оператор **целого деления** `~/` делит два числа и **усекает** результат до целого числа (отбрасывает десятичную часть).

```dart
int a = 10;
int b = 3;
int result = a ~/ b; // 3  (не 3.333...)
```

Используйте `~/` когда вам нужно целое частное без остатка.
