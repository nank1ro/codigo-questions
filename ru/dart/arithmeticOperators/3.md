# --description--

Оператор **умножения** `*` умножает два значения вместе.

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

Умножение `int` на другой `int` дает `int`. Умножение с `double` дает `double`.

# --instructions--

Завершите код, чтобы умножить `8` на `6` и сохраните результат в `product`.
