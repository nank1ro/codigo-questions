# --description--

Оператор **вычитания** `-` вычитает одно значение из другого.

```dart
int a = 10;
int b = 4;
int diff = a - b; // 6
```

Результат может быть отрицательным, если второй операнд больше первого.

# --instructions--

Завершите код, чтобы вычесть `9` из `20` и сохраните результат в `diff`.

# --seed--

  print(diff);
