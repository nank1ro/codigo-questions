# --description--

Dart следует стандартному математическому **порядку операций** (приоритету): умножение и деление оцениваются перед сложением и вычитанием.

```dart
int result = 2 + 3 * 4; // 14, не 20
```

Используйте скобки, чтобы изменить порядок:

```dart
int result = (2 + 3) * 4; // 20
```
