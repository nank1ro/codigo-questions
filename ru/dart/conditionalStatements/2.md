# --description--

Можно использовать `print()` внутри блока `if`, чтобы вывести сообщение при выполнении условия.

```dart
bool isRaining = true;
if (isRaining) {
  print('Take an umbrella');
}
```

Если `isRaining` равно `false`, ничего не выводится.
