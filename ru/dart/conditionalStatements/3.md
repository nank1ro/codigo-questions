# --description--

Оператор **`if-else`** выполняет блок `if`, когда условие равно `true`, и блок `else`, когда оно равно `false`.

```dart
int temperature = 10;
if (temperature > 20) {
  print('Warm');
} else {
  print('Cold');
}
// prints: Cold
```

Всегда выполняется ровно одна из двух ветвей.
