# --description--

**`else if`** позволяет последовательно проверять несколько условий. Выполняется первая ветвь, условие которой равно `true`, остальные пропускаются.

```dart
int score = 75;
if (score >= 90) {
  print('A');
} else if (score >= 70) {
  print('B');
} else {
  print('C');
}
// prints: B
```
