# --description--

Оператор **`if`** выполняет блок кода только тогда, когда условие равно `true`.

```dart
int age = 20;
if (age >= 18) {
  print('Adult');
}
```

Код внутри фигурных скобок выполняется только тогда, когда условие `age >= 18` равно `true`.
