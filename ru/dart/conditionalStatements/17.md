# --solutions--

```dart
String trafficLight(String color) {
  switch (color) {
    case 'green':
      return 'Go';
    case 'yellow':
      return 'Slow down';
    case 'red':
      return 'Stop';
    default:
      return 'Invalid';
  }
}

void main() {
  print(trafficLight('green'));
}
```
