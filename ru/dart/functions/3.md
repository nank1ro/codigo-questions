# --description--

Функции могут принимать **параметры** — значения, передаваемые при вызове функции. Параметры объявляются внутри скобок с указанием их типа и имени:

```dart
int square(int n) {
  return n * n;
}

void main() {
  print(square(4)); // prints 16
}
```

Каждый параметр имеет тип (`int`) и имя (`n`). Несколько параметров разделяются запятыми.
