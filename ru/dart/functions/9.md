# --instructions--

Напишите функцию `greetUser`, которая принимает параметр `String` с именем `name` со **значением по умолчанию** `"World"` и возвращает строку `"Hello, <name>!"`.

Пример:
```dart
print(greetUser());          // Hello, World!
print(greetUser("Alice"));   // Hello, Alice!
```
