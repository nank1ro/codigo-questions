# --instructions--

Напишите **рекурсивную** функцию `factorial`, которая принимает параметр `int` с именем `n` и возвращает `n!` (факториал `n`).

Факториал `n` определяется следующим образом:
- `factorial(0) = 1`
- `factorial(n) = n * factorial(n - 1)` для `n > 0`

Пример:
```dart
print(factorial(5)); // 120
print(factorial(3)); // 6
```

# --solutions--

```dart
int factorial(int n) {
  if (n == 0) return 1;
  return n * factorial(n - 1);
}
```
