# --description--

Функции могут **возвращать значение** в место вызова. Тип возвращаемого значения объявляется перед именем функции, а ключевое слово `return` используется для отправки значения обратно:

```dart
int getAge() {
  return 25;
}
```

Тип перед именем функции (`int`) сообщает Dart, какого рода значение вернёт функция. Можно использовать `String`, `int`, `double`, `bool` и другие.
