# --description--

Dart поддерживает **стрелочные функции** с синтаксисом `=>`. Когда тело функции представляет собой одно выражение, его можно сократить с помощью `=>` вместо `{ return ...; }`:

```dart
// Обычная функция
int double(int n) {
  return n * 2;
}

// Стрелочная функция — тот же результат
int double(int n) => n * 2;
```

Стрелочные функции делают код более лаконичным. `=>` заменяет как фигурные скобки, так и ключевое слово `return`.
