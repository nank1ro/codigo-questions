# --description--

**Функция** — это повторно используемый блок кода, который выполняет определённую задачу. В Dart функция определяется с указанием типа возвращаемого значения, имени и пары скобок `()`.

Когда функция не возвращает никакого значения, её тип возвращаемого значения — `void`:

```dart
void sayHello() {
  print("Hello!");
}
```

Функция вызывается написанием её имени с `()`:

```dart
sayHello(); // prints Hello!
```
