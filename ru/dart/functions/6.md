# --instructions--

Что означает тип возвращаемого значения `void` в функции Dart?

```dart
void sayHi() {
  print("Hi!");
}
```

# --answers--

- Функция возвращает число ноль
- Функция не возвращает никакого значения
- Функция возвращает пустую строку
- Функция может вернуть любой тип
