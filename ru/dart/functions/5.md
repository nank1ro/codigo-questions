# --description--

Dart поддерживает **именованные параметры** — параметры, заключённые в фигурные скобки `{}`. Именованные параметры должны вызываться по имени и могут иметь значения по умолчанию или быть помечены как `required`:

```dart
void printInfo({required String name, int age = 0}) {
  print("$name is $age years old");
}

void main() {
  printInfo(name: "Alice", age: 30);
  // prints Alice is 30 years old
}
```

Именованные параметры улучшают читаемость кода, особенно когда функция имеет много параметров.
