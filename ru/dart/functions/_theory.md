**Функция** — это повторно используемый блок кода, который выполняет определённую задачу. В Dart функция определяется с указанием типа возвращаемого значения, имени и пары скобок `()`.

Когда функция не возвращает никакого значения, её тип возвращаемого значения — `void`:

```dart
void sayHello() {
  print("Hello!");
}
```

Функция вызывается написанием её имени с `()`:

```dart
sayHello(); // prints Hello!
```

---

Функции могут **возвращать значение** в место вызова. Тип возвращаемого значения объявляется перед именем функции, а ключевое слово `return` используется для отправки значения обратно:

```dart
int getAge() {
  return 25;
}
```

Тип перед именем функции (`int`) сообщает Dart, какого рода значение вернёт функция. Можно использовать `String`, `int`, `double`, `bool` и другие.

---

Функции могут принимать **параметры** — значения, передаваемые при вызове функции. Параметры объявляются внутри скобок с указанием их типа и имени:

```dart
int square(int n) {
  return n * n;
}

void main() {
  print(square(4)); // prints 16
}
```

Каждый параметр имеет тип (`int`) и имя (`n`). Несколько параметров разделяются запятыми.

---

Dart поддерживает **стрелочные функции** с синтаксисом `=>`. Когда тело функции представляет собой одно выражение, его можно сократить с помощью `=>` вместо `{ return ...; }`:

```dart
// Обычная функция
int double(int n) {
  return n * 2;
}

// Стрелочная функция — тот же результат
int double(int n) => n * 2;
```

Стрелочные функции делают код более лаконичным. `=>` заменяет как фигурные скобки, так и ключевое слово `return`.

---

Dart поддерживает **именованные параметры** — параметры, заключённые в фигурные скобки `{}`. Именованные параметры должны вызываться по имени и могут иметь значения по умолчанию или быть помечены как `required`:

```dart
void printInfo({required String name, int age = 0}) {
  print("$name is $age years old");
}

void main() {
  printInfo(name: "Alice", age: 30);
  // prints Alice is 30 years old
}
```

Именованные параметры улучшают читаемость кода, особенно когда функция имеет много параметров.
