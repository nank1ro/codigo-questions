# --instructions--

В чём разница между константой __времени компиляции__ и константой __времени выполнения__?

# --answers--

- Константы __времени компиляции__ содержат фиксированные значения, известные до запуска программы, тогда как константы __времени выполнения__ получают фиксированное значение только во время выполнения программы
- Константы __времени компиляции__ содержат фиксированные значения, известные после запуска программы, тогда как константы __времени выполнения__ получают фиксированное значение до запуска программы
- Между ними нет никакой разницы
