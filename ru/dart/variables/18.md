# --instructions--

Всегда ли необходимо указывать тип переменной в Dart?

# --answers--

- Тип переменной должен быть указан всегда
- Dart поддерживает __вывод типов__, поэтому способен определить его самостоятельно
- Dart не поддерживает типы
