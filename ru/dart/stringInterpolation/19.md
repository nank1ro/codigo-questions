# --instructions--

Используйте `${}`, чтобы вставить число `500` в пустое место с помощью интерполяции строк

# --seed--

```dart
void main() {
    print("There are  items");
}
```

# --solutions--

```dart
void main() {
    print("There are ${500} items");
}
```

# --output--

There are 500 items
