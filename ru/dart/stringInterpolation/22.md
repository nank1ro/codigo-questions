# --instructions--

Что не так с этим кодом?
```dart
const manchesterUnited = 2;
const barcelona = 2
println("The final score is $manchesterUnited : $barcelona")
```

# --answers--

- С кодом всё в порядке
- manchesterUnited и barcelona не имеют значений
- Мы не можем использовать переменные внутри фигурных скобок
- Мы не можем использовать $ дважды в одной строке
