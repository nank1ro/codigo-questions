# --description--

Если то, что следует после знака `$`, не распознаётся как идентификатор программы, вы столкнётесь с ошибкой

# --instructions--

Запустите, чтобы увидеть ошибку

# --seed--

```dart
void main() {
    print("$5 friends");
}
```

# --solutions--

```dart
void main() {
    print("$5 friends");
```
