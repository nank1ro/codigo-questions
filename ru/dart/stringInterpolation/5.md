# --description--

Если то, что следует после знака `$`, не распознаётся как идентификатор программы, Dart выдаст ошибку компиляции: после `$` в строке всегда должен идти идентификатор или выражение в фигурных скобках (`${}`).

# --instructions--

Что произойдёт, если попытаться запустить этот код?

```dart
void main() {
    print("$5 friends");
}
```

# --answers--

- Код не скомпилируется, так как `$5` — это некорректная интерполяция строки
- Будет выведено `$5 friends`
- Будет выведено `5 friends`
- Ничего не будет выведено
