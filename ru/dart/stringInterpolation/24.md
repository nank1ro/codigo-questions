# --instructions--

Что не так с этим кодом?
```dart
const pieces = 5;
const label = "we have $pieces pieces";
```

# --answers--

- Мы не можем сохранять шаблоны строк в переменных
- Отсутствуют символы $
- Мы не можем использовать переменные внутри ${}
- С кодом всё в порядке
