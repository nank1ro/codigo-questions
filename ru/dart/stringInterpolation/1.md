# --description--

_Интерполяция_ строк — это программный способ создания строки.
В Dart мы можем использовать знак `+` (конкатенация) для отображения двух или более строк вместе, например:
```dart
print("Hello " + "Dart!");
// prints "Hello Dart!"
```
