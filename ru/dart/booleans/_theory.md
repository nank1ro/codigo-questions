__Булево значение__ — это тип данных, который может принимать только одно из двух значений: `true` или `false`.
В Dart булев тип объявляется ключевым словом `bool`.

```dart
bool isRaining = true;
```

Переменная `isRaining` хранит значение `true`, что означает, что в настоящее время идет дождь.
Булевы значения всегда пишутся в нижнем регистре: `true` и `false`.

---

Вы можете вывести булево значение с помощью `print()`, как и любую другую переменную.
Когда вы выводите булево значение, Dart отображает текст `true` или `false`.

```dart
bool isSunny = true;
print(isSunny); // true
```

---

Булева переменная также может хранить значение `false`.
Используйте `false`, когда что-то не является верным.

```dart
bool isLoggedIn = false;
print(isLoggedIn); // false
```

Как и `true`, `false` должно быть написано в нижнем регистре.

---

__Оператор отрицания__ `!` (также называемый "not" или "не") переворачивает булево значение.
Применение `!` к `true` дает `false`, а применение `!` к `false` дает `true`.

```dart
bool isActive = true;
print(!isActive); // false
```

Это полезно, когда вы хотите проверить противоположность условия.

---

Булевы значения используются во всем программировании для представления условий и принятия решений.
Всякий раз, когда вашей программе нужно выбирать между двумя возможностями, вовлечено булево значение.

Примеры:
- Вошел ли пользователь в систему? (`isLoggedIn`)
- Дверь открыта? (`isDoorOpen`)
- Был ли заказ отправлен? (`isShipped`)
