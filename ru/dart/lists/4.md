# --description--

Свойство `.length` возвращает количество элементов в списке:

```dart
var scores = [95, 87, 72, 100];
print(scores.length); // 4
```

Пустой список имеет длину `0`:

```dart
var empty = [];
print(empty.length); // 0
```
