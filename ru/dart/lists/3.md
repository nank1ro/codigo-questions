# --description--

Метод `.add()` добавляет один элемент в **конец** списка:

```dart
var nums = [1, 2, 3];
nums.add(4);
print(nums); // [1, 2, 3, 4]
```

Обратите внимание, что `.add()` изменяет список **на месте** и возвращает `void`.
