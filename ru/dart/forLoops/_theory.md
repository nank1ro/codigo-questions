Цикл `for` повторяет блок кода заданное количество раз. Базовый синтаксис:

```dart
for (initialization; condition; update) {
  // body
}
```

**initialization**: выполняется один раз перед началом цикла (например, `int i = 0`)

**condition**: проверяется перед каждой итерацией; цикл останавливается, когда значение ложно

**update**: выполняется после каждой итерации (например, `i++`)

---

Переменную цикла можно использовать внутри тела для отслеживания текущего счётчика. Например, накопление суммы:

```dart
int total = 0;
for (int i = 1; i <= 5; i++) {
  total += i;
}
```

После цикла `total` равно 15.

---

Цикл `for` может считать **в обратном направлении**, используя декремент (`i--`) и условие `>=`:

```dart
for (int i = 5; i >= 1; i--) {
  print(i); // 5, 4, 3, 2, 1
}
```

Это полезно для обратного отсчёта или обратной итерации.

---

Цикл `for-in` перебирает каждый элемент коллекции без необходимости использовать индекс:

```dart
List<String> fruits = ['apple', 'banana', 'cherry'];
for (var fruit in fruits) {
  print(fruit);
}
```

На каждой итерации следующий элемент присваивается переменной цикла (здесь `fruit`).

---

Инструкция `break` немедленно завершает цикл при выполнении условия:

```dart
for (int i = 0; i < 10; i++) {
  if (i == 5) break; // stops at 5
  print(i);
}
```

Это удобно при поиске значения — цикл останавливается сразу после его нахождения.
