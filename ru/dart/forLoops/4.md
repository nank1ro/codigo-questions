# --description--
Цикл `for-in` перебирает каждый элемент коллекции без необходимости использовать индекс:

```dart
List<String> fruits = ['apple', 'banana', 'cherry'];
for (var fruit in fruits) {
  print(fruit);
}
```

На каждой итерации следующий элемент присваивается переменной цикла (здесь `fruit`).
