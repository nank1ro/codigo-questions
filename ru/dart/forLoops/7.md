# --instructions--
Что делает цикл `for-in` в Dart?

# --answers--

- Перебирает каждый элемент коллекции
- Итерирует фиксированное количество раз с использованием индекса
- Повторяет код, пока логическое условие не станет истинным
- Выполняет код только если коллекция не пуста
