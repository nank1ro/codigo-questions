# --description--

Оператор **ИЛИ** `||` объединяет два логических выражения. Он возвращает `true`, если **хотя бы одна** сторона имеет значение `true`. Он возвращает `false` только когда обе стороны имеют значение `false`.

```dart
bool a = true;
bool b = false;
bool result = a || b; // true

bool c = false;
bool d = false;
bool result2 = c || d; // false
```

# --instructions--

Объявите переменную `bool` верхнего уровня с именем `isPassed`, которая имеет значение `true` только если переменная `score` больше `70` ИЛИ `bonus` больше `10`. Используйте переменные `score` = `65` и `bonus` = `15`.
