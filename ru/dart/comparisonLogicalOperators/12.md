# --instructions--

Объявите две переменные `int` верхнего уровня: `num1` со значением `20` и `num2` со значением `15`. Затем объявите переменную `bool` верхнего уровня с именем `isGreater`, которая хранит результат сравнения, больше ли `num1` чем `num2` с помощью оператора `>`.

# --solutions--

```dart
int high = 20;
int low = 12;
bool isHigher = high > low;

void main() {
  print(isHigher);
}
```

# --asserts--

`isGreater` должна быть `true` и быть типом `bool`

```dart
  test('test1', () {
    expect(isHigher, true, reason: '--err-t1--');
    expect(isHigher, TypeMatcher<bool>(), reason: '--err-t2--');
  });
```
