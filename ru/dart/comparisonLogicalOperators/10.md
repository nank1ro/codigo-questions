# --instructions--

Объявите две переменные `int` верхнего уровня: `x` со значением `8` и `y` со значением `8`. Затем объявите переменную `bool` верхнего уровня с именем `areEqual`, которая хранит результат сравнения `x` и `y` с помощью оператора `==`.

# --solutions--

```dart
int x = 8;
int y = 8;
bool areEqual = x == y;

void main() {
  print(areEqual);
}
```

# --asserts--

`areEqual` должна быть `true` и быть типом `bool`

```dart
  test('test1', () {
    expect(areEqual, true, reason: '--err-t1--');
    expect(areEqual, TypeMatcher<bool>(), reason: '--err-t2--');
  });
```
