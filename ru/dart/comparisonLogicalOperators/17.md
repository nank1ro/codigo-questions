# --instructions--

Объявите две переменные `int` верхнего уровня: `x` со значением `15` и `y` со значением `15`. Затем объявите переменную `bool` верхнего уровня с именем `isEqual`, которая хранит результат сравнения `x` и `y` с помощью оператора `>=`.

# --solutions--

```dart
int age = 35;
bool isWorkingAge = age >= 18 && age <= 65;

void main() {
  print(isWorkingAge);
}
```

# --asserts--

`isEqual` должна быть `true` и быть типом `bool`

```dart
  test('test1', () {
    expect(isWorkingAge, true, reason: '--err-t1--');
    expect(isWorkingAge, TypeMatcher<bool>(), reason: '--err-t2--');
  });
```
