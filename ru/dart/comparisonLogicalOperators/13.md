# --instructions--

Объявите две переменные `int` верхнего уровня: `value1` со значением `6` и `value2` со значением `8`. Затем объявите переменную `bool` верхнего уровня с именем `isSmaller`, которая хранит результат сравнения, меньше ли `value1` чем `value2` с помощью оператора `<`.

# --solutions--

```dart
int price = 30;
int budget = 50;
bool canAfford = price < budget;

void main() {
  print(canAfford);
}
```

# --asserts--

`isSmaller` должна быть `true` и быть типом `bool`

```dart
  test('test1', () {
    expect(canAfford, true, reason: '--err-t1--');
    expect(canAfford, TypeMatcher<bool>(), reason: '--err-t2--');
  });
```
