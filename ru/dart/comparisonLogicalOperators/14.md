# --description--

Оператор **И** `&&` объединяет два логических выражения. Он возвращает `true` только когда **обе** стороны имеют значение `true`. Если хотя бы одна сторона имеет значение `false`, результат будет `false`.

```dart
bool a = true;
bool b = true;
bool result = a && b; // true

bool c = true;
bool d = false;
bool result2 = c && d; // false
```

# --instructions--

Объявите две переменные `int` верхнего уровня: `age` со значением `25` и `score` со значением `80`. Затем объявите переменную `bool` верхнего уровня с именем `qualifies`, которая имеет значение `true` только если `age` больше или равно `18` И `score` больше или равно `60`.

# --solutions--

```dart
int age = 25;
int score = 80;
bool qualifies = age >= 18 && score >= 60;

void main() {
  print(qualifies);
}
```
