# --description--

Оператор **неравенства** `!=` возвращает `true`, когда два значения **различны**, и `false`, когда они одинаковы.

```dart
int x = 3;
int y = 7;
bool result = x != y; // true
```

Это противоположность `==`.

# --instructions--

Заполните код, чтобы проверить, не равно ли `4` `9`, и сохраните результат в `isDifferent`.
