# --instructions--

Объявите переменную `bool` верхнего уровня с именем `isNotLocked`. Используйте оператор НЕ `!`, чтобы инвертировать значение переменной `isLocked`, которая имеет значение `true`. Результат должен быть `false`.

# --solutions--

```dart
bool isOffline = false;
bool isOnline = !isOffline;

void main() {
  print(isOnline);
}
```

# --asserts--

`isNotLocked` должна быть `false` и быть типом `bool`

  test('test1', () {

  });
