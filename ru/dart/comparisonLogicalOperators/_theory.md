Оператор **равенства** `==` сравнивает два значения и возвращает `true`, если они равны, или `false`, если они не равны.

```dart
int a = 5;
int b = 5;
bool result = a == b; // true
```

Результат сравнения всегда является значением `bool`.

---

Оператор **неравенства** `!=` возвращает `true`, когда два значения **различны**, и `false`, когда они одинаковы.

```dart
int x = 3;
int y = 7;
bool result = x != y; // true
```

Это противоположность `==`.

---

Оператор **больше** `>` возвращает `true`, когда левое значение строго больше правого значения.

```dart
int a = 8;
int b = 3;
bool result = a > b; // true
```

Если левое значение равно или меньше правого, он возвращает `false`.

---

Оператор **меньше** `<` возвращает `true`, когда левое значение строго меньше правого значения.

```dart
int a = 2;
int b = 9;
bool result = a < b; // true
```

Если левое значение равно или больше правого, он возвращает `false`.

---

Оператор **больше или равно** `>=` возвращает `true`, когда левое значение больше **или равно** правому значению.

```dart
int score = 50;
bool passed = score >= 50; // true
```

В отличие от `>`, этот оператор также возвращает `true`, когда оба значения равны.
