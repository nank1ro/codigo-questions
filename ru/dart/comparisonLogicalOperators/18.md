# --instructions--

Объявите две переменные `int` верхнего уровня: `a` со значением `12` и `b` со значением `18`. Затем объявите переменную `bool` верхнего уровня с именем `isSmaller`, которая хранит результат сравнения, меньше ли `a` или равно `b` с помощью оператора `<=`.

# --solutions--

```dart
int x = 5;
int y = 3;
int z = 0;
bool result = !(x > 10) || (y < 5 && z == 0);

void main() {
  print(result);
}
```

# --asserts--
