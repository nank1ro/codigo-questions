# --instructions--

Объявите две переменные `int` верхнего уровня: `a` со значением `10` и `b` со значением `5`. Затем объявите переменную `bool` верхнего уровня с именем `areNotEqual`, которая хранит результат сравнения `a` и `b` с помощью оператора `!=`.

# --solutions--

```dart
int a = 5;
int b = 9;
bool notEqual = a != b;

void main() {
  print(notEqual);
}
```

# --asserts--

`areNotEqual` должна быть `true` и быть типом `bool`

```dart
  test('test1', () {
    expect(notEqual, true, reason: '--err-t1--');
    expect(notEqual, TypeMatcher<bool>(), reason: '--err-t2--');
  });
```
