# --description--

Оператор **больше или равно** `>=` возвращает `true`, когда левое значение больше **или равно** правому значению.

```dart
int score = 50;
bool passed = score >= 50; // true
```

В отличие от `>`, этот оператор также возвращает `true`, когда оба значения равны.

# --instructions--
