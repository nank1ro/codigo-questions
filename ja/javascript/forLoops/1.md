# --description--

`while`ループを使ってコードを繰り返す方法は知っています。
このプログラムのように、`hello`を表示する文を繰り返します
```javascript
var counter = 0;

while (counter < 5) {
    console.log("hello");
    counter++;
}
```
しかし、`for`ループでも同じことができます：
```javascript
for (let i = 0; i < 5; i++) {
    console.log("hello");
}
```
