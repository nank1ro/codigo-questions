# --description--

**不等価** `!=` 比較演算子に進みましょう。
この演算子は、2つの式が等しく**ない**かどうかを示す**ブール値**（`true` または `false`）を返します。例えば：
```javascript
console.log(2 != 2);
// false を出力
console.log(2 != 3);
// true を出力
```
これは*等価*演算子の正反対です
