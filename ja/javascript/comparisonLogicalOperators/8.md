# --description--

**or** `||` 論理演算子に進みましょう。
この演算子は、*true*と評価される最初のオペランドを返し、すべてが*false*の場合は最後のオペランドを返します。
```javascript
console.log(2 == 2 || 2 == 3);
// true を出力
console.log(1 == 2 || 1 == 3);
// false を出力
```
