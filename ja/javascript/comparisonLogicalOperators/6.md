# --description--

**以下** `<=` 比較演算子に進みましょう。
この演算子は、一方の式がもう一方以下かどうかを示す**ブール値**（`true` または `false`）を返します。例えば：
```javascript
console.log(2 <= 2);
// true を出力
console.log(3 <= 2);
// false を出力
console.log(3 <= 4);
// true を出力
```
