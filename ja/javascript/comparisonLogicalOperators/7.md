# --description--

次に**論理**演算子を見てみましょう。まず__AND__ `&&`から始めます。
この演算子は、*false*と評価される最初のオペランドを返し、すべてが*true*の場合は最後のオペランドを返します。
```javascript
console.log(2 == 2 && 2 == 3);
// false を出力
console.log(1 == 1 && 1 == 1.0);
// true を出力
```
