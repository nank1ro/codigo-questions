# --description--

最後に**not** `!` 論理演算子を見てみましょう。
この演算子は、式の論理状態を反転したブール値を返します。
```javascript
console.log(!true);
// false を出力
console.log(!false);
// true を出力
console.log(!(2 == 2));
// false を出力
```
