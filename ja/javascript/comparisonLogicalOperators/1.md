# --description--

**等価** `==` 比較演算子から始めましょう。
この演算子は、2つの式が等しいかどうかを示す**ブール値**（`true` または `false`）を返します。例えば：
```javascript
console.log(2 == 2);
// true を出力
console.log(2 == 3);
// false を出力
```
