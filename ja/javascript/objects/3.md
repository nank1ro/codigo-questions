# --description--

配列と同様に、辞書は_ミュータブル（変更可能）_です。
つまり、作成後に変更することができます（定数として宣言されていない場合）。
この利点の1つは、作成後に辞書に新しい_キーと値のペア_を追加できることです。以下のように記述します:
```javascript
dictName[newKeyName] = newValue;
```
