# --description--

辞書変数はミュータブル（変更可能）なので、さまざまな方法で変更できます。
'delete'キーワードを使って辞書から要素を削除できます:
```javascript
delete dictName[keyName];
```
これにより、辞書からキー`keyName`とそれに関連付けられた値が削除されます。

# --instructions--

辞書からキー`age`を削除してください
