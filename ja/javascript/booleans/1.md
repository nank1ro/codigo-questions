# --description--

JavaScriptには`boolean`と呼ばれる基本的なブーリアン型があります。
ブーリアン値は論理値とも呼ばれ、trueまたはfalseの2つの値しか取ることができません。
JavaScriptでは任意の式を評価でき、`true`または`false`のいずれかの結果を得ることができます。
