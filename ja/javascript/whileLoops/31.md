# --instructions--

`counter += 1`は何をしますか？
```javascript
var counter = 1;

while (counter < 5) {
    console.log(counter);
    counter += 1;
}
```

# --answers--

- ループがコードブロックを実行するたびにcounterを1増加させる
- counter変数を作成する
- 毎回counterを2増加させる
- 毎回counterを3増加させる
