# --instructions--

このループが停止しないことをどうやって知ることができますか？
```javascript
var isLoading = true;

while (isLoading) {
    console.log("Loading...");
    isLoading = true;
}
```

# --answers--

- コードブロック内で変数がfalseに更新されないから
- すべてのwhileループは停止しないから
- 変数がfalseに更新されるから
