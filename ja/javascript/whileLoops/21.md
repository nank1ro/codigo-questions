# --instructions--

このループが停止することをどうやって知ることができますか？
```javascript
var isLoading = true;

while (isLoading) {
    console.log("Loading...");
    isLoading = false;
}
```

# --answers--

- ループ内の変数がfalseに更新される
- ループは絶対に停止しない
