# --description--

「__Hello, World!__」は、新しいプログラミング言語を学ぶ際の伝統的な最初のプログラムです。
`console.log()`関数を使って、標準出力デバイス（画面）にデータを出力します。
JavaScriptで画面に`Hello, World!`と表示するには、次のように書きます
```javascript
console.log("Hello, World!")
```
