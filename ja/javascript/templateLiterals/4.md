# --description--

次に、波括弧の中に異なる種類の値を追加して、1つのconsole.log文として表示します。ここでは`${5}`のようにします

# --instructions--

空欄を埋めて、次の出力を得てください
```javascript
5 friends
```

# --seed--

```javascript
console.log(` friends`);
```

# --solutions--

```javascript
console.log(`${5} friends`);
```

# --output--

5 friends
