# --instructions--

このコードの何が間違っていますか？
```javascript
var manchesterUnited = 2;
var barcelona = 2;
console.log(`The final score is ${manchesterUnited} : ${barcelona}`);
```

# --answers--

- コードに問題はない
- manchesterUnitedとbarcelonaに値がない
- 波括弧の中で変数を使うことはできない
- 同じ文字列の中で${}を2回使うことはできない
