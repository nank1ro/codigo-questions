# --description--

_ドット構文_を使用してインスタンスのプロパティにアクセスできます。
ドット構文では、インスタンス名の直後にプロパティ名をピリオド`.`で区切って、スペースなしで記述します：
```javascript
someInstance.someProperty
```
同じ構文を使用して、プロパティの値を更新することもできます。
