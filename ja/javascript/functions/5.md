# --description--

関数のパラメータに値を代入することで、任意のパラメータに_デフォルト_値を定義できます。
デフォルト値が定義されている場合、関数を呼び出す際にそのパラメータを省略できます
```javascript
function someFunction(parameterWithoutDefault, parameterWithDefault = 12) {
    // do stuff here
}
```
