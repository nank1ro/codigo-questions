# --description--

コードの一部を少しだけ異なる値で再利用したい場面を考えたことがあるかもしれません。
コード全体を書き直す代わりに、関数を定義すれば、繰り返し使用できるのでずっとすっきりします。
JavaScriptでは、`function`キーワードの後に関数名を記述します：
```javascript
function sayHi() {
    console.log("Hello!");
}
sayHi();
// prints "Hello!"
```
