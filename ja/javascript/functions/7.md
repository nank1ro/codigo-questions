# --description--

関数には、関数の動作を説明する_オプションのコメント_を追加できます：
```javascript
// Prints 'Hello World' to the console.
function helloWorld() {
    console.log("Hello, World!");
}
```
単一行コメントには`//`を、複数行コメントには`/** */`を使用できます
