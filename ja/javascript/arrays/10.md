# --description--

JavaScriptでは`for..of`キーワードを使用して、配列を非常に簡単にループ処理できます:
```javascript
var numbers = [1, 2, 3];
for (num of numbers) {
    console.log(num);
}
// prints 1, 2, 3
```
`for`キーワードの後に変数名を指定すると、その変数に配列の各要素の値が順番に代入されます。

# --instructions--

配列の各要素を`2`倍にして出力してください
