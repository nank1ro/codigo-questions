# --description--

配列の個々の要素にはインデックスを使ってアクセスできます。
インデックスは、配列内の要素の位置を特定するアドレスのようなものです。
インデックスは配列名の直後に、角括弧の中に記述します:
```javascript
arrayName[index];
```
配列のインデックスは`0`から始まります。`1`では**ありません**！配列の最初の要素には次のようにアクセスします: `arrayName[0]`。
配列の2番目の要素はインデックス1にあります: `arrayName[1]`。
