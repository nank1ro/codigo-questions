# --description--

配列の要素は任意の型にすることができます。
```javascript
var arrayName = ["one", 2, true];
```
実際、上記には順に文字列、整数、ブール値が含まれています。
さらに、配列の中に配列を含めることもできます！

# --instructions--

配列から値を出力してください

# --seed--

```javascript
var array1 = ["a", "b", "c"];
var array2 = ["x", "y", "z"];
var array3 = [array1, array2];
console.log([/]);
```
