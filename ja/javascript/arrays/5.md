# --description--

配列は固定長である必要はありません。
いつでも配列の末尾に要素を追加できます！
配列に要素を追加するには`push`関数を使用します:
```javascript
var letters = ["a", "b"];
letters.push("c");
console.log(letters);
// Prints ["a", "b", "c"]
```

# --instructions--
