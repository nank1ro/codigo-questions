演算子は、変数や値に対して操作を行うために使用されます。
まずは算術演算子から始めましょう。特に**加算**演算子 `+` について見ていきます。
これは2つの数値を足すために使用されます。例えば：
```javascript
console.log(5 + 3);
// 8 と表示される
```

---

次に**減算**演算子 `-` を見ていきましょう。
これはある数値から別の数値を引くために使用されます。例えば：
```javascript
console.log(5 - 3);
// 2 と表示される
```

---

次に**乗算**演算子 `*` を見ていきましょう。
これは2つの数値を掛け合わせるために使用されます。例えば：
```javascript
console.log(5 * 3);
// 15 と表示される
```

---

次に**除算**演算子 `/` を見ていきましょう。
これは2つの数値を割るために使用されます。例えば：
```javascript
console.log(10 / 5);
// 2 と表示される
```

---

次に**剰余**演算子 `%` を見ていきましょう。
これは2つの数値の除算後の余りを求めるために使用されます。例えば：
```javascript
console.log(5 % 2);
// 1 と表示される
```
5を2で割ると商は2、余りは1なので、結果は1になります
```javascript
console.log(9 % 3);
// 0 と表示される
```
9を3で割ると商は3、余りは0なので、結果は0になります
