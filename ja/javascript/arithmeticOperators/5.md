# --description--

次に**剰余**演算子 `%` を見ていきましょう。
これは2つの数値の除算後の余りを求めるために使用されます。例えば：
```javascript
console.log(5 % 2);
// 1 と表示される
```
5を2で割ると商は2、余りは1なので、結果は1になります
```javascript
console.log(9 % 3);
// 0 と表示される
```
9を3で割ると商は3、余りは0なので、結果は0になります
