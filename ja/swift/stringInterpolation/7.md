# --description--

文字列補間の中で、丸括弧を使って何度でも値を挿入できます

# --instructions--

ここでは`2`、次に`8`を入れてみましょう

# --seed--

```swift
print("\([/]) new friends and \([/]) old friends")
```

# --answers--

- 2
- 8

# --solutions--

```swift
print("\(2) new friends and \(8) old friends")
```

# --output--

2 new friends and 8 old friends
