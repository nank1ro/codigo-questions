# --description--

次に、異なる種類の値を波括弧の中に追加して、1つのprint文として表示します。ここでは`\(5)`を使います

# --instructions--

空白を埋めて、次の出力を得てください
```swift
5 friends
```

# --seed--

```swift
print(" friends")
```

# --solutions--

```swift
print("\(5) friends")
```

# --output--

5 friends
