# --instructions--

このループが停止しないとどうしてわかりますか？
```swift
var isLoading = true

while isLoading {
    print("Loading...")
    isLoading = true
}
```

# --answers--

- コードブロック内で変数がfalseに更新されないから
- すべてのwhileループは停止しないから
- 変数がfalseに更新されるから
