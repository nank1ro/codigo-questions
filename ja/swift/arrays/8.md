# --description--

`Any`型を指定すると、配列の要素は任意の型にすることができます：
```swift
var arrayName: [Any] = ["one", 2, true]
```
実際、上の例では順に文字列、整数、ブール値が含まれています。
また、配列の中に配列を入れることもできます！

# --instructions--

配列から値を出力してください
