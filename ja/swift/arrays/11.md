# --description--

**タプル**は配列に似ていますが、要素に名前を付けてその名前で参照することができます。
タプルを作成するには丸括弧`()`を使います
```swift
var person = (firstName: "John", lastName: "Smith")
var firstName = person.firstName // John
var lastName = person.lastName // Smith
```
