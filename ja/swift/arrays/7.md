# --description--

Swiftでは配列を自由にスライスできます！
```swift
// Grabs the first two items
listName[..<2]
// Grabs the fourth through last items
listName[3...]
```
配列スライスに配列の最初または最後の要素が含まれる場合、その要素のインデックスを含める必要はありません
