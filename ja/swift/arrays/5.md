# --description--

配列は固定長である必要はありません。
いつでも配列の末尾に要素を追加できます！
配列に要素を追加するには`append`関数を使います：
```swift
var letters = ["a", "b"]
letters.append("c")
print(letters)
// Prints ["a", "b", "c"]
```
