# --description--

Swiftでは`for..in`キーワードを使って非常に簡単に配列をループ処理できます：
```swift
var numbers = [1, 2, 3]
for num in numbers {
    print(num)
}
// prints 1, 2, 3
```
`for`キーワードの後に変数名が続き、配列の各要素の値が順番に代入されます。
