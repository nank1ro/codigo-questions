# --description--

配列は、複数の異なる情報を1つの変数名のもとにシーケンスとして格納するためのデータ型です。
配列は1つまたは複数の型の値を格納し、**インデックス**を使ってそれらの値を区別します。
以下の形式の式で配列に要素を割り当てることができます：
```swift
var arrayName: [itemsType] = [item1, item2]
```
`itemsType`は配列内の要素の型を表し、例えば`Int`、`String`、`Any`などが使えます。
