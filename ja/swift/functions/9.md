# --description--

関数には、関数の動作を説明する_オプションのコメント_を追加できます:
```swift
/// Prints 'Hello World' to the console.
func helloWorld() {
    print("Hello, World!")
}
```
1行コメントには`///`を、複数行コメントには`/** */`を使用できます
