# --description--

コードの一部を、少しだけ異なる値で再利用したいと思う場面を考えたことがあるかもしれません。
コード全体を書き直す代わりに、関数を定義する方がはるかにすっきりしており、繰り返し使用できます。
Swiftでは、`func`キーワードの後に関数名を書いて関数を定義します:
```swift
func say_hi() {
    print("Hello!")
}
say_hi() // prints "Hello!"
```
