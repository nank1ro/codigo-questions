# --description--

_ドット構文_を使ってインスタンスのプロパティにアクセスできます。
ドット構文では、インスタンス名の直後にピリオド`.`で区切って、スペースなしでプロパティ名を記述します:
```swift
someInstance.someProperty
```
同じ構文を使ってプロパティの値を更新することもできます

# --instructions--

`Animal`インスタンスに`female`の性別を割り当ててください。
`Animal`インスタンスの性別を出力してください
