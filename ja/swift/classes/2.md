# --description--

_構造体_と_クラス_は、プログラムコードの構成要素となる汎用的で柔軟な構造です。
基本的なクラスや構造体は、`class`または`struct`キーワードとその名前だけで構成されます。例えば:
```swift
class ClassName {
    // class definition
}
struct ClassName {
    // structure definition
}
```
