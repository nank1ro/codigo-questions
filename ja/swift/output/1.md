# --description--

"__Hello, World!__"は、新しいプログラミング言語を学ぶ際の伝統的な最初のプログラムです。
`print()`関数を使って、標準出力デバイス（画面）にデータを出力します。
Swiftで画面に`Hello, World!`と表示するには、次のように書きます
```swift
print("Hello, World!")
```

# --instructions--

プログラムは画面に文字列`Hello, Swift!`を表示する必要があります。
