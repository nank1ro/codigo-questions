# --description--

個々の列挙値を`switch`文でマッチさせることができます:
```swift
let color = Colors.red
switch color {
    case .blue:
        print("Blue")
    case .red:
        print("Red")
    case .green:
        print("Green")
}
// prints "Red"
```
すべての列挙ケースに対して`case`を用意する必要がない場合は、明示的に対処されていないケースをカバーするために`default`ケースを使用できることを覚えておいてください
