# --description--

`enumeration`（列挙型）は、関連する値のグループに共通の型を定義し、コード内でそれらの値を型安全な方法で扱えるようにします。
列挙型は`enum`キーワードを使って宣言します:
```swift
enum Colors {
    case blue
    case red
    case green
}
```
列挙型で定義された値（`blue`、`red`、`green`など）は、その_列挙ケース_です。
`case`キーワードを使って新しい列挙ケースを導入します。
