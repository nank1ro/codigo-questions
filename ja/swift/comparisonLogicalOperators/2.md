# --description--

次に**不等値**比較演算子 `!=` を見てみましょう。
2つの式が等しく**ない**かどうかを**ブール値**（`true` または `false`）で返します。例えば：
```swift
print(2 != 2) // false
print(2 != 3) // true
```
これは*等値*演算子の正反対です
