# --description--

次に**論理**演算子を見てみましょう。まず __AND__ `&&` から始めます。
*false* と評価される最初のオペランドを返すか、すべてが *true* の場合は最後のオペランドを返します。
```swift
print(2 == 2 && 2 == 3) // false
print(1 == 1 && 1 == 1.0) // true
```
