# --description--

次に**or** `||` 論理演算子を見てみましょう。
*true* と評価される最初のオペランドを返すか、すべてが *false* の場合は最後のオペランドを返します。
```swift
print(2 == 2 || 2 == 3) // true
print(1 == 2 || 1 == 3) // false
```
