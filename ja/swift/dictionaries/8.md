# --description--

配列と同様に、`for..in`キーワードを使って辞書の要素をループ処理できます。
ループでキーと値の両方を取得するには、プロパティを指定する必要はありません:
```swift
for (key, value) in dictName {
    print("\(key): \(value)")
}
```
