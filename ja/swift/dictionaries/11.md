# --description--

以前、`removeValue()`メソッドを使って辞書から_キーと値のペア_を削除する方法を見ました。
キーに`nil`値を代入することで要素を削除することもできます
```swift
dictName[keyName] = nil
// keyNameが辞書dictNameから削除されます
```

# --instructions--

辞書から2番目のキーを削除してください
