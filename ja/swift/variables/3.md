# --description--

他の変数の値を変数に代入することもできます。ここでは、`y`変数に`x`の値を与えることができます。
```swift
var x = 5
var y = x
print(y) // prints 5
```

# --instructions--

`y`という名前の新しい変数を作成し、変数`x`の値を代入してください。
