# --description--

次は**剰余**演算子 `%` を見てみましょう。
2つの数値の除算後の余りを求めるために使用されます。例えば：
```swift
print(5 % 2) // 1
```
5を2で割ると商が2で余りが1なので、結果は1になります
```swift
print(9 % 3) // 0
```
9を3で割ると商が3で余りが0なので、結果は0になります
