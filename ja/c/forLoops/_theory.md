`while`ループを使ってコードを繰り返す方法を知っています。
例えば、次のプログラムは`hello`を表示する文を繰り返します
```c
int counter = 0;

while (counter < 5) {
    printf("Hello\n");
    counter++;
}
```
しかし、`for`ループでも同じことができます:
```c
for (int i = 0; i < 5; i++) {
    printf("Hello\n");
}
```

---

`for`ループでは、ループを何回実行するかを指定できます

---

`<`を使うと次の数値を含まずにループし、`<=`を使うと次の数値を含んでループできます

---

`i`という変数はカウンター変数です。
好きな名前を付けることができます。
ループの現在の繰り返し回数を数えます
