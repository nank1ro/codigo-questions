# --description--

`while`ループを使ってコードを繰り返す方法を知っています。
例えば、次のプログラムは`hello`を表示する文を繰り返します
```c
int counter = 0;

while (counter < 5) {
    printf("Hello\n");
    counter++;
}
```
しかし、`for`ループでも同じことができます:
```c
for (int i = 0; i < 5; i++) {
    printf("Hello\n");
}
```
