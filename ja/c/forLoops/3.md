# --description--

`<`を使うと次の数値を含まずにループし、`<=`を使うと次の数値を含んでループできます

# --instructions--

コードブロックを`3`回実行するために、次のコードの`for`ループを完成させてください

# --seed--

```c
for (int i = 1; i[/]3; i++) {
        printf("%d\n", i);
}
```
