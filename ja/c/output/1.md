# --description--

"__Hello, World!__"は、新しいプログラミング言語を学ぶ際の伝統的な最初のプログラムです。
`printf()`関数を使用して、標準出力デバイス（画面）にデータを出力します。
すべてのCプログラムはライブラリを使用しており、必要な関数を実行する機能を提供します。例えば、画面に出力する関数は`stdio.h`ヘッダーファイルで定義されています。
最初に実行されるコードは、常にmain関数の中にあります。
Cで画面に`Hello, World!`と表示するには、次のように書きます
```c
printf("Hello, World!");
```
最後に`0`を返して、プログラムが正常に終了したことを示します
