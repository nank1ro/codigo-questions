# --description--

C言語では、`printf`関数と式を使って、2つ以上の値を一緒に表示できます。例えば：
```c
>>> char c = 'C';
>>> printf("Hello %c!\n", c);
Hello C!
```
`%c`は1文字を表示するために使います

# --instructions--

`%s`式を使って名前を表示してください（`%s`は_文字配列_または_文字列_を表示するために使います）
