# --description--

Cには**べき乗**演算子がないため、`math.h` ライブラリに含まれる `pow()` 関数を使用する必要があります。
べき乗は底の繰り返しの乗算に対応します。つまり、指数 *n* の **b** は、*n* 個の底を掛け合わせた積です:
![exponentiation](https://bit.ly/3zcz6Lt)
```
>>> pow(5, 2);
25
```
