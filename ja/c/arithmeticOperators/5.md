# --description--

次は**剰余** `%` 演算子です。
2つの数値の除算後の余りを求めるために使用されます。例えば:
```
>>> 5 % 2
1
```
5を2で割ると商が2、余りが1になるため、結果は1になります
```
>>> 9 % 3
0
```
9を3で割ると商が3、余りが0になるため、結果は0になります
