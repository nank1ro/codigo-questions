# --description--

次は `floor()` 関数を使った**切り捨て除算**です。
この関数は商の整数部分を返します。例えば:
```
>>> 5.0 / 2
2.5
>>> floor(2.5)
2.0
```
整数除算とも呼ばれます。結果の値は整数ですが、結果の*型*は必ずしもintではありません。
