変数に値を代入するには `=` 記号を使うことを既に学びました。例えば：
```c
int a = 5;
```

---

既に初期化された変数 `total` があります
```c
int total = 5;
```
変数 `total` に数値 `2` を加えたい場合、次のように書くことができます
```c
total = total + 2;
```
これで動きます！しかし、同じことをより短く書く方法があります：
```c
total += 2;
```
`+=` 記号は**加算代入**と呼ばれます。
変数の値に値を加え、その結果をその変数に代入します。

---

加算代入と同様に、**減算代入** `-=` があります。
機能は同じで、唯一の違いは減算を行うことです。
つまり、以下は全く同じです
```c
num = num - 5;
// is equal to
num -= 5;
```

---

**乗算代入** `*=` 演算子を見てみましょう。
変数に値を掛け、その結果をその変数に代入します。
つまり、以下は全く同じです
```c
num = num * 5;
// is equal to
num *= 5;
```

---

**除算代入** `/=` 演算子を見てみましょう。
変数を値で割り、その結果をその変数に代入します。
つまり、以下は全く同じです
```c
num = num / 5;
// is equal to
num /= 5;
```

---

**剰余代入** `%=` 演算子を見てみましょう。
変数と値の剰余を計算し、その結果をその変数に代入します。
つまり、以下は全く同じです
```c
num = num % 5;
// is equal to
num %= 5;
```
