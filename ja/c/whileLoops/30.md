# --instructions--

このループが停止しないことをどうやって分かりますか？
```c
is_loading = true;

while (is_loading) {
    printf("Loading...\n");
    is_loading = true;
}
```

# --answers--

- コードブロック内で変数がfalseに更新されないから
- すべてのwhileループは停止しないから
- 変数がfalseに更新されるから
