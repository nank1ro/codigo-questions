# --instructions--

`counter += 1`は何をしますか？
```c
int counter = 1;

while (counter < 5) {
    printf("%d\n", counter);
    counter += 1;
}
```

# --answers--

- ループがコードブロックを実行するたびにcounterを1ずつ増やす
- counter変数を作成する
- 毎回counterを2ずつ増やす
- 毎回counterを3ずつ増やす
