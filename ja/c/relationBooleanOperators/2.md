# --description--

**不等価** `!=` 関係演算子を見てみましょう。
これは**ブール値**、真 `1` または偽 `0` を返し、2つの式が等しく**ない**かどうかを示します。例えば：
```c
>>> 2 != 2
0
>>> 2 != 3
1
```
これは*等価*演算子の正反対です
