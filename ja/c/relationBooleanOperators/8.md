# --description--

**or** `||` ブール演算子を見てみましょう。
これは*真*と評価される最初のオペランドを返すか、すべてが*偽*の場合は最後のオペランドを返します。
```c
>>> 2 == 2 || 2 == 3
1
>>> 1 == 2 || 1 == 3
0
```
