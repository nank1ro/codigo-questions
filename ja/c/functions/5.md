# --description--

関数には、その関数が何をするかを説明する_オプションのコメント_を追加できます：
```c
/*
 * Function:  hello_world
 * --------------------
 * prints "Hello, World!" to the screen
 */
function hello_world() {
    printf("Hello, World!\n");
}
```
`//`を使って1行コメントを、`/* */`を使って複数行コメントを書くことができます

# --instructions--

関数に有効なコメントを追加してください
