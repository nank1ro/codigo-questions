# --instructions--

このコードスニペットの__問題点__は何ですか？
```c
greet("Daniel");
void greet(char *name) {
    printf("Hello %s!\n", name);
};
```

# --answers--

- 定義する前に`greet()`を使用することはできません
- `greet()`を2回使用することはできません
- `greet()`は何も返しません
- `name`をパラメータとして使用することはできません
