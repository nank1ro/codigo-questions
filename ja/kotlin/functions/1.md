# --description--

関数とは、特定のタスクを実行し、プログラム全体で再利用できるコードのブロックです。
Kotlinでは、`fun` キーワードに続けて関数名と括弧を使って関数を定義します:
```kotlin
fun greet() {
    println("Hello!")
}
```
関数を呼び出す（実行する）には、名前の後に括弧を付けます:
```kotlin
greet() // prints Hello!
```
値を返さない関数は暗黙的に `Unit` を返します。
