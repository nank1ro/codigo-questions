演算子は、変数や値に対して演算を行うために使用されます。
まず算術演算子から始めましょう。特に**加算**演算子 `+` についてです。
これは2つの数値を足すために使用されます。例えば：
```kotlin
println(5 + 3) // 8
```

---

次は**減算**演算子 `-` を見てみましょう。
これはある数値から別の数値を引くために使用されます。例えば：
```kotlin
println(5 - 3) // 2
```

---

次は**乗算**演算子 `*` を見てみましょう。
これは2つの数値を掛け合わせるために使用されます。例えば：
```kotlin
println(5 * 3) // 15
```

---

次は**除算**演算子 `/` を見てみましょう。
これは2つの数値を割るために使用されます。例えば：
```kotlin
println(10 / 5) // 2
```

---

次は**剰余**演算子 `%` を見てみましょう。
これは2つの数値の除算後の余りを求めるために使用されます。例えば：
```kotlin
println(5 % 2) // 1
```
これは1と評価されます。なぜなら、5を2で割ると商が2で余りが1になるからです
```kotlin
println(9 % 3) // 0
```
こちらは0と評価されます。なぜなら、9を3で割ると商が3で余りが0になるからです
