# --description--

`Set`は、特定の順序を持たない__一意な__要素のコレクションです。

```kotlin
val numbers = setOf(1, 1, 2) // [1]
println(numbers)
// prints [1, 2]
```

__[1]__では数値__1__を2回含むセットを作成しようとしていますが、各要素は一意でなければならないため、2番目の__1__は自動的に破棄されます。
