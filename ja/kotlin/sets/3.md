# --description--

Kotlinには2種類の`Set`があります：

- `Set`は作成後に変更できません。
- `MutableSet`は作成後に変更でき、要素の追加、削除、更新が可能です。

```kotlin
val numbers = setOf(1, 2, 3)
numbers.add(4) // [1]
```
__[1]__は`Set`が_読み取り専用_であるため、エラーをスローします。

変更可能なセットを作成するには`mutableSetOf`キーワードを使用します
```kotlin
val numbers = mutableSetOf(1, 2, 3)
numbers.add(4)
println(numbers)
// prints [1, 2, 3, 4]
```
