# --description--

`Set`の最も一般的な操作は、`in`または`contains()`を使ったメンバーシップのテストです。

```kotlin
val numbers = setOf(1, 2, 3)
println(2 in numbers) // prints true
println(numbers.contains(5)) // prints false
```

上記のように、`in`と`contains`は渡された要素がセットに存在するかどうかを示す`Bool`を返します。
