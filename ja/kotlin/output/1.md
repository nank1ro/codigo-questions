# --description--

"__Hello, World!__"は、新しいプログラミング言語を学ぶ際の伝統的な最初のプログラムです。
`println()`関数を使用して、標準出力デバイス（画面）にデータを出力します。
Kotlinで画面に`Hello, World!`を表示するには、次のように書きます
```kotlin
println("Hello, World!")
```
