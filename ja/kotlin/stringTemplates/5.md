# --description--

`$`記号の後にプログラムの識別子として認識されないものが続く場合、特に何も起こりません

# --instructions--

空欄を埋めて、次の出力を得てください
```kotlin
5 friends
```

# --seed--

```kotlin
fun main() {
    println(" friends")
}
```

# --solutions--

```kotlin
fun main() {
    println("$5 friends")
}
```
