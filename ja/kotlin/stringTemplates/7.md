# --description--

波括弧を使って、文字列テンプレート内に何度でも値を挿入できます

# --instructions--

`2`と`8`で試してみましょう

# --seed--

```kotlin
println("[/] new friends and [/] old friends")
```

# --answers--

- $2
- $8
- ${2}
- ${8}

# --solutions--

```kotlin
println("${2} new friends and ${8} old friends")
```

# --output--

2 new friends and 8 old friends
