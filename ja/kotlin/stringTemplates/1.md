# --description--

文字列_テンプレート_は、プログラム的に文字列を生成する方法です。
Kotlinでは、`+`記号（連結）を使用して、2つ以上の文字列を一緒に表示できます。例えば：
```kotlin
println("Hello " + "Kotlin!")
// prints "Hello Kotlin!"
```
