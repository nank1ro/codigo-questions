# --instructions--

`counter += 1` は何をしますか？
```kotlin
var counter = 1

while (counter < 5) {
    println(counter)
    counter += 1
}
```

# --answers--

- ループがコードブロックを実行するたびにcounterを1増やす
- counter変数を作成する
- 毎回counterを2増やす
- 毎回counterを3増やす
