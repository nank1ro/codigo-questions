# --instructions--

このループが停止しないことをどのようにして知ることができますか？
```kotlin
var isLoading = true

while (isLoading) {
    println("Loading...")
    isLoading = true
}
```

# --answers--

- コードブロック内で変数がfalseに更新されないから
- すべてのwhileループは停止しないから
- 変数がfalseに更新されるから
