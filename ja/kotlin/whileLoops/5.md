# --description--

_while_ ループ内での `break` の使用例を示します:
```kotlin
var i = 0
while (i < 3) {
  i++
  if (i == 2) break // [1]
  println(i)
}
// prints 1
```

__[1]__ でご覧の通り、`i` が _2_ に等しいとき、ループを _break_ します。実際に数字の2と3は表示されません。

# --instructions--

空欄を `while` ループで埋めて、`n` が __3__ に等しいときにループを停止してください。
