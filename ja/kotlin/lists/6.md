# --description--

もう一つの便利なリスト操作は、指定された要素がリストに含まれているかどうかを調べる`contains`メソッドです。
例えば、名前のリストがある場合、`contains`メソッドを使って指定された名前がリストに存在するかどうかを確認できます。
```kotlin
val names = listOf("Thomas", "Donald", "Scarlett")
println(names.contains("Scarlett"))
// prints true
```
