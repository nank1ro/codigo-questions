# --description--

**以上** `>=` 比較演算子を見ていきましょう。
この演算子は、一方の式がもう一方以上であるかどうかを示す**ブール値**（`true` または `false`）を返します。例えば：
```kotlin
println(2 >= 2) // true
println(3 >= 2) // true
println(3 >= 4) // false
```
