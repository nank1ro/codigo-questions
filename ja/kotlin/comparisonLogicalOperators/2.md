# --description--

**不等価** `!=` 比較演算子を見ていきましょう。
この演算子は、2つの式が等しく**ない**かどうかを示す**ブール値**（`true` または `false`）を返します。例えば：
```kotlin
println(2 != 2) // false
println(2 != 3) // true
```
これは*等価*演算子の正反対です
