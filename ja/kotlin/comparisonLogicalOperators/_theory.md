**等価** `==` 比較演算子から始めましょう。
この演算子は、2つの式が等しいかどうかを示す**ブール値**（`true` または `false`）を返します。例えば：
```kotlin
println(2 == 2) // true
println(2 == 3) // false
```

---

**不等価** `!=` 比較演算子を見ていきましょう。
この演算子は、2つの式が等しく**ない**かどうかを示す**ブール値**（`true` または `false`）を返します。例えば：
```kotlin
println(2 != 2) // false
println(2 != 3) // true
```
これは*等価*演算子の正反対です

---

**より大きい** `>` 比較演算子を見ていきましょう。
この演算子は、一方の式がもう一方より大きいかどうかを示す**ブール値**（`true` または `false`）を返します。例えば：
```kotlin
println(2 > 2) // false
println(3 > 2) // true
```

---

**より小さい** `<` 比較演算子を見ていきましょう。
この演算子は、一方の式がもう一方より小さいかどうかを示す**ブール値**（`true` または `false`）を返します。例えば：
```kotlin
println(2 < 2) // false
println(2 < 3) // true
```

---

**以上** `>=` 比較演算子を見ていきましょう。
この演算子は、一方の式がもう一方以上であるかどうかを示す**ブール値**（`true` または `false`）を返します。例えば：
```kotlin
println(2 >= 2) // true
println(3 >= 2) // true
println(3 >= 4) // false
```

---

**以下** `<=` 比較演算子を見ていきましょう。
この演算子は、一方の式がもう一方以下であるかどうかを示す**ブール値**（`true` または `false`）を返します。例えば：
```kotlin
println(2 <= 2) // true
println(3 <= 2) // false
println(3 <= 4) // true
```

---

次に**論理**演算子を見ていきましょう。最初は __AND__ `&&` です。
この演算子は、*false* と評価される最初のオペランドを返し、すべてが *true* の場合は最後のオペランドを返します。
```kotlin
println(2 == 2 && 2 == 3) // false
println(1 == 1 && 1.0 == 1.0) // true
```

---

**or** `||` 論理演算子を見ていきましょう。
この演算子は、*true* と評価される最初のオペランドを返し、すべてが *false* の場合は最後のオペランドを返します。
```kotlin
println(2 == 2 || 2 == 3) // true
println(1 == 2 || 1 == 3) // false
```

---

最後に **not** `!` 論理演算子を見ていきましょう。
この演算子は、式の論理状態を反転させたブール値を返します。
```kotlin
println(!true) // false
println(!false) // true
println(!(2 == 2)) // false
```
