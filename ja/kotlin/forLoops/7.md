# --description--

コードブロックを単に`n`回繰り返す必要がある場合は、`repeat(times: Int)`関数を使用できます。

```kotlin
repeat(3) {
    println("repeat")
}
// prints repeat 3 times
```

インデックスにアクセスすることもできます
```kotlin
repeat(3) { index ->
    println(index)
}
// prints 0, 1, 2
```
