# --description--

Kotlinでは、反復可能なコレクションに対して`for-in`を使用し、各要素に対して指定されたクロージャを呼び出すこともできます：
```kotlin
// this is a list, we'll see about that soon
val numbers = listOf(2, 4, 6, 8, 10)
for (num in numbers) {
    println(num)
}
// prints (2, 4, 6, 8, 10)
```
