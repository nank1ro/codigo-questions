変数はデータ値を格納するためのコンテナです。
Kotlinのすべての変数はオブジェクトです。
変数を作成するには、スペースを含まないように注意しながら__名前__を付ける必要があります。
変数は、最初に値を代入した時点で作成されます。
Kotlinでは、`val`（_value_の略）キーワードで定数を宣言し、`var`（_variable_の略）キーワードで変数を宣言します。
定数の値は一度設定すると変更できませんが、変数は後から別の値に設定できます。
`x`という名前の変数を作成する例は次のとおりです：
```kotlin
var x = 1
```
このようにして、`x`という名前の変数に値`1`を代入しました。
変数`x`を出力すると、数値`1`が返されます：
```kotlin
println(x) // prints 1
```

---

変数は、格納する値を変更できるため、このように呼ばれます。
`=`を使って新しい値を与えることで、`x`を更新できます。
```kotlin
var x = 1
println(x) // prints 1
x = 2
println(x) // prints 2
```

---

他の変数の値を変数に代入することもできます。ここでは、`y`変数に`x`の値を代入できます。
```kotlin
var x = 5
var y = x
println(y) // prints 5
```

---

変数を更新すると、以前の値は失われます。ここでは、`x`変数を2回表示して、値がどのように更新されるかを確認できます。
```kotlin
var x = 5
println(x) // prints 5
x = 10
println(x) // prints 10
```

---

Kotlinでは、文字列変数はダブルクォートを使用してのみ宣言できます：
```kotlin
val x = "May"
```

---

複数の単語からなる変数名を使いたい場合は、**キャメルケース**を使用します。
これは、フレーズの途中の各単語を大文字で始める書き方です。
