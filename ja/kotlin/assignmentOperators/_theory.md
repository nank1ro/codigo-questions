代入演算子は変数に値を代入するために使われます。
最も基本的な代入演算子は `=` で、右側の値を左側の変数に代入します:
```kotlin
var x = 10
```
ここでは、`10` が変数 `x` に代入されます。
既存の変数に新しい値を再代入することもできます:
```kotlin
x = 20
```
これで `x` は値 `20` を保持します。

---

`+=` 演算子は、変数に値を加算してその結果を変数に戻す省略記法です。
以下のように書く代わりに:
```kotlin
x = x + 5
```
次のように書けます:
```kotlin
x += 5
```
どちらの式も同じことをします: `x` の値を `5` 増やします。

---

`-=` 演算子は、変数から値を減算してその結果を変数に戻します。
以下のように書く代わりに:
```kotlin
x = x - 3
```
次のように書けます:
```kotlin
x -= 3
```
これにより `x` の値が `3` 減少します。

---

`*=` 演算子は、変数に値を乗算してその結果を変数に戻します。
以下のように書く代わりに:
```kotlin
x = x * 4
```
次のように書けます:
```kotlin
x *= 4
```
これにより `x` が `4` 倍され、結果が `x` に格納されます。

---

`/=` 演算子は、変数を値で除算してその結果を変数に戻します。
以下のように書く代わりに:
```kotlin
x = x / 2
```
次のように書けます:
```kotlin
x /= 2
```
注意: 両方のオペランドが `Int` の場合、Kotlin は**整数除算**を行います（小数部分は切り捨てられます）:
```kotlin
var x = 7
x /= 2
// x は 3 になります（3.5 ではありません）
```
