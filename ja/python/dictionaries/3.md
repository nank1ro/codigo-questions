# --description--

リストと同様に、辞書は_ミュータブル（変更可能）_です。
つまり、作成後に変更することができます。
この利点の一つは、作成後に辞書に新しい_キーと値のペア_を追加できることです：
```python
dict_name[new_key_name] = new_value
```
