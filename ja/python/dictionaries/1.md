# --description--

**辞書（Dictionaries）**はリストやタプルに似ていますが、インデックスの代わりに*キー*を使って値にアクセスします。
キーには任意の文字列や数値を使用できます。
辞書は波括弧で囲みます。以下のようになります：
```python
d = {"key1": 1, "key2": 2, "key3": 3}
```
これは3つの*キーと値のペア*を持つ`d`という辞書です。
キー`key1`は値`1`を、`key2`は`2`を指し、以下同様です。
