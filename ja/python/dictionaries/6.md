# --description--

辞書のすべてのキーを一覧表示したい場合はどうすればよいでしょうか？
`keys()`メソッドがあります。

# --seed--

```python
user = {"first_name": "John", "last_name": "Hood", "age": 30}
print([/][/])
```

# --answers--

- user
- .keys()
- .keys
- .key
- .allKeys()

# --solutions--

```python
user = {"first_name": "John", "last_name": "Hood", "age": 30}
print(user.keys())
```

# --output--

dict_keys(['first_name', 'last_name', 'age'])
