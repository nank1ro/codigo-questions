# --description--

Pythonでは`for..in`キーワードを使って非常に簡単にリストをループ処理できます：
```python
>>> numbers = [1, 2, 3]
>>> for num in numbers:
>>>     print(num)
1
2
3
```
`for`キーワードの後に変数名が続き、リストの各要素の値が順番に代入されます。
