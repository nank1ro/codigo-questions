# --description--

# --instructions--

`tuple()`関数を使ってリストをタプルに変換してください

# --seed--

```python
list1 = ["dog", "bear", "cat"]
tuple1 = [/]
print(tuple1)
```

# --answers--

- tuple[list1]
- tuple(list1)

# --solutions--

```python
list1 = ["dog", "bear", "cat"]
tuple1 = tuple(list1)
print(tuple1)
```

# --output--
