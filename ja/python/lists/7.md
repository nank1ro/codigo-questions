# --description--

文字列もリストとまったく同じようにスライスできます！実際、文字列は文字のリストと考えることができます：各文字はインデックス`0`から始まるリスト内の連続した要素です。
```python
list_name[:2]
# Grabs the first two items
list_name[3:]
# Grabs the fourth through last items
```
リストスライスにリスト（または文字列）の最初または最後の要素が含まれる場合、その要素のインデックスを省略できます。

# --instructions--

- `first_name`に`full_name`のインデックス`0`からインデックス`4`を*含まない*までのスライスを代入してください。
- `last_name`に`full_name`のインデックス`4`から文字列の末尾までのスライスを代入してください

# --seed--

```python
full_name = "Sam Smith"
first_name = [/][/]
last_name = [/][/]
print(first_name)
print(last_name)
```
