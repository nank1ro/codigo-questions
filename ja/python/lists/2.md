# --description--

リストの個々の要素にはインデックスを使ってアクセスできます。
インデックスとは、リスト内の要素の位置を識別するアドレスのようなものです。
インデックスはリスト名の直後に角括弧で囲んで記述します：
```python
list_name[index]
```

リストのインデックスは`0`から始まり、`1`からでは**ありません**！リストの最初の要素にはこのようにアクセスします：`list_name[0]`。
リストの2番目の要素はインデックス1にあります：`list_name[1]`。
