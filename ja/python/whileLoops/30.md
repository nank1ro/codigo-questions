# --instructions--

このループが停止しないとどうやってわかりますか？
```python
is_loading = True

while is_loading:
    print("Loading...")
    is_loading = True
```

# --answers--

- コードブロック内で変数がFalseに更新されないから
- すべてのwhileループは停止しないから
- 変数がFalseに更新されるから
