# --instructions--

このコードスニペットの__問題点__は何ですか？
```python
greet('Mathew')

def greet(name):
    """This function greets to the person passed in as a parameter"""
    print(f"Hello, {name}. Good morning!")

greet('Alex')
```

# --answers--

- `greet()`を定義する前に使用することはできません
- `greet()`を2回使用することはできません
- `greet()`は何も返しません
- `name`をパラメータとして使用することはできません
