# --description--

**切り捨て除算** `//` 演算子を見てみましょう。
この演算子は商の整数部分を返します。例えば：
```python
>>> 5.0 / 2
2.5
>>> 5.0 // 2
2.0
```
整数除算とも呼ばれます。結果の値は整数ですが、結果の*型*は必ずしもintとは限りません。
