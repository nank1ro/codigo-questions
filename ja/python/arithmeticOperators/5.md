# --description--

**剰余** `%` 演算子を見てみましょう。
これは2つの数の除算の余りを求めるために使用されます。例えば：
```python
>>> 5 % 2
1
```
5を2で割ると商が2で余りが1なので、結果は1になります
```python
>>> 9 % 3
0
```
こちらは9を3で割ると商が3で余りが0なので、結果は0になります
