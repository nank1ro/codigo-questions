# --description--

**不等価** `!=` 関係演算子に進みましょう。
2つの式が等しく**ない**かどうかを示す**ブール値**（`True` または `False`）を返します。例：
```python
>>> 2 != 2
False
>>> 2 != 3
True
```
これは*等価*演算子の正反対です
