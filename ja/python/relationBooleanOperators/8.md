# --description--

次は **or** ブール演算子です。
*True* と評価される最初のオペランドを返すか、すべてが *False* の場合は最後のオペランドを返します。
```python
>>> 2 == 2 or 2 == 3
True
>>> 1 == 2 or 1 == 3
False
```
