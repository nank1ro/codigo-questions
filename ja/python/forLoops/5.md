# --description--

`range()`関数は、デフォルトで0から始まり、1ずつ増加し、指定された数値の前で停止する数列を返します。
この関数の構文は次の通りです:
```python
range(start, stop, step)
```
`start`と`step`は省略可能ですが、`stop`は必須です
