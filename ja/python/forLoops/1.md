# --description--

`while`ループを使ってコードを繰り返す方法は知っています。
例えば、次のプログラムは`hello`を表示する文を繰り返します
```python
counter = 0

while counter < 5:
    print("hello")
    counter += 1
```
しかし、`for`ループでも同じことができます:
```python
for i in range(5):
    print("hello")
```
