# --description--

**切り捨て除算代入** `//=` 演算子を見てみましょう。
変数を値で切り捨て除算し、その結果を変数に代入します。
つまり、以下の2つはまったく同じです
```python
num = num // 5
# is equal to
num //= 5
```
