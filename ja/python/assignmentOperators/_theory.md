変数に値を代入するには `=` 記号を使えることはすでに学びました。例えば：
```python
a = 5
```

---

すでに初期化された変数 `total` があります
```python
total = 5
```
変数 `total` に数値 `2` を加えたい場合、次のように書くことができます
```python
total = total + 2
```
これで動作しますが、同じことをするもっと短い書き方があります：
```python
total += 2
```
記号 `+=` は**加算代入**と呼ばれます。
変数の値に値を加算し、その結果を変数に代入します。

---

加算代入と同様に、**減算代入** `-=` があります。
機能は同じで、唯一の違いは減算を行うことです。
つまり、以下の2つはまったく同じです
```python
num = num - 5
# is equal to
num -= 5
```

---

**乗算代入** `*=` 演算子を見てみましょう。
変数に値を掛けて、その結果を変数に代入します。
つまり、以下の2つはまったく同じです
```python
num = num * 5
# is equal to
num *= 5
```

---

**除算代入** `/=` 演算子を見てみましょう。
変数を値で割り、その結果を変数に代入します。
つまり、以下の2つはまったく同じです
```python
num = num / 5
# is equal to
num /= 5
```

---

**累乗代入** `**=` 演算子を見てみましょう。
変数を指定された累乗で計算し、その結果を変数に代入します。
つまり、以下の2つはまったく同じです
```python
num = num ** 5
# is equal to
num **= 5
```

---

**剰余代入** `%=` 演算子を見てみましょう。
変数と値の剰余を計算し、その結果を変数に代入します。
つまり、以下の2つはまったく同じです
```python
num = num % 5
# is equal to
num %= 5
```

---

**切り捨て除算代入** `//=` 演算子を見てみましょう。
変数を値で切り捨て除算し、その結果を変数に代入します。
つまり、以下の2つはまったく同じです
```python
num = num // 5
# is equal to
num //= 5
```
