# --description--

加算代入と同様に、**減算代入** `-=` があります。
機能は同じで、唯一の違いは減算を行うことです。
つまり、以下の2つはまったく同じです
```python
num = num - 5
# is equal to
num -= 5
```

# --instructions--

*減算代入*演算子を使って変数 `result` から値 `15` を減算し、その値を出力してください

# --seed--

```python
result = 75
result [/][/]
print(result)
```
