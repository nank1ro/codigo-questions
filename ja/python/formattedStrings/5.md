# --description--

次に、異なる種類の値を波括弧の中に入れることで、1つのprint文として表示します。ここでは`{5}`のようにします

# --instructions--

空欄を埋めて、以下の出力を得てください
```python
5 friends
```

# --seed--

```python
print(f" friends")
```
