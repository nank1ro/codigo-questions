# --description--
`for-in` ループはインデックスを使わずにコレクションのすべての要素を反復処理します：

```dart
List<String> fruits = ['apple', 'banana', 'cherry'];
for (var fruit in fruits) {
  print(fruit);
}
```

各反復で次の要素がループ変数（ここでは `fruit`）に代入されます。

# --instructions--
リスト内のすべての数値を出力する `for-in` ループを完成させてください。
