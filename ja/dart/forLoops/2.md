# --description--
現在のカウントを追跡するために、ループ変数をボディの中で使うことができます。例えば、合計を累積する場合：

```dart
int total = 0;
for (int i = 1; i <= 5; i++) {
  total += i;
}
```

ループの後、`total` は 15 になります。

# --instructions--
`for` ループを使って 1 から 5 までの数を合計するコードを完成させてください。
