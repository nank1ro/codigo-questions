# --description--
`for` ループはデクリメント（`i--`）と `>=` 条件を使って**下向きに**カウントできます：

```dart
for (int i = 5; i >= 1; i--) {
  print(i); // 5, 4, 3, 2, 1
}
```

これはカウントダウンや逆順の反復に役立ちます。
