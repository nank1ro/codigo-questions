# --description--

Dartにおいて、**リスト**は要素の順序付きコレクションです。リストを作成する最も簡単な方法は `[]` リテラル構文です：

```dart
List<int> numbers = [1, 2, 3];
```

`var` による型推論も使用できます：

```dart
var fruits = ['apple', 'banana', 'cherry'];
```

型アノテーション `List<String>` は、リストのすべての要素が `String` でなければならないことをDartに伝えます。
