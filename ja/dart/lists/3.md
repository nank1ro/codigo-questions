# --description--

`.add()` メソッドはリストの**末尾**に1つの要素を追加します：

```dart
var nums = [1, 2, 3];
nums.add(4);
print(nums); // [1, 2, 3, 4]
```

`.add()` はリストを**その場で**変更し、`void` を返すことに注意してください。
