# --description--

Dartのリストは**ゼロインデックス**です。つまり、最初の要素はインデックス `0`、2番目はインデックス `1`、というようになります。

```dart
var colors = ['red', 'green', 'blue'];
print(colors[0]); // red
print(colors[2]); // blue
```

インデックスによる要素へのアクセスは `list[index]` 構文で行います。
