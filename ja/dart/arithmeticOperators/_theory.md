**加算演算子** `+` は2つの値を加算します。

```dart
int a = 3;
int b = 4;
int sum = a + b; // 7
```

`int` または `double` の値で `+` を使用できます。

---

**減算演算子** `-` は一方の値からもう一方の値を減算します。

```dart
int a = 10;
int b = 4;
int diff = a - b; // 6
```

2番目のオペランドが最初のオペランドより大きい場合、結果は負になる可能性があります。

---

**乗算演算子** `*` は2つの値を乗算します。

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

`int` を別の `int` に乗算すると `int` が生成されます。`double` で乗算すると `double` が生成されます。

---

**除算演算子** `/` は一方の値を別の値で除算します。Dartでは、`/` は結果が整数の場合でも**常に `double` を返します**。

```dart
int a = 10;
int b = 4;
double result = a / b; // 2.5

double exact = 10 / 2; // 5.0  (not 5!)
```

`/` の結果を格納するには常に `double` 変数を使用してください。

---

**整数除算演算子** `~/` は2つの数を除算し、結果を整数に**切り詰めます**（小数部分を破棄します）。

```dart
int a = 10;
int b = 3;
int result = a ~/ b; // 3  (not 3.333...)
```

余りのない整数商が必要な場合は `~/` を使用してください。
