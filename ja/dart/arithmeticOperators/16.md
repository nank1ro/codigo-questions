# --description--

Dartは標準的な数学的**演算の順序**（優先度）に従います：乗算と除算は加算と減算の前に評価されます。

```dart
int result = 2 + 3 * 4; // 14, not 20
```

括弧を使用して順序を変更します：

```dart
int result = (2 + 3) * 4; // 20
```

# --instructions--

トップレベル `int` 変数 `result` を宣言して `3 + 4 * 2 - 1` を評価します。Dartに默定の演算子の優先度を適用させます（括弧は追加しないでください）。
