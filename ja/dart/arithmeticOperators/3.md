# --description--

**乗算演算子** `*` は2つの値を乗算します。

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

`int` を別の `int` に乗算すると `int` が生成されます。`double` で乗算すると `double` が生成されます。
