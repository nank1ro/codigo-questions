# --description--

文字列補間を使って、波括弧で好きなだけ値を挿入できます

# --instructions--

`2`と`8`を使って確認してください

# --seed--

```dart
print("[/] new friends and [/] old friends");
```

# --answers--

- $2
- $8
- ${2}
- ${8}

# --solutions--

```dart
print("${2} new friends and ${8} old friends");
```

# --output--

2 new friends and 8 old friends
