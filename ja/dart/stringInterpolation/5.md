# --description--

`$`記号の後に続くものがプログラムの識別子として認識できない場合、Dart はコンパイルエラーを発生させます。文字列内の `$` の後には必ず識別子または波括弧 (`${}`) で囲んだ式が続かなければなりません。

# --instructions--

このコードを実行しようとすると何が起きますか？

```dart
void main() {
    print("$5 friends");
}
```

# --answers--

- `$5` が無効な文字列補間のためコンパイルに失敗します
- `$5 friends` と出力されます
- `5 friends` と出力されます
- 何も出力されません
