# --description--

文字列_補間_（interpolation）は、プログラム的に文字列を生成する方法です。
Dartでは`+`記号（連結）を使って、2つ以上の文字列を一緒に表示できます。例えば：
```dart
print("Hello " + "Dart!");
// prints "Hello Dart!"
```
