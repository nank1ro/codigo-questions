# --description--

ブーリアン変数は値`false`も保持できます。
何かが成立しない場合は`false`を使用します。

```dart
bool isLoggedIn = false;
print(isLoggedIn); // false
```

`true`と同じように、`false`は小文字で書かなければなりません。

# --instructions--

`isExpired`というブーリアン変数を宣言し、値`false`を割り当てて、それを出力してください。
