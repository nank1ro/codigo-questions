# --description--

__否定演算子__`!`（「not」とも呼ばれる）はブーリアン値を反転します。
`!`を`true`に適用すると`false`になり、`!`を`false`に適用すると`true`になります。

```dart
bool isActive = true;
print(!isActive); // false
```

これは条件の反対をチェックしたい場合に便利です。

# --instructions--

変数`isBlocked`が`false`に設定されている場合、`!`を使用してその否定値を出力してください。
