# --description--

ブーリアンはプログラミング全体で条件を表し、決定を駆動するために使用されます。
プログラムが2つの可能性の間で決定する必要があるときはいつでも、ブーリアンが関与しています。

例：
- ユーザーはログインしていますか？（`isLoggedIn`）
- ドアは開いていますか？（`isDoorOpen`）
- 注文は発送されましたか？（`isShipped`）

# --instructions--

ブーリアンはプログラムで通常何に使用されていますか？

# --answers--

- 整数を格納する
- 真または偽のいずれかの条件を表す
- テキストを格納する
- 算術演算を実行する
