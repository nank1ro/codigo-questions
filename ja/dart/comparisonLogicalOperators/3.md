# --description--

**より大きい演算子** `>` は、左の値が右の値より厳密に大きい場合に `true` を返します。

```dart
int a = 8;
int b = 3;
bool result = a > b; // true
```

左の値が右の値以下の場合は `false` を返します。

# --instructions--

`15` が `6` より大きいかどうかを確認するコードを完成させ、結果を `isGreater` に保存します。
