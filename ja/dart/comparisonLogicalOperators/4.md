# --description--

**より小さい演算子** `<` は、左の値が右の値より厳密に小さい場合に `true` を返します。

```dart
int a = 2;
int b = 9;
bool result = a < b; // true
```

左の値が右の値以上の場合は `false` を返します。

# --instructions--
