# --description--

**以上演算子** `>=` は、左の値が右の値より大きい**または等しい**場合に `true` を返します。

```dart
int score = 50;
bool passed = score >= 50; // true
```

`>` と異なり、この演算子は両方の値が等しい場合にも `true` を返します。

# --instructions--

`7` が `7` 以上かどうかを確認するコードを完成させ、結果を `isAtLeastSeven` に保存します。
