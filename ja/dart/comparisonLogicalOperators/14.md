# --description--

**AND演算子** `&&` は2つのブール式を組み合わせます。**両方**の側が `true` のときだけ `true` を返します。どちらか一方の側が `false` の場合、結果は `false` です。

```dart
bool a = true;
bool b = true;
bool result = a && b; // true

bool c = true;
bool d = false;
bool result2 = c && d; // false
```
