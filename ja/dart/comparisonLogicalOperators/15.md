# --description--

**OR演算子** `||` は2つのブール式を組み合わせます。**少なくとも1つ**の側が `true` のとき `true` を返します。**両方**の側が `false` のときだけ `false` を返します。

```dart
bool a = false;
bool b = true;
bool result = a || b; // true

bool c = false;
bool d = false;
bool result2 = c || d; // false
```

# --instructions--
