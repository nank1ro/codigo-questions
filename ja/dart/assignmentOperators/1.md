# --description--

**代入演算子** `=` は値を変数に格納します。

```dart
int x = 5;
String name = 'Alice';
```

割り当てられた後、変数は再度割り当てられるまでその値を保持します。

# --instructions--

コードを完成させて、変数 `answer` に値 `42` を割り当ててください。

# --seed--

```dart
void main() {
  int answer [/] 42;
  print(answer);
}
```
