**代入演算子** `=` は値を変数に格納します。

```dart
int x = 5;
String name = 'Alice';
```

割り当てられた後、変数は再度割り当てられるまでその値を保持します。

---

**加算代入演算子** `+=` は値を変数に加え、結果をその変数に格納します。

```dart
int score = 10;
score += 5; // score は今 15
```

これは `score = score + 5` の省略形です。

---

**減算代入演算子** `-=` は変数から値を減らし、結果をその変数に格納します。

```dart
int lives = 3;
lives -= 1; // lives は今 2
```

これは `lives = lives - 1` の省略形です。

---

**乗算代入演算子** `*=` は変数に値を掛け、結果をその変数に格納します。

```dart
int side = 4;
side *= 3; // side は今 12
```

これは `side = side * 3` の省略形です。
