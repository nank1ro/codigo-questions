# --description--

これら2つのコメントの書き方に加えて、Dartには_ドキュメント_コメントがあります。

_ドキュメント_コメントは、`///`または`/**`で始まる複数行または単一行のコメントです。連続する行に`///`を使用すると、複数行のドキュメントコメントと同じ効果があります。

_ドキュメント_コメントは、ドキュメントを生成できるため非常に便利です。
特定のコードブロックが何をするかを明確にするために、コードに_ドキュメント_コメントを追加することをお勧めします。

_ドキュメント_コメント内では、アナライザは角括弧で囲まれていないすべてのテキストを無視します。
角括弧を使用して、_クラス_、_メソッド_、_フィールド_、_変数_、_関数_、_パラメータ_を参照できます。

以下は例です：
```dart
/// A domesticated South American camelid (Lama glama).
///
/// Just like any other animal, llamas need to eat,
/// so don't forget to [feed] them some [Food].
class Llama {
  String? name;

  /// Feeds your llama [food].
  ///
  /// The typical llama eats one bale of hay per week.
  void feed(Food food) {
    // ...
  }

  /// Exercises your llama with an [activity] for
  /// [timeLimit] minutes.
  void exercise(Activity activity, int timeLimit) {
    // ...
  }
}
```

生成されたドキュメントでは、`[feed]`は`feed`メソッドのドキュメントへのリンクになり、`[Food]`は`Food`クラスのドキュメントへのリンクになります。
一方、`[activity]`と`[timeLimit]`はそれぞれ`activity`と`timeLimit`のドキュメントへのリンクになります。
