# --description--

コメントブロック、つまり_複数行_コメントを作成することもできます。
_複数行_コメントは`/*`で始まり、`*/`で終わります。
```dart
/*
This is a multi-line
comment
*/
```

# --instructions--

_複数行_コメントを追加するためにコードを完成させてください。
