# --description--

カウンター変数は `while` ループが何回実行されるかを制御します。

```dart
int count = 0;
while (count < 5) {
  count++;
}
print(count); // 5
```

カウンターは `0` から始まり、`count < 5` が `false` になるまでループが各繰り返しでインクリメントします。

# --instructions--

`1` から `5` までカウントし、各値を出力するループを完成させてください。
