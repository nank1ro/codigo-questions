# --description--

`break` キーワードはループの条件に関係なく、ループを即座に終了させます。

```dart
int i = 0;
while (true) {
  if (i == 3) break;
  print(i);
  i++;
}
```

`break` がなければ、`while (true)` ループは永遠に実行され続けます。ここでは `i` が `3` に達したときに停止します。

# --instructions--

`0` から数値を出力し、`i` が `4` に等しくなったときに `break` を使って停止するループを完成させてください。
