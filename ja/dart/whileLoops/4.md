# --description--

`while` ループの条件には任意のブール式を使用できます。条件が `false` と評価された瞬間にループが終了します。

```dart
int n = 100;
while (n > 1) {
  n ~/= 2;
}
print(n); // 0
```

ここでは、整数除算 (`~/`) を使って各繰り返しで `n` を半分にしています。

# --instructions--

`n` が `1` より大きい間、(`~/= 2` を使って) `n` を半分にし続け、その後結果を出力する `while` ループを完成させてください。
