`while` ループは、条件が `true` である限りコードブロックを繰り返します。

```dart
int i = 0;
while (i < 3) {
  print(i);
  i++;
}
```

ループは各繰り返しの**前**に条件を確認します。`i < 3` が `false` になると、ループが停止します。

---

カウンター変数は `while` ループが何回実行されるかを制御します。

```dart
int count = 0;
while (count < 5) {
  count++;
}
print(count); // 5
```

カウンターは `0` から始まり、`count < 5` が `false` になるまでループが各繰り返しでインクリメントします。

---

`do-while` ループは `while` ループに似ていますが、**必ず少なくとも一度ボディを実行します** — 条件は各繰り返しの*後*に確認されます。

```dart
int i = 0;
do {
  print(i);
  i++;
} while (i < 3);
```

最初から条件が `false` であっても、ボディは一度実行されます。

---

`while` ループの条件には任意のブール式を使用できます。条件が `false` と評価された瞬間にループが終了します。

```dart
int n = 100;
while (n > 1) {
  n ~/= 2;
}
print(n); // 0
```

ここでは、整数除算 (`~/`) を使って各繰り返しで `n` を半分にしています。

---

`break` キーワードはループの条件に関係なく、ループを即座に終了させます。

```dart
int i = 0;
while (true) {
  if (i == 3) break;
  print(i);
  i++;
}
```

`break` がなければ、`while (true)` ループは永遠に実行され続けます。ここでは `i` が `3` に達したときに停止します。
