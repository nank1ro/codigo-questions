# --instructions--

`int` パラメータ `n` を取り `n!`（`n` の階乗）を返す**再帰的**関数 `factorial` を記述してください。

`n` の階乗は次のように定義されます：
- `factorial(0) = 1`
- `factorial(n) = n * factorial(n - 1)` （`n > 0` の場合）

例：
```dart
print(factorial(5)); // 120
print(factorial(3)); // 6
```

```dart
int factorial(int n) {
  if (n == 0) return 1;
  return n * factorial(n - 1);
}
```
