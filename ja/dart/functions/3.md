# --description--

関数は**パラメータ**を受け取ることができます — 関数を呼び出すときに渡される値です。パラメータは型と名前を伴って括弧の中に宣言されます：

```dart
int square(int n) {
  return n * n;
}

void main() {
  print(square(4)); // prints 16
}
```

各パラメータには型（`int`）と名前（`n`）があります。複数のパラメータはカンマで区切られます。
