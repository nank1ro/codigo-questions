# --description--

Dartは `=>` 構文を使った**アロー関数**をサポートしています。関数本体が単一の式の場合、`{ return ...; }` の代わりに `=>` で短縮できます：

```dart
// 通常の関数
int double(int n) {
  return n * 2;
}

// アロー関数 — 同じ結果
int double(int n) => n * 2;
```

アロー関数はコードをより簡潔にします。`=>` は波括弧と `return` キーワードの両方を置き換えます。
