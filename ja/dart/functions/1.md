# --description--

**関数**とは、特定のタスクを実行する再利用可能なコードブロックです。Dartでは、戻り値の型、名前、`()` のペアで関数を定義します。

関数が値を返さない場合、戻り値の型は `void` になります：

```dart
void sayHello() {
  print("Hello!");
}
```

関数を呼び出すには、名前の後に `()` を記述します：

```dart
sayHello(); // prints Hello!
```
