# --description--

関数は呼び出し元に**値を返す**ことができます。関数名の前に戻り値の型を宣言し、`return` キーワードを使って値を返します：

```dart
int getAge() {
  return 25;
}
```

関数名の前の型（`int`）は、関数が返す値の種類をDartに伝えます。`String`、`int`、`double`、`bool` などを使用できます。
