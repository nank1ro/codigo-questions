# --instructions--

`int` パラメータ `n` を取り、`1` から `n` までの整数（両端を含む）を格納した `List<int>` を返す関数 `range` を記述してください。

例：
```dart
print(range(3)); // [1, 2, 3]
print(range(5)); // [1, 2, 3, 4, 5]
```

# --solutions--

```dart
List<int> range(int n) {
  List<int> result = [];
  for (int i = 1; i <= n; i++) {
    result.add(i);
  }
  return result;
}
```
