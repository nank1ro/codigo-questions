# --description--

変数は値を格納するためのコンテナです。
Dartのすべての変数はオブジェクト（`Object`）です。
変数を作成するには、スペースを含めてはいけないという点を考慮して、__名前__を付ける必要があります。
次の例を見てみましょう:

```dart
int number = 1;
```

この文は`int`型の`number`という変数を宣言しています。
そして、変数の値を数値`1`に設定します。
宣言の`int`の部分は__型アノテーション__として知られており、Dartに変数の型を明示的に伝えます。
