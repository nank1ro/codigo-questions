# --description--

Dartは__型推論__をサポートしています。
Dartは変数の型を推論できるため、型を明示する必要はありません。
`var`キーワードは、Dartに最も適切な型を使用するよう指示します。

```dart
var number = 5;
```

数値`5`が`int`型であることをDartに伝える必要はありません。
Dartが型を推論し、`number`を`int`型にします。
