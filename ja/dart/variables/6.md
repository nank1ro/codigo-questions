# --description--

リスクを冒してでも、`dynamic`型を使用して言語の__型安全性__を完全に無視することができます。
これにより、変数に任意の型の値を代入できます。

```dart
dynamic number;
number = 1; // OK
number = 3.14159; // OK
number = '10'; // OK
```

このアプローチは、エラーがコードの_アナライザー_によってではなく、_ランタイム_（プログラムの実行時）にのみ検出されるため、強く推奨されません。

# --instructions--

変数を文字列`Hello`で更新してください。
