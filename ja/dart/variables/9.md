# --description--

定数変数を宣言した後は、その値を変更することはできません。例えば:

```dart
const number = 2;
number = 3; // Error
```

このコードは次のエラーを生成します:
> Constant variables can't be assigned a value.

つまり、定数変数の値を変更することはできません。

# --instructions--

定数`number`の値を変更して、発生するエラーを確認してください。
