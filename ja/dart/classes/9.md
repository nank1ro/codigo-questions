# --asserts--

`Point(3, 5)`は`x`を`3`、`y`を`5`に設定する

```dart
  test('test1', () {
    final p = Point(3, 5);
    expect(p.x, 3, reason: '--err-t1--');
    expect(p.y, 5, reason: '--err-t1--');
  });
```

`Point.origin()`は`x`と`y`の両方を`0`に設定する

```dart
  test('test2', () {
    final p = Point.origin();
    expect(p.x, 0, reason: '--err-t2--');
    expect(p.y, 0, reason: '--err-t2--');
  });
```

`Point(-1, 4)`は`x`を`-1`に設定する

```dart
  test('test3', () {
    final p = Point(-1, 4);
    expect(p.x, -1, reason: '--err-t3--');
  });
```
