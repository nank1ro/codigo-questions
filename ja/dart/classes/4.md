# --description--

**ゲッター**はアクセスするときにプロパティのように見える特別なメソッドです。`get`キーワードを使って定義します:

```dart
class Circle {
  double radius;
  Circle(this.radius);

  double get area => 3.14 * radius * radius;
}

void main() {
  final c = Circle(5);
  print(c.area); // フィールドのようにアクセスし、メソッド呼び出しではない
}
```

ゲッターはフィールドから計算された値に便利です。
