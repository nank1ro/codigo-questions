# --description--

"__Hello, World!__"は、新しいプログラミング言語を学び始めるときの伝統的な最初のプログラムです。
`print()`関数を使って、標準出力デバイス（画面）にデータを出力します。
Dartで画面に`Hello, World!`と表示するには、次のように書きます
```dart
print("Hello, World!")
```
