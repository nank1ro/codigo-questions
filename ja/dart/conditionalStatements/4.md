# --description--

**`else if`** を使うと複数の条件を順番にテストできます。条件が `true` になる最初の分岐が実行され、残りはスキップされます。

```dart
int score = 75;
if (score >= 90) {
  print('A');
} else if (score >= 70) {
  print('B');
} else {
  print('C');
}
// prints: B
```
