# --description--

条件が満たされたときにメッセージを表示するために、`if` ブロック内で `print()` を使用できます。

```dart
bool isRaining = true;
if (isRaining) {
  print('Take an umbrella');
}
```

`isRaining` が `false` の場合、何も出力されません。
