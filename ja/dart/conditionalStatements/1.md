# --description--

**`if` 文**は条件が `true` のときだけコードブロックを実行します。

```dart
int age = 20;
if (age >= 18) {
  print('Adult');
}
```

波括弧の中のコードは、条件 `age >= 18` が `true` と評価された場合にのみ実行されます。
