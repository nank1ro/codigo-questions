# --description--

你可以通过在参数后面赋值来为函数的任何参数定义_默认_值。
如果定义了默认值，调用函数时可以省略该参数
```javascript
function someFunction(parameterWithoutDefault, parameterWithDefault = 12) {
    // do stuff here
}
```
