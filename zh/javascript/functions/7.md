# --description--

在函数中，我们可以添加一个_可选注释_来解释函数的作用：
```javascript
// Prints 'Hello World' to the console.
function helloWorld() {
    console.log("Hello, World!");
}
```
我们可以使用 `//` 来写单行注释，使用 `/** */` 来写多行注释
