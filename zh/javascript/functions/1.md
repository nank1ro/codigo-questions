# --description--

你可能遇到过这样的情况：想要重复使用一段代码，只是需要改变一些值。
与其重写整段代码，不如定义一个函数，这样可以反复使用，代码也更简洁。
在 JavaScript 中，我们使用 `function` 关键字后跟函数名来定义函数：
```javascript
function sayHi() {
    console.log("Hello!");
}
sayHi();
// prints "Hello!"
```
