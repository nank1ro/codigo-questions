# --description--

数组不必具有固定长度。
你可以随时将项目添加到数组的末尾！
要向数组添加项目，我们使用 `push` 函数：
```javascript
var letters = ["a", "b"];
letters.push("c");
console.log(letters);
// Prints ["a", "b", "c"]
```
