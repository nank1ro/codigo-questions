# --description--

数组索引的行为与任何其他变量名相同。
它既可以用于访问值，也可以用于赋值。
你已经看到了如何像这样访问数组索引：
```javascript
var names = ["Jeremiah", "Barney", "Ivan", "Noel"];
// Prints the value "Jeremiah"
console.log(names[0]);
```
以下是赋值的方式：
```javascript
var names = ["Jeremiah", "Barney", "Ivan", "Noel"];
// Assign the new value "Jordan"
names[0] = "Jordan";
// Prints the value "Jordan"
console.log(names[0]);
```
