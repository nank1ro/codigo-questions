# --description--

数组是一种数据类型，可用于在单个变量名下按顺序存储不同信息的集合。
数组存储一种或多种类型的多个值，并使用**索引**来区分这些值。
你可以使用以下形式的表达式将项目分配给数组：
```javascript
var arrayName = [item1, item2];
```
