# --description--

在 JavaScript 中，我们可以随意切片数组！
```javascript
// Grabs the first two items
listName.slice(0, 2);
// Grabs the fourth through last items
listName.slice(3);
```
如果你的数组切片包含数组中的第一个或最后一个项目，则不必包含该项目的索引
