# --description--

在 JavaScript 中，我们可以使用 `for..of` 关键字以非常简单的方式遍历数组：
```javascript
var numbers = [1, 2, 3];
for (num of numbers) {
    console.log(num);
}
// prints 1, 2, 3
```
`for` 关键字后面跟着一个变量名，它将依次被赋予每个数组项目的值。
