# --description--

让我们从**相等** `==` 比较运算符开始。
它返回一个**布尔值**（`true` 或 `false`），表示两个表达式是否相等，例如：
```javascript
console.log(2 == 2);
// prints true
console.log(2 == 3);
// prints false
```
