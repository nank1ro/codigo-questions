# --description--

让我们继续学习**不等于** `!=` 比较运算符。
它返回一个**布尔值**（`true` 或 `false`），表示两个表达式是否**不**相等，例如：
```javascript
console.log(2 != 2);
// prints false
console.log(2 != 3);
// prints true
```
它与*相等*运算符完全相反
