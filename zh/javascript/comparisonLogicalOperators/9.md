# --description--

最后让我们学习**非** `!` 逻辑运算符。
它返回一个布尔值，是表达式逻辑状态的反转。
```javascript
console.log(!true);
// prints false
console.log(!false);
// prints true
console.log(!(2 == 2));
// prints false
```
