# --description--

现在让我们来看**逻辑**运算符，先从第一个叫做 __AND__（与）`&&` 的开始。
它返回第一个求值为 *false* 的操作数，如果所有操作数都为 *true*，则返回最后一个。
```javascript
console.log(2 == 2 && 2 == 3);
// prints false
console.log(1 == 1 && 1 == 1.0);
// prints true
```
