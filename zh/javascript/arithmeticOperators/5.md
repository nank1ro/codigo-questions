# --description--

让我们了解**取余** `%` 运算符。
它用于求两个数字相除后的余数，例如：
```javascript
console.log(5 % 2);
// prints 1
```
结果为 1，因为 5 除以 2 的商为 2，余数为 1
```javascript
console.log(9 % 3);
// prints 0
```
这个结果为 0，因为 9 除以 3 的商为 3，余数为 0
