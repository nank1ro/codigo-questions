运算符用于对变量和值执行操作。
让我们从算术运算符开始，特别是**加法** `+` 运算符。
它用于将两个数字相加，例如：
```javascript
console.log(5 + 3);
// prints 8
```

---

让我们继续了解**减法** `-` 运算符。
它用于将一个数字从另一个数字中减去，例如：
```javascript
console.log(5 - 3);
// prints 2
```

---

让我们了解**乘法** `*` 运算符。
它用于将两个数字相乘，例如：
```javascript
console.log(5 * 3);
// prints 15
```

---

让我们了解**除法** `/` 运算符。
它用于将两个数字相除，例如：
```javascript
console.log(10 / 5);
// prints 2
```

---

让我们了解**取余** `%` 运算符。
它用于求两个数字相除后的余数，例如：
```javascript
console.log(5 % 2);
// prints 1
```
结果为 1，因为 5 除以 2 的商为 2，余数为 1
```javascript
console.log(9 % 3);
// prints 0
```
这个结果为 0，因为 9 除以 3 的商为 3，余数为 0
