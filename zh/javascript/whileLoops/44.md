# --instructions--

这段代码块会被执行吗？
```javascript
var shouldRepeat = false;

do {
    console.log("run code");
} while (shouldRepeat);
```

# --answers--

- 是的，因为 do-while 会在检查循环条件之前先执行一次
- 不会，因为 shouldRepeat 是 false
- 这段代码块会被执行两次
- 这段代码块会被执行三次
