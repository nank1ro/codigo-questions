# --instructions--

`counter += 1` 做了什么？
```javascript
var counter = 1;

while (counter < 5) {
    console.log(counter);
    counter += 1;
}
```

# --answers--

- 每次循环运行代码块时将 counter 增加 1
- 创建 counter 变量
- 每次将 counter 增加 2
- 每次将 counter 增加 3
