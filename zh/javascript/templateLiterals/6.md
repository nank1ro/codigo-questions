# --description--

我们可以在模板字面量中任意多次使用 `${}` 来插入值

# --instructions--

在这里用 `2` 和 `8` 来试试

# --seed--

```javascript
console.log(`${[/]} new friends and ${[/]} old friends`);
```

