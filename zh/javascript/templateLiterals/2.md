# --description--

模板字面量允许我们显示表达式，例如将字符串和数字组合在一起，而无需使用 `+` 号。
要创建有效的模板字面量，你需要使用反引号 \`\` 而不是双引号或单引号。
你可以使用美元符号和花括号 `${expression}` 在模板字面量中插入变量

# --seed--

```javascript
console.log([/]);
```
