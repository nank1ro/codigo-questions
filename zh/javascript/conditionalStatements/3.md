# --description--

我们刚刚学习了如何在条件满足时执行一段代码块，现在让我们看看如何在第一个条件不满足时执行另一段代码块。
如果天气好，我们就出去玩；否则，我们就待在家里。
在 JavaScript 中，我们可以使用 `else` 语句，如下所示：
```javascript
var niceWeather = true;
if (niceWeather) {
    // play outside
} else {
    // stay home
}
```
