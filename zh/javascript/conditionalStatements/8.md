# --description--

_空值合并运算符_ `a ?? b` 会解包可选值 `a`（如果它包含一个值），或者在 `a` 为 `nil` 时返回默认值 `b`。
表达式 `a` 始终是可选类型。
表达式 `b` 必须与 `a` 中存储的类型匹配。
空值合并运算符是以下代码的简写形式：
```javascript
a != nil ? a! : b;
```
