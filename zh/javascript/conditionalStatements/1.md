# --description--

当我们只想在满足特定条件时执行代码时，就需要进行决策判断。
假设我们只想在天气好的时候出去玩。
在编程中，我们可以保存一个布尔变量 `niceWeather`，并在该变量为 `true` 时执行出去玩的操作，如下所示：
```javascript
var niceWeather = true;
if (niceWeather) {
    // play outside
}
```
