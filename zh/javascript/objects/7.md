# --description--

与数组一样，我们可以使用 `Object.entries()` 方法遍历字典元素。
```javascript
var user = {"first_name": "John", "last_name": "Hood", "age": 30};
for (const [key, value] of Object.entries(user)) {
    console.log(`${key}: ${value}`);
}
```
