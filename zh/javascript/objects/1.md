# --description--

**对象**与数组类似，但你通过查找*键*而不是索引来访问值。
键可以是任何字符串或数字。
对象用花括号括起来，如下所示：
```javascript
var objectName = {"key1": 1, "key2": 2, "key3": 3};
```
这是一个名为 `objectName` 的字典，包含三个*键值对*。
键 `key1` 指向值 `1`，`key2` 指向 `2`，以此类推。
