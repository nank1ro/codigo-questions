# --description--

如果我们想要一个包含多个单词的变量名，我们使用**驼峰命名法（camelCase）**。
它是一种书写方式，短语中间的每个单词的首字母大写

# --instructions--

填空以创建一个*驼峰命名法*的变量名。

# --seed--

```javascript
first[/]ame
```

- -
- _
- N
- n
