# --solutions--

```javascript
class Animal {
    constructor(legs = 4) {
        this.legs = legs;
    }
}
class Dog extends Animal {
    constructor(breed) {
        super();
        this.breed = breed;
    }
}
let max = new Dog();
max.breed = "Chihuahua";
console.log(max.breed);
```

# --output--

Chihuahua
