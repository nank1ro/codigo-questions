# --description--

JavaScript 是一种面向对象的编程语言，这意味着它操作的是称为对象的编程结构。
你可以将对象看作是一个包含数据和函数的单一数据结构；对象的函数被称为它的方法。
例如，当你调用：
```javascript
arrayName.push("value");
```
JavaScript 会检查 `arrayName` 是否有 `push()` 方法（所有数组都有该方法），如果找到则执行该方法。
