# --description--

你可以使用_点语法_来访问实例的属性。
在点语法中，你在实例名后面紧跟属性名，用句点 `.` 分隔，中间没有空格：
```javascript
someInstance.someProperty
```
使用相同的语法，我们还可以更新属性的值
