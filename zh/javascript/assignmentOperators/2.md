# --description--

我们已经有一个初始化的变量 `total`
```javascript
var total = 5;
```
假设我们想将数字 `2` 加到变量 `total` 上，我们可以这样写
```javascript
total = total + 2;
```
没问题，这样可以！但有一种更简短的写法来实现同样的功能：
```javascript
total += 2;
```
符号 `+=` 被称为**加法赋值**运算符。
它将一个值加到变量的值上，并将结果赋给该变量。
