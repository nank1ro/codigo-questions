我们已经学过，可以使用 `=` 号来给变量赋值，例如：
```javascript
let a = 5;
```

---

我们已经有一个初始化的变量 `total`
```javascript
var total = 5;
```
假设我们想将数字 `2` 加到变量 `total` 上，我们可以这样写
```javascript
total = total + 2;
```
没问题，这样可以！但有一种更简短的写法来实现同样的功能：
```javascript
total += 2;
```
符号 `+=` 被称为**加法赋值**运算符。
它将一个值加到变量的值上，并将结果赋给该变量。

---

就像加法赋值一样，我们还有**减法赋值** `-=`。
功能是一样的，唯一的区别是它执行的是减法运算。
所以下面两种写法完全相同
```javascript
var num = num - 5;
// is equal to
num -= 5;
```

---

让我们来看看**乘法赋值** `*=` 运算符。
它将变量乘以一个值，并将结果赋给该变量。
所以下面两种写法完全相同
```javascript
var num = num * 5;
// is equal to
num *= 5;
```

---

让我们来看看**除法赋值** `/=` 运算符。
它将变量除以一个值，并将结果赋给该变量。
所以下面两种写法完全相同
```javascript
num = num / 5;
// is equal to
num /= 5;
```

---

让我们来看看**取余赋值** `%=` 运算符。
它计算变量与一个值的余数，并将结果赋给该变量。
所以下面两种写法完全相同
```javascript
num = num % 5;
// is equal to
num %= 5;
```
