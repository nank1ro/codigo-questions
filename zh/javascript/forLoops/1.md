# --description--

我们知道如何使用 `while` 循环来重复执行代码。
就像这个程序重复输出 `hello` 一样
```javascript
var counter = 0;

while (counter < 5) {
    console.log("hello");
    counter++;
}
```
但我们也可以使用 `for` 循环来实现同样的效果：
```javascript
for (let i = 0; i < 5; i++) {
    console.log("hello");
}
```
