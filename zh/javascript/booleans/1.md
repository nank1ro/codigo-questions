# --description--

JavaScript 有一种基本的布尔类型，叫做 `boolean`。
布尔值被称为逻辑值，因为它们只能是 true 或 false。
你可以对 JavaScript 中的任何表达式求值，得到两个答案之一：`true` 或 `false`。
