# --description--

当子类构造函数需要调用父类构造函数时，在**初始化列表**中使用 `super` 关键字：

```dart
class Vehicle {
  String brand;
  Vehicle(this.brand);
}

class Car extends Vehicle {
  int doors;
  Car(String brand, this.doors) : super(brand);
}
```

`super(brand)` 将 `brand` 参数传递给 `Vehicle` 的构造函数。
