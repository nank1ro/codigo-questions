# --description--

**getter** 是一种特殊方法，用于读取计算值或私有值，访问方式与属性相同。使用 `get` 关键字定义：

```dart
class Circle {
  double radius;

  Circle(this.radius);

  double get area => 3.14159 * radius * radius;
}
```

像访问字段一样访问 getter：`circle.area`（无括号）。
