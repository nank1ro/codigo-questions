# --description--

**构造函数**是在从类创建（实例化）对象时运行的特殊方法。构造函数与类同名：

```dart
class Animal {
  String name;

  Animal(String name) {
    this.name = name;
  }
}
```

使用 `new` 关键字（在 Dart 中可省略）或直接用类名创建对象：

```dart
var dog = Animal('Rex');
```
