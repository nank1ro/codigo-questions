# --description--

**方法**是定义在类内部的函数。方法描述对象的行为：

```dart
class Animal {
  String name;

  Animal(this.name);

  void speak() {
    print('$name 发出声音。');
  }
}
```

使用点符号在对象上调用方法：`dog.speak()`。
