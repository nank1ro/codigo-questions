# --description--

**抽象类**是不能直接实例化的类。它作为基类定义契约——子类必须实现的方法。抽象方法通过省略方法体来标记：

```dart
abstract class Shape {
  double area(); // 抽象方法——无方法体
}

class Circle extends Shape {
  double radius;
  Circle(this.radius);

  @override
  double area() => 3.14159 * radius * radius;
}
```

直接实例化 `Shape()` 会导致错误。
