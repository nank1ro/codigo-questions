# --description--

**继承**允许一个类（**子类**）扩展另一个类（**父类**）并复用其字段和方法。使用 `extends` 关键字：

```dart
class Animal {
  String name;
  Animal(this.name);

  void speak() {
    print('...');
  }
}

class Dog extends Animal {
  Dog(String name) : super(name);

  @override
  void speak() {
    print('Woof!');
  }
}
```

`Dog` 从 `Animal` 继承 `name` 并重写 `speak` 方法。
