# --description--

**方法重写**允许子类提供父类中已有方法的自定义实现。使用 `@override` 注解：

```dart
class Shape {
  double area() => 0;
}

class Circle extends Shape {
  double radius;
  Circle(this.radius);

  @override
  double area() => 3.14159 * radius * radius;
}
```

`@override` 注解告知 Dart（及其他开发者）你有意替换父类方法。
