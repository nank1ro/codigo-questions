# --description--

**类**是创建对象的蓝图。在 Dart 中，使用 `class` 关键字、类名以及一对花括号来定义类：

```dart
class Animal {
  // 字段和方法
}
```

按照惯例，类名使用 **PascalCase** 命名法（每个单词首字母大写）。
