# --description--

**静态成员**属于类本身，而不属于任何特定实例。使用 `static` 关键字声明，直接通过类名访问：

```dart
class MathHelper {
  static const double pi = 3.14159;

  static double circleArea(double r) => pi * r * r;
}

// 无需创建对象即可访问：
print(MathHelper.pi);
print(MathHelper.circleArea(5));
```

静态字段和方法由所有实例共享。
