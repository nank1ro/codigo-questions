# --description--

**工厂构造函数**使用 `factory` 关键字，允许控制对象的创建——例如返回缓存的实例或子类型：

```dart
class Logger {
  static final Logger _instance = Logger._internal();

  factory Logger() => _instance;

  Logger._internal();
}
```

每次调用 `Logger()` 都返回同一个实例（单例模式）。
