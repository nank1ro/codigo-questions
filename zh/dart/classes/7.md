# --description--

Dart 支持**命名构造函数**，可以定义创建对象的额外方式。命名构造函数写作 `类名.构造函数名`：

```dart
class Point {
  double x;
  double y;

  Point(this.x, this.y);

  Point.origin()
      : x = 0,
        y = 0;
}
```

然后可以用 `var p = Point.origin();` 创建原点处的对象。
