# --description--

可以在 `if` 块内使用 `print()` 在条件满足时输出消息。

```dart
bool isRaining = true;
if (isRaining) {
  print('Take an umbrella');
}
```

如果 `isRaining` 为 `false`，则不输出任何内容。
