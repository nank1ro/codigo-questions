# --description--

**`if` 语句**仅在条件为 `true` 时执行代码块。

```dart
int age = 20;
if (age >= 18) {
  print('Adult');
}
```

只有当条件 `age >= 18` 为 `true` 时，花括号内的代码才会执行。
