# --description--

**`if-else` 语句**在条件为 `true` 时执行 `if` 块，在条件为 `false` 时执行 `else` 块。

```dart
int temperature = 10;
if (temperature > 20) {
  print('Warm');
} else {
  print('Cold');
}
// prints: Cold
```

两个分支中始终恰好执行一个。
