# --description--

**`else if`** 允许按顺序测试多个条件。第一个条件为 `true` 的分支会被执行，其余分支会被跳过。

```dart
int score = 75;
if (score >= 90) {
  print('A');
} else if (score >= 70) {
  print('B');
} else {
  print('C');
}
// prints: B
```
