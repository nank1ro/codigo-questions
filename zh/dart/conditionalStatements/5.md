# --description--

**三元运算符** `condition ? expr1 : expr2` 是编写简单 `if-else` 表达式的简洁方式。

```dart
int x = 10;
String label = x > 0 ? 'positive' : 'non-positive';
print(label); // positive
```

如果条件为 `true`，则使用 `expr1`；否则使用 `expr2`。
