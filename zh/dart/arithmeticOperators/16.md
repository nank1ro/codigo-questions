# --description--

Dart 遵循标准数学**操作顺序**（优先级）：乘法和除法在加法和减法之前评估。

```dart
int result = 2 + 3 * 4; // 14，不是 20
```

使用括号改变顺序：

```dart
int result = (2 + 3) * 4; // 20
```

# --instructions--

声明一个顶级 `int` 变量，名称为 `result`，计算 `3 + 4 * 2 - 1`。让 Dart 应用其默认的操作优先级（不添加括号）。
