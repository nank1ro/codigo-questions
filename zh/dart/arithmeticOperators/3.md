# --description--

**乘法运算符** `*` 将两个值相乘。

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

将 `int` 与另一个 `int` 相乘会产生 `int`。与 `double` 相乘会产生 `double`。
