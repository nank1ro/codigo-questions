# --description--

**整数除法运算符** `~/` 将两个数相除并**截断**结果为整数（丢弃小数部分）。

```dart
int a = 10;
int b = 3;
int result = a ~/ b; // 3  (不是 3.333...)
```

当你需要整数商而不需要余数时，使用 `~/`。
