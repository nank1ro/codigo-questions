# --description--

字符串_插值_是一种以编程方式生成字符串的方法。
在 Dart 中，我们可以使用 `+` 号（连接）将两个或多个字符串组合在一起显示，例如：
```dart
print("Hello " + "Dart!");
// prints "Hello Dart!"
```
