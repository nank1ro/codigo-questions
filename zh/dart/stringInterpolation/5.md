# --description--

如果 `$` 符号后面的内容无法被识别为程序标识符，Dart 将产生编译错误：字符串中的 `$` 后面必须始终跟着一个标识符或用花括号 (`${}`) 包裹的表达式。

# --instructions--

当你尝试运行这段代码时会发生什么？

```dart
void main() {
    print("$5 friends");
}
```

# --answers--

- 编译失败，因为 `$5` 是无效的字符串插值
- 输出 `$5 friends`
- 输出 `5 friends`
