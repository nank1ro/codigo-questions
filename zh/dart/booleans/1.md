# --description--

__布尔值__是一种数据类型，只能保存两个值之一：`true` 或 `false`。
在 Dart 中，布尔类型使用关键字 `bool` 声明。

```dart
bool isRaining = true;
```

变量 `isRaining` 存储值 `true`，表示当前正在下雨。
布尔值始终以小写形式书写：`true` 和 `false`。
