# --description--

__否定运算符__ `!`（也称为"not"或"非"）翻转布尔值。
将 `!` 应用于 `true` 得到 `false`，将 `!` 应用于 `false` 得到 `true`。

```dart
bool isActive = true;
print(!isActive); // false
```

当你想检查条件的相反情况时，这很有用。
