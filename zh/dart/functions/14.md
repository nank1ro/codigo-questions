# --instructions--

编写**递归**函数 `factorial`，接受名为 `n` 的 `int` 参数，并返回 `n!`（`n` 的阶乘）。

`n` 的阶乘定义如下：
- `factorial(0) = 1`
- `factorial(n) = n * factorial(n - 1)`，当 `n > 0`

示例：
```dart
print(factorial(5)); // 120
print(factorial(3)); // 6
```

```dart
int factorial(int n) {
  if (n == 0) return 1;
  return n * factorial(n - 1);
}
```
