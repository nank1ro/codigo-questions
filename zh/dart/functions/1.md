# --description--

**函数**是一个可重复使用的代码块，用于执行特定任务。在 Dart 中，定义函数时需要指定返回类型、名称和一对括号 `()`。

当函数不返回任何值时，其返回类型为 `void`：

```dart
void sayHello() {
  print("Hello!");
}
```

通过写出函数名称加 `()` 来调用函数：

```dart
sayHello(); // prints Hello!
```
