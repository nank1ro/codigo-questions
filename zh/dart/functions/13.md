# --instructions--

编写函数 `range`，接受名为 `n` 的 `int` 参数，并返回包含从 `1` 到 `n`（含）的整数的 `List<int>`。

示例：
```dart
print(range(3)); // [1, 2, 3]
print(range(5)); // [1, 2, 3, 4, 5]
```

# --solutions--

```dart
List<int> range(int n) {
  List<int> result = [];
  for (int i = 1; i <= n; i++) {
    result.add(i);
  }
  return result;
}
```
