# --description--

你还可以创建注释块，即_多行_注释。
_多行_注释以 `/*` 开始，以 `*/` 结束。
```dart
/*
This is a multi-line
comment
*/
```

# --instructions--

补全代码以添加一个_多行_注释。
# --seed--

```dart

Comment line 1
Comment line 2
[/]
```
