Dart 和许多其他编程语言一样，允许你为代码添加注释。
这使你可以在代码旁边编写任何文本。
注释会被编译器忽略。

Dart 支持_单行_注释、_多行_注释和_文档_注释。

以下是编写_单行_注释的方式：
```dart
// This is a comment. It is not executed.
```
