# --description--

**等于运算符** `==` 比较两个值，如果它们相等则返回 `true`，如果它们不相等则返回 `false`。

```dart
int a = 5;
int b = 5;
bool result = a == b; // true
```

比较的结果总是一个 `bool` 值。
