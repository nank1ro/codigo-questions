# --instructions--

声明两个顶级 `int` 变量：`x` 值为 `15`，`y` 值为 `15`。然后声明一个顶级 `bool` 变量 `isEqual`，它使用 `>=` 运算符存储 `x` 和 `y` 比较的结果。

# --solutions--

```dart
int age = 35;
bool isWorkingAge = age >= 18 && age <= 65;

void main() {
  print(isWorkingAge);
}
```

# --asserts--

`isEqual` 必须是 `true` 且类型为 `bool`

```dart
  test('test1', () {
    expect(isWorkingAge, true, reason: '--err-t1--');
    expect(isWorkingAge, TypeMatcher<bool>(), reason: '--err-t2--');
  });
```
