# --instructions--

声明一个顶级 `bool` 变量 `isNotLocked`。使用 NOT 运算符 `!` 反转值为 `true` 的变量 `isLocked` 的值。结果应该是 `false`。

# --solutions--

```dart
bool isOffline = false;
bool isOnline = !isOffline;

void main() {
  print(isOnline);
}
```

# --asserts--

`isNotLocked` 必须是 `false` 且类型为 `bool`
