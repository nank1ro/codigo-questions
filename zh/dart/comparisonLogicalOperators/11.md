# --instructions--

声明两个顶级 `int` 变量：`a` 值为 `10`，`b` 值为 `5`。然后声明一个顶级 `bool` 变量 `areNotEqual`，它使用 `!=` 运算符存储 `a` 和 `b` 比较的结果。

# --solutions--

```dart
int a = 5;
int b = 9;
bool notEqual = a != b;

void main() {
  print(notEqual);
}
```

# --asserts--

`areNotEqual` 必须是 `true` 且类型为 `bool`

```dart
  test('test1', () {
    expect(notEqual, true, reason: '--err-t1--');
    expect(notEqual, TypeMatcher<bool>(), reason: '--err-t2--');
  });
```
