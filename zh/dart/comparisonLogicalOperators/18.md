# --instructions--

声明两个顶级 `int` 变量：`a` 值为 `12`，`b` 值为 `18`。然后声明一个顶级 `bool` 变量 `isSmaller`，它使用 `<=` 运算符存储 `a` 是否小于或等于 `b` 的比较结果。

# --solutions--

```dart
int x = 5;
int y = 3;
int z = 0;
bool result = !(x > 10) || (y < 5 && z == 0);

void main() {
  print(result);
}
```
