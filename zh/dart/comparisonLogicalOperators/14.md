# --description--

**AND 运算符** `&&` 组合两个布尔表达式。只有当**两个**边都是 `true` 时，它才返回 `true`。如果任一边是 `false`，结果就是 `false`。

```dart
bool a = true;
bool b = true;
bool result = a && b; // true

bool c = true;
bool d = false;
bool result2 = c && d; // false
```

# --instructions--

声明两个顶级 `int` 变量：`age` 值为 `25`，`score` 值为 `80`。然后声明一个顶级 `bool` 变量 `qualifies`，只有当 `age` 大于或等于 `18` 且 `score` 大于或等于 `60` 时，它才是 `true`。

# --solutions--

```dart
int age = 25;
int score = 80;
bool qualifies = age >= 18 && score >= 60;

void main() {
  print(qualifies);
}
```

# --asserts--
