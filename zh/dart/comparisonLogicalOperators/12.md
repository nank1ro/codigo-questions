# --instructions--

声明两个顶级 `int` 变量：`num1` 值为 `20`，`num2` 值为 `15`。然后声明一个顶级 `bool` 变量 `isGreater`，它使用 `>` 运算符存储 `num1` 是否大于 `num2` 的比较结果。

# --solutions--

```dart
int high = 20;
int low = 12;
bool isHigher = high > low;

void main() {
  print(isHigher);
}
```

# --asserts--

`isGreater` 必须是 `true` 且类型为 `bool`

```dart
  test('test1', () {
    expect(isHigher, true, reason: '--err-t1--');
    expect(isHigher, TypeMatcher<bool>(), reason: '--err-t2--');
  });
```
