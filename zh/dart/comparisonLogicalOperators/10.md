# --instructions--

声明两个顶级 `int` 变量：`x` 值为 `8`，`y` 值为 `8`。然后声明一个顶级 `bool` 变量 `areEqual`，它使用 `==` 运算符存储 `x` 和 `y` 比较的结果。

# --solutions--

```dart
int x = 8;
int y = 8;
bool areEqual = x == y;

void main() {
  print(areEqual);
}
```

# --asserts--

`areEqual` 必须是 `true` 且类型为 `bool`

```dart
  test('test1', () {
    expect(areEqual, true, reason: '--err-t1--');

  });
```
