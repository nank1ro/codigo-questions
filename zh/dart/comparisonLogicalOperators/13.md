# --instructions--

声明两个顶级 `int` 变量：`value1` 值为 `6`，`value2` 值为 `8`。然后声明一个顶级 `bool` 变量 `isSmaller`，它使用 `<` 运算符存储 `value1` 是否小于 `value2` 的比较结果。

# --solutions--

```dart
int price = 30;
int budget = 50;
bool canAfford = price < budget;

void main() {
  print(canAfford);
}
```

# --asserts--

`isSmaller` 必须是 `true` 且类型为 `bool`
