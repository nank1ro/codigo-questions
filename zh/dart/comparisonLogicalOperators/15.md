# --description--

**OR 运算符** `||` 组合两个布尔表达式。如果**至少一个**边是 `true`，它就返回 `true`。只有当两个边都是 `false` 时，它才返回 `false`。

```dart
bool a = true;
bool b = false;
bool result = a || b; // true

bool c = false;
bool d = false;
bool result2 = c || d; // false
```

# --instructions--

声明一个顶级 `bool` 变量 `isPassed`，只有当变量 `score` 大于 `70` 或 `bonus` 大于 `10` 时，它才是 `true`。使用变量 `score` = `65` 和 `bonus` = `15`。
