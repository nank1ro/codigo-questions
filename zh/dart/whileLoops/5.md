# --description--

`break` 关键字立即退出循环，而不管循环条件如何。

```dart
int i = 0;
while (true) {
  if (i == 3) break;
  print(i);
  i++;
}
```

没有 `break`，`while (true)` 循环将永远运行。这里当 `i` 到达 `3` 时循环停止。
