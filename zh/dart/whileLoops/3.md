# --description--

`do-while` 循环与 `while` 循环类似，但它**至少执行一次循环体** — 条件在每次迭代*之后*检查。

```dart
int i = 0;
do {
  print(i);
  i++;
} while (i < 3);
```

即使条件从一开始就是 `false`，循环体也会执行一次。
