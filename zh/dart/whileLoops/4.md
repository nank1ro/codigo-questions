# --description--

`while` 循环的条件可以包含任何布尔表达式。一旦条件求值为 `false`，循环就结束。

```dart
int n = 100;
while (n > 1) {
  n ~/= 2;
}
print(n); // 0
```

这里使用整除运算符（`~/`）在每次迭代时将 `n` 减半。
