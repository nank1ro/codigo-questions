# --description--

只要条件为 `true`，`while` 循环就会重复执行代码块。

```dart
int i = 0;
while (i < 3) {
  print(i);
  i++;
}
```

循环在每次迭代**前**检查条件。当 `i < 3` 变为 `false` 时，循环停止。
