# --description--

`int` 类型允许存储整数。
要存储小数，我们可以使用 `double` 类型：

```dart
double pi = 3.14159;
```

这个示例与前一个类似。但是这次，变量的类型是 `double`，这种类型允许以高精度存储小数。
