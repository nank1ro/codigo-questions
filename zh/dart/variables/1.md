# --description--

变量是用于存储值的容器。
Dart 中的每个变量都是一个对象（`Object`）。
要创建一个变量，我们需要给它一个__名称__，并且名称不能包含空格。
请看以下示例：

```dart
int number = 1;
```

这条语句声明了一个名为 `number` 的 `int` 类型变量。
然后将变量的值设置为数字 `1`。
声明中的 `int` 部分被称为__类型注解__，它明确告诉 Dart 变量的类型。
