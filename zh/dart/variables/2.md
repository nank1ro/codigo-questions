# --description--

在前面的示例中，我们看到了如何创建一个变量：

```dart
int number = 1;
```

不要被 `=` 符号所迷惑。
它不是数学中的等号，而是被称为__赋值运算符__，因为它将一个值赋给变量。
而等号实际上是 `==`。
