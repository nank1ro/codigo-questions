# --description--

Dart 支持__类型推断__。
不需要指明变量的类型，因为 Dart 可以推断其类型。
`var` 关键字告诉 Dart 使用最合适的类型。

```dart
var number = 5;
```

不需要告诉 Dart 数字 `5` 是 `int` 类型。
Dart 会推断类型并将 `number` 设为 `int` 类型。
