# --description--

以下是一个只能在运行时获取的值的示例：

```dart
final currentHour = DateTime.now().hour;
```

`DateTime.now()` 是一个获取代码执行时当前日期和时间的函数。
通过 `hour` 字段，我们可以访问从当天开始已经过去的小时数。

由于 `hour` 的值根据代码执行的时间不同而不同，这可以被定义为_运行时_值。

如果你尝试更改 `final` 变量的值，会得到一个错误。
