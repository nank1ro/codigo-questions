# --description--

在 Dart 中，**列表**是一种有序的元素集合。创建列表最简单的方式是使用 `[]` 字面量语法：

```dart
List<int> numbers = [1, 2, 3];
```

也可以使用 `var` 进行类型推断：

```dart
var fruits = ['apple', 'banana', 'cherry'];
```

类型注解 `List<String>` 告诉 Dart 列表中的每个元素都必须是 `String` 类型。
