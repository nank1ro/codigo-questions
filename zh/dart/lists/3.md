# --description--

`.add()` 方法将单个元素追加到列表的**末尾**：

```dart
var nums = [1, 2, 3];
nums.add(4);
print(nums); // [1, 2, 3, 4]
```

注意，`.add()` **就地**修改列表，返回值为 `void`。
