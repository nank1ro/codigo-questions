# --description--

Dart 中的列表是**从零开始索引**的，即第一个元素在索引 `0` 处，第二个在索引 `1` 处，依此类推。

```dart
var colors = ['red', 'green', 'blue'];
print(colors[0]); // red
print(colors[2]); // blue
```

通过索引访问元素使用 `list[index]` 语法。
