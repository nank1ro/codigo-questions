**赋值运算符** `=` 将值存储在变量中。

```dart
int x = 5;
String name = 'Alice';
```

赋值后，变量会保持该值，直到被重新赋值。

---

**加法赋值运算符** `+=` 将值添加到变量并将结果存储回该变量。

```dart
int score = 10;
score += 5; // score 现在是 15
```

这是 `score = score + 5` 的简写。

---

**减法赋值运算符** `-=` 从变量中减去值并将结果存储回该变量。

```dart
int lives = 3;
lives -= 1; // lives 现在是 2
```

这是 `lives = lives - 1` 的简写。

---

**乘法赋值运算符** `*=` 将变量与值相乘并将结果存储回该变量。

```dart
int side = 4;
side *= 3; // side 现在是 12
```

这是 `side = side * 3` 的简写。

---

赋值运算符将算术运算与赋值组合为一个步骤。

例如，`x += 3` 是 `x = x + 3` 的简写。
