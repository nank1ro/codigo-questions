# --description--
`break` 语句在满足条件时立即退出循环：

```dart
for (int i = 0; i < 10; i++) {
  if (i == 5) break; // stops at 5
  print(i);
}
```

当搜索某个值并希望找到后立即停止时，这非常有用。
