# --description--
`for` 循环可以通过使用递减（`i--`）和 `>=` 条件来**倒序**计数：

```dart
for (int i = 5; i >= 1; i--) {
  print(i); // 5, 4, 3, 2, 1
}
```

这在倒计时或反向迭代时非常有用。
