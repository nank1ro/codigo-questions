# --description--

在函数中，我们可以添加一个_可选的注释_来解释函数的功能：
```c
/*
 * Function:  hello_world
 * --------------------
 * prints "Hello, World!" to the screen
 */
function hello_world() {
    printf("Hello, World!\n");
}
```
我们可以使用 `//` 进行单行注释，使用 `/* */` 进行多行注释

# --instructions--
