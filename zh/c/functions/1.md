# --description--

你可能遇到过这样的情况：你想重复使用一段代码，只是使用不同的值。
与其重写整段代码，不如定义一个函数，这样可以重复调用，代码也更加简洁。
在 C 语言中，我们使用 `return_type`（返回类型）后跟 `function`（函数）名称，例如：
```c
void say_hello() {
    printf("Hello!\n");
}

int main() {
    say_hello();
    // prints "Hello!"
    return 0;
}
```
