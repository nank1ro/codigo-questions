# --instructions--

这段代码有什么问题？
```c
int manchester_united = 0;
int barcelona = 1;
printf("The score is %i : %i\n", manchester_united, barcelona);
```

# --answers--

- 代码没有问题
- manchester_united 和 barcelona 没有值
- 我们不能在格式化字符串中使用变量
- 我们不能在同一个字符串中使用两次 %i
