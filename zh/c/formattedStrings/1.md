# --description--

在 C 语言中，我们使用 `printf` 函数配合表达式来将两个或多个值组合在一起显示，例如：
```c
>>> char c = 'C';
>>> printf("Hello %c!\n", c);
Hello C!
```
`%c` 用于显示单个字符
