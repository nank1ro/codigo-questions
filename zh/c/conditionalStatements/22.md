# --instructions--

我们怎么知道 `Hello World` 总是会被打印？
```c
bool online = false;
if (online) {
    printf("User is online\n");
}
printf("Hello World\n");
```

# --answers--

- 因为它不在条件语句内部
- 因为 if 语句不能有超过一行代码
- 因为没有 else 语句
- 因为没有 else if 语句
