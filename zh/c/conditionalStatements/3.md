# --description--

我们刚刚学习了如何在条件成立时执行代码块，现在让我们看看当第一个条件不成立时如何执行另一个代码块。
如果天气好，我们出去玩；否则，我们待在家里。
在 C 语言中，我们可以使用 `else` 语句，如下所示：
```c
bool nice_weather = false;
if (nice_weather) {
    // play outside
} else {
    // stay home
}
```
