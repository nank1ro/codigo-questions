# --description--

让我们继续之前的例子。
```c
bool nice_weather = true;
if (nice_weather) {
    // play outside
}
```
我们已经看到 `if` 语句只有在条件为 `true` 时才执行代码块。
另一个需要注意的重要内容是**花括号** `{}`，它表示一个代码块。
