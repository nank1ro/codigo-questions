# --description--

让我们来看使用 `floor()` 函数进行的**向下取整除法**。
该函数返回商的整数部分，例如：
```
>>> 5.0 / 2
2.5
>>> floor(2.5)
2.0
```
也称为整数除法。结果值是一个整数，尽管结果的*类型*不一定是 int。
