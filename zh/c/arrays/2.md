# --description--

你可以通过索引访问数组中的单个元素。
索引就像一个地址，用于标识元素在数组中的位置。
索引紧跟在数组名之后，放在方括号内，如下所示：
```
list_name[index];
```

数组索引从 `0` 开始，**而不是** `1`！你可以这样访问数组的第一个元素：`list_name[0]`。
数组的第二个元素在索引 1 处：`list_name[1]`。
