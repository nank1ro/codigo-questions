# --description--

你可以通过获取数组的 `sizeof` 来计算数组的字节长度，然后需要将其除以单个元素的大小。
这样做是可行的，因为数组中的每个元素都具有相同的类型，因此大小也相同。
得到的*长度*就是数组包含的元素个数
