# --description--

最后让我们学习**not** `!` 布尔运算符。
它返回一个布尔值，该值是表达式逻辑状态的相反值。
```c
>>> !true
0
>>> !false
1
>>> !(2 == 2)
0
```

# --instructions--

打印一个布尔值，检查 `num` 是否**不**等于 `5`

# --seed--

```c
int num = 5;
printf("%d\n", [/](num[/]5));
```
