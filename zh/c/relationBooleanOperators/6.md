# --description--

让我们继续学习**小于或等于** `<=` 关系运算符。
它返回一个**布尔值**，true `1` 或 false `0`，表示一个表达式是否小于或等于另一个，例如：
```c
>>> 2 <= 2
1
>>> 3 <= 2
0
>>> 3 <= 4
1
```
