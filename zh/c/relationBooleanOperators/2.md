# --description--

让我们继续学习**不等于** `!=` 关系运算符。
它返回一个**布尔值**，true `1` 或 false `0`，表示两个表达式是否**不**相等，例如：
```c
>>> 2 != 2
0
>>> 2 != 3
1
```
它与*等于*运算符的结果完全相反
