# --description--

让我们继续学习**or** `||` 布尔运算符。
它返回第一个求值为 *true* 的操作数，如果所有操作数都为 *false*，则返回最后一个。
```c
>>> 2 == 2 || 2 == 3
1
>>> 1 == 2 || 1 == 3
0
```
