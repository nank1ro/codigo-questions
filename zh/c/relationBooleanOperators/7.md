# --description--

现在让我们来看**布尔**运算符，从第一个叫做 __and__ `&&` 的开始。
它返回第一个求值为 *false* 的操作数，如果所有操作数都为 *true*，则返回最后一个。
```c
>>> 2 == 2 && 2 == 3
0
>>> 1 == 1 && 1 == 1.0
1
```
