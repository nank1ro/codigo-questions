我们知道如何使用 `while` 循环来重复执行代码。
就像这个程序重复语句来显示 `hello`
```c
int counter = 0;

while (counter < 5) {
    printf("Hello\n");
    counter++;
}
```
但我们也可以使用 `for` 循环来实现同样的效果：
```c
for (int i = 0; i < 5; i++) {
    printf("Hello\n");
}
```

---

在 `for` 循环中，我们可以指定循环运行的次数

---

我们可以使用 `<` 循环到下一个数字（不包含），或者使用 `<=` 循环到下一个数字（包含）

---

名为 `i` 的变量是计数器变量。
我们可以给它任意名称。
它记录当前循环执行到第几次
