# --description--

我们知道如何使用 `while` 循环来重复执行代码。
就像这个程序重复语句来显示 `hello`
```c
int counter = 0;

while (counter < 5) {
    printf("Hello\n");
    counter++;
}
```
但我们也可以使用 `for` 循环来实现同样的效果：
```c
for (int i = 0; i < 5; i++) {
    printf("Hello\n");
}
```
