# --description--

Kotlin 中有两种类型的 `Set`：

- `Set` 创建后不可修改。
- `MutableSet` 创建后可以修改，即你可以添加、删除或更新其中的元素。

```kotlin
val numbers = setOf(1, 2, 3)
numbers.add(4) // [1]
```
__[1]__ 会抛出错误，因为 `Set` 是_只读的_。

要创建一个可修改的集合，请使用 `mutableSetOf` 关键字
```kotlin
val numbers = mutableSetOf(1, 2, 3)
numbers.add(4)
println(numbers)
// prints [1, 2, 3, 4]
```
