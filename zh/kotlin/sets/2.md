# --description--

`Set` 是一个没有特定顺序的__唯一__元素集合。

```kotlin
val numbers = setOf(1, 1, 2) // [1]
println(numbers)
// prints [1, 2]
```

在 __[1]__ 处，我们尝试创建一个包含两次数字 __1__ 的集合，但如你所见，每个元素必须是唯一的，第二个 __1__ 会被自动丢弃。
