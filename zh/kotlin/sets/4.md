# --description--

最常见的 `Set` 操作是使用 `in` 或 `contains()` 来测试成员资格。

```kotlin
val numbers = setOf(1, 2, 3)
println(2 in numbers) // prints true
println(numbers.contains(5)) // prints false
```

如上所示，`in` 和 `contains` 返回一个 `Bool` 值，表示传入的元素是否存在于集合中。
