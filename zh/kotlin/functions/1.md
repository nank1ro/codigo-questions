# --description--

函数是执行特定任务的代码块，可以在程序中重复使用。
在 Kotlin 中，使用 `fun` 关键字定义函数，后跟函数名和括号：
```kotlin
fun greet() {
    println("Hello!")
}
```
要调用（执行）函数，使用函数名加括号：
```kotlin
greet() // prints Hello!
```
不返回值的函数会隐式返回 `Unit`。
