赋值运算符用于将值赋给变量。
最基本的赋值运算符是 `=`，它将右边的值赋给左边的变量：
```kotlin
var x = 10
```
这里，`10` 被赋给变量 `x`。
你也可以给现有变量重新赋一个新值：
```kotlin
x = 20
```
现在 `x` 保存的值是 `20`。

---

`+=` 运算符是一种简写，它将一个值加到变量上并将结果赋回给该变量。
不必写：
```kotlin
x = x + 5
```
可以写：
```kotlin
x += 5
```
两个表达式的效果相同：将 `x` 的值增加 `5`。

---

`-=` 运算符从变量中减去一个值并将结果赋回给该变量。
不必写：
```kotlin
x = x - 3
```
可以写：
```kotlin
x -= 3
```
这会将 `x` 的值减少 `3`。

---

`*=` 运算符将变量乘以一个值并将结果赋回给该变量。
不必写：
```kotlin
x = x * 4
```
可以写：
```kotlin
x *= 4
```
这会将 `x` 乘以 `4` 并将结果存回 `x`。

---

`/=` 运算符将变量除以一个值并将结果赋回给该变量。
不必写：
```kotlin
x = x / 2
```
可以写：
```kotlin
x /= 2
```
注意：当两个操作数都是 `Int` 类型时，Kotlin 执行**整数除法**（小数部分会被丢弃）：
```kotlin
var x = 7
x /= 2
// x 现在是 3（而不是 3.5）
```
