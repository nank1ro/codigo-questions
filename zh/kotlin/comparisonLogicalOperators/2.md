# --description--

让我们继续学习**不等于** `!=` 比较运算符。
它返回一个**布尔值**（`true` 或 `false`），表示两个表达式是否**不**相等，例如：
```kotlin
println(2 != 2) // false
println(2 != 3) // true
```
它与*等于*运算符的结果正好相反
