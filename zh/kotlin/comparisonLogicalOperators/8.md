# --description--

让我们继续学习**或** `||` 逻辑运算符。
它返回第一个求值为 *true* 的操作数，或者如果所有操作数都为 *false* 则返回最后一个。
```kotlin
println(2 == 2 || 2 == 3) // true
println(1 == 2 || 1 == 3) // false
```
