# --description--

让我们继续学习**大于或等于** `>=` 比较运算符。
它返回一个**布尔值**（`true` 或 `false`），表示一个表达式是否大于或等于另一个，例如：
```kotlin
println(2 >= 2) // true
println(3 >= 2) // true
println(3 >= 4) // false
```
