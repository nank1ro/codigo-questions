# --description--

字符串_模板_是一种以编程方式生成字符串的方法。
在 Kotlin 中，我们可以使用 `+` 号（连接符）将两个或多个字符串连接在一起显示，例如：
```kotlin
println("Hello " + "Kotlin!")
// prints "Hello Kotlin!"
```
