# --description--

如果 `$` 符号后面的内容不能被识别为程序标识符，则不会发生任何特殊处理

# --instructions--

填写空白处以获得以下输出
```kotlin
5 friends
```

# --seed--

```kotlin
fun main() {
    println(" friends")
}
```

# --solutions--

```kotlin
fun main() {
    println("$5 friends")
}
```

5 friends
