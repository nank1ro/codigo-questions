# --description--

让我们来看看**取余** `%` 运算符。
它用于求两个数字相除后的余数，例如：
```kotlin
println(5 % 2) // 1
```
结果为 1，因为 5 除以 2 的商为 2，余数为 1
```kotlin
println(9 % 3) // 0
```
这个结果为 0，因为 9 除以 3 的商为 3，余数为 0
