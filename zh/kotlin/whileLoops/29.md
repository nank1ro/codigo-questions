# --instructions--

`counter += 1` 的作用是什么？
```kotlin
var counter = 1

while (counter < 5) {
    println(counter)
    counter += 1
}
```

# --answers--

- 每次循环运行代码块时将 counter 增加 1
- 它创建了 counter 变量
- 每次将 counter 增加 2
- 每次将 counter 增加 3
