# --description--

使用 _while_ 还有第二种方式，即与 `do` 关键字结合使用。
```kotlin
do {
  // Code to be repeated
} while (Boolean-expression)
```
如你所见，`do-while` 与 `while` 循环非常相似，但有一个重要区别：
> 循环体在条件求值之前先执行一次。

换句话说，`do-while` 的循环体至少会执行一次，即使条件表达式最初为 `false`。

相反，如果条件第一次就为 `false`，`while` 循环的循环体则永远不会运行。
