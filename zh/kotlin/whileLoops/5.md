# --description--

以下是在 _while_ 循环中使用 `break` 的示例：
```kotlin
var i = 0
while (i < 3) {
  i++
  if (i == 2) break // [1]
  println(i)
}
// prints 1
```

如你在 __[1]__ 处所见，当 `i` 等于 _2_ 时，我们_中断_了循环。事实上数字 2 和 3 从未被打印。
