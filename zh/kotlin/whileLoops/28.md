# --instructions--

我们如何知道这个循环不会停止？
```kotlin
var isLoading = true

while (isLoading) {
    println("Loading...")
    isLoading = true
}
```

# --answers--

- 因为变量在代码块内没有更新为 false
- 因为所有 while 循环都不会停止
- 因为变量更新为了 false
