# --description--

让我们分析这段代码。
```kotlin
var counter = 0 // [1]
while (counter < 100) { // [2]
    counter += 10 // [3]
    println(counter)
}
```
- __[1]__：我们将 `counter` 变量初始化为 __0__。
- __[2]__：_while_ 的条件表达式表示："只要 counter 小于 _100_，就重复执行循环体中的语句"。
- __[3]__：`+=` 运算符将 _10_ 加到 `counter` 上，并在一次操作中将结果赋给 `counter`。

上面代码的输出是 _10_、_20_、_30_、_40_、_50_、_60_、_70_、_80_、_90_、_100_
