# --description--

"__Hello, World!__" 是在新语言中开始编程的传统第一个程序。
我们使用 `println()` 函数将数据输出到标准输出设备（屏幕）。
要在屏幕上用 Kotlin 打印 `Hello, World!`，我们可以写
```kotlin
println("Hello, World!")
```
