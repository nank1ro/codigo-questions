# --description--

在 Kotlin 中，我们可以使用 `for..in` 关键字以非常简单的方式遍历列表：
```kotlin
val numbers = listOf(1, 2, 3)
for (num in numbers) {
    println(num)
}
// prints 1, 2, 3
```
`for` 关键字后面跟着一个变量名，它将依次被赋予列表中每个元素的值。
