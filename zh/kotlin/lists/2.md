# --description--

列表是一个有特定顺序的元素集合。Kotlin 中有两种类型的列表：

- `List` 创建后不可修改。
- `MutableList` 创建后可以修改，即你可以添加、删除或更新其中的元素。

```kotlin
val numbers = listOf(1, 3, 5)
numbers.add(7) // [1]
```
__[1]__ 会抛出错误，因为 `List` 是_只读的_。

要创建一个可修改的列表，请使用 `mutableListOf` 关键字
```kotlin
val numbers = mutableListOf(1, 3, 5)
numbers.add(7)
println(numbers)
// prints [1, 3, 5, 7]
```
