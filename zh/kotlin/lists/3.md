# --description--

你可以通过索引访问列表中的单个元素。
索引就像一个地址，标识元素在列表中的位置。
索引直接跟在列表名称后面，放在方括号中，如下所示：
```kotlin
listName[index]
```

列表索引从 `0` 开始，**不是** `1`！你可以这样访问列表的第一个元素：`listName[0]` 或 `listName.get(0)` 甚至 `listName.first()`。
列表中的第二个元素在索引 __1__ 处：`listName[1]`。
