# --description--

另一个有用的列表操作是 `contains` 方法，用于查找给定元素是否在列表中。
例如，如果你有一个名字列表，可以使用 `contains` 方法来判断某个名字是否在列表中。
```kotlin
val names = listOf("Thomas", "Donald", "Scarlett")
println(names.contains("Scarlett"))
// prints true
```
