# --description--

可变列表不必具有固定的长度。
你可以随时向列表末尾添加元素！
要向可变列表添加元素，我们使用 `add` 函数或 `+=` 快捷方式：
```kotlin
val letters = mutableListOf("a", "b")
letters.add("c")
println(letters)
// prints [a, b, c]
```
