# --description--

列表是一种数据类型，可以用来将多个不同的信息按顺序存储在一个变量名下。
列表存储一种或多种类型的多个值，并使用**索引**来区分这些值。
你可以用以下表达式将元素赋给列表：
```kotlin
val listName = listOf<itemsType>(item1, item2)
```
`itemsType` 代表列表中元素的类型，例如可以是 `Int`、`String`、`Any`...
