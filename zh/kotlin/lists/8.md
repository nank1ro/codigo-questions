# --description--

正如我们在前面的例子中看到的，我们可以使用 `add` 函数逐个添加元素。
但如果我们需要一次性添加另一个列表的所有元素，可以简单地使用 `addAll` 函数或 `+=` 快捷方式：
```kotlin
val letters = mutableListOf("a", "b")
val newLetters = listOf("c", "d", "e")
letters.addAll(newLetters)
println(letters)
// prints [a, b, c, d, e]
```
