# --description--

_Elvis 运算符_ `a ?: b` 如果可选值 `a` 包含值，则将其解包；如果 `a` 为 `null`，则返回默认值 `b`。
表达式 `a` 始终是可选类型。
表达式 `b` 必须与 `a` 中存储的类型匹配。
Elvis 运算符是以下代码的简写形式：
```kotlin
if (a != null) a else b
```
