# --description--

我们还可以在一个条件语句（`if`、`else if` 或 `else`）中嵌套另一个条件语句，以创建更复杂的结构。
```kotlin
var num = 4
if (num < 3) {
    println("the number is lower than 3")
} else {
    if (num == 3) {
        println("the number is 3")
    } else if (num == 4) {
        println("the number is 4")
    } else {
        println("the number is greather than 4")
    }
}
```
这段代码的输出是 `the number is 4`。
