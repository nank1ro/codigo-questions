# --description--

_范围_ 是由一对端点定义的值区间。
定义范围有两种基本方式：

```kotlin
var firstRange = 1..3           // [1]
var secondRange = 1 until 3     // [2]
println(firstRange)
println(secondRange)

/* prints
1..3
1..2
*/
```

- __[1]__ 使用 `..` 语法会将两个边界都包含在结果范围中。
- __[2]__ `until` 排除末端。输出显示 _3_ 不在范围内。
