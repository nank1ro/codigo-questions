# --description--

如果你只需要重复执行一段代码 `n` 次，可以使用 `repeat(times: Int)` 函数。

```kotlin
repeat(3) {
    println("repeat")
}
// prints repeat 3 times
```

你甚至可以通过以下方式访问索引
```kotlin
repeat(3) { index ->
    println(index)
}
// prints 0, 1, 2
```
