# --description--

你可以遍历一个 __String__。
```kotlin
for (char in 'abc') print(char + 1)
// prints bdc
```

在上面的示例中，我们打印了每个字符 + 1，所以 `'a'` 变成了 `'b'`，`'b'` 变成了 `'c'`，以此类推。

这是因为字符以对应其 [ASCII 码](https://en.wikipedia.org/wiki/ASCII) 的数字形式存储。

因此，将整数加到字符上会产生一个对应新码值的新字符。
