# --description--

在 Kotlin 中，我们也可以使用 `for-in` 来遍历可迭代集合，对每个元素调用给定的闭包：
```kotlin
// this is a list, we'll see about that soon
val numbers = listOf(2, 4, 6, 8, 10)
for (num in numbers) {
    println(num)
}
// prints (2, 4, 6, 8, 10)
```
