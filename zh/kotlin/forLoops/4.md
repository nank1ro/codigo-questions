# --description--

范围的默认 _步长_ 为 __1__，但你可以显式设置其他值。

你可以使用 `step` 关键字来定义 `for` 循环的 __步长__。

```kotlin
for (i in 1..10 step 2) {
    println(i)
}
// prints 1, 3, 5, 7, 9
```

如你所见，代码块以 _2_ 而不是 _1_ 的步长运行，完全改变了我们的输出。
