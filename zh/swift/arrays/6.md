# --description--

有时你只想访问数组的一部分。
考虑以下代码：
```swift
let numbers = [1, 2, 3, 4]
let slice = numbers[1...2]
print(slice)
// prints [2, 3]
```
首先，我们创建一个名为 `numbers` 的数组。
然后，我们取数组的一个子部分并存储在 slice 数组中。
我们通过在数组名称后定义要包含的索引来实现：`numbers[1...2]`。
在 Swift 中，我们可以使用 `...` 来包含最后一个索引，也可以使用 `..<` 来排除最后一个索引
