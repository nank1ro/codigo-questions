# --description--

数组不必有固定的长度。
你可以随时向数组末尾添加元素！
要向数组添加元素，我们使用 `append` 函数：
```swift
var letters = ["a", "b"]
letters.append("c")
print(letters)
// Prints ["a", "b", "c"]
```
