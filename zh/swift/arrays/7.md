# --description--

在 Swift 中，我们可以随意切片数组！
```swift
// Grabs the first two items
listName[..<2]
// Grabs the fourth through last items
listName[3...]
```
如果你的数组切片包含数组中的第一个或最后一个元素，则不必包含该元素的索引
