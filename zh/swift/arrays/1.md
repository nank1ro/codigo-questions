# --description--

数组是一种数据类型，可以用来将多条不同的信息作为序列存储在一个变量名下。
数组可以存储一种或多种类型的多个值，并使用**索引**来区分这些值。
你可以使用以下表达式将元素赋值给数组：
```swift
var arrayName: [itemsType] = [item1, item2]
```
`itemsType` 表示数组中元素的类型，例如可以是 `Int`、`String`、`Any` 等...
