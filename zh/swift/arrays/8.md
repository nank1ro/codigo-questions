# --description--

如果我们指定 `Any` 类型，数组元素可以是任何类型：
```swift
var arrayName: [Any] = ["one", 2, true]
```
事实上，上面我们依次有一个字符串、一个整数和一个布尔值。
但我们也可以在数组中嵌套数组！
