# --description--

当我们希望仅在满足特定条件时才执行代码时，就需要进行决策判断。
假设我们只想在天气好的时候去外面玩。
在编程中，我们可以保存一个布尔变量 `niceWeather`，并在该变量为 `true` 时执行去外面玩的操作，如下所示：
```swift
var niceWeather = true
if niceWeather {
    // play outside
}
```
