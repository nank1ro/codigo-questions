# --description--

_空合运算符_ `a ?? b` 对可选类型 `a` 进行解包，如果 `a` 包含一个值则返回该值，如果 `a` 为 `nil` 则返回默认值 `b`。
表达式 `a` 始终是可选类型。
表达式 `b` 必须与 `a` 中存储的类型匹配。
空合运算符是以下代码的简写形式：
```swift
a != nil ? a! : b
```
