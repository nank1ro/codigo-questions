# --description--

三元条件运算符是一种特殊的运算符，由三部分组成，形式为 `question ? answer1 : answer2`。
它是根据 `question` 是 true 还是 false 来计算两个表达式之一的快捷方式。
如果 `question` 为 true，则计算 `answer1` 并返回其值；否则，计算 `answer2` 并返回其值。
```swift
let a = 10, b = 20, c: Int
if (a < b) {
    c = a
} else {
    c = b
}
print(c)
```
上面代码的简写形式为：
```swift
let a = 10, b = 20, c: Int
c = a < b ? a : b
print(c)
```
`c` 被设置为等于 `a`，因为条件 `a < b` 为 true
