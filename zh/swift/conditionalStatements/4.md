# --description--

假设我们还有另一个条件需要检查，如以下示例所示：
```swift
var num = 3
if num == 2 {
    print("the number is 2")
} else if num == 3 {
    print("the number is 3")
} else {
    print("do something else")
}
```
这段代码的输出是 `the number is 3`。
首先，检查数字是否等于 2，结果为 false。
然后继续检查第二个语句，判断 `num` 是否等于 3，结果为 true，因此执行后面的代码块，打印 `the number is 3`
