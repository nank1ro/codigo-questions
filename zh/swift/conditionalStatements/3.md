# --description--

我们刚刚学习了如何在满足条件时执行代码块，现在让我们看看如何在第一个条件不满足时执行另一个代码块。
如果天气好，我们去外面玩；否则，我们待在家里。
在 Swift 中，我们可以使用 `else` 语句，如下所示：
```swift
var niceWeather = true
if niceWeather {
    // play outside
} else {
    // stay home
}
```
