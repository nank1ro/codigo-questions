# --description--

与数组一样，字典是_可变的_。
这意味着它们在创建后可以被修改（如果没有被声明为常量）。
这样做的一个好处是，我们可以在字典创建后向其中添加新的_键/值对_，如下所示：
```swift
dictName[newKeyName] = newValue
```
