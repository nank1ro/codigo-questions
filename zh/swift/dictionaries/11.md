# --description--

之前我们了解了如何使用 `removeValue()` 方法从字典中删除_键值对_。
我们也可以通过将键的值赋为 `nil` 来删除元素
```swift
dictName[keyName] = nil
// keyName has been removed from the dictionary dictName
```
