# --description--

**字典**与数组和元组类似，但你通过查找*键*而不是索引来访问值。
键可以是任何字符串或数字。
字典用方括号括起来，如下所示：
```swift
var dictionaryName: [String: Int] = ["key1": 1, "key2": 2, "key3": 3]
```
这是一个名为 `dictionaryName` 的字典，包含三个*键值对*。
键 `key1` 指向值 `1`，`key2` 指向 `2`，依此类推。
