# --description--

如果我们想列出字典的所有值怎么办？
可以使用 `values` 属性。

# --seed--

```swift
var user: [String: Any] = ["first_name": "John", "last_name": "Hood", "age": 30]
print([/][/])
```

# --answers--

- user
- .values()
- .values
- .value
- .allValues()

# --solutions--

```swift
var user: [String: Any] = ["first_name": "John", "last_name": "Hood", "age": 30]
print(user.values)
```

# --output--

["John", "Hood", 30]
