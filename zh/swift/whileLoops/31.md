# --instructions--

`counter += 1` 做了什么？
```swift
var counter = 1

while counter < 5 {
    print(counter)
    counter += 1
}
```

# --answers--

- 每次循环运行代码块时，它将 counter 增加 1
- 它创建了 counter 变量
- 它每次将 counter 增加 2
- 它每次将 counter 增加 3
