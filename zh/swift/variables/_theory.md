变量是用于存储数据值的容器。
Swift 中的每个变量都是一个对象。
要创建一个变量，我们需要给它一个**名称**，请注意名称中不能包含空格。
变量在你第一次给它赋值时被创建。
在 Swift 中，你可以使用 `let` 关键字声明常量，使用 `var` 关键字声明变量。
常量的值一旦设定就不能更改，而变量可以在之后被设置为不同的值。
下面是一个创建名为 `x` 的变量的示例：
```swift
var x = 1
```
这样我们就将值 `1` 赋给了名为 `x` 的变量。
如果我们打印变量 `x`，我们会得到数字 `1`：
```swift
print(x) // prints 1
```

---

变量之所以被称为变量，是因为它们存储的值可以改变。
我们可以使用 `=` 来更新 `x` 并给它一个新值。
```swift
var x = 1
print(x) // prints 1
x = 2
print(x) // prints 2
```

---

我们也可以将其他变量的值赋给变量。在这里，我们可以将 `x` 的值赋给变量 `y`
```swift
var x = 5
var y = x
print(y) // prints 5
```

---

当我们更新一个变量时，它会忘记之前的值。在这里我们可以两次显示变量 `x`，并看到它的值是如何更新的。
```swift
var x = 5
print(x) // prints 5
x = 10
print(x) // prints 10
```

---

在 Swift 中，字符串变量只能使用双引号来声明：
```swift
let x = "May"
```

---

如果我们想要一个由多个单词组成的变量名，我们使用**驼峰命名法**。
驼峰命名法是一种书写方式，短语中间的每个单词都以大写字母开头。
