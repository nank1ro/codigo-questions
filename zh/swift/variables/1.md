# --description--

变量是用于存储数据值的容器。
Swift 中的每个变量都是一个对象。
要创建一个变量，我们需要给它一个**名称**，请注意名称中不能包含空格。
变量在你第一次给它赋值时被创建。
在 Swift 中，你可以使用 `let` 关键字声明常量，使用 `var` 关键字声明变量。
常量的值一旦设定就不能更改，而变量可以在之后被设置为不同的值。
下面是一个创建名为 `x` 的变量的示例：
```swift
var x = 1
```
这样我们就将值 `1` 赋给了名为 `x` 的变量。
如果我们打印变量 `x`，我们会得到数字 `1`：
```swift
print(x) // prints 1
```
