# --description--

你可能遇到过这样的情况：想要重复使用一段代码，只是需要改变一些值。
与其重写整段代码，不如定义一个函数，这样可以反复调用，代码更加简洁。
在 Swift 中，我们使用 `func` 关键字后跟函数名来定义函数：
```swift
func say_hi() {
    print("Hello!")
}
say_hi() // prints "Hello!"
```
