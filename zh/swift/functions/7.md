# --description--

你可以通过在参数类型后赋值来为函数中的任何参数定义_默认_值。
如果定义了默认值，调用函数时可以省略该参数
```swift
func someFunction(parameterWithoutDefault: Int, parameterWithDefault: Int = 12) {
    // do stuff here
}
```
