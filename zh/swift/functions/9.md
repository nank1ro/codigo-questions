# --description--

在函数中，我们可以添加一个_可选的注释_来解释函数的功能：
```swift
/// Prints 'Hello World' to the console.
func helloWorld() {
    print("Hello, World!")
}
```
我们可以使用 `///` 进行单行注释，使用 `/** */` 进行多行注释
