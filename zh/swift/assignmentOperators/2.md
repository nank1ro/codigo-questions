# --description--

我们已经有一个初始化的变量 `total`
```swift
var total = 5
```
假设我们想将数字 `2` 加到变量 `total` 上，我们可以这样写
```swift
total = total + 2
```
没问题，可以运行！但有一种更简短的方式做同样的事：
```swift
total += 2
```
符号 `+=` 称为**加法赋值**。
它将一个值加到变量的值上，并将结果赋给该变量。
