我们已经学过，可以使用 `=` 号将值赋给变量，例如：
```swift
let a = 5
```

---

我们已经有一个初始化的变量 `total`
```swift
var total = 5
```
假设我们想将数字 `2` 加到变量 `total` 上，我们可以这样写
```swift
total = total + 2
```
没问题，可以运行！但有一种更简短的方式做同样的事：
```swift
total += 2
```
符号 `+=` 称为**加法赋值**。
它将一个值加到变量的值上，并将结果赋给该变量。

---

与加法赋值一样，我们有**减法赋值** `-=`。
功能相同，唯一的区别是它执行减法运算。
因此以下两种写法完全相同
```swift
var num = num - 5
// is equal to
num -= 5
```

---

让我们来看看**乘法赋值** `*=` 运算符。
它将变量乘以一个值，并将结果赋给该变量。
因此以下两种写法完全相同
```swift
var num = num * 5
// is equal to
num *= 5
```

---

让我们来看看**除法赋值** `/=` 运算符。
它将变量除以一个值，并将结果赋给该变量。
因此以下两种写法完全相同
```swift
num = num / 5
// is equal to
num /= 5
```

---

让我们来看看**取余赋值** `%=` 运算符。
它计算变量与一个值的余数，并将结果赋给该变量。
因此以下两种写法完全相同
```swift
num = num % 5
// is equal to
num %= 5
```
