# --description--

我们知道如何使用 `while` 循环来重复代码。
例如这个程序重复语句来显示 `hello`
```swift
var counter = 0

while counter < 5 {
    print("hello")
    counter += 1;
}
```
但我们可以用 `for` 循环来做同样的事情：
```swift
for i in 0..<5 {
    print("hello")
}
```
