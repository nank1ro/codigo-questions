# --description--

你可以使用_点语法_来访问实例的属性。
在点语法中，你在实例名称后面直接写属性名，用句点 `.` 分隔，中间没有空格：
```swift
someInstance.someProperty
```
使用同样的语法，我们也可以更新属性的值
