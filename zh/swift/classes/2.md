# --description--

_结构体_ 和 _类_ 是通用的、灵活的结构，它们构成了程序代码的基本组件。
一个基本的类或结构体仅由 `class` 或 `struct` 关键字及其名称组成，例如：
```swift
class ClassName {
    // class definition
}
struct ClassName {
    // structure definition
}
```
