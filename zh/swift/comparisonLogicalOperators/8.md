# --description--

让我们继续学习**或** `||` 逻辑运算符。
它返回第一个计算结果为 *true* 的操作数，如果所有操作数都为 *false*，则返回最后一个。
```swift
print(2 == 2 || 2 == 3) // true
print(1 == 2 || 1 == 3) // false
```
