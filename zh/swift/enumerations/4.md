# --description--

对于某些枚举，拥有该枚举所有成员的集合是很有用的。
你可以通过在枚举名称后面写 `: CaseIterable` 来启用此功能。
Swift 将所有成员的集合作为枚举类型的 `allCases` 属性暴露出来：
```swift
enum Colors: CaseIterable {
    case blue, red, green
}
for color in Colors.allCases {
    print(color)
}
// prints blue, red, green
```
