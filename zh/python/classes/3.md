# --description--

让我们用其他内容替换 `pass`。
`__init__()` 方法是类所必需的，它用于__初始化__所创建的对象。
`__init__()` 至少接受一个参数 `self`，它指向正在创建的对象。
你可以把 `__init__()` 看作是类创建每个对象时的启动方法。
