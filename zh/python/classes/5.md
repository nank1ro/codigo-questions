# --description--

`__init__()` 的第一个参数用于引用实例对象，按照惯例，该参数称为 `self`。
如果你添加额外的参数（例如，为动物添加 `gender` 和 `legs`），在 `__init__()` 的方法体中将它们分别赋值给 `self.gender` 和 `self.legs`，这样当你创建 `Animal` 类的实例对象时，你需要为每个实例提供 gender 和 legs，它们将与你创建的特定实例关联
