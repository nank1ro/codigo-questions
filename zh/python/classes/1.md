# --description--

Python 是一种面向对象的编程语言，这意味着它操作称为对象的编程构造。
你可以把对象看作一个包含数据和函数的单一数据结构；对象的函数称为它的方法。
例如，当你调用：
```python
dict_name.items()
```
Python 会检查 `my_dict` 是否有 `items()` 方法（所有字典都有该方法），如果找到就执行该方法。
