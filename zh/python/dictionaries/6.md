# --description--

如果我们想列出字典的所有键怎么办？
可以使用 `keys()` 方法。

# --seed--

```python
user = {"first_name": "John", "last_name": "Hood", "age": 30}
print([/][/])
```

# --answers--

- user
- .keys()
- .keys
- .key
- .allKeys()

# --solutions--

```python
user = {"first_name": "John", "last_name": "Hood", "age": 30}
print(user.keys())
```

# --output--

dict_keys(['first_name', 'last_name', 'age'])
