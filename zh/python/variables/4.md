# --description--

当我们更新一个变量时，它会忘记之前的值。在这里我们可以两次显示变量 `x`，看看它的值是如何更新的。
```python
>>> x = 5
>>> print(x)
5
>>> x = 10
>>> print(x)
10
```

# --instructions--

将变量 `x` 的值更新为字符串 `"updated"`。
