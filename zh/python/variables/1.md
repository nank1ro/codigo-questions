# --description--

变量是用于存储数据值的容器。
Python 中的每个变量都是一个对象，与其他编程语言不同，Python 没有声明变量的命令。
要创建一个变量，我们需要给它一个**名称**，并且注意名称不能包含空格。
变量在你第一次给它赋值时就被创建了。
创建一个名为 `x` 的变量的示例如下：
```python
x = 1
```
这样我们就将值 `1` 赋给了名为 `x` 的变量。
如果我们打印变量 `x`，我们会得到数字 `1`：
```python
>>> print(x)
1
```
