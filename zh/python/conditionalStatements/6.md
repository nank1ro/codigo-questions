# --description--

我们还可以在一个条件语句（`if`、`elif` 或 `else`）内部嵌套另一个条件语句，以创建更复杂的结构。
```python
num = 4
if (num < 3):
    print("the number is lower than 3")
else:
    if (num == 3):
        print("the number is 3")
    elif (num == 4):
        print("the number is 4")
    else:
        print("the number is greather than 4")
```
这段代码的输出是 `the number is 4`。
