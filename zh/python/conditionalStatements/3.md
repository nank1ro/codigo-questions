# --description--

我们刚刚学习了如何在条件成立时执行一段代码块，现在让我们来看看如何在第一个条件不成立时执行另一段代码块。
如果天气好，我们就出去玩；否则，我们就待在家里。
在 Python 中，我们可以使用 `else` 语句，如下所示：
```python
nice_weather = True
if (nice_weather):
    # play outside
else:
    # stay home
```
