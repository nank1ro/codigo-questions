# --instructions--

这段代码有什么__问题__？
```python
greet('Mathew')

def greet(name):
    """This function greets to the person passed in as a parameter"""
    print(f"Hello, {name}. Good morning!")

greet('Alex')
```

# --answers--

- 我们不能在定义 greet() 之前使用它
- 我们不能使用 greet() 两次
- greet() 没有返回任何值
- 我们不能使用 name 作为参数
