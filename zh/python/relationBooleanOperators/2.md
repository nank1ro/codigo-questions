# --description--

让我们继续学习**不等于** `!=` 关系运算符。
它返回一个**布尔值**（`True` 或 `False`），表示两个表达式是否**不**相等，例如：
```python
>>> 2 != 2
False
>>> 2 != 3
True
```
它与*等于*运算符恰好相反
