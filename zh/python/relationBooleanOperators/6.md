# --description--

让我们继续学习**小于或等于** `<=` 关系运算符。
它返回一个**布尔值**（`True` 或 `False`），表示一个表达式是否小于或等于另一个，例如：
```python
>>> 2 <= 2
True
>>> 3 <= 2
False
>>> 3 <= 4
True
```
