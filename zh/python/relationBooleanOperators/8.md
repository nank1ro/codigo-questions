# --description--

让我们继续学习 **or** 布尔运算符。
它返回第一个计算结果为 *True* 的操作数，如果所有操作数都为 *False*，则返回最后一个。
```python
>>> 2 == 2 or 2 == 3
True
>>> 1 == 2 or 1 == 3
False
```
