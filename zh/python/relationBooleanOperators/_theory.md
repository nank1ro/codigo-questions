让我们从**等于** `==` 关系运算符开始。
它返回一个**布尔值**（`True` 或 `False`），表示两个表达式是否相等，例如：
```python
>>> 2 == 2
True
>>> 2 == 3
False
```

---

让我们继续学习**不等于** `!=` 关系运算符。
它返回一个**布尔值**（`True` 或 `False`），表示两个表达式是否**不**相等，例如：
```python
>>> 2 != 2
False
>>> 2 != 3
True
```
它与*等于*运算符恰好相反

---

让我们继续学习**大于** `>` 关系运算符。
它返回一个**布尔值**（`True` 或 `False`），表示一个表达式是否大于另一个，例如：
```python
>>> 2 > 2
False
>>> 3 > 2
True
```

---

让我们继续学习**小于** `<` 关系运算符。
它返回一个**布尔值**（`True` 或 `False`），表示一个表达式是否小于另一个，例如：
```python
>>> 2 < 2
False
>>> 2 < 3
True
```

---

让我们继续学习**大于或等于** `>=` 关系运算符。
它返回一个**布尔值**（`True` 或 `False`），表示一个表达式是否大于或等于另一个，例如：
```python
>>> 2 >= 2
True
>>> 3 >= 2
True
>>> 3 >= 4
False
```

---

让我们继续学习**小于或等于** `<=` 关系运算符。
它返回一个**布尔值**（`True` 或 `False`），表示一个表达式是否小于或等于另一个，例如：
```python
>>> 2 <= 2
True
>>> 3 <= 2
False
>>> 3 <= 4
True
```

---

现在让我们来看**布尔**运算符，让我们从第一个叫做 `and` 的开始。
它返回第一个计算结果为 *False* 的操作数，如果所有操作数都为 *True*，则返回最后一个。
```python
>>> 2 == 2 and 2 == 3
False
>>> 1 == 1 and 1 == 1.0
True
```

---

让我们继续学习 **or** 布尔运算符。
它返回第一个计算结果为 *True* 的操作数，如果所有操作数都为 *False*，则返回最后一个。
```python
>>> 2 == 2 or 2 == 3
True
>>> 1 == 2 or 1 == 3
False
```

---

最后让我们学习 **not** 布尔运算符。
它返回一个布尔值，该值是表达式逻辑状态的反转。
```python
>>> not True
False
>>> not False
True
>>> not 2 == 2
False
```
