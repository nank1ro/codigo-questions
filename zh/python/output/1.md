# --description--

"__Hello, World!__" 是学习新编程语言时的传统第一个程序。
我们使用 `print()` 函数将数据输出到标准输出设备（屏幕）。
要在 Python 中在屏幕上打印 `Hello, World!`，我们可以这样写
```python
print("Hello, World!")
```
