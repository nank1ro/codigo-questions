# --description--

让我们看看**取模** `%` 运算符。
它用于求两个数字相除后的余数，例如：
```python
>>> 5 % 2
1
```
结果为 1，因为 5 除以 2 的商为 2，余数为 1
```python
>>> 9 % 3
0
```
这个结果为 0，因为 9 除以 3 的商为 3，余数为 0
