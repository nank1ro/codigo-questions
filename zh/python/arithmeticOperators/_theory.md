运算符用于对变量和值执行操作。
让我们从算术运算符开始，特别是**加法** `+` 运算符。
它用于将两个数字相加，例如：
```python
>>> 5+3
8
```

---

接下来让我们看看**减法** `-` 运算符。
它用于从一个数字中减去另一个数字，例如：
```python
>>> 5-3
2
```

---

让我们看看**乘法** `*` 运算符。
它用于将两个数字相乘，例如：
```python
>>> 5*3
15
```

---

让我们看看**除法** `/` 运算符。
它用于将两个数字相除，例如：
```python
>>> 10/5
2
```

---

让我们看看**取模** `%` 运算符。
它用于求两个数字相除后的余数，例如：
```python
>>> 5 % 2
1
```
结果为 1，因为 5 除以 2 的商为 2，余数为 1
```python
>>> 9 % 3
0
```
这个结果为 0，因为 9 除以 3 的商为 3，余数为 0

---

让我们看看**幂运算** `**` 运算符。
幂运算对应于底数的重复乘法：即 **b** 的 *n* 次幂是 *n* 个底数相乘的积：
![exponentiation](https://bit.ly/3zcz6Lt)
```python
>>> 5 ** 2
25
```

---

让我们看看**整除** `//` 运算符。
此运算符返回商的整数部分，例如：
```python
>>> 5.0 / 2
2.5
>>> 5.0 // 2
2.0
```
也称为整数除法。结果值是一个整数，但结果的*类型*不一定是 int。
