# --description--

让我们看看**整除** `//` 运算符。
此运算符返回商的整数部分，例如：
```python
>>> 5.0 / 2
2.5
>>> 5.0 // 2
2.0
```
也称为整数除法。结果值是一个整数，但结果的*类型*不一定是 int。
