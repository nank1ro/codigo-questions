# --description--

我们知道如何使用 `while` 循环来重复代码。
就像这个程序重复语句来显示 `hello`
```python
counter = 0

while counter < 5:
    print("hello")
    counter += 1
```
但我们也可以使用 `for` 循环来做同样的事情：
```python
for i in range(5):
    print("hello")
```
