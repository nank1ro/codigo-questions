# --description--

`range()` 函数返回一个数字序列，默认从 0 开始，默认递增 1，在指定数字之前停止。
该函数的语法如下：
```python
range(start, stop, step)
```
`start` 和 `step` 是可选的，而 `stop` 是必需的
