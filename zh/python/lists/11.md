# --description--

**元组**与列表类似，但速度更快。
然而，元组的值不能被修改。
我们通常将元组用于在程序运行期间保持不变的**只读**数据。
要创建元组，我们使用圆括号 `()`

# --instructions--

创建一个有效的元组
