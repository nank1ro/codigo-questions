# --description--

列表索引的行为与其他变量名一样！它既可以用于访问值，也可以用于赋值。
你已经看过如何这样访问列表索引：
```python
names = ["Jeremiah", "Barney", "Ivan", "Noel"]
names[0] # Gets the value "Jeremiah"
```
以下是赋值的方式：
```python
names = ["Jeremiah", "Barney", "Ivan", "Noel"]
names[0] = "Jordan"
names[0] # Gets the new value "Jordan"
```
