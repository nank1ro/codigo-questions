# --description--

列表是一种数据类型，你可以用它在一个变量名下按顺序存储一组不同的信息。
列表可以存储任何类型的多个值，并使用**索引**来区分这些值。
你可以用以下形式的表达式为列表赋值：
```python
list_name = [item1, item2]
```
