# --description--

列表不必有固定的长度。
你可以随时向列表末尾添加元素！
要向列表添加元素，我们使用 `append` 方法：
```python
>>> letters = ["a", "b"]
>>> letters.append("c")
>>> print(letters)
['a', 'b', 'c']
```
