# --description--

有时，你只想访问列表的一部分。
请看以下代码：
```python
>>> numbers = [1, 2, 3, 4]
>>> slice = numbers[1:3]
>>> print(slice)
[2, 3]
```
首先，我们创建一个名为 `numbers` 的列表。
然后，我们取列表的一个子部分并将其存储在 slice 列表中。
我们通过在列表名之后定义想要包含的索引来实现：`numbers[1:3]`。
在 Python 中，当我们以这种方式指定列表的一部分时，会包含第一个索引 `1` 处的元素，但不包含第二个索引 `3` 处的元素。
