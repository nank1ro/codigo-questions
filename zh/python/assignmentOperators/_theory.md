我们已经知道，可以使用 `=` 号来给变量赋值，例如：
```python
a = 5
```

---

我们已经有一个初始化的变量 `total`
```python
total = 5
```
假设我们想把数字 `2` 加到变量 `total` 上，我们可以这样写
```python
total = total + 2
```
没问题，这样可以！但有一种更简短的写法可以做同样的事：
```python
total += 2
```
符号 `+=` 称为**加法赋值**。
它将一个值加到变量的值上，并将结果赋给该变量。

---

与加法赋值类似，我们还有**减法赋值** `-=`。
功能是一样的，唯一的区别是它执行减法运算。
所以以下两种写法完全相同
```python
num = num - 5
# is equal to
num -= 5
```

---

让我们看看**乘法赋值** `*=` 运算符。
它将变量乘以一个值，并将结果赋给该变量。
所以以下两种写法完全相同
```python
num = num * 5
# is equal to
num *= 5
```

---

让我们看看**除法赋值** `/=` 运算符。
它将变量除以一个值，并将结果赋给该变量。
所以以下两种写法完全相同
```python
num = num / 5
# is equal to
num /= 5
```

---

让我们看看**幂赋值** `**=` 运算符。
它将变量提升到指定的幂次方，并将结果赋给该变量。
所以以下两种写法完全相同
```python
num = num ** 5
# is equal to
num **= 5
```

---

让我们看看**取模赋值** `%=` 运算符。
它计算变量与一个值的取模运算，并将结果赋给该变量。
所以以下两种写法完全相同
```python
num = num % 5
# is equal to
num %= 5
```

---

让我们看看**整除赋值** `//=` 运算符。
它将变量进行整除运算，并将结果赋给该变量。
所以以下两种写法完全相同
