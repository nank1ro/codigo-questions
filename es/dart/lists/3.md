# --description--

El método `.add()` agrega un único elemento al **final** de una lista:

```dart
var nums = [1, 2, 3];
nums.add(4);
print(nums); // [1, 2, 3, 4]
```

Ten en cuenta que `.add()` modifica la lista **en el lugar** y devuelve `void`.
