# --description--

La propiedad `.length` devuelve el número de elementos de una lista:

```dart
var scores = [95, 87, 72, 100];
print(scores.length); // 4
```

Una lista vacía tiene longitud `0`:

```dart
var empty = [];
print(empty.length); // 0
```
