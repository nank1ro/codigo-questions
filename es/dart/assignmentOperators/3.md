# --description--

The **subtraction assignment operator** `-=` resta un valor de una variable y almacena el resultado de nuevo en esa variable.

```dart
int lives = 3;
lives -= 1; // lives is now 2
```

Esta es una abreviatura para `lives = lives - 1`.

# --instructions--

Completa el código para restar `5` de la variable `stock` usando `-=`.
