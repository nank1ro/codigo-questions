# --description--

El **operador de asignación de multiplicación** `*=` multiplica una variable por un valor y almacena el resultado de nuevo en esa variable.

```dart
int side = 4;
side *= 3; // side is now 12
```

Esta es una abreviatura para `side = side * 3`.
