# --description--

The **assignment operator** `=` almacena un valor en una variable.

```dart
int x = 5;
String name = 'Alice';
```

Una vez asignada, la variable mantiene ese valor hasta que se reasigna.

# --instructions--

Completa el código para asignar el valor `42` a la variable `answer`.

# --seed--

```dart
void main() {
  int answer [/] 42;
  print(answer);
}
```
