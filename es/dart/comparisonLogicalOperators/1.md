# --description--

El **operador de igualdad** `==` compara dos valores y devuelve `true` si son iguales, o `false` si no lo son.

```dart
int a = 5;
int b = 5;
bool result = a == b; // true
```

El resultado de una comparación siempre es un valor `bool`.
