# --description--

> Antes de comenzar, quería decirte que estoy muy feliz de enseñarte Dart.
Con este mensaje aprovecho para hacerte saber que la aplicación que estás usando fue escrita justo en Dart, usando el framework Flutter. Así que es un honor para mí poder compartir mi experiencia en el lenguaje.

Dart, como muchos otros lenguajes de programación, te permite documentar tu código.
Esto te permite escribir cualquier texto junto al código.
Los comentarios son ignorados por el compilador.

Dart soporta comentarios de _una sola línea_, comentarios de _múltiples líneas_ y comentarios de _documentación_.

Así es como se escribe un comentario de _una sola línea_:
```dart
// Este es un comentario. No se ejecuta.
```

# --instructions--

¿Qué tipos de comentarios soporta el lenguaje Dart?

# --answers--

- Comentarios de _una sola línea_, comentarios de _múltiples líneas_ y comentarios de _documentación_
- Comentarios de _múltiples líneas_ y de _una sola línea_
- Dart no soporta comentarios
- Comentarios de _una sola línea_ y comentarios de _documentación_
