# --description--

La palabra clave `break` sale de un bucle inmediatamente, independientemente de la condición del bucle.

```dart
int i = 0;
while (true) {
  if (i == 3) break;
  print(i);
  i++;
}
```

Sin `break`, el bucle `while (true)` se ejecutaría para siempre. Aquí se detiene cuando `i` llega a `3`.
