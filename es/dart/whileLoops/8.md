# --instructions--

¿Qué garantiza un bucle `do-while` en comparación con un bucle `while`?

# --answers--

- El cuerpo se ejecuta al menos una vez, incluso si la condición es inicialmente `false`
- El cuerpo se ejecuta como máximo una vez
- La condición se comprueba antes de que el cuerpo se ejecute
- El bucle se ejecuta indefinidamente a menos que se use `break`
