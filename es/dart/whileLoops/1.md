# --description--

Un bucle `while` repite un bloque de código mientras una condición sea `true`.

```dart
int i = 0;
while (i < 3) {
  print(i);
  i++;
}
```

El bucle comprueba la condición **antes** de cada iteración. Cuando `i < 3` se vuelve `false`, el bucle se detiene.
