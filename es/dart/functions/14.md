# --instructions--

Escribe una función **recursiva** `factorial` que tome un parámetro `int` `n` y devuelva `n!` (el factorial de `n`).

El factorial de `n` se define como:
- `factorial(0) = 1`
- `factorial(n) = n * factorial(n - 1)` para `n > 0`

Ejemplo:
```dart
print(factorial(5)); // 120
print(factorial(3)); // 6
```

# --solutions--

```dart
int factorial(int n) {
  if (n == 0) return 1;
  return n * factorial(n - 1);
}
```
