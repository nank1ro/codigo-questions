# --instructions--

¿Qué significa el tipo de retorno `void` en una función de Dart?

```dart
void sayHi() {
  print("Hi!");
}
```

# --answers--

- La función devuelve el número cero
- La función no devuelve ningún valor
- La función devuelve una cadena vacía
- La función puede devolver cualquier tipo
