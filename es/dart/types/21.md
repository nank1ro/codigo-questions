# --instructions--

Indica la forma correcta de convertir la variable `numero` a un entero.
```dart
final numero = 3.14;
```

# --answers--

```dart
final entero = numero.toInt();
```

```dart
final entero = int(numero);
```

```dart
final int entero = numero;
```

```dart
final entero = numero as int;
```
