# --description--
La instrucción `break` sale de un bucle de inmediato cuando se cumple una condición:

```dart
for (int i = 0; i < 10; i++) {
  if (i == 5) break; // stops at 5
  print(i);
}
```

Esto es útil cuando se busca un valor y se quiere parar en cuanto se encuentre.
