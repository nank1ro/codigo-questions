# --instructions--
¿Qué hace el bucle `for-in` en Dart?

# --answers--

- Itera sobre cada elemento de una colección
- Itera un número fijo de veces usando un índice
- Se repite hasta que una condición booleana se vuelve verdadera
- Ejecuta código solo si la colección no está vacía
