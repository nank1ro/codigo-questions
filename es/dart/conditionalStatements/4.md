# --description--

**`else if`** permite probar múltiples condiciones en secuencia. Se ejecuta la primera rama cuya condición sea `true`, y el resto se omite.

```dart
int score = 75;
if (score >= 90) {
  print('A');
} else if (score >= 70) {
  print('B');
} else {
  print('C');
}
// prints: B
```
