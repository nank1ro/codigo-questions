# --description--

Puedes usar `print()` dentro de un bloque `if` para mostrar un mensaje cuando se cumple una condición.

```dart
bool isRaining = true;
if (isRaining) {
  print('Take an umbrella');
}
```

Si `isRaining` es `false`, no se imprime nada.
