# --description--

Una **sentencia `if-else`** ejecuta el bloque `if` cuando la condición es `true`, y el bloque `else` cuando es `false`.

```dart
int temperature = 10;
if (temperature > 20) {
  print('Warm');
} else {
  print('Cold');
}
// prints: Cold
```

Exactamente una de las dos ramas siempre se ejecuta.
