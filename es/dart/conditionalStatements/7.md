# --instructions--

¿Qué hace una sentencia `if` en Dart?

# --answers--

- Siempre ejecuta el código dentro de su bloque
- Ejecuta su bloque solo cuando la condición es `true`
- Repite su bloque mientras la condición sea `true`
- Define una nueva función
