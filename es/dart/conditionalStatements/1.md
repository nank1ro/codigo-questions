# --description--

Una **sentencia `if`** ejecuta un bloque de código solo cuando una condición es `true`.

```dart
int age = 20;
if (age >= 18) {
  print('Adult');
}
```

El código dentro de las llaves se ejecuta solo si la condición `age >= 18` se evalúa como `true`.
