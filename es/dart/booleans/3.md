# --description--

Una variable booleana también puede contener el valor `false`.
Usa `false` cuando algo no es el caso.

```dart
bool isLoggedIn = false;
print(isLoggedIn); // false
```

Igual que `true`, `false` debe escribirse en minúsculas.
