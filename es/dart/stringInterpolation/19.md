# --instructions--

Usa `${}` para insertar el número `500` en el espacio vacío usando interpolación de cadena 

# --seed--

```dart
void main() {
    print("There are  items");
}
```

# --solutions--

```dart
void main() {
    print("There are ${500} items");
}
```

# --output--

There are 500 items
