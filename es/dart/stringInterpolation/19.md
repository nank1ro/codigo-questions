# --instructions--

Usa `${}` para insertar el número `500` en el espacio vacío usando interpolación de cadena 

# --seed--

```dart
void main() {
    print("Hay  elementos");
}
```

# --solutions--

```dart
void main() {
    print("Hay ${500} elementos");
}
```

# --output--

Hay 500 elementos
