# --instructions--

¿Qué está mal con este código?
```dart
const manchesterUnited = 2;
const barcelona = 2
println("The final score is $manchesterUnited : $barcelona")
```

# --answers--

- No hay nada malo en el código
- manchesterUnited y barcelona no tienen valores
- No podemos usar variables dentro de las llaves
- No podemos usar $ dos veces en la misma cadena
