# --description--

Una _interpolación_ de String es una forma programática de generar una String.
En Dart podemos usar el signo `+` (concatenación) para mostrar dos o más strings juntos, como:
```dart
print("Hello " + "Dart!");
// prints "Hello Dart!"
```
