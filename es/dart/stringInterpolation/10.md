# --description--

Podemos usar llaves para insertar valores tantas veces como queramos usando la interpolación de cadenas

# --instructions--

Míralo aquí con `2` y luego con `8`

# --seed--

```dart
print("[/] nuevos amigos y [/] amigos antiguos");
```

# --answers--

- $2
- $8
- ${2}
- ${8}

# --solutions--

```dart
print("${2} nuevos amigos y ${8} amigos antiguos");
```

# --output--

2 nuevos amigos y 8 amigos antiguos
