# --description--

Podemos usar llaves para insertar valores tan a menudo como queramos usando la interpolación de cadena

# --instructions--

Míralo aquí con `2` y luego con `8`

# --seed--

```dart
print("[/] new friends and [/] old friends");
```

# --answers--

- $2
- $8
- ${2}
- ${8}

# --solutions--

```dart
print("${2} new friends and ${8} old friends");
```

# --output--

2 new friends and 8 old friends
