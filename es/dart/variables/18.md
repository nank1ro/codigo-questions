# --instructions--

¿Es siempre necesario especificar el tipo de una variable en Dart?

# --answers--

- El tipo de una variable siempre debe especificarse
- Dart soporta __inferencia de tipos__, por lo que puede inferirlo por sí solo
- Dart no soporta tipos
