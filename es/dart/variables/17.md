# --instructions--

¿Siempre es necesario especificar el tipo de una variable en Dart?

# --answers--

- El tipo de una variable siempre debe especificarse
- Dart admite la __inferencia de tipos__, por lo que es capaz de inferirlo por sí mismo
- Dart no admite tipos
