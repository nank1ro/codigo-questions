# --instructions--

¿Cuál es la diferencia entre una constante __en tiempo de compilación__ y una __de tiempo de ejecución__?

# --answers--

- Las constantes __en tiempo de compilación__ contienen valores fijos conocidos antes de ejecutar el programa, mientras que las constantes __de tiempo de ejecución__ obtienen un valor fijo solo durante la ejecución del programa
- Las constantes __en tiempo de compilación__ contienen valores fijos conocidos después de ejecutar el programa, mientras que las constantes __de tiempo de ejecución__ obtienen un valor fijo antes de ejecutar el programa
- No hay diferencia entre las dos
