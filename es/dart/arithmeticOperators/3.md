# --description--

El **operador de multiplicación** `*` multiplica dos valores juntos.

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

Multiplicar un `int` por otro `int` produce un `int`. Multiplicar con un `double` produce un `double`.
