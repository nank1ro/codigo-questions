# --description--

Dart sigue el **orden estándar de operaciones** (precedencia): la multiplicación y la división se evalúan antes que la suma y la resta.

```dart
int result = 2 + 3 * 4; // 14, not 20
```

Usa paréntesis para cambiar el orden:

```dart
int result = (2 + 3) * 4; // 20
```
