# --description--

El **operador de resta** `-` resta un valor de otro.

```dart
int a = 10;
int b = 4;
int diff = a - b; // 6
```

El resultado puede ser negativo si el segundo operando es mayor que el primero.

# --instructions--

Completa el código para restar `9` de `20` y almacena el resultado en `diff`.
