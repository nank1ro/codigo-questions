De manera similar, cuando la condición es `false`, el bloque de código dentro de la sentencia `if` se omite y la ejecución continúa después de la sentencia condicional.

# --instructions--

¿Qué muestra este código en la salida?
```c
if (false) {
    printf("Hello!\n");
}
```
