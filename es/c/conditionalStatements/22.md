# --description--

Es importante entender que solo el código dentro de las llaves de la sentencia `if` está condicionado. El código fuera de las llaves siempre se ejecuta, sin importar la condición.

# --instructions--

¿Cómo sabemos que `Hello World` siempre se imprimirá?
```c
bool online = false;
if (online) {
    printf("User is online\n");
}
printf("Hello World\n");
```

# --answers--

- Porque no está dentro de la sentencia condicional
- Porque la sentencia if no puede tener más de una línea
- Porque no hay una sentencia else
- Porque no hay una sentencia else if
