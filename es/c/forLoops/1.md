# --description--

Sabemos cómo repetir código usando un bucle `while`.
Como este programa repitiendo sentencias para mostrar `hello`
```c
int counter = 0;

while (counter < 5) {
    printf("Hello\n");
    counter++;
}
```
Pero podemos hacer lo mismo con bucles `for`:
```c
for (int i = 0; i < 5; i++) {
    printf("Hello\n");
}
```
