# --instructions--

¿Cómo sabemos que el bucle `while` salta su bloque de código?
```c
bool is_loading = false;

while (is_loading) {
    printf("Loading...\n");
}
```

# --answers--

- La variable en la condición se compara con true
- La variable en la condición se compara con false

# --solutions--
