¿Cómo sabemos que este es un bucle infinito?
```c
bool is_logged = false;

while (true) {
    printf("Please login\n");
}
```

# --answers--

- Este bucle no se ejecutará porque la condición es false
- No hay una variable en la condición
- Este bucle se ejecutará una vez
