# --description--

Veamos el operador **asignación de módulo** `%=`.
Calcula el módulo de la variable y un valor y asigna el resultado a esa variable.
Entonces los siguientes son exactamente lo mismo
```c
num = num % 5;
// es igual a
num %= 5;
```
