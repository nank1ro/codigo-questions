# --instructions--

¿Qué está mal con este código?
```c
int pieces = 5;
char label[] = "we have %i pieces", 5;
```

# --answers--

- No podemos guardar cadenas formateadas en variables de cadena
- Falta el carácter f
- No podemos usar variables dentro de %
- Nada está mal con el código
