# --instructions--

¿Qué está mal con este código?
```c
int manchester_united = 0;
int barcelona = 1;
printf("The score is %i : %i\n", manchester_united, barcelona);
```

# --answers--

- Nada está mal con el código
- manchester_united y barcelona no tienen valores
- No podemos usar variables dentro de la cadena formateada
- No podemos usar %i dos veces en la misma cadena
