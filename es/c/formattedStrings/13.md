# --instructions--

¿Qué está mal con este código?
```c
printf(0, "%i friends");
```

# --answers--

- El valor de la expresión debe venir después de la cadena
- La expresión no es un número
- La cadena le faltan comillas " "
- La expresión debe obtener un nombre de variable
