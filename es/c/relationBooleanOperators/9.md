# --description--

Terminemos con el operador booleano **no** `!`.
Devuelve un booleano que es el inverso del estado lógico de una expresión.
```c
>>> !true
0
>>> !false
1
>>> !(2 == 2)
0
```

Imprime un booleano verificando si `num` **NO** es igual a `5`
