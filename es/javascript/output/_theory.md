"__¡Hola, mundo!__" es el primer programa tradicional para comenzar a programar en un nuevo lenguaje.
Usamos la función `console.log()` para mostrar datos en el dispositivo de salida estándar (pantalla).
Para imprimir `¡Hola, mundo!` en la pantalla con JavaScript podemos escribir
```javascript
console.log("¡Hola, mundo!")
```
