# --description--

Un arreglo no tiene que tener una longitud fija.
¡Puedes agregar elementos al final de un arreglo en cualquier momento!
Para agregar un elemento a un arreglo usamos la función `push`:
```javascript
var letters = ["a", "b"];
letters.push("c");
console.log(letters);
// Prints ["a", "b", "c"]
```
