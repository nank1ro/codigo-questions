# --description--

Un array no tiene que tener una longitud fija.
¡Puedes añadir elementos al final de un array en cualquier momento!
Para agregar un elemento a un array usamos la función `push`:
```javascript
var letters = ["a", "b"];
letters.push("c");
console.log(letters);
// Prints ["a", "b", "c"]
```
