# --instructions--

¿Qué está mal con este código?
```javascript
while (isLoading) {
    console.log("Loading...");
    isLoading = false;
}
```

# --answers--

- isLoading no es un nombre de variable válido
- El bucle while tiene una sentencia print dentro del bloque de código
- El bloque de código no tiene la indentación correcta
