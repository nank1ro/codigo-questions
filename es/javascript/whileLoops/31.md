# --instructions--

¿Qué hace `counter += 1`?
```javascript
var counter = 1;

while (counter < 5) {
    console.log(counter);
    counter += 1;
}
```

# --answers--

- Aumenta counter en uno cada vez que el bucle ejecuta su bloque de código
- Crea la variable counter
- Aumenta counter en dos cada vez
- Aumenta counter en tres cada vez
