# --instructions--

¿Se ejecutará alguna vez este bloque de código?
```javascript
var shouldRepeat = false;

do {
    console.log("run code");
} while (shouldRepeat);
```

# --answers--

- Sí porque el do-while se ejecuta una vez antes de considerar la condición del bucle
- No porque shouldRepeat es false
- Este bloque de código se ejecutará dos veces
- Este bloque de código se ejecutará tres veces
