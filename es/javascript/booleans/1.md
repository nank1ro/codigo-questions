# --description--

JavaScript tiene un tipo Boolean básico, llamado `boolean`.
Los valores booleanos se conocen como lógicos, porque solo pueden ser verdaderos o falsos.
Puedes evaluar cualquier expresión en JavaScript y obtener una de dos respuestas, `true` o `false`.
