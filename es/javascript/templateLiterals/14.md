# --instructions--

¿Qué está mal con este código?
```javascript
console.log(`$(10} friends`);
```

# --answers--

- Las expresiones deben estar dentro de ${}
- El valor dentro de ${} no es un número
- A la cadena le faltan las comillas invertidas ``
- El valor dentro de ${} debe ser un nombre de variable
