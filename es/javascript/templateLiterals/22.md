# --instructions--

¿Qué está mal con este código?
```javascript
var pieces = 5;
var label = `we have ${pieces} pieces`;
```

# --answers--

- No podemos guardar literales de plantilla en variables
- Faltan los caracteres ${}
- No podemos usar variables dentro de ${}
- Nada está mal con el código
