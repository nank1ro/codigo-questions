# --instructions--

Usa `${}` para insertar el número `500` en el espacio vacío del literal de plantilla

# --seed--

```javascript
console.log(`There are  items`);
```

# --solutions--

```javascript
console.log(`There are ${500} items`);
```

# --output--

There are 500 items
