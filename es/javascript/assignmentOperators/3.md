# --description--

Al igual que en la asignación de suma, tenemos la **asignación de decremento** `-=`.
La funcionalidad es la misma, la única diferencia es que realiza la resta.
Así que los siguientes son exactamente iguales
```javascript
var num = num - 5;
// es igual a
num -= 5;
```
