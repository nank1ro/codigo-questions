Continuemos con el operador de comparación **no igual** `!=`.
Devuelve un **booleano** (`true` o `false`) indicando si dos expresiones **NO** son iguales, por ejemplo:
```javascript
console.log(2 != 2);
// prints false
console.log(2 != 3);
// prints true
```
Es exactamente lo opuesto al operador *igual*
