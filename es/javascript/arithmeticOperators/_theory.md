Los operadores se usan para realizar operaciones sobre variables y valores.
Comencemos con los operadores aritméticos, en particular con el operador de **suma** `+`.
Se usa para sumar dos números, como:
```javascript
console.log(5 + 3);
// prints 8
```

---

Continuemos con el operador de **resta** `-`.
Se usa para restar un número de otro, como:
```javascript
console.log(5 - 3);
// prints 2
```

---

Veamos el operador de **multiplicación** `*`.
Se usa para multiplicar dos números, como:
```javascript
console.log(5 * 3);
// prints 15
```

---

Veamos el operador de **división** `/`.
Se usa para dividir dos números, como:
```javascript
console.log(10 / 5);
// prints 2
```

---

Veamos el operador de **resto** `%`.
Se usa para encontrar el resto después de una división entre dos números, como:
```javascript
console.log(5 % 2);
// prints 1
```
Esto evalúa a 1 porque 5 dividido por 2 tiene un cociente de 2 y un resto de 1
```javascript
console.log(9 % 3);
// prints 0
```
Este otro evalúa a 0 porque 9 dividido por 3 tiene un cociente de 3 y deja un resto de 0
