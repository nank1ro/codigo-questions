# --description--

Sabemos cómo repetir código usando un bucle `while`.
Como este programa que repite instrucciones para mostrar `hello`
```javascript
var counter = 0;

while (counter < 5) {
    console.log("hello");
    counter++;
}
```
Pero podemos hacer lo mismo con bucles `for`:
```javascript
for (let i = 0; i < 5; i++) {
    console.log("hello");
}
```
