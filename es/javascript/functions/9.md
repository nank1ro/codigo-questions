# --instructions--

¿Cuál es el __problema__ con este fragmento de código?
```javascript
console.log(greet("Mathew"));
// This function greets to the person passed in as argument
const greet = (name) => `Hello, ${name}. Good morning!`;
console.log(greet("Alex"));
```

# --answers--

- No podemos usar greet() antes de definirla
- No podemos usar greet() dos veces
- greet() no devuelve nada
- No podemos usar name como parámetro
