# --description--

En Python podemos recorrer una lista de manera muy sencilla usando las palabras clave `for..in`:
```python
>>> numeros = [1, 2, 3]
>>> for num in numeros:
>>>     print(num)
1
2
3
```
Un nombre de variable sigue a la palabra clave `for`, se le asignará el valor de cada elemento de la lista por turno.
