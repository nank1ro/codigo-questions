# --description--

Una lista no tiene que tener una longitud fija.
¡Puedes agregar elementos al final de una lista en cualquier momento!
Para agregar un elemento a una lista usamos la palabra clave `append`:
```python
>>> letters = ["a", "b"]
>>> letters.append("c")
>>> print(letters)
['a', 'b', 'c']
```
