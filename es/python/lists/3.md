# --description--

Una lista no tiene que tener una longitud fija.
Puedes agregar elementos al final de una lista en cualquier momento que desees.
Para agregar un elemento a una lista usamos la palabra clave `append`:
```python
>>> letras = ["a", "b"]
>>> letras.append("c")
>>> print(letras)
['a', 'b', 'c']
```
