# --instructions--

¿Cómo sabemos que el bucle `while` omite su bloque de código?
```python
esta_cargando = False

while esta_cargando:
    print("Cargando...")
```

# --answers--

- La variable en la condición se compara con True
- La variable en la condición se compara con False

# --solutions--

- La variable en la condición se compara con True
