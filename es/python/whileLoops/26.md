# --instructions--

¿Qué hace `contador += 1`?
```python
contador = 1

while contador < 5:
    print(contador)
    contador += 1
```

# --answers--

- Incrementa contador en uno cada vez que el bucle ejecuta su bloque de código
- Crea la variable contador
- Incrementa contador en dos cada vez
- Incrementa contador en tres cada vez
