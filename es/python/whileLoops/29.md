¿Qué hay de malo en este código?
```python
while esta_cargando:
    print("Cargando...")
    esta_cargando = False
```

# --answers--

- esta_cargando no es un nombre de variable válido
- El bucle while tiene una sentencia print dentro del bloque de código
- El bloque de código no tiene la sangría correcta
