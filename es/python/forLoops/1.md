# --description--

Sabemos repetir cdigo usando un bucle `while`.
Como este programa que repite sentencias para mostrar `hello`
```python
counter = 0

while counter < 5:
    print("hello")
    counter += 1
```
Pero podemos hacer lo mismo con bucles `for`:
```python
for i in range(5):
    print("hello")
```
