# --description--

La indentación en Python es muy importante para definir la estructura del código.
Un código que está al mismo nivel de indentación que la declaración `if` no es parte del bloque condicional.

# --instructions--

¿Cómo sabemos que `Hello World` siempre se imprimirá?
```python
online = False;
if (online):
    print("User is online")
print("Hello World")
```

# --answers--

- Por su indentación
- Porque la instrucción if no puede tener más de una línea
- Porque no hay una sentencia else
- Porque no hay una sentencia elif
