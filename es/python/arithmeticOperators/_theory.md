Los operadores se utilizan para realizar operaciones sobre variables y valores.
Comencemos con los operadores aritméticos, en particular con el operador de **suma** `+`.
Se utiliza para sumar dos números, por ejemplo:
```python
>>> 5+3
8
```

---

Continuemos con el operador de **resta** `-`.
Se utiliza para restar un número de otro, por ejemplo:
```python
>>> 5-3
2
```

---

Veamos el operador de **multiplicación** `*`.
Se utiliza para multiplicar dos números, por ejemplo:
```python
>>> 5*3
15
```

---

Veamos el operador de **división** `/`.
Se utiliza para dividir dos números, por ejemplo:
```python
>>> 10/5
2
```

---

Veamos el operador de **módulo** `%`.
Se utiliza para encontrar el resto después de una división entre dos números, por ejemplo:
```python
>>> 5 % 2
1
```
Esto evalúa a 1 porque 5 dividido por 2 tiene un cociente de 2 y un resto de 1
```python
>>> 9 % 3
0
```
Este otro evalúa a 0 porque 9 dividido por 3 tiene un cociente de 3 y deja un resto de 0

---

Veamos el operador de **exponenciación** `**`.
La exponenciación corresponde a la multiplicación repetida de la base: es decir, **b** con exponente *n* es el producto de multiplicar *n* bases:
![exponentiation](https://bit.ly/3zcz6Lt)
```python
>>> 5 ** 2
25
```

---

Veamos el operador de **división entera** `//`.
Este operador devuelve la parte entera del cociente, por ejemplo:
```python
>>> 5.0 / 2
2.5
>>> 5.0 // 2
2.0
```
También se conoce como división entera. El valor resultante es un número entero, aunque el *tipo* del resultado no necesariamente es int.
