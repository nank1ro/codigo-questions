# --description--

Veamos el operador **módulo** `%`.
Se utiliza para encontrar el resto después de una división entre dos números, como:
```python
>>> 5 % 2
1
```
Esto se evalúa a 1 porque 5 dividido por 2 tiene un cociente de 2 y un resto de 1
```python
>>> 9 % 3
0
```
Esto otro se evalúa a 0 porque 9 dividido por 3 tiene un cociente de 3 y deja un resto de 0
