# --description--

Veamos el operador de **módulo** `%`.
Se utiliza para encontrar el resto después de una división entre dos números, por ejemplo:
```python
>>> 5 % 2
1
```
Esto evalúa a 1 porque 5 dividido por 2 tiene un cociente de 2 y un resto de 1
```python
>>> 9 % 3
0
```
Este otro evalúa a 0 porque 9 dividido por 3 tiene un cociente de 3 y deja un resto de 0
