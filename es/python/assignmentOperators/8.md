# --description--

Veamos el operador de **asignación de división entera** `//=`.

Divide la variable por un valor usando división entera y asigna el resultado a esa variable.
Por lo tanto, los siguientes son exactamente iguales
```python
numero = numero // 5
# es igual a
numero //= 5
```
