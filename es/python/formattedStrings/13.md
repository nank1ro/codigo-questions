# --instructions--

¿Qué tiene mal este código?
```python
print("{10} amigos"f)
```

# --answers--

- El carácter f debe ir antes de la cadena
- El valor entre las llaves no es un número
- A la cadena le faltan sus comillas " "
- El valor dentro de las llaves debe ser un nombre de variable
