En Python podemos usar el signo `+` para mostrar dos o más cadenas juntas, como:
```python
>>> print("Hola " + "Python!")
Hola Python!
```

---

Cada declaración de cadena formateada consta de dos partes, primero el carácter `f`, luego la cadena que queremos formatear

---

A continuación, añadimos diferentes tipos de valores entre llaves para que se muestren como una sola declaración print. Como aquí, con `{5}`

---

Pero usar el signo `+` para sumar un número como '10' a una cadena como ` "amigos"` produce un error ya que son diferentes tipos de valores

---

Las cadenas formateadas nos permiten mostrar expresiones como sumar una cadena a un número, sin ningún error.

---

Podemos usar llaves para insertar valores tantas veces como queramos dentro de la cadena formateada

---

Insertar variables como `amigos` entre las llaves también muestra su valor

---

Las cadenas formateadas se usan mejor en declaraciones print, ya que no podemos almacenarlas en variables como las cadenas normales.
