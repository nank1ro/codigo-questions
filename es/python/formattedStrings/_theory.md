En Python podemos usar el signo `+` para mostrar dos o más cadenas juntas, como:
```python
>>> print("Hello " + "Python!")
Hello Python!
```

---

Pero usar el signo `+` para sumar un número como '10' a una cadena como ` "friends"` produce un error ya que son diferentes tipos de valores

---

Las cadenas formateadas nos permiten mostrar expresiones como sumar una cadena a un número, sin ningún error.

---

Cada declaración de cadena formateada consta de dos partes, primero el carácter `f`, luego la cadena que queremos formatear

---

A continuación, agregamos el diferentes tipo de valor entre llaves para que se muestre como una declaración de impresión. Como aquí, con `{5}`

---

Insertar variables como `friends` entre las llaves también muestra su valor

---

Podemos usar llaves para insertar valores tantas veces como queramos dentro de la cadena formateada

---

Las cadenas formateadas se usan mejor en declaraciones de impresión, ya que no podemos almacenarlas en variables como cadenas normales.
