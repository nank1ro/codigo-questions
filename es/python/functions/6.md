# --instructions--

¿Cuál es el __problema__ con este fragmento de código?
```python
greet('Mathew')

def greet(nombre):
    """Esta función saluda a la persona pasada como parámetro"""
    print(f"Hello, {nombre}. Good morning!")

greet('Alex')
```

# --answers--

- No podemos usar greet() antes de definirlo
- No podemos usar greet() dos veces
- greet() no devuelve nada
- No podemos usar nombre como parámetro
