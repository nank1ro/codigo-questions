# --description--

Continuemos con el **Menor que** `<` comparison operador.
It devuelve un **booleano** (`verdadero` o `falso`) stating whether uno expresion es Menor que el other, Por ejemplo:
```swift
print(2 < 2) // falso
print(2 < 3) // verdadero
```
