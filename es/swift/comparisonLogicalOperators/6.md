# --description--

Continuemos con el **Menor o igual que** `<=` comparison operador.
It devuelve un **booleano** (`verdadero` o `falso`) stating whether uno expresion es Menor o igual que el other, Por ejemplo:
```swift
print(2 <= 2) // verdadero
print(3 <= 2) // falso
print(3 <= 4) // verdadero
```
