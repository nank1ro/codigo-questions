# --description--

Continuemos con el **no equal** `!=` comparison operador.
It devuelve un **booleano** (`verdadero` o `falso`) stating whether dos expresiones son **no** equal, Por ejemplo:
```swift
print(2 != 2) // falso
print(2 != 3) // verdadero
```
It es exactly el opposite de el *equal* operador
