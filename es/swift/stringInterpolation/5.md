# --description--

A continuación, agregamos el tipo diferente de valor entre llaves para que se muestre como una declaración de impresión. Como aquí, con `\(5)`

# --instructions--

Completa el espacio vacío para obtener esta salida
```swift
5 friends
```

# --seed--

```swift
print(" friends")
```

# --solutions--

```swift
print("\(5) friends")
```

# --output--

5 friends
