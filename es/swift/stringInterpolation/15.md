# --instructions--

¿Qué hay de malo con este código?
```swift
print("\(10} friends")
```

# --answers--

- La interpolación de cadena debe estar dentro de \()
- El valor dentro de \() no es un número
- La cadena no tiene comillas " "
- El valor dentro de \() debe ser un nombre de variable
