# --description--

Podemos usar paréntesis para insertar valores tantas veces como queramos dentro de la interpolación de cadenas

# --instructions--

Véalo aquí con `2` y luego con `8`

# --seed--

```swift
print("\([/]) new friends and \([/]) old friends")
```

# --answers--

- 8

# --solutions--

```swift
print("\(2) new friends and \(8) old friends")
```
