# --instructions--

¿Qué hay de malo con este código?
```swift
let pieces = 5
let label = "we have \(pieces) pieces"
```

# --answers--

- No podemos guardar interpolaciones de cadenas en variables
- Los caracteres \() faltan
- No podemos usar variables dentro de \()
- Nada está mal con el código
