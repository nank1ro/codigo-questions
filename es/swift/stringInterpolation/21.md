# --instructions--

¿Qué hay de malo con este código?
```swift
let manchesterUnited = 2
let barcelona = 2
print("The final score is \(manchesterUnited) : \(barcelona)")
```

# --answers--

- Nada está mal con el código
- manchesterUnited y barcelona no tienen valores
- No podemos usar variables dentro de los paréntesis
- No podemos usar \() dos veces en la misma cadena
