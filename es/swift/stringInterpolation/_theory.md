In Swift we puede usar el `+` sign un display dos o mas cadenas together, like:
```swift
print("Hello " + "Swift!") // prints "Hello Swift!"
```

---

pero usando el sign `+` un Anade un numero like '10' un a cadena like ` "friends"` produces un error como ellos son un diferente kind de valores

---

cadena interpolation allow us un display expresiones like anadiendo un cadena un a numero, sin cualquier error.

---

Every cadena interpolation sentencia consists de dos parts, el `\()` donde we Inserta el numero o variable, y el normal cadena

---

Next, we Anade el diferente kind de valor en curly braces asi que it'll display como uno Imprime sentencia. Like here, con `\(5)`

---

Inserting variables like `friends` entre el round brackets displays su valor too

---

We puede usar round brackets un Inserta valores como frecuentemente como we like inside el cadena interpolation

---

cadena interpolations son best usado en Imprime sentencias, pero we puede tambien almacenar los en variables like normal strings.
