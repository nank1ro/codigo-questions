# --description--

We already tiene un initialized variable `total`
```swift
var total = 5
```
permitir's say ese we querer un Anade el numero `2` un el variable `total`, we puede Escribe
```swift
total = total + 2
```
Okay, lo funciona! pero ahi es un shorter version un hacer el mismo thing:
```swift
total += 2
```
The sign `+=` es llamo **addition assignment**.
It anade un valor un el variable valor y assigns el resultado un ese variable.
