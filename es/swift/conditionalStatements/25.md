# --instructions--

¿Cómo sabemos que `Hello World` siempre se imprimirá?
```swift
var online = False;
if online {
    print("User is online")
}
print("Hello World")
```

# --answers--

- Porque no está dentro de un bloque de código
- Porque la instrucción if no puede tener más de una línea
- Porque no hay una instrucción else
- Porque no hay una instrucción else if
