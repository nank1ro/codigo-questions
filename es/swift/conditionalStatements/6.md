# --description--

We puede tambien nest un conditional sentencia (`si`, `sino si` o `sino`) inside another conditional sentencia, un Crea un mas complex structure.
```swift
var num = 4
if num < 3 {
    print("the number is lower than 3")
} else {
    if num == 3 {
        print("the number is 3")
    } else if num == 4 {
        print("the number is 4")
    } else {
        print("the number is greather than 4")
    }
}
```
y el salida de este codigo es `el numero es 4`.
