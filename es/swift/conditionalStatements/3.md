# --description--

We solo vio como un ejecutar un block de codigo si un condicion occurs, ahora permitir's ver como un ejecutar another block de codigo si el primero condicion fails.
We ir un jugar outside si el weather es nice; otherwise, we quedarse home.
In Swift we puede usar el `sino` sentencia, like:
```swift
var niceWeather = true
if niceWeather {
    // play outside
} else {
    // quedarse home
}
```
