# --description--

The _nil-coalescing operator_ `a ?? b` unwraps un optional `a` si lo contiene un valor, o devuelve un default valor `b` si `a` es `nil`.
The expresion `a` es siempre de un optional tipo.
The expresion `b` debe match el tipo ese es stored inside a.
The nil-coalescing operador es shorthand para el codigo below:
```swift
a != nil ? a! : b
```
