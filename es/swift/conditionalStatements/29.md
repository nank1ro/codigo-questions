# --instructions--

¿Cuál de las instrucciones print es parte del bloque de código de la instrucción?
```swift
var online = false
if online {
    print("Hello User!")
}
print("Hello World")
```

# --answers--

- print("Hello User!")
- print(Hello User!)
- print("Hello World")
- print(Hello World)
