# --description--

Variables son llamo en este way porque el valor ellos almacenar puede cambiar.
We puede Actualiza `x` por usando `=` y dando lo un nuevo valor.
```swift
var x = 1
print(x) // prints 1
x = 2
print(x) // prints 2
```
