# --description--

Cuando actualizamos una variable, olvida su valor anterior. Aquí podemos mostrar la variable `x` dos veces y ver cómo se actualiza su valor.
```swift
var x = 5
print(x) // prints 5
x = 10
print(x) // prints 10
```
