# --description--

You podria tiene considero el situation donde you habria like un reuse un piece de codigo, solo con un pocos diferente valores.
Instead de rewriting el whole codigo, it's much cleaner un Define un funcion, cual puede entonces ser usado repeatedly.
In Swift we usar el `func` palabra clave siguio por el nombre de el funcion:
```swift
func say_hi() {
    print("Hello!")
}
say_hi() // prints "Hello!"
```
