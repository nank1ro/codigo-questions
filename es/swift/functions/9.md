# --description--

In functions we puede Anade un _optional comment_ ese explains que el funcion hace:
```swift
// / Prints 'Hello World' a the console.
func helloWorld() {
    print("Hello, World!")
}
```
We puede usar `///` para un single line comentario y `/** */` para un multi line comentario
