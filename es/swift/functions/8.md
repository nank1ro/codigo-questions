# --description--

A _variadic parameter_ accepts zero o mas valores de un specified tipo.
You usar un variadic parametro un specify ese el parametro puede ser paso un varying numero de entrada valores cuando el funcion es llamo.
Escribe variadic parametros por inserting three period characters `...` despues el parametro's tipo nombre.
los valores paso un a variadic parametro son hizo available within el funcion's body como un arreglo de el appropriate tipo.
para ejemplo, un variadic parametro con un nombre de `numeros` y un tipo de `Double...` es hizo available within el funcion's body como un constante arreglo llamo numeros de tipo `[Double]`
