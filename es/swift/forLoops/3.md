# --description--

Podemos usar `..<` para hacer un bucle hasta el siguiente número excluido, o `...` para hacer un bucle hasta el siguiente número incluido

# --instructions--

Completa el bucle `for` en el siguiente código para ejecutar el bloque de código `3` veces

# --seed--

```swift
for i in 1[/]3 {
    print(i)
}
```
