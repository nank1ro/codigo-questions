# --description--

In Swift we tiene tambien el `forEach` bucle.
In fact, `forEach` llama el dado closure en cada element en el sequence en el mismo order como un `para-in` bucle:
```swift
// this es an arreglo, we'll ver about that soon
let numbers: [Int] = [1, 3, 5, 7, 9] 
numbers.forEach { num in 
    print(num)
}
```
usando el `forEach` metodo es distinct de un `para-in` bucle en dos importante ways:
1. The `break` o `continue` sentencia no puede ser usado un exit el current llamar de el body closure o un skip subsequent llama.
2. usando el `Devuelve` sentencia en el body closure podra solo exit el closure y no el outer scope, y lo no podra skip subsequent llama.
