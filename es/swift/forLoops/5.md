# --description--

La función `stride()` devuelve una secuencia de números.
Requiere los parámetros _from_, _to_ y _by_.
Esta es la sintaxis de la función:
```swift
stride(from:to:by:)
```
Tenga en cuenta que el valor `to` se excluye

# --instructions--

Completa la función `stride()` en el siguiente código para imprimir los siguientes números `(3, 6, 9, 12)`

# --seed--

```swift
for i in stride(from: [/], to: [/], by: [/]) {
    print(i)
}
```
