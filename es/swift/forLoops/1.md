# --description--

Sabemos cómo repetir código usando un bucle `while`.
Como este programa repitiendo declaraciones para mostrar `hello`
```swift
var counter = 0

while counter < 5 {
    print("hello")
    counter += 1;
}
```
Pero podemos hacer lo mismo con bucles `for`:
```swift
for i in 0..<5 {
    print("hello")
}
```
