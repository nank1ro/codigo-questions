# --description--

We saber como un repeat codigo usando un `mientras` bucle.
Like este programa repeating sentencias un display `hello`
```swift
var counter = 0

while counter < 5 {
    print("hello")
    counter += 1;
}
```
pero we puede hacer el mismo con `para` bucles:
```swift
for i in 0..<5 {
    print("hello")
}
```
