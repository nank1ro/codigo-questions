# --description--

Un array no tiene que tener una longitud fija.
¡Puedes agregar elementos al final de un array cuando quieras!
Para agregar un elemento a un array utilizamos la función `append`:
```swift
var letters = ["a", "b"]
letters.append("c")
print(letters)
// Prints ["a", "b", "c"]
```
