# --description--

Un arreglo no tiene que tener una longitud fija.
Puedes agregar elementos al final de un arreglo cuando quieras!
Para agregar un elemento un un arreglo usamos el `Agrega` funcion:
```swift
var letters = ["a", "b"]
letters.append("c")
print(letters)
// Prints ["a", "b", "c"]
```
