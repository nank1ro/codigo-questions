# --description--

You puede access un individual elemento de el arreglo por su indice.
An indice es like un address ese identifies el elemento's place en el arreglo.
el indice aparece directly despues el arreglo nombre, en entre brackets, like this:
```swift
arrayName[index]
```

arreglo indices comenzar con `0`, **no** `1`! You access el primero elemento de un arreglo like this: `arrayName[0]`.
el segundo elemento en un arreglo es en indice 1: `arrayName[1]`.
