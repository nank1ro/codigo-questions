# --description--

In Swift we puede bucle through un arreglo en un muy simple way usando el `para..in` palabras clave:
```swift
var numbers = [1, 2, 3]
for num in numbers {
    print(num)
}
// prints 1, 2, 3 
```
A variable nombre sigue el `para` palabra clave, lo podra ser assigned el valor de cada arreglo elemento en turn.
