# --description--

Un indice de arreglo se comporta como cualquier otro nombre de variable.
Se puede usar para acceder asi como para asignar valores.
Viste como acceder un un indice de arreglo asi:
```swift
var names = ["Jeremiah", "Barney", "Ivan", "Noel"]
// Imprime el valor "Jeremiah"
print(names[0])
```
Asi es como funciona una asignacion:
```swift
var names = ["Jeremiah", "Barney", "Ivan", "Noel"]
// Asigna el nuevo valor "Jordan"
names[0] = "Jordan"
// Imprime el valor "Jordan"
print(names[0])
```
