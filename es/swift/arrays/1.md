# --description--

Los arreglos son un tipo de dato que puedes usar para almacenar una coleccion de diferentes piezas de informacion como una secuencia bajo un solo nombre de variable.
An arreglo almacena multiples valores de uno o multiples tipos y usa **indexes** un distinguish estos valores.
Puedes asignar elementos un un arreglo con una expresion de la forma:
```swift
var arrayName: [itemsType] = [item1, item2]
```
`itemsType` representa el tipo de los elementos dentro del arreglo, por ejemplo, puede ser `Int`, `cadena`, `Any`...
