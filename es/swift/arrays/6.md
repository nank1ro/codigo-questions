# --description--

a veces, you solo querer un access un portion de un arreglo.
considerar el siguiendo codigo:
```swift
let numbers = [1, 2, 3, 4]
let slice = numbers[1...2]
print(slice)
// prints [2, 3]
```
primero, we Crea un arreglo llamo `numeros`.
entonces, we tomar un subsection de el arreglo y almacenar lo en el slice arreglo.
We hacer este por defining el indices we querer un incluir despues el nombre de el arreglo: `numeros[1...2]`.
In Swift we puede incluir el ultimo indice usando `...`, pero we puede tambien exclude el ultimo indice usando `..<`
