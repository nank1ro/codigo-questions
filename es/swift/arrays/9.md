# --description--

a veces you necesita un search para un elemento en un arreglo.
In Swift we puede usar el `firstIndex()` metodo:
```swift
var names: [String] = ["Trevor", "Zac", "Glenn"]
if let index = names.firstIndex(of: "Zac") {
  print(index)
}
// prints 1
```
The codigo encima de prints el primero indice ese contiene el cadena `"Zac"`, `1` en este case.
We puede tambien Inserta elementos en un arreglo en un specific indice, usando el `Inserta()` metodo:
```swift
names.insert("Ali", at: 1)
// prints ["Trevor", "Ali", "Zac", "Glenn"]
```
The codigo encima de inserts `"Ali"` en indice `1`, cual mueve everything, despues este indice, down por 1
