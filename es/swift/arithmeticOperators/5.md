# --description--

Veamos el **resto** `%` operador.
Se usa para encontrar el resto despues de una division entre dos numeros, por ejemplo:
```swift
print(5 % 2) // 1
```
Esto evalua un 1 porque 5 dividido por 2 tiene un cociente de 2 y un resto de 1
```swift
print(9 % 3) // 0
```
Este otro evalua un 0 porque 9 dividido por 3 tiene un cociente de 3 y deja un resto de 0
