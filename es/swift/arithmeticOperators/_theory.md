Los operadores se utilizan para realizar operaciones en variables y valores.
Comencemos con los operadores aritméticos, en particular con el operador de **adición** `+`.
Se utiliza para sumar dos números, como:
```swift
print(5 + 3) // 8
```

---

Continuemos con el operador de **sustracción** `-`.
Se utiliza para restar un número de otro, como:
```swift
print(5 - 3) // 2
```

---

Veamos el operador de **multiplicación** `*`.
Se utiliza para multiplicar dos números entre sí, como:
```swift
print(5 * 3) // 15
```

---

Veamos el operador de **división** `/`.
Se utiliza para dividir dos números entre sí, como:
```swift
print(10 / 5) // 2
```

---

Veamos el operador de **resto** `%`.
Se utiliza para encontrar el resto después de una división entre dos números, como:
```swift
print(5 % 2) // 1
```
Esto se evalúa como 1 porque 5 dividido por 2 tiene un cociente de 2 y un resto de 1
```swift
print(9 % 3) // 0
```
Este otro se evalúa como 0 porque 9 dividido por 3 tiene un cociente de 3 y deja un resto de 0
