# --description--

_Structures_ y _classes_ son general-purpose, flexible constructs ese become el construyendo blocks de your program's codigo.
A basic clase|struct consists solo de el `clase` o `struct` palabra clave y su nombre, Por ejemplo:
```swift
class ClassName {
    // clase definition
}
struct ClassName {
    // structure definition
}
```
