# --instructions--

¿Qué hay de malo con este código?
```swift
while isLoading {
    print("Loading...")
    isLoading = false
}
```

# --answers--

- isLoading no es un nombre de variable válido
- El bucle while tiene una instrucción print dentro del bloque de código
- El bloque de código no tiene la sangría correcta
