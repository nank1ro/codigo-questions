# --instructions--

¿Cómo sabemos que este bucle se detiene?
```swift
var isLoading = true

while isLoading {
    print("Loading...")
    isLoading = false
}
```

# --answers--

- La variable en el bucle se actualiza a false
- El bucle nunca se detendrá
