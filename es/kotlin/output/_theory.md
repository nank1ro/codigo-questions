"__¡Hola, Mundo!__" es el primer programa tradicional para comenzar a programar en un nuevo lenguaje.
Usamos la función `println()` para mostrar datos en el dispositivo de salida estándar (pantalla).
Para imprimir `Hello, World!` en la pantalla con Kotlin podemos escribir
```kotlin
println("Hello, World!")
```

---

`println()` moves the cursor to a new line after displaying its parameter, so subsequent output appears on the next line.
You can use `print()` instead, which leaves the cursor on the same line.
