# --description--

Una función de expresión única usa `=` en lugar de un cuerpo de bloque:
```kotlin
fun triple(n: Int) = n * 3
```
Esto es una forma abreviada de:
```kotlin
fun triple(n: Int): Int {
    return n * 3
}
```

# --instructions--

Completa la función de expresión única que suma dos enteros.
