# --instructions--

¿Cuáles son las principales diferencias entre una `List` y un `Set`?

# --answers--

- Un `Set` no puede contener duplicados
- Una `List` no puede contener duplicados
- No hay diferencias
- Un `Set` puede realizar operaciones de _diagrama de Venn_
