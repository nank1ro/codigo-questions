# --instructions--

Un **objeto companion** es un singleton asociado a una clase. Permite definir propiedades y métodos que pertenecen a la clase en sí, no a las instancias — similar a los miembros `static` en otros lenguajes:

```kotlin
class MathHelper {
    companion object {
        fun square(n: Int): Int = n * n
    }
}
println(MathHelper.square(4)) // 16
```

Crea una clase `Counter` con un objeto companion que tenga una propiedad `var` llamada `total` de tipo `Int` (inicializada en `0`) y un método `increment()` que suma `1` a `total`.

# --asserts--

`Counter.total` debe comenzar en `0`

```kotlin
    tryCatch(Counter.total == 0)
```

Después de un `increment()`, `total` debe ser `1`

```kotlin
    Counter.increment()
    tryCatch(Counter.total == 1)
```

Después de dos llamadas más a `increment()`, `total` debe ser `3`

```kotlin
    Counter.increment()
    Counter.increment()
    tryCatch(Counter.total == 3)
```
