# --description--

En Kotlin también podemos usar el `for-in` para colecciones iterables llamando al cierre dado en cada elemento:
```kotlin
// this is a list, we'll see about that soon
val numbers = listOf(2, 4, 6, 8, 10)
for (num in numbers) {
    println(num)
}
// prints (2, 4, 6, 8, 10)
```
