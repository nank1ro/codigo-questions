# --description--

En caso de que simplemente necesites repetir un bloque de código `n` veces, puedes usar la función `repeat(times: Int)`.

```kotlin
repeat(3) {
    println("repeat")
}
// prints repeat 3 times
```

Incluso puedes acceder al índice con
```kotlin
repeat(3) { index ->
    println(index)
}
// prints 0, 1, 2
```
