# --instructions--

¿Qué está mal en este código?
```kotlin
while (isLoading) {
    println("Loading...")
    isLoading = false
}
```

# --answers--

- isLoading no es un nombre de variable válido
- El bucle while tiene una declaración de impresión dentro del bloque de código
- El bloque de código no tiene la indentación correcta
