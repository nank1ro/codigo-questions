# --description--

Aquí hay un ejemplo del uso de `break` dentro de un bucle _while_:
```kotlin
var i = 0
while (i < 3) {
  i++
  if (i == 2) break // [1]
  println(i)
}
// prints 1
```

Como puedes ver en __[1]__ cuando `i` es igual a _2_, _rompemos_ el bucle. De hecho, los números 2 y 3 nunca se imprimen.
