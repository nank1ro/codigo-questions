# --description--

Una plantilla de cadena _template_ es una forma programática de generar una cadena.
En Kotlin podemos usar el signo `+` (concatenación) para mostrar dos o más cadenas juntas, como:
```kotlin
println("Hello " + "Kotlin!")
// prints "Hello Kotlin!"
```
