# --instructions--

¿Qué está mal en este código?
```kotlin
println("$(10} friends")
```

# --answers--

- El identificador debe estar dentro de ${}
- El valor dentro de ${} no es un número
- A la cadena le faltan sus comillas " "
- El valor dentro de ${} debe ser un nombre de variable
