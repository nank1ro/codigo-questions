# --instructions--

¿Cómo sabemos que `Hello World` siempre se imprimirá?
```kotlin
var online = false;
if (online) {
    println("User is online")
}
println("Hello World")
```

# --answers--

- Porque no está dentro de un bloque de código
- Porque la sentencia if no puede tener más de una línea
- Porque no hay una sentencia else
- Porque no hay una sentencia else if

# --solutions--

- Because it's not inside a block of code
