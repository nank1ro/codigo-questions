# --description--

Classes can contain **methods** — functions that belong to the class and operate on its properties:

```kotlin
class Counter(var count: Int) {
    fun increment() {
        count++
    }
    fun value(): Int {
        return count
    }
}
```

Methods are called using dot notation on an instance:

```kotlin
val c = Counter(0)
c.increment()
println(c.value()) // 1
```
