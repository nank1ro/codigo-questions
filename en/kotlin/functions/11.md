# --description--

A single-expression function uses `=` instead of a block body:
```kotlin
fun triple(n: Int) = n * 3
```
This is shorthand for:
```kotlin
fun triple(n: Int): Int {
    return n * 3
}
```

# --instructions--

Complete the single-expression function that adds two integers.
