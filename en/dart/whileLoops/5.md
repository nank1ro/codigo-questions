# --description--

The `break` keyword exits a loop immediately, regardless of the loop condition.

```dart
int i = 0;
while (true) {
  if (i == 3) break;
  print(i);
  i++;
}
```

Without `break`, the `while (true)` loop would run forever. Here it stops when `i` reaches `3`.
