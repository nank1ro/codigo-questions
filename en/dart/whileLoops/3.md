# --description--

A `do-while` loop is similar to a `while` loop, but it **always executes the body at least once** — the condition is checked *after* each iteration.

```dart
int i = 0;
do {
  print(i);
  i++;
} while (i < 3);
```

Even if the condition is `false` from the start, the body runs once.
