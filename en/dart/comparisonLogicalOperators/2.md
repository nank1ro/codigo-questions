# --description--

The **not equal operator** `!=` returns `true` when two values are **different**, and `false` when they are the same.

```dart
int x = 3;
int y = 7;
bool result = x != y; // true
```

It is the opposite of `==`.

# --instructions--

Complete the code to check if `4` is not equal to `9` and store the result in `isDifferent`.
