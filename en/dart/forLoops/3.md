# --description--
A `for` loop can count **downward** by using a decrement (`i--`) and a `>=` condition:

```dart
for (int i = 5; i >= 1; i--) {
  print(i); // 5, 4, 3, 2, 1
}
```

This is useful for countdowns or reverse iteration.
