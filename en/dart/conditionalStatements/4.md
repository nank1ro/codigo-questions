# --description--

**`else if`** lets you test multiple conditions in sequence. The first branch whose condition is `true` is executed, and the rest are skipped.

```dart
int score = 75;
if (score >= 90) {
  print('A');
} else if (score >= 70) {
  print('B');
} else {
  print('C');
}
// prints: B
```
