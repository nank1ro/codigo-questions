# --description--

If what follows the `$` sign isn't recognizable as a program identifier, Dart will raise a compile error: the `$` in a string must always be followed by an identifier or an expression in curly braces (`${}`).

# --instructions--

What happens when you try to run this code?

```dart
void main() {
    print("$5 friends");
}
```

# --answers--

- It fails to compile because `$5` is invalid string interpolation
- It prints `$5 friends`
- It prints `5 friends`
- It prints nothing
