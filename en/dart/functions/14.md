# --instructions--

Write a **recursive** function `factorial` that takes an `int` parameter `n` and returns `n!` (the factorial of `n`).

The factorial of `n` is defined as:
- `factorial(0) = 1`
- `factorial(n) = n * factorial(n - 1)` for `n > 0`

Example:
```dart
print(factorial(5)); // 120
print(factorial(3)); // 6
```

# --solutions--

```dart
int factorial(int n) {
  if (n == 0) return 1;
  return n * factorial(n - 1);
}
```
