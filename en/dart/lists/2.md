# --description--

Lists in Dart are **zero-indexed**, meaning the first element is at index `0`, the second at index `1`, and so on.

```dart
var colors = ['red', 'green', 'blue'];
print(colors[0]); // red
print(colors[2]); // blue
```

Accessing an element by index is done with the `list[index]` syntax.
