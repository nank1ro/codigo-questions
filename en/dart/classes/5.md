# --description--

A **method** is a function defined inside a class. Methods describe the behaviour of an object:

```dart
class Animal {
  String name;

  Animal(this.name);

  void speak() {
    print('$name makes a sound.');
  }
}
```

You call a method on an object using dot notation: `dog.speak()`.
