# --description--

The **assignment operator** `=` stores a value in a variable.

```dart
int x = 5;
String name = 'Alice';
```

Once assigned, the variable holds that value until it is reassigned.

# --instructions--

Complete the code to assign the value `42` to the variable `answer`.

# --seed--

```dart
void main() {
  int answer [/] 42;
  print(answer);
}
```
