The **assignment operator** `=` stores a value in a variable.

```dart
int x = 5;
String name = 'Alice';
```

Once assigned, the variable holds that value until it is reassigned.

---

The **addition assignment operator** `+=` adds a value to a variable and stores the result back in that variable.

```dart
int score = 10;
score += 5; // score is now 15
```

This is shorthand for `score = score + 5`.

---

The **subtraction assignment operator** `-=` subtracts a value from a variable and stores the result back in that variable.

```dart
int lives = 3;
lives -= 1; // lives is now 2
```

This is shorthand for `lives = lives - 1`.

---

The **multiplication assignment operator** `*=` multiplies a variable by a value and stores the result back in that variable.

```dart
int side = 4;
side *= 3; // side is now 12
```

This is shorthand for `side = side * 3`.

---

Assignment operators combine an arithmetic operation with an assignment in a single step.

For example, `x += 3` is shorthand for `x = x + 3`.
