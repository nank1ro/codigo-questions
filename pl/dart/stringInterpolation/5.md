# --description--

Jeśli to, co następuje po znaku `$`, nie jest rozpoznawalne jako identyfikator programu, Dart zgłosi błąd kompilacji: po `$` w ciągu znaków zawsze musi pojawić się identyfikator lub wyrażenie ujęte w nawiasy klamrowe (`${}`).

# --instructions--

Co stanie się, gdy spróbujesz uruchomić ten kod?

```dart
void main() {
    print("$5 friends");
}
```

# --answers--

- Nie kompiluje się, ponieważ `$5` to nieprawidłowa interpolacja ciągu znaków
- Wypisuje `$5 friends`
- Wypisuje `5 friends`
- Nic nie wypisuje
