# --instructions--

Użyj `${}` aby wstawić liczbę `500` w puste miejsce używając interpolacji ciągów

# --seed--

```dart
void main() {
    print("There are  items");
}
```

# --solutions--

```dart
void main() {
    print("There are ${500} items");
}
```
