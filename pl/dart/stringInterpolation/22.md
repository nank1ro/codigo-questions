# --instructions--

Co jest nie tak z tym kodem?
```dart
const manchesterUnited = 2;
const barcelona = 2
println("The final score is $manchesterUnited : $barcelona")
```

# --answers--

- Nic nie jest nie tak z kodem
- manchesterUnited i barcelona nie mają wartości
- Nie możemy używać zmiennych wewnątrz nawiasów klamrowych
- Nie możemy używać $ dwa razy w tym samym ciągu
