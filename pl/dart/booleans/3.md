# --description--

Zmienna logiczna może również przechowywać wartość `false`.
Użyj `false`, gdy coś nie jest prawdą.

```dart
bool isLoggedIn = false;
print(isLoggedIn); // false
```

Tak jak `true`, `false` musi być napisane małymi literami.
