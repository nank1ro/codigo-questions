# --description--

Metoda `.add()` dołącza pojedynczy element na **końcu** listy:

```dart
var nums = [1, 2, 3];
nums.add(4);
print(nums); // [1, 2, 3, 4]
```

Należy pamiętać, że `.add()` modyfikuje listę **w miejscu** i zwraca `void`.
