# --description--

Właściwość `.length` zwraca liczbę elementów na liście:

```dart
var scores = [95, 87, 72, 100];
print(scores.length); // 4
```

Pusta lista ma długość `0`:

```dart
var empty = [];
print(empty.length); // 0
```
