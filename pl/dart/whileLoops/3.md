# --description--

Pętla `do-while` jest podobna do pętli `while`, ale **zawsze wykonuje ciało przynajmniej raz** — warunek jest sprawdzany *po* każdej iteracji.

```dart
int i = 0;
do {
  print(i);
  i++;
} while (i < 3);
```

Nawet jeśli warunek jest `false` od początku, ciało wykona się raz.
