# --description--

Pętla `while` powtarza blok kodu tak długo, jak warunek jest `true`.

```dart
int i = 0;
while (i < 3) {
  print(i);
  i++;
}
```

Pętla sprawdza warunek **przed** każdą iteracją. Gdy `i < 3` staje się `false`, pętla się zatrzymuje.
