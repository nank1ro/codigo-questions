# --description--

Zmienna licznikowa kontroluje, ile razy pętla `while` się wykonuje.

```dart
int count = 0;
while (count < 5) {
  count++;
}
print(count); // 5
```

Licznik zaczyna od `0`, a pętla zwiększa go przy każdej iteracji, dopóki `count < 5` nie stanie się `false`.
