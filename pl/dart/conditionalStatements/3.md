# --description--

Instrukcja **`if-else`** wykonuje blok `if`, gdy warunek jest `true`, a blok `else`, gdy jest `false`.

```dart
int temperature = 10;
if (temperature > 20) {
  print('Warm');
} else {
  print('Cold');
}
// prints: Cold
```

Zawsze wykonywana jest dokładnie jedna z dwóch gałęzi.

# --instructions--

Uzupełnij instrukcję `if-else` tak, aby drukowała `'Positive'`, gdy `number` jest większy niż `0`, a `'Non-positive'` w przeciwnym razie.
