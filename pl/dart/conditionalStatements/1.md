# --description--

Instrukcja **`if`** wykonuje blok kodu tylko wtedy, gdy warunek jest `true`.

```dart
int age = 20;
if (age >= 18) {
  print('Adult');
}
```

Kod wewnątrz nawiasów klamrowych jest wykonywany tylko wtedy, gdy warunek `age >= 18` jest `true`.
