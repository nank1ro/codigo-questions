# --description--

Możesz użyć `print()` wewnątrz bloku `if`, aby wyświetlić wiadomość, gdy warunek jest spełniony.

```dart
bool isRaining = true;
if (isRaining) {
  print('Take an umbrella');
}
```

Jeśli `isRaining` jest `false`, nic nie jest drukowane.
