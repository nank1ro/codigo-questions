# --description--

**`else if`** pozwala testować wiele warunków po kolei. Wykonywana jest pierwsza gałąź, której warunek jest `true`, a pozostałe są pomijane.

```dart
int score = 75;
if (score >= 90) {
  print('A');
} else if (score >= 70) {
  print('B');
} else {
  print('C');
}
// prints: B
```
