# --description--
Instrukcja `break` natychmiast kończy pętlę, gdy spełniony jest warunek:

```dart
for (int i = 0; i < 10; i++) {
  if (i == 5) break; // stops at 5
  print(i);
}
```

Jest to przydatne, gdy szukamy wartości i chcemy zatrzymać się po jej znalezieniu.
