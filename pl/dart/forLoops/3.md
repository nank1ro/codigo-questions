# --description--
Pętla `for` może liczyć **wstecz** używając dekrementacji (`i--`) i warunku `>=`:

```dart
for (int i = 5; i >= 1; i--) {
  print(i); // 5, 4, 3, 2, 1
}
```

Jest to przydatne przy odliczaniu wstecz lub iteracji odwrotnej.
