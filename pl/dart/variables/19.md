# --instructions--

Jaka jest różnica między stałą __czasu kompilacji__ a stałą __czasu wykonania__?

# --answers--

- Stałe __czasu kompilacji__ zawierają stałe wartości znane przed uruchomieniem programu, natomiast stałe __czasu wykonania__ uzyskują stałą wartość dopiero podczas wykonywania programu
- Stałe __czasu kompilacji__ zawierają stałe wartości znane po uruchomieniu programu, natomiast stałe __czasu wykonania__ uzyskują stałą wartość przed uruchomieniem programu
- Nie ma różnicy między nimi
