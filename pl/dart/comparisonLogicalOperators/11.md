# --instructions--

Deklaruj dwie zmienne `int` na poziomie górnym: `a` o wartości `10` i `b` o wartości `5`. Następnie deklaruj zmienną `bool` na poziomie górnym o nazwie `areNotEqual`, która przechowuje wynik porównania `a` i `b` za pomocą operatora `!=`.

# --solutions--

```dart
int a = 5;
int b = 9;
bool notEqual = a != b;

void main() {
  print(notEqual);
}
```

# --asserts--

`areNotEqual` musi być `true` i być typu `bool`

```dart
  test('test1', () {
    expect(notEqual, true, reason: '--err-t1--');
    expect(notEqual, TypeMatcher<bool>(), reason: '--err-t2--');
  });
```
