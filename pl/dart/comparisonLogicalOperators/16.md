# --instructions--

Deklaruj zmienną `bool` na poziomie górnym o nazwie `isNotLocked`. Użyj operatora NOT `!`, aby odwrócić wartość zmiennej `isLocked`, która jest `true`. Wynikiem powinno być `false`.

# --solutions--

```dart
bool isOffline = false;
bool isOnline = !isOffline;

void main() {
  print(isOnline);
}
```

# --asserts--

`isNotLocked` musi być `false` i być typu `bool`
