# --instructions--

Deklaruj dwie zmienne `int` na poziomie górnym: `x` o wartości `15` i `y` o wartości `15`. Następnie deklaruj zmienną `bool` na poziomie górnym o nazwie `isEqual`, która przechowuje wynik porównania `x` i `y` za pomocą operatora `>=`.

# --solutions--

```dart
int age = 35;
bool isWorkingAge = age >= 18 && age <= 65;

void main() {
  print(isWorkingAge);
}
```

# --asserts--

`isEqual` musi być `true` i być typu `bool`

```dart
  test('test1', () {
    expect(isWorkingAge, true, reason: '--err-t1--');

  });
```
