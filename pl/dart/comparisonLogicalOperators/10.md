# --instructions--

Deklaruj dwie zmienne `int` na poziomie górnym: `x` o wartości `8` i `y` o wartości `8`. Następnie deklaruj zmienną `bool` na poziomie górnym o nazwie `areEqual`, która przechowuje wynik porównania `x` i `y` za pomocą operatora `==`.

# --solutions--

```dart
int x = 8;
int y = 8;
bool areEqual = x == y;

void main() {
  print(areEqual);
}
```

# --asserts--

`areEqual` musi być `true` i być typu `bool`

```dart
  test('test1', () {
    expect(areEqual, true, reason: '--err-t1--');
    expect(areEqual, TypeMatcher<bool>(), reason: '--err-t2--');
  });
```
