# --description--

Operator **mniejszości** `<` zwraca `true`, gdy lewa wartość jest ściśle mniejsza niż prawa wartość.

```dart
int a = 2;
int b = 9;
bool result = a < b; // true
```

Jeśli lewa wartość jest równa lub większa od prawej, zwraca `false`.
