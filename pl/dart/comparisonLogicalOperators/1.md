# --description--

Operator **równości** `==` porównuje dwie wartości i zwraca `true`, jeśli są równe, lub `false`, jeśli nie.

```dart
int a = 5;
int b = 5;
bool result = a == b; // true
```

Wynik porównania jest zawsze wartością `bool`.
