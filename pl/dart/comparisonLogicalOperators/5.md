# --description--

Operator **większy lub równy** `>=` zwraca `true`, gdy lewa wartość jest większa **lub równa** prawej wartości.

```dart
int score = 50;
bool passed = score >= 50; // true
```

W przeciwieństwie do `>`, ten operator zwraca również `true`, gdy obie wartości są równe.
