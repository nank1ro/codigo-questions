# --description--

Operator **OR** `||` łączy dwa wyrażenia logiczne. Zwraca `true` jeśli **co najmniej jedna** strona jest `true`. Zwraca `false` tylko wtedy, gdy obie strony są `false`.

```dart
bool a = true;
bool b = false;
bool result = a || b; // true

bool c = false;
bool d = false;
bool result2 = c || d; // false
```

# --instructions--

Deklaruj zmienną `bool` na poziomie górnym o nazwie `isPassed`, która jest `true` tylko wtedy, gdy zmienna `score` jest większa niż `70` LUB `bonus` jest większy niż `10`. Użyj zmiennych `score` = `65` i `bonus` = `15`.
