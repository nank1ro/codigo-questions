# --instructions--

Deklaruj dwie zmienne `int` na poziomie górnym: `num1` o wartości `20` i `num2` o wartości `15`. Następnie deklaruj zmienną `bool` na poziomie górnym o nazwie `isGreater`, która przechowuje wynik porównania, czy `num1` jest większy niż `num2` za pomocą operatora `>`.

# --solutions--

```dart
int high = 20;
int low = 12;
bool isHigher = high > low;

void main() {
  print(isHigher);
}
```

# --asserts--

`isGreater` musi być `true` i być typu `bool`

```dart
  test('test1', () {
    expect(isHigher, true, reason: '--err-t1--');
    expect(isHigher, TypeMatcher<bool>(), reason: '--err-t2--');
  });
```
