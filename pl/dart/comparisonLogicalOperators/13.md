# --instructions--

Deklaruj dwie zmienne `int` na poziomie górnym: `value1` o wartości `6` i `value2` o wartości `8`. Następnie deklaruj zmienną `bool` na poziomie górnym o nazwie `isSmaller`, która przechowuje wynik porównania, czy `value1` jest mniejszy niż `value2` za pomocą operatora `<`.

# --solutions--

```dart
int price = 30;
int budget = 50;
bool canAfford = price < budget;

void main() {
  print(canAfford);
}
```

# --asserts--

`isSmaller` musi być `true` i być typu `bool`

```dart
    expect(canAfford, true, reason: '--err-t1--');
    expect(canAfford, TypeMatcher<bool>(), reason: '--err-t2--');
  });
```
