# --instructions--

Deklaruj dwie zmienne `int` na poziomie górnym: `a` o wartości `12` i `b` o wartości `18`. Następnie deklaruj zmienną `bool` na poziomie górnym o nazwie `isSmaller`, która przechowuje wynik porównania, czy `a` jest mniejszy lub równy `b` za pomocą operatora `<=`.

# --solutions--

```dart
int x = 5;
int y = 3;
int z = 0;
bool result = !(x > 10) || (y < 5 && z == 0);

void main() {
  print(result);
}
```
