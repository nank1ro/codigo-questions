# --description--

Operator **większości** `>` zwraca `true`, gdy lewa wartość jest ściśle większa niż prawa wartość.

```dart
int a = 8;
int b = 3;
bool result = a > b; // true
```

Jeśli lewa wartość jest równa lub mniejsza od prawej, zwraca `false`.
