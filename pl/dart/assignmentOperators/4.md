# --description--

Operator **mnożenia z przypisaniem** `*=` mnoży zmienną przez wartość i przechowuje wynik z powrotem w tej zmiennej.

```dart
int side = 4;
side *= 3; // side jest teraz 12
```

To jest skrót dla `side = side * 3`.
