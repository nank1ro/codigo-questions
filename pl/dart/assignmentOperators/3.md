# --description--

Operator **odejmowania z przypisaniem** `-=` odejmuje wartość od zmiennej i przechowuje wynik z powrotem w tej zmiennej.

```dart
int lives = 3;
lives -= 1; // lives jest teraz 2
```

To jest skrót dla `lives = lives - 1`.
