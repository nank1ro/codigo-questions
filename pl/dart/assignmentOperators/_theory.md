Operator **przypisania** `=` przechowuje wartość w zmiennej.

```dart
int x = 5;
String name = 'Alice';
```

Po przypisaniu zmienna przechowuje tę wartość, dopóki nie zostanie przypisana ponownie.

---

Operator **dodawania z przypisaniem** `+=` dodaje wartość do zmiennej i przechowuje wynik z powrotem w tej zmiennej.

```dart
int score = 10;
score += 5; // score jest teraz 15
```

To jest skrót dla `score = score + 5`.

---

Operator **odejmowania z przypisaniem** `-=` odejmuje wartość od zmiennej i przechowuje wynik z powrotem w tej zmiennej.

```dart
int lives = 3;
lives -= 1; // lives jest teraz 2
```

To jest skrót dla `lives = lives - 1`.

---

Operator **mnożenia z przypisaniem** `*=` mnoży zmienną przez wartość i przechowuje wynik z powrotem w tej zmiennej.

```dart
int side = 4;
side *= 3; // side jest teraz 12
```

To jest skrót dla `side = side * 3`.

---

Operatory przypisania łączą operację arytmetyczną z przypisaniem w jednym kroku.

Na przykład, `x += 3` to skrót dla `x = x + 3`.
