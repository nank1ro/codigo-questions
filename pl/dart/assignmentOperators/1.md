# --description--

Operator **przypisania** `=` przechowuje wartość w zmiennej.

```dart
int x = 5;
String name = 'Alice';
```

Po przypisaniu zmienna przechowuje tę wartość, dopóki nie zostanie przypisana ponownie.

# --instructions--

Uzupełnij kod, aby przypisać wartość `42` do zmiennej `answer`.

# --seed--

```dart
void main() {
  int answer [/] 42;
  print(answer);
}
```
