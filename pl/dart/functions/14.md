# --instructions--

Napisz **rekurencyjną** funkcję `factorial`, która przyjmuje parametr `int` o nazwie `n` i zwraca `n!` (silnię `n`).

Silnia `n` jest zdefiniowana jako:
- `factorial(0) = 1`
- `factorial(n) = n * factorial(n - 1)` dla `n > 0`

Przykład:
```dart
print(factorial(5)); // 120
print(factorial(3)); // 6
```

# --solutions--

```dart
int factorial(int n) {
  if (n == 0) return 1;
  return n * factorial(n - 1);
}
```
