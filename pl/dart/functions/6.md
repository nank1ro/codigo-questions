# --instructions--

Co oznacza typ zwracany `void` w funkcji Dart?

```dart
void sayHi() {
  print("Hi!");
}
```

# --answers--

- Funkcja zwraca liczbę zero
- Funkcja nie zwraca żadnej wartości
- Funkcja zwraca pusty ciąg znaków
- Funkcja może zwrócić dowolny typ
