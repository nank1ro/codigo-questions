# --instructions--

Napisz funkcję `greetUser`, która przyjmuje parametr `String` o nazwie `name` z **wartością domyślną** `"World"` i zwraca ciąg znaków `"Hello, <name>!"`.

Przykład:
```dart
print(greetUser());          // Hello, World!
print(greetUser("Alice"));   // Hello, Alice!
```
