# --description--

Słowo kluczowe `is` pozwala sprawdzić, czy zmienna jest określonego typu. Możesz
 również sprawdzić, czy zmienna nie jest określonego typu za pomocą słowa kluczowego `is!`
```dart
final number = 3.14;
print(number is! int); // true
```
