# --description--

**Konstruktor fabryczny** używa słowa kluczowego `factory` i pozwala kontrolować tworzenie obiektów — na przykład zwracać buforowaną instancję lub podtyp:

```dart
class Logger {
  static final Logger _instance = Logger._internal();

  factory Logger() => _instance;

  Logger._internal();
}
```

Każde wywołanie `Logger()` zwraca tę samą instancję (wzorzec singleton).

# --instructions--

Utwórz klasę `Counter` z prywatnym polem `_count` zainicjalizowanym wartością `0`. Dodaj metodę instancji `increment()`, która zwiększa `_count` o `1`, oraz getter `count` zwracający `_count`. Użyj konstruktora `factory` o nazwie `Counter.start()`, który zawsze zwraca nowy obiekt `Counter` zainicjalizowany z `_count = 0` (używając prywatnego konstruktora nazwanego `Counter._internal()`).

Początkowa wartość `count` wynosi 0

```dart
    test('increment', () {
      var c = Counter.start();
      c.increment();
      expect(c.count, 1, reason: '--err-t2--');
    });
```

Wielokrotne wywołania `increment` działają poprawnie

```dart
    test('multiple increments', () {

      c.increment();

    });
```

```dart
class Counter {
  int _count = 0;

  Counter._internal();

  factory Counter.start() => Counter._internal();

  void increment() {
    _count++;
  }

  int get count => _count;
}
```
