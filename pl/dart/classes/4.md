# --description--

Dart oferuje skróconą składnię konstruktorów, która automatycznie przypisuje parametry do zmiennych instancji. Zamiast:

```dart
class Point {
  int x;
  int y;

  Point(int x, int y) {
    this.x = x;
    this.y = y;
  }
}
```

Można napisać:

```dart
class Point {
  int x;
  int y;

  Point(this.x, this.y);
}
```

Ta krótsza forma nosi nazwę **inicjalizujących parametrów formalnych**.

# --instructions--

Która z poniższych opcji poprawnie używa skróconej składni inicjalizujących parametrów formalnych?

# --answers--

- `Point(this.x, this.y);`
- `Point(x, y) { self.x = x; self.y = y; }`
- `Point(x, y) => x + y;`
- `Point.init(x, y);`

# --solutions--

- `Point(this.x, this.y);`
