# --description--

Klasa może mieć **zmienne instancji** (zwane też polami), które przechowują dane każdego obiektu. Deklaruje się je wewnątrz ciała klasy:

```dart
class Animal {
  String name;
  int age;
}
```

Każdy obiekt stworzony z tej klasy będzie miał własne pola `name` i `age`.
