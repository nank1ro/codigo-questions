# --description--

Dart podąża standardową matematyczną **kolejnością operacji** (priorytetem): mnożenie i dzielenie są oceniane przed dodawaniem i odejmowaniem.

```dart
int result = 2 + 3 * 4; // 14, nie 20
```

Użyj nawiasów aby zmienić kolejność:

```dart
int result = (2 + 3) * 4; // 20
```

Zadeklaruj zmienną najwyższego poziomu `int` o nazwie `result`, która ewaluuje `3 + 4 * 2 - 1`. Pozwól Dart na zastosowanie domyślnego priorytetu operatorów (nie dodawaj nawiasów).
