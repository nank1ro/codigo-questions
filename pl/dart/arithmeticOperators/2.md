# --description--

Operator **odejmowania** `-` odejmuje jedną wartość od drugiej.

```dart
int a = 10;
int b = 4;
int diff = a - b; // 6
```

Wynik może być ujemny, jeśli drugi operand jest większy niż pierwszy.
