# --description--

Operator **mnożenia** `*` mnoży dwie wartości razem.

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

Mnożenie `int` przez inny `int` daje `int`. Mnożenie z `double` daje `double`.
