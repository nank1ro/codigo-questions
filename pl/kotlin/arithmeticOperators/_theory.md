Operatory służą do wykonywania operacji na zmiennych i wartościach.
Zacznijmy od operatorów arytmetycznych, w szczególności od operatora **dodawania** `+`.
Służy on do dodawania dwóch liczb, np.:
```kotlin
println(5 + 3) // 8
```

---

Przejdźmy do operatora **odejmowania** `-`.
Służy on do odejmowania jednej liczby od drugiej, np.:
```kotlin
println(5 - 3) // 2
```

---

Poznajmy operator **mnożenia** `*`.
Służy on do mnożenia dwóch liczb, np.:
```kotlin
println(5 * 3) // 15
```

---

Poznajmy operator **dzielenia** `/`.
Służy on do dzielenia dwóch liczb, np.:
```kotlin
println(10 / 5) // 2
```

---

Poznajmy operator **reszty z dzielenia** `%`.
Służy on do wyznaczania reszty po podzieleniu dwóch liczb, np.:
```kotlin
println(5 % 2) // 1
```
To wyrażenie daje wynik 1, ponieważ 5 podzielone przez 2 daje iloraz 2 i resztę 1
```kotlin
println(9 % 3) // 0
```
To wyrażenie daje wynik 0, ponieważ 9 podzielone przez 3 daje iloraz 3 i resztę 0
