# --description--

W Kotlinie możemy używać `for-in` również dla iterowalnych kolekcji, wywołując podane domknięcie na każdym elemencie:
```kotlin
// this is a list, we'll see about that soon
val numbers = listOf(2, 4, 6, 8, 10)
for (num in numbers) {
    println(num)
}
// prints (2, 4, 6, 8, 10)
```
