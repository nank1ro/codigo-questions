# --description--

Jeśli po prostu musisz powtórzyć blok kodu `n` razy, możesz użyć funkcji `repeat(times: Int)`.

```kotlin
repeat(3) {
    println("repeat")
}
// prints repeat 3 times
```

Możesz nawet uzyskać dostęp do indeksu za pomocą
```kotlin
repeat(3) { index ->
    println(index)
}
// prints 0, 1, 2
```
