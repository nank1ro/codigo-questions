# --description--

Oto przykład użycia `break` w pętli _while_:
```kotlin
var i = 0
while (i < 3) {
  i++
  if (i == 2) break // [1]
  println(i)
}
// prints 1
```

Jak widać w __[1]__, gdy `i` jest równe _2_, _przerywamy_ pętlę. W rzeczywistości liczby 2 i 3 nigdy nie są wyświetlane.
