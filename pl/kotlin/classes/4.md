# --description--

Klasy mogą zawierać **metody** — funkcje należące do klasy, które operują na jej właściwościach:

```kotlin
class Counter(var count: Int) {
    fun increment() {
        count++
    }
    fun value(): Int {
        return count
    }
}
```

Metody są wywoływane przy użyciu notacji kropkowej na instancji:

```kotlin
val c = Counter(0)
c.increment()
println(c.value()) // 1
```

# --asserts--

```kotlin
    val account = BankAccount(0.0)
    account.deposit(100.0)
    tryCatch(account.balance == 100.0)
```

```kotlin
    account.withdraw(30.0)
    tryCatch(account.balance == 70.0)
```

class BankAccount(var balance: Double) {
    fun deposit(amount: Double) {
        balance += amount
    }
    fun withdraw(amount: Double) {
        balance -= amount
    }
}
