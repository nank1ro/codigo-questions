# --description--

Szablon tekstowy (_String template_) to programowy sposób generowania ciągu znaków.
W Kotlinie możemy użyć znaku `+` (konkatenacja) do wyświetlania dwóch lub więcej ciągów razem, na przykład:
```kotlin
println("Hello " + "Kotlin!")
// prints "Hello Kotlin!"
```
