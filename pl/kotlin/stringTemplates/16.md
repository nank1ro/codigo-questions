# --instructions--

Co jest nie tak z tym kodem?
```kotlin
println("$(10} friends")
```

# --answers--

- Identyfikator powinien być wewnątrz ${}
- Wartość wewnątrz ${} nie jest liczbą
- Ciągowi znaków brakuje cudzysłowów " "
- Wartość wewnątrz ${} musi być nazwą zmiennej
