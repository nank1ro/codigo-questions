# --description--

Funkcje mogą zwracać wartości `Boolean`, co jest przydatne do sprawdzania warunków:
```kotlin
fun isEven(n: Int): Boolean {
    return n % 2 == 0
}
println(isEven(4)) // prints true
println(isEven(7)) // prints false
```
Funkcja `Boolean` zwraca `true` lub `false`.
