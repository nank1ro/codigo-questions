# --description--

Wiemy, jak powtarzać kod za pomocą pętli `while`.
Jak ten program powtarzający instrukcje, aby wyświetlić `hello`
```javascript
var counter = 0;

while (counter < 5) {
    console.log("hello");
    counter++;
}
```
Ale możemy zrobić to samo za pomocą pętli `for`:
```javascript
for (let i = 0; i < 5; i++) {
    console.log("hello");
}
```
