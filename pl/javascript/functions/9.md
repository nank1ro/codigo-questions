# --instructions--

Jaki jest __problem__ z tym fragmentem kodu?
```javascript
console.log(greet("Mathew"));
// This function greets to the person passed in as argument
const greet = (name) => `Hello, ${name}. Good morning!`;
console.log(greet("Alex"));
```

# --answers--

- Nie możemy użyć greet() przed jej zdefiniowaniem
- Nie możemy użyć greet() dwa razy
- greet() nic nie zwraca
- Nie możemy użyć name jako parametru
