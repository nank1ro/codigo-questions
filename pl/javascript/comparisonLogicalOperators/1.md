# --description--

Zacznijmy od operatora porównania **równości** `==`.
Zwraca on wartość **logiczną** (`true` lub `false`) określającą, czy dwa wyrażenia są równe, na przykład:
```javascript
console.log(2 == 2); 
// prints true
console.log(2 == 3);
// prints false
```
