# --description--

Kontynuujmy z operatorem logicznym **lub** `||`.
Zwraca on pierwszy operand, który przyjmuje wartość *true*, lub ostatni, jeśli wszystkie są *false*.
```javascript
console.log(2 == 2 || 2 == 3);
// prints true
console.log(1 == 2 || 1 == 3);
// prints false
```
