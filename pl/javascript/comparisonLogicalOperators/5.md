# --description--

Kontynuujmy z operatorem porównania **większy lub równy** `>=`.
Zwraca on wartość **logiczną** (`true` lub `false`) określającą, czy jedno wyrażenie jest większe lub równe drugiemu, na przykład:
```javascript
console.log(2 >= 2); 
// prints true
console.log(3 >= 2);
// prints true
console.log(3 >= 4);
// prints false
```
