# --description--

Kontynuujmy z operatorem porównania **nierówności** `!=`.
Zwraca on wartość **logiczną** (`true` lub `false`) określającą, czy dwa wyrażenia są **NIERÓWNE**, na przykład:
```javascript
console.log(2 != 2);
// prints false
console.log(2 != 3); 
// prints true
```
Jest to dokładne przeciwieństwo operatora *równości*
