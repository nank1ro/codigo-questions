# --description--

Zapoznajmy się z operatorem **reszty z dzielenia** `%`.
Służy on do znajdowania reszty po podzieleniu dwóch liczb, np.:
```javascript
console.log(5 % 2);
// prints 1
```
To daje 1, ponieważ 5 podzielone przez 2 daje iloraz 2 i resztę 1
```javascript
console.log(9 % 3);
// prints 0
```
To daje 0, ponieważ 9 podzielone przez 3 daje iloraz 3 i resztę 0
