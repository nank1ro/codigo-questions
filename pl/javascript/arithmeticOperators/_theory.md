Operatory służą do wykonywania operacji na zmiennych i wartościach.
Zacznijmy od operatorów arytmetycznych, a w szczególności od operatora **dodawania** `+`.
Służy on do dodawania dwóch liczb, np.:
```javascript
console.log(5 + 3);
// prints 8
```

---

Przejdźmy do operatora **odejmowania** `-`.
Służy on do odejmowania jednej liczby od drugiej, np.:
```javascript
console.log(5 - 3);
// prints 2
```

---

Zapoznajmy się z operatorem **mnożenia** `*`.
Służy on do mnożenia dwóch liczb, np.:
```javascript
console.log(5 * 3);
// prints 15
```

---

Zapoznajmy się z operatorem **dzielenia** `/`.
Służy on do dzielenia dwóch liczb, np.:
```javascript
console.log(10 / 5);
// prints 2
```

---

Zapoznajmy się z operatorem **reszty z dzielenia** `%`.
Służy on do znajdowania reszty po podzieleniu dwóch liczb, np.:
```javascript
console.log(5 % 2);
// prints 1
```
To daje 1, ponieważ 5 podzielone przez 2 daje iloraz 2 i resztę 1
```javascript
console.log(9 % 3);
// prints 0
```
To daje 0, ponieważ 9 podzielone przez 3 daje iloraz 3 i resztę 0
