# --instructions--

Co jest nie tak z tym kodem?
```javascript
while (isLoading) {
    console.log("Loading...");
    isLoading = false;
}
```

# --answers--

- isLoading nie jest poprawną nazwą zmiennej
- Pętla while ma instrukcję drukowania wewnątrz bloku kodu
- Blok kodu nie ma poprawnego wcięcia
