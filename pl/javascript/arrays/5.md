# --description--

Tablica nie musi mieć stałej długości.
Możesz dodawać elementy na końcu tablicy w dowolnym momencie!
Aby dodać element do tablicy, używamy funkcji `push`:
```javascript
var letters = ["a", "b"];
letters.push("c");
console.log(letters);
// Prints ["a", "b", "c"]
```
