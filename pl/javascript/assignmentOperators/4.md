# --description--

Poznajmy operator **przypisania z mnożeniem** `*=`.
Mnoży zmienną przez wartość i przypisuje wynik do tej zmiennej.
Poniższe zapisy są równoważne:
```javascript
var num = num * 5;
// jest równoważne z
num *= 5;
```
