# --description--

Poznajmy operator **przypisania z dzieleniem** `/=`.
Dzieli zmienną przez wartość i przypisuje wynik do tej zmiennej.
Poniższe zapisy są równoważne:
```javascript
num = num / 5;
// jest równoważne z
num /= 5;
```
