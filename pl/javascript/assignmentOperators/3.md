# --description--

Podobnie jak przy przypisaniu z dodawaniem, mamy **operator przypisania z odejmowaniem** `-=`.
Działanie jest takie samo, jedyna różnica polega na tym, że wykonuje odejmowanie.
Poniższe zapisy są równoważne:
```javascript
var num = num - 5;
// jest równoważne z
num -= 5;
```
