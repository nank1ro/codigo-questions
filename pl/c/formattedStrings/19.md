# --instructions--

Co jest nie tak z tym kodem?
```c
int manchester_united = 0;
int barcelona = 1;
printf("The score is %i : %i\n", manchester_united, barcelona);
```

# --answers--

- Nic nie jest nie tak z tym kodem
- manchester_united i barcelona nie mają wartości
- Nie możemy używać zmiennych wewnątrz sformatowanego ciągu znaków
- Nie możemy używać %i dwa razy w tym samym ciągu znaków
