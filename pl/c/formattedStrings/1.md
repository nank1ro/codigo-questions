# --description--

W języku C używamy funkcji `printf` z wyrażeniem, aby wyświetlić dwie lub więcej wartości razem, na przykład:
```c
>>> char c = 'C';
>>> printf("Hello %c!\n", c);
Hello C!
```
`%c` służy do wyświetlania pojedynczego znaku

# --instructions--

Wyświetl imię używając wyrażenia `%s` (służącego do wyświetlania _tablicy znaków_ lub _ciągu znaków_)
