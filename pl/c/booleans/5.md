# --instructions--

Dlaczego `true` nie jest ciągiem znaków?
```c
bool a = true;
```

# --answers--

- Jest przechowywane wewnątrz zmiennej
- Nie ma wokół niego cudzysłowów i typ nie jest char
- Zaczyna się wielką literą
