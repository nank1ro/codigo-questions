# --instructions--

Co robi `counter += 1`?
```c
int counter = 1;

while (counter < 5) {
    printf("%d\n", counter);
    counter += 1;
}
```

# --answers--

- Zwiększa counter o jeden za każdym razem, gdy pętla wykonuje swój blok kodu
- Tworzy zmienną counter
- Zwiększa counter o dwa za każdym razem
- Zwiększa counter o trzy za każdym razem
