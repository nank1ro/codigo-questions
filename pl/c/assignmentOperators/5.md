# --description--

Zobaczmy operator **przypisania dzielenia** `/=`.
Dzieli zmienną przez wartość i przypisuje wynik do tej zmiennej.
Zatem poniższe są dokładnie tym samym
```c
num = num / 5;
// is equal to
num /= 5;
```
