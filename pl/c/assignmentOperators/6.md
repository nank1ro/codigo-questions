# --description--

Zobaczmy operator **przypisania modulo** `%=`.
Oblicza resztę z dzielenia zmiennej przez wartość i przypisuje wynik do tej zmiennej.
Zatem poniższe są dokładnie tym samym
```c
num = num % 5;
// is equal to
num %= 5;
```
