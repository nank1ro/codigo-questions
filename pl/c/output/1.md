# --description--

"__Hello, World!__" to tradycyjny pierwszy program dla osób zaczynających naukę programowania w nowym języku.
Używamy funkcji `printf()` do wyświetlania danych na standardowym urządzeniu wyjściowym (ekranie).
Każdy program w C korzysta z bibliotek, które umożliwiają wykonywanie niezbędnych funkcji, na przykład funkcja wydruku na ekranie jest zdefiniowana w pliku nagłówkowym `stdio.h`.
Pierwszy kod, który zostanie wykonany, zawsze będzie znajdował się w funkcji main.
Aby wydrukować `Hello, World!` na ekranie w C, możemy napisać
```c
printf("Hello, World!");
```
Na koniec zwracamy `0`, aby wskazać, że nasz program zakończył się sukcesem
