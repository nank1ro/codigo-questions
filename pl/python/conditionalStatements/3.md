Idziemy bawić się na zewnątrz, jeśli pogoda jest ładna; w przeciwnym razie zostajemy w domu.
W Pythonie możemy użyć instrukcji `else`, jak:
```python
nice_weather = True
if (nice_weather):
    # play outside
else:
    # stay home
```
