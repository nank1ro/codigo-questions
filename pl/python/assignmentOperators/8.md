# --description--

Poznajmy operator **przypisania dzielenia całkowitego** `//=`.
Dzieli zmienną całkowicie przez wartość i przypisuje wynik do tej zmiennej.
Poniższe są dokładnie równoważne:
```python
num = num // 5
# is equal to
num //= 5
```
