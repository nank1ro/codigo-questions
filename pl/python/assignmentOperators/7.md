# --description--

Poznajmy operator **przypisania modulo** `%=`.
Oblicza modulo zmiennej i wartości, a następnie przypisuje wynik do tej zmiennej.
Poniższe są dokładnie równoważne:
```python
num = num % 5
# is equal to
num %= 5
```
