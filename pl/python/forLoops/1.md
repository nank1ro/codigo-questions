# --description--

Wiemy, jak powtarzać kod za pomocą pętli `while`.
Na przykład ten program wielokrotnie wyświetla `hello`:
```python
counter = 0

while counter < 5:
    print("hello")
    counter += 1
```
To samo możemy zrobić za pomocą pętli `for`:
```python
for i in range(5):
    print("hello")
```
