# --description--

Podobnie jak w listach, możemy iterować po elementach słownika używając słów kluczowych `for..in`.
Aby uzyskać zarówno klucz, jak i wartość w pętli, możemy użyć metody `items()`:
```python
for key, value in dict_name:
    print(key, value)
```
