# --description--

W Pythonie możemy bardzo prosto iterować po liście za pomocą słów kluczowych `for..in`:
```python
>>> numbers = [1, 2, 3]
>>> for num in numbers:
>>>     print(num)
1
2
3
```
Po słowie kluczowym `for` następuje nazwa zmiennej, której zostanie przypisana wartość każdego elementu listy po kolei.
