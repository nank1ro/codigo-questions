# --description--

Lista nie musi mieć stałej długości.
Możesz dodawać elementy na końcu listy w dowolnym momencie!
Aby dodać element do listy, używamy słowa kluczowego `append`:
```python
>>> letters = ["a", "b"]
>>> letters.append("c")
>>> print(letters)
['a', 'b', 'c']
```
