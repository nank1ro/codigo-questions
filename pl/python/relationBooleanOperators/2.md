# --description--

Przejdźmy do relacyjnego operatora **nierówności** `!=`.
Zwraca on wartość **logiczną** (`True` lub `False`) określającą, czy dwa wyrażenia **NIE** są równe, na przykład:
```python
>>> 2 != 2
False
>>> 2 != 3
True
```
Jest to dokładne przeciwieństwo operatora *równości*
