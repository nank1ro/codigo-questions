# --description--

Poznajmy operator **modulo** `%`.
Służy on do znajdowania reszty z dzielenia dwóch liczb, na przykład:
```python
>>> 5 % 2
1
```
Wynik wynosi 1, ponieważ 5 podzielone przez 2 daje iloraz 2 i resztę 1
```python
>>> 9 % 3
0
```
Ten wynik wynosi 0, ponieważ 9 podzielone przez 3 daje iloraz 3 i resztę 0
