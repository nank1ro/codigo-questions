Operatory służą do wykonywania operacji na zmiennych i wartościach.
Zacznijmy od operatorów arytmetycznych, a w szczególności od operatora **dodawania** `+`.
Służy on do dodawania dwóch liczb, na przykład:
```python
>>> 5+3
8
```

---

Przejdźmy dalej do operatora **odejmowania** `-`.
Służy on do odejmowania jednej liczby od drugiej, na przykład:
```python
>>> 5-3
2
```

---

Poznajmy operator **mnożenia** `*`.
Służy on do mnożenia dwóch liczb, na przykład:
```python
>>> 5*3
15
```

---

Poznajmy operator **dzielenia** `/`.
Służy on do dzielenia dwóch liczb, na przykład:
```python
>>> 10/5
2
```

---

Poznajmy operator **modulo** `%`.
Służy on do znajdowania reszty z dzielenia dwóch liczb, na przykład:
```python
>>> 5 % 2
1
```
Wynik wynosi 1, ponieważ 5 podzielone przez 2 daje iloraz 2 i resztę 1
```python
>>> 9 % 3
0
```
Ten wynik wynosi 0, ponieważ 9 podzielone przez 3 daje iloraz 3 i resztę 0

---

Poznajmy operator **potęgowania** `**`.
Potęgowanie odpowiada wielokrotnemu mnożeniu podstawy: tzn. **b** z wykładnikiem *n* jest iloczynem mnożenia *n* podstaw:
![exponentiation](https://bit.ly/3zcz6Lt)
```python
>>> 5 ** 2
25
```

---

Poznajmy operator **dzielenia całkowitego** `//`.
Ten operator zwraca całkowitą część ilorazu, na przykład:
```python
>>> 5.0 / 2
2.5
>>> 5.0 // 2
2.0
```
Zwane też dzieleniem całkowitym. Wynik jest liczbą całkowitą, choć *typ* wyniku niekoniecznie jest int.
