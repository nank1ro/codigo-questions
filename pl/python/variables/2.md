# --description--

Zmienne są tak nazwane, ponieważ wartość, którą przechowują, może się zmieniać.
Możemy zaktualizować `x` używając `=` i nadając mu nową wartość.
```python
>>> x = 1
>>> print(x)
1
>>> x = 2
>>> print(x)
2
```
