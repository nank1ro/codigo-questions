# --description--

Tablica nie musi mieć stałej długości.
Możesz dodawać elementy na koniec tablicy w dowolnym momencie!
Aby dodać element do tablicy, używamy funkcji `append`:
```swift
var letters = ["a", "b"]
letters.append("c")
print(letters)
// Prints ["a", "b", "c"]
```
