# --description--

W przypadku niektórych wyliczeń przydatne jest posiadanie kolekcji wszystkich przypadków tego wyliczenia.
Umożliwia to napisanie `: CaseIterable` po nazwie wyliczenia.
Swift udostępnia kolekcję wszystkich przypadków jako właściwość `allCases` typu wyliczenia:
```swift
enum Colors: CaseIterable {
    case blue, red, green
}
for color in Colors.allCases {
    print(color)
}
// prints blue, red, green
```

# --instructions--

Iteruj po wyliczeniu, aby wydrukować każdy przypadek.
`$0` to skrócona nazwa argumentu, która może być używana do odwoływania się do wartości argumentów domknięcia.
`$0` odnosi się do pierwszego argumentu domknięcia, `$1` do drugiego argumentu i tak dalej
