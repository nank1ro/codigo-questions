# --description--

Przyjrzyjmy się operatorowi **przypisania dzielenia** `/=`.
Dzieli zmienną przez wartość i przypisuje wynik do tej zmiennej.
Poniższe są dokładnie tym samym:
```swift
num = num / 5
// is equal to
num /= 5
```
