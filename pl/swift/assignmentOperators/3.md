# --description--

Podobnie jak w przypisaniu dodawania, mamy **przypisanie odejmowania** `-=`.
Funkcjonalność jest taka sama, jedyną różnicą jest to, że wykonuje odejmowanie.
Poniższe są dokładnie tym samym:
```swift
var num = num - 5
// is equal to
num -= 5
```
