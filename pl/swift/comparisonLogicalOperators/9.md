# --description--

Zakończmy operatorem logicznym **negacji** `!`.
Zwraca on wartość logiczną będącą odwrotnością stanu logicznego wyrażenia.
```swift
print(!true) // false
print(!false) // true
print(!(2 == 2)) // false
```
