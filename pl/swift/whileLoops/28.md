Co powoduje awarię tego programu?
```swift
while true {

}
```

# --answers--

- Kod ma pętlę nieskończoną
- Nic nie powoduje awarii tego kodu
- Pętla while nie ma poprawnego wcięcia
- Blok kodu nie ma poprawnego wcięcia
