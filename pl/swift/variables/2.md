# --description--

Zmienne nazywane są w ten sposób, ponieważ wartość, którą przechowują, może się zmieniać.
Możemy zaktualizować `x` używając `=` i podając nową wartość.
```swift
var x = 1
print(x) // prints 1
x = 2
print(x) // prints 2
```
