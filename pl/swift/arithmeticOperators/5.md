# --description--

Przyjrzyjmy się operatorowi **reszty z dzielenia** `%`.
Służy on do znajdowania reszty po dzieleniu dwóch liczb, na przykład:
```swift
print(5 % 2) // 1
```
To daje 1, ponieważ 5 podzielone przez 2 daje iloraz 2 i resztę 1
```swift
print(9 % 3) // 0
```
To daje 0, ponieważ 9 podzielone przez 3 daje iloraz 3 i resztę 0
