# --description--

Wiemy już, jak powtarzać kod za pomocą pętli `while`.
Na przykład ten program powtarza instrukcje, aby wyświetlić `hello`
```swift
var counter = 0

while counter < 5 {
    print("hello")
    counter += 1;
}
```
Ale to samo możemy zrobić za pomocą pętli `for`:
```swift
for i in 0..<5 {
    print("hello")
}
```
