# --description--

Funkcja `stride()` zwraca sekwencję liczb.
Wymaga parametrów _from_, _to_ i _by_.
Oto składnia tej funkcji:
```swift
stride(from:to:by:)
```
Pamiętaj, że wartość `to` jest wykluczona

# --instructions--

Uzupełnij funkcję `stride()` w poniższym kodzie, aby wydrukować następujące liczby `(3, 6, 9, 12)`

# --seed--

```swift
for i in stride(from: [/], to: [/], by: [/]) {
    print(i)
}
```
