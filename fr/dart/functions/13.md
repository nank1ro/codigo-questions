# --instructions--

Écrivez une fonction `range` qui prend un paramètre `int` `n` et retourne une `List<int>` contenant les entiers de `1` à `n` (inclus).

Exemple :
```dart
print(range(3)); // [1, 2, 3]
print(range(5)); // [1, 2, 3, 4, 5]
```

# --solutions--

```dart
List<int> range(int n) {
  List<int> result = [];
  for (int i = 1; i <= n; i++) {
    result.add(i);
  }
  return result;
}
```
