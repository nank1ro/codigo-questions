# --instructions--

Écrivez une fonction **récursive** `factorial` qui prend un paramètre `int` `n` et retourne `n!` (la factorielle de `n`).

La factorielle de `n` est définie comme :
- `factorial(0) = 1`
- `factorial(n) = n * factorial(n - 1)` pour `n > 0`

Exemple :
```dart
print(factorial(5)); // 120
print(factorial(3)); // 6
```

# --solutions--

```dart
int factorial(int n) {
  if (n == 0) return 1;
  return n * factorial(n - 1);
}
```
