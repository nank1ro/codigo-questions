# --instructions--

Que signifie le type de retour `void` dans une fonction Dart ?

```dart
void sayHi() {
  print("Hi!");
}
```

# --answers--

- La fonction retourne le nombre zéro
- La fonction ne retourne aucune valeur
- La fonction retourne une chaîne vide
- La fonction peut retourner n'importe quel type
