# --instructions--

Écrivez une fonction `greetUser` qui prend un paramètre `String` `name` avec une **valeur par défaut** de `"World"` et retourne la chaîne `"Hello, <name>!"`.

Exemple :
```dart
print(greetUser());          // Hello, World!
print(greetUser("Alice"));   // Hello, Alice!
```
