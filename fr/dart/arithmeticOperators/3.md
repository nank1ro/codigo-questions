# --description--

L'**opérateur de multiplication** `*` multiplie deux valeurs ensemble.

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

Multiplier un `int` par un autre `int` produit un `int`. Multiplier avec un `double` produit un `double`.
