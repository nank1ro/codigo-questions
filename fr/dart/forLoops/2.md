# --description--
Vous pouvez utiliser la variable de boucle dans le corps pour suivre le compte courant. Par exemple, pour accumuler une somme :

```dart
int total = 0;
for (int i = 1; i <= 5; i++) {
  total += i;
}
```

Après la boucle, `total` est égal à 15.
