# --instructions--
Que fait la boucle `for-in` en Dart ?

# --answers--

- Itère sur chaque élément d'une collection
- Itère un nombre fixe de fois en utilisant un index
- Se répète jusqu'à ce qu'une condition booléenne devienne vraie
- Exécute du code uniquement si la collection n'est pas vide
