# --description--
L'instruction `break` quitte une boucle immédiatement lorsqu'une condition est remplie :

```dart
for (int i = 0; i < 10; i++) {
  if (i == 5) break; // stops at 5
  print(i);
}
```

C'est utile lorsqu'on recherche une valeur et qu'on veut s'arrêter dès qu'elle est trouvée.
