# --description--
Une boucle `for` peut compter **à rebours** en utilisant un décrément (`i--`) et une condition `>=` :

```dart
for (int i = 5; i >= 1; i--) {
  print(i); // 5, 4, 3, 2, 1
}
```

C'est utile pour les comptes à rebours ou l'itération en sens inverse.
