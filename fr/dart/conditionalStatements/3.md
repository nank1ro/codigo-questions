# --description--

Une **instruction `if-else`** exécute le bloc `if` lorsque la condition est `true`, et le bloc `else` lorsqu'elle est `false`.

```dart
int temperature = 10;
if (temperature > 20) {
  print('Warm');
} else {
  print('Cold');
}
// prints: Cold
```

Exactement l'une des deux branches s'exécute toujours.
