# --instructions--

Que fait une instruction `if` en Dart ?

# --answers--

- Elle exécute toujours le code dans son bloc
- Elle exécute son bloc uniquement lorsque la condition est `true`
- Elle répète son bloc tant que la condition est `true`
- Elle définit une nouvelle fonction
