# --description--

Une **instruction `if`** exécute un bloc de code uniquement lorsqu'une condition est `true`.

```dart
int age = 20;
if (age >= 18) {
  print('Adult');
}
```

Le code à l'intérieur des accolades s'exécute uniquement si la condition `age >= 18` est évaluée à `true`.
