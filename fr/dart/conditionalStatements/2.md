# --description--

Tu peux utiliser `print()` à l'intérieur d'un bloc `if` pour afficher un message lorsqu'une condition est remplie.

```dart
bool isRaining = true;
if (isRaining) {
  print('Take an umbrella');
}
```

Si `isRaining` est `false`, rien n'est affiché.
