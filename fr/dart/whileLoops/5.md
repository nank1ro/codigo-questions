# --description--

Le mot-clé `break` quitte une boucle immédiatement, indépendamment de la condition de la boucle.

```dart
int i = 0;
while (true) {
  if (i == 3) break;
  print(i);
  i++;
}
```

Sans `break`, la boucle `while (true)` s'exécuterait indéfiniment. Ici, elle s'arrête quand `i` atteint `3`.
