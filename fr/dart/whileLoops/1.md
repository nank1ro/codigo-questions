# --description--

Une boucle `while` répète un bloc de code tant qu'une condition est `true`.

```dart
int i = 0;
while (i < 3) {
  print(i);
  i++;
}
```

La boucle vérifie la condition **avant** chaque itération. Quand `i < 3` devient `false`, la boucle s'arrête.
