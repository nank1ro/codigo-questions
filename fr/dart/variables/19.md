# --instructions--

Quelle est la différence entre une constante de __compilation__ et une constante d'__exécution__ ?

# --answers--

- Les constantes de __compilation__ contiennent des valeurs fixes connues avant l'exécution du programme, tandis que les constantes d'__exécution__ obtiennent une valeur fixe uniquement pendant l'exécution du programme

- Il n'y a pas de différence entre les deux
