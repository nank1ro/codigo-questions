# --instructions--

Est-il toujours nécessaire de spécifier le type d'une variable en Dart ?

# --answers--

- Le type d'une variable doit toujours être spécifié
- Dart supporte l'__inférence de type__, il est donc capable de le déduire par lui-même
- Dart ne supporte pas les types
