# --instructions--

Qu'est-ce qui ne va pas avec ce code ?
```dart
const pieces = 5;
const label = "we have $pieces pieces";
```

# --answers--

- On ne peut pas sauvegarder des modèles de chaînes dans des variables
- Les caractères $ sont manquants
- On ne peut pas utiliser de variables à l'intérieur de ${}
- Rien ne va mal avec le code
