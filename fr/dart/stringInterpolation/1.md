# --description--

Une _interpolation_ de chaîne est un moyen programmatique de générer une chaîne.
En Dart, nous pouvons utiliser le signe `+` (concaténation) pour afficher deux ou plusieurs chaînes ensemble, comme :
```dart
print("Hello " + "Dart!");
// prints "Hello Dart!"
```
