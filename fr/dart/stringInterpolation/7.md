# --description--

Nous pouvons utiliser des accolades pour insérer des valeurs autant de fois que nous le souhaitons en utilisant l'interpolation de chaînes

# --instructions--

Voyez ici avec `2` puis avec `8`

# --seed--

```dart
print("[/] new friends and [/] old friends");
```

# --answers--

- $2
- $8
- ${2}
- ${8}

# --solutions--

```dart
print("${2} new friends and ${8} old friends");
```

# --output--

2 new friends and 8 old friends
