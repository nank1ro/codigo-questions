# --instructions--

Qu'est-ce qui ne va pas avec ce code ?
```dart
print("$(10} friends");
```

# --answers--

- L'expression devrait être à l'intérieur de ${}
- La valeur à l'intérieur de ${} n'est pas un nombre
- La chaîne n'a pas ses guillemets " "
- La valeur à l'intérieur de ${} doit être un nom de variable
