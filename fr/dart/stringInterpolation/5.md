# --description--

Si ce qui suit le signe `$` n'est pas reconnaissable comme un identificateur de programme, vous allez rencontrer une erreur

# --instructions--

Exécutez pour voir l'erreur

# --seed--

```dart
void main() {
    print("$5 friends");
}
```

# --solutions--
