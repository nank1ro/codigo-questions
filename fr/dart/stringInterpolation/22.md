# --instructions--

Qu'est-ce qui ne va pas avec ce code ?
```dart
const manchesterUnited = 2;
const barcelona = 2
println("The final score is $manchesterUnited : $barcelona")
```

# --answers--

- Rien ne va mal avec le code
- manchesterUnited et barcelona n'ont pas de valeurs
- On ne peut pas utiliser de variables à l'intérieur des accolades
- On ne peut pas utiliser $ deux fois dans la même chaîne
