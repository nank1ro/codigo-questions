# --instructions--

Utilisez `${}` pour insérer le nombre `500` dans l'espace vide en utilisant l'interpolation de chaînes

# --seed--

```dart
void main() {
    print("There are  items");
}
```

# --solutions--

```dart
void main() {
    print("There are ${500} items");
}
```

# --output--

There are 500 items
