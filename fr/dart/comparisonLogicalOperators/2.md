# --description--

L'**opérateur de différence** `!=` renvoie `true` lorsque deux valeurs sont **différentes**, et `false` lorsqu'elles sont identiques.

```dart
int x = 3;
int y = 7;
bool result = x != y; // true
```

Il est l'opposé de `==`.

# --instructions--

Complete the code to check if `4` is not equal to `9` and store the result in `isDifferent`.
