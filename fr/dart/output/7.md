# --instructions--

Pourquoi ceci n'est-il pas une valeur de type chaîne de caractères ?
```dart
Hello World!
```

# --answers--

- Parce qu'il y a un espace entre les mots
- Parce que les mots commencent par une majuscule
- Parce qu'il n'y a pas de guillemets doubles autour
