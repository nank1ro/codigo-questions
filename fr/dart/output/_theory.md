"__Bonjour, le monde !__" est le programme traditionnel pour commencer la programmation dans un nouveau langage.
Nous utilisons la fonction `print()` pour afficher des données sur le périphérique de sortie standard (écran).
Pour imprimer `Hello, World!` à l'écran avec Dart, nous pouvons écrire
```dart
print("Hello, World!")
```
