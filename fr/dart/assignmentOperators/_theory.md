The **assignment operator** `=` stocke une valeur dans une variable.

```dart
int x = 5;
String name = 'Alice';
```

Une fois assignée, la variable conserve cette valeur jusqu'à ce qu'elle soit réaffectée.

---

The **addition assignment operator** `+=` ajoute une valeur à une variable et stocke le résultat dans cette variable.

```dart
int score = 10;
score += 5; // score is now 15
```

C'est un raccourci pour `score = score + 5`.

---

The **subtraction assignment operator** `-=` soustrait une valeur d'une variable et stocke le résultat dans cette variable.

```dart
int lives = 3;
lives -= 1; // lives is now 2
```

C'est un raccourci pour `lives = lives - 1`.

---

The **multiplication assignment operator** `*=` multiplie une variable par une valeur et stocke le résultat dans cette variable.

```dart
int side = 4;
side *= 3; // side is now 12
```

C'est un raccourci pour `side = side * 3`.

---

Les opérateurs d'affectation combinent une opération arithmétique avec une affectation en une seule étape.

Par exemple, `x += 3` est un raccourci pour `x = x + 3`.
