# --description--

L'**opérateur d'affectation** `=` stocke une valeur dans une variable.

```dart
int x = 5;
String name = 'Alice';
```

Une fois assignée, la variable conserve cette valeur jusqu'à ce qu'elle soit réaffectée.

# --instructions--

Complétez le code pour affecter la valeur `42` à la variable `answer`.

# --seed--

```dart
void main() {
  int answer [/] 42;
  print(answer);
}
```

# --solutions--
