# --description--

Comme n'importe quelle autre variable, vous pouvez afficher une valeur booléenne avec `print()`.
Lorsque vous affichez un booléen, Dart génère le texte `true` ou `false`.

```dart
bool isSunny = true;
print(isSunny); // true
```
