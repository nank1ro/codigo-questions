# --description--

La propriété `.length` retourne le nombre d'éléments d'une liste :

```dart
var scores = [95, 87, 72, 100];
print(scores.length); // 4
```

Une liste vide a une longueur de `0` :

```dart
var empty = [];
print(empty.length); // 0
```
