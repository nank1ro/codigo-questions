# --description--

La méthode `.add()` ajoute un seul élément à la **fin** d'une liste :

```dart
var nums = [1, 2, 3];
nums.add(4);
print(nums); // [1, 2, 3, 4]
```

Notez que `.add()` modifie la liste **en place** et retourne `void`.
