Nous avons déjà appris que pour assigner une valeur à une variable, nous pouvons utiliser le signe `=`, comme :
```python
a = 5
```

---

Nous avons déjà une variable initialisée `total`
```python
total = 5
```
Disons que nous voulons ajouter le nombre `2` à la variable `total`, nous pouvons écrire
```python
total = total + 2
```
D'accord, cela fonctionne ! Mais il existe une version plus courte pour faire la même chose :
```python
total += 2
```
Le signe `+=` s'appelle **assignation par addition**.
Il ajoute une valeur à la valeur de la variable et assigne le résultat à cette variable.

---

Tout comme dans l'assignation par addition, nous avons l'**assignation par décrémentation** `-=`.
La fonctionnalité est la même, la seule différence est qu'elle effectue la soustraction.
Donc les suivants sont exactement les mêmes
```python
num = num - 5
# est égal à
num -= 5
```

---

Voyons l'opérateur **assignation par multiplication** `*=`.
Il multiplie la variable par une valeur et assigne le résultat à cette variable.
Donc les suivants sont exactement les mêmes
```python
num = num * 5
# est égal à
num *= 5
```

---

Voyons l'opérateur **assignation par division** `/=`.
Il divise la variable par une valeur et assigne le résultat à cette variable.
Donc les suivants sont exactement les mêmes
```python
num = num / 5
# est égal à
num /= 5
```

---

Voyons l'opérateur **assignation par puissance** `**=`.
Il élève la variable à une puissance spécifiée et assigne le résultat à la variable.
Donc les suivants sont exactement les mêmes
```python
num = num ** 5
# est égal à
num **= 5
```

---

Voyons l'opérateur **assignation par modulo** `%=`.
Il calcule le modulo de la variable et d'une valeur et assigne le résultat à cette variable.
Donc les suivants sont exactement les mêmes
```python
num = num % 5
# est égal à
num %= 5
```

---

Voyons l'opérateur **assignation par division entière** `//=`.
Il divise la variable par une valeur (division entière) et assigne le résultat à cette variable.
Donc les suivants sont exactement les mêmes
```python
num = num // 5
# est égal à
num //= 5
```
