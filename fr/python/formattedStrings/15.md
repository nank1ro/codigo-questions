# --instructions--

Qu'est-ce qui ne va pas avec ce code ?
```python
print("{10} friends"f)
```

# --answers--

- Le caractère f devrait venir avant la chaîne
- La valeur entre les accolades n'est pas un nombre
- La chaîne manque ses guillemets " "
- La valeur à l'intérieur des accolades doit être un nom de variable
