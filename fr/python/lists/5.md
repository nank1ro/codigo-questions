# --description--

Une liste n'a pas besoin d'avoir une longueur fixe.
Vous pouvez ajouter des éléments à la fin d'une liste à tout moment !
To add an item to a list we use the `append` keyword:
```python
>>> letters = ["a", "b"]
>>> letters.append("c")
>>> print(letters)
['a', 'b', 'c']
```
