# --description--

En Python, nous pouvons parcourir une liste de manière très simple en utilisant les mots-clés `for..in` :
```python
>>> numbers = [1, 2, 3]
>>> for num in numbers:
>>>     print(num)
1
2
3
```
A variable name follows the `for` keyword, it will be assigned the value of each list item in turn.
