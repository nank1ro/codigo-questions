Vous avez peut-être envisagé une situation où vous aimeriez réutiliser un morceau de code avec juste quelques valeurs différentes.
Au lieu de réécrire tout le code, il est bien plus propre de définir une fonction, qui peut alors être utilisée à plusieurs reprises.
En Python, nous utilisons le mot-clé `def` suivi du nom de la fonction :
```python
def say_hi():
    print("Hello!")
```

---

Les parenthèses de la __définition de fonction__ n'ont pas besoin d'être vides.
En leur sein, nous pouvons spécifier des paramètres

---

Parfois, nous voulons qu'une fonction __retourne__ une valeur.
Eh bien, il y a le mot-clé `return`

---

Dans les fonctions, nous pouvons ajouter un _commentaire optionnel_ qui explique ce que la fonction fait :
```python
"""
Prints 'Hello World' to the console.
"""
```
