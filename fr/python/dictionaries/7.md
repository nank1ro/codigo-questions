# --description--

Que faire si nous voulons lister toutes les valeurs du dictionnaire ?
Eh bien, il y a la méthode `values()` .

# --seed--

```python
user = {"first_name": "John", "last_name": "Hood", "age": 30}
print([/][/])
```

# --answers--

- user
- .keys()
- .values()
- .values
- .allValues()

# --solutions--

```python
user = {"first_name": "John", "last_name": "Hood", "age": 30}
print(user.values())
```
