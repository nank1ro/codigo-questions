# --description--

Continuons avec l'opérateur relationnel **supérieur ou égal** `>=`.
Il retourne un **Booléen** (`True` ou `False`) indiquant si une expression est supérieure ou égale à l'autre, par exemple :
```python
>>> 2 >= 2
True
>>> 3 >= 2
True
>>> 3 >= 4
False
```
