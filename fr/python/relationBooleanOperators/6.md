# --description--

Continuons avec l'opérateur relationnel **inférieur ou égal** `<=`.
Il retourne un **Booléen** (`True` ou `False`) indiquant si une expression est inférieure ou égale à l'autre, par exemple :
```python
>>> 2 <= 2
True
>>> 3 <= 2
False
>>> 3 <= 4
True
```
