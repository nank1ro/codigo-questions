# --description--

Voyons l'opérateur **modulo** `%`.
Il est utilisé pour trouver le reste après une division entre deux nombres, comme :
```python
>>> 5 % 2
1
```
Ceci s'évalue à 1 car 5 divisé par 2 a un quotient de 2 et un reste de 1
```python
>>> 9 % 3
0
```
Ceci s'évalue à 0 car 9 divisé par 3 a un quotient de 3 et laisse un reste de 0
