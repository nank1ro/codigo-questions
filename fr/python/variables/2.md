# --description--

Les variables s'appellent ainsi parce que la valeur qu'elles stockent peut changer.
Nous pouvons mettre à jour `x` en utilisant `=` et en lui donnant une nouvelle valeur.
```python
>>> x = 1
>>> print(x)
1
>>> x = 2
>>> print(x)
2
```
