# --description--

Quand nous mettons à jour une variable, elle oublie sa valeur précédente. Ici, nous pouvons afficher la variable `x` deux fois et voir comment sa valeur se met à jour.
```python
>>> x = 5
>>> print(x)
5
>>> x = 10
>>> print(x)
10
```
