# --description--

Nous savons comment répéter du code en utilisant une boucle `while`.
Comme ce programme qui répète les instructions pour afficher `hello`
```python
counter = 0

while counter < 5:
    print("hello")
    counter += 1
```
Mais nous pouvons faire la même chose avec des boucles `for` :
```python
for i in range(5):
    print("hello")
```
