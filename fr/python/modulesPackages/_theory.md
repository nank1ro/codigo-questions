Un __module__ est un fichier qui peut contenir n'importe quel type de code Python.
Les modules nous permettent de réutiliser du code encore et encore.

---

Il y a beaucoup de __modules intégrés__ qui peuvent nous aider avec toutes sortes de choses, comme travailler avec des dates et des heures.
Pour inclure un module, nous utilisons le mot-clé `import`

---

Nous pouvons aussi importer uniquement les fonctions spécifiques dont nous avons besoin d'un module en utilisant les mots-clés `from` et `import`

---

Nous pouvons aussi renommer une fonction en l'important avec le mot-clé `as`
