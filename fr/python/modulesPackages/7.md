# --instructions--

Les packages ne sont rien de plus que des dossiers contenant un fichier `__init__.py`.
Que se passe-t-il si nous oublions d'ajouter un tel fichier ?

# --answers--

- Python won't recognize the folder as a package
- We can still import packages but not subpackages
- We can still import functions
