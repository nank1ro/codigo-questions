# --description--

« __Hello, World!__ » est le programme traditionnel pour commencer la programmation dans un nouveau langage.
Nous utilisons la fonction `console.log()` pour produire des données sur le dispositif de sortie standard (écran).
Pour imprimer `Hello, World!` sur l'écran avec JavaScript, nous pouvons écrire
```javascript
console.log("Hello, World!")
```
