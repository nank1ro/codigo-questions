# --description--

JavaScript possède un type booléen de base, appelé `boolean`.
Les valeurs booléennes sont appelées logiques, car elles ne peuvent être que vraies ou fausses.
Vous pouvez évaluer n'importe quelle expression en JavaScript et obtenir l'une de deux réponses, `true` ou `false`.
