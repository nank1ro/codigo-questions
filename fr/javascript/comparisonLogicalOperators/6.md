# --description--

Continuons avec l'opérateur de comparaison **inférieur ou égal à** `<=`.
Il retourne un **booléen** (`true` ou `false`) indiquant si une expression est inférieure ou égale à l'autre, par exemple :
```javascript
console.log(2 <= 2);
// prints true
console.log(3 <= 2);
// prints false
console.log(3 <= 4);
// prints true
```
