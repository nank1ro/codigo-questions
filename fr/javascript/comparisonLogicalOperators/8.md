# --description--

Continuons avec l'opérateur logique **ou** `||`.
Il retourne le premier opérande qui évalue à *vrai* ou le dernier s'ils sont tous *faux*.
```javascript
console.log(2 == 2 || 2 == 3);
// prints true
console.log(1 == 2 || 1 == 3);
// prints false
```
