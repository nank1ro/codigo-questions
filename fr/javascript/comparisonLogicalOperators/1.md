# --description--

Commençons par l'opérateur de comparaison **égal** `==`.
Il retourne un **booléen** (`true` ou `false`) indiquant si deux expressions sont égales, par exemple :
```javascript
console.log(2 == 2);
// prints true
console.log(2 == 3);
// prints false
```
