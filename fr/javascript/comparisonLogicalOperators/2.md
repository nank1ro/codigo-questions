# --description--

Continuons avec l'opérateur de comparaison **non égal** `!=`.
Il retourne un **booléen** (`true` ou `false`) indiquant si deux expressions sont **PAS** égales, par exemple :
```javascript
console.log(2 != 2);
// prints false
console.log(2 != 3);
// prints true
```
C'est exactement l'opposé de l'opérateur *égal*
