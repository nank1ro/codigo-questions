# --description--

Finissons avec l'opérateur logique **non** `!`.
Il retourne un booléen qui est l'inverse de l'état logique d'une expression.
```javascript
console.log(!true);
// prints false
console.log(!false);
// prints true
console.log(!(2 == 2));
// prints false
```
