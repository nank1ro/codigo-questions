# --description--

Voyons maintenant les opérateurs **logiques**, commençons par le premier appelé __ET__ `&&`.
Il retourne le premier opérande qui évalue à *faux* ou le dernier s'ils sont tous *vrais*.
```javascript
console.log(2 == 2 && 2 == 3);
// prints false
console.log(1 == 1 && 1 == 1.0);
// prints true
```
