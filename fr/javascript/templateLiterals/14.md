# --instructions--

Quel est le problème avec ce code ?
```javascript
console.log(`$(10} friends`);
```

# --answers--

- Les expressions doivent être à l'intérieur de ${}
- La valeur à l'intérieur de ${} n'est pas un nombre
- La chaîne de caractères n'a pas ses backticks ``
- La valeur à l'intérieur de ${} doit être un nom de variable
