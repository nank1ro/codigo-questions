# --description--

Ensuite, nous ajoutons différents types de valeurs dans les accolades pour qu'elles s'affichent comme une seule déclaration console.log. Comme ici, avec `${5}`

# --instructions--

Remplissez l'espace vide pour obtenir cette sortie
```javascript
5 friends
```

# --seed--

```javascript
console.log(` friends`);
```

# --solutions--

```javascript
console.log(`${5} friends`);
```

# --output--

5 friends
