# --instructions--

Quel est le problème avec ce code ?
```javascript
var pieces = 5;
var label = `we have ${pieces} pieces`;
```

# --answers--

- On ne peut pas enregistrer les template literals dans des variables
- Les caractères ${} sont manquants
- On ne peut pas utiliser de variables à l'intérieur de ${}
- Il n'y a aucun problème avec le code
