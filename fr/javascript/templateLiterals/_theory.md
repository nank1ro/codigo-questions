En JavaScript, nous pouvons utiliser le signe `+` pour afficher deux ou plusieurs chaînes ensemble, comme :
```javascript
console.log("Hello " + "JavaScript!") // prints "Hello JavaScript!"
```

---

Les littéraux de modèle nous permettent d'afficher des expressions comme l'ajout d'une chaîne à un nombre, sans utiliser le signe `+`.
To create a valid template literal you have to use the backticks \`\` instead of double or single quotes.
Vous pouvez insérer des variables dans les littéraux de modèle en utilisant le signe dollar et les accolades `${expression}`

---

Chaque déclaration de littéral de modèle se compose de deux parties : `${}` où nous insérons le nombre ou la variable, et la chaîne normale

---

Ensuite, nous ajoutons différents types de valeurs dans les accolades pour qu'elles s'affichent comme une seule déclaration console.log. Comme ici, avec `${5}`

---

L'insertion de variables comme `friends` entre les parenthèses rondes affiche également leur valeur

---

Nous pouvons utiliser `${}` pour insérer des valeurs aussi souvent que nous le souhaitons dans les littéraux de modèle

---

Les littéraux de modèle sont mieux utilisés dans les instructions d'impression, mais nous pouvons aussi les stocker dans des variables comme des chaînes normales.
