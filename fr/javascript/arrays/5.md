# --description--

Un tableau n'a pas besoin d'avoir une longueur fixe.
Vous pouvez ajouter des éléments à la fin d'un tableau à tout moment!
Pour ajouter un élément à un tableau, nous utilisons la fonction `push`:
```javascript
var letters = ["a", "b"];
letters.push("c");
console.log(letters);
// Prints ["a", "b", "c"]
```
