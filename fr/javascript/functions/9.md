# --instructions--

Quel est le __problème__ avec ce fragment de code ?
```javascript
console.log(greet("Mathew"));
// This function greets to the person passed in as argument
const greet = (name) => `Hello, ${name}. Good morning!`;
console.log(greet("Alex"));
```

# --answers--

- Nous ne pouvons pas utiliser greet() avant de la définir
- Nous ne pouvons pas utiliser greet() deux fois
- greet() ne retourne rien
- Nous ne pouvons pas utiliser name comme paramètre
