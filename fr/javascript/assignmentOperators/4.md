# --description--

Voyons l'opérateur **multiplication assignment** `*=`.
Il multiplie la variable par une valeur et assigne le résultat à cette variable.
Donc les éléments suivants sont exactement les mêmes
```javascript
var num = num * 5;
// is equal to
num *= 5;
```
