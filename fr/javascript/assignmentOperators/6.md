# --description--

Voyons l'opérateur **remainder assignment** `%=`.
Il calcule le reste de la variable et d'une valeur et assigne le résultat à cette variable.
Donc les éléments suivants sont exactement les mêmes
```javascript
num = num % 5;
// is equal to
num %= 5;
```
