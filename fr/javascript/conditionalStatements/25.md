# --instructions--

Comment savons-nous que `Hello World` sera toujours affiché ?
```javascript
var online = False;
if (online) {
    console.log("User is online");
}
console.log("Hello World");
```

# --answers--

- Car ce n'est pas dans un bloc de code
- Car l'instruction if ne peut pas avoir plus d'une ligne
- Car il n'y a pas d'instruction else
- Car il n'y a pas d'instruction else if
