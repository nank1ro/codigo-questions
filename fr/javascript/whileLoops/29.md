# --instructions--

Quel est le problème avec ce code ?
```javascript
while (isLoading) {
    console.log("Loading...");
    isLoading = false;
}
```

# --answers--

- isLoading n'est pas un nom de variable valide
- La boucle while a une déclaration print dans le bloc de code
- Le bloc de code n'a pas l'indentation correcte
