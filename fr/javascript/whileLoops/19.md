# --instructions--

Comment savons-nous que c'est une boucle infinie ?
```javascript
var isLogged = false;

while (true) {
    console.log("Please login");
}
```

# --answers--

- Cette boucle ne s'exécutera pas car la condition est false
- Il n'y a pas de variable dans la condition
- Cette boucle s'exécutera une fois
- Cette boucle s'exécutera deux fois
