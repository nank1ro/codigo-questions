# --instructions--

What does `counter += 1` do?
```javascript
var counter = 1;

while (counter < 5) {
    console.log(counter);
    counter += 1;
}
```

# --answers--

- Il augmente le compteur d'un à chaque fois que la boucle exécute son bloc de code
- Il crée la variable de compteur
- Il augmente le compteur de deux à chaque fois
- Il augmente le compteur de trois à chaque fois
