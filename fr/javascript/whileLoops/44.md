# --instructions--

Ce bloc de code sera-t-il jamais exécuté ?
```javascript
var shouldRepeat = false;

do {
    console.log("run code");
} while (shouldRepeat);
```

# --answers--

- Oui parce que le do-while s'exécute une fois avant de considérer la condition de la boucle
- Non parce que shouldRepeat est false
- Ce bloc de code sera exécuté deux fois
- Ce bloc de code sera exécuté trois fois
