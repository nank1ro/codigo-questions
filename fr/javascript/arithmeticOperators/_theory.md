Les opérateurs sont utilisés pour effectuer des opérations sur des variables et des valeurs.
Commençons par les opérateurs arithmétiques, en particulier avec l'opérateur **addition** `+`.
Il est utilisé pour ajouter deux nombres, comme:
```javascript
console.log(5 + 3);
// prints 8
```

---

Continuons avec l'opérateur **subtraction** `-`.
Il est utilisé pour soustraire un nombre d'un autre, comme:
```javascript
console.log(5 - 3);
// prints 2
```

---

Voyons l'opérateur **multiplication** `*`.
Il est utilisé pour multiplier deux nombres ensemble, comme:
```javascript
console.log(5 * 3);
// prints 15
```

---

Voyons l'opérateur **division** `/`.
Il est utilisé pour diviser deux nombres ensemble, comme:
```javascript
console.log(10 / 5);
// prints 2
```

---

Voyons l'opérateur **remainder** `%`.
Il est utilisé pour trouver le reste après une division entre deux nombres, comme:
```javascript
console.log(5 % 2);
// prints 1
```
Cela évalue à 1 car 5 divisé par 2 a un quotient de 2 et un reste de 1
```javascript
console.log(9 % 3);
// prints 0
```
Cela évalue à 0 car 9 divisé par 3 a un quotient de 3 et laisse un reste de 0
