# --description--

Voyons l'opérateur **remainder** `%`.
Il est utilisé pour trouver le reste après une division entre deux nombres, comme:
```javascript
console.log(5 % 2);
// prints 1
```
Cela évalue à 1 car 5 divisé par 2 a un quotient de 2 et un reste de 1
```javascript
console.log(9 % 3);
// prints 0
```
Cela évalue à 0 car 9 divisé par 3 a un quotient de 3 et laisse un reste de 0
