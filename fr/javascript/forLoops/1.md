# --description--

Nous savons comment répéter du code en utilisant une boucle `while`.
Comme ce programme qui répète les instructions pour afficher `bonjour`
```javascript
var counter = 0;

while (counter < 5) {
    console.log("hello");
    counter++;
}
```
Mais nous pouvons faire la même chose avec les boucles `for` :
```javascript
for (let i = 0; i < 5; i++) {
    console.log("hello");
}
```
