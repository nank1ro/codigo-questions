# --description--

Une fonction à expression unique utilise `=` au lieu d'un corps de bloc :
```kotlin
fun triple(n: Int) = n * 3
```
C'est l'abréviation de :
```kotlin
fun triple(n: Int): Int {
    return n * 3
}
```

# --instructions--

Complétez la fonction à expression unique qui additionne deux entiers.
