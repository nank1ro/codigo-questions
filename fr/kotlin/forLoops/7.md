# --description--

Si vous avez simplement besoin de répéter un bloc de code `n` fois, vous pouvez utiliser la fonction `repeat(times: Int)`.

```kotlin
repeat(3) {
    println("repeat")
}
// prints repeat 3 times
```

Vous pouvez même accéder à l'index avec
```kotlin
repeat(3) { index ->
    println(index)
}
// prints 0, 1, 2
```
