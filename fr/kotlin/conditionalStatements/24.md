# --instructions--

Comment savons-nous que `Hello World` sera toujours affiché ?
```kotlin
var online = false;
if (online) {
    println("User is online")
}
println("Hello World")
```

# --answers--

- Parce qu'il n'est pas à l'intérieur d'un bloc de code
- Parce que l'instruction if ne peut pas avoir plus d'une ligne
- Parce qu'il n'y a pas d'instruction else
- Parce qu'il n'y a pas d'instruction else if
