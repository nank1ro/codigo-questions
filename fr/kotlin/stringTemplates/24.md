# --instructions--

Qu'y a-t-il de mal avec ce code ?
```kotlin
val pieces = 5
val label = "we have $pieces pieces"
```

# --answers--

- Nous ne pouvons pas enregistrer de modèles de chaînes dans des variables
- Les caractères $ manquent
- Nous ne pouvons pas utiliser de variables à l'intérieur de ${}
- Rien ne va pas avec le code
