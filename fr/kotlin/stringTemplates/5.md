# --description--

Si ce qui suit le signe `$` n'est pas reconnaissable comme un identificateur de programme, rien de spécial ne se passe

# --instructions--

Remplissez l'espace vide pour obtenir cette sortie
```kotlin
5 friends
```

    println(" friends")
