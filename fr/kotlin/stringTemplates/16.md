# --instructions--

Qu'y a-t-il de mal avec ce code ?
```kotlin
println("$(10} friends")
```

# --answers--

- L'identificateur devrait être à l'intérieur de ${}
- La valeur à l'intérieur de ${} n'est pas un nombre
- La chaîne manque de guillemets " "
- La valeur à l'intérieur de ${} doit être un nom de variable
