# --instructions--

Qu'y a-t-il de mal avec ce code ?
```kotlin
val manchesterUnited = 2
val barcelona = 2
println("The final score is $manchesterUnited : $barcelona")
```

# --answers--

- Rien ne va pas avec le code
- manchesterUnited et barcelona n'ont pas de valeurs
- Nous ne pouvons pas utiliser de variables entre les accolades
- Nous ne pouvons pas utiliser $ deux fois dans la même chaîne
