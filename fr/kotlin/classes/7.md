# --instructions--

Un **objet companion** est un singleton associé à une classe. Il permet de définir des propriétés et des méthodes qui appartiennent à la classe elle-même plutôt qu'aux instances — similaire aux membres `static` dans d'autres langages :

```kotlin
class MathHelper {
    companion object {
        fun square(n: Int): Int = n * n
    }
}
println(MathHelper.square(4)) // 16
```

Créez une classe `Counter` avec un objet companion qui contient une propriété `var` nommée `total` de type `Int` (initialisée à `0`) et une méthode `increment()` qui ajoute `1` à `total`.

# --asserts--

`Counter.total` doit commencer à `0`

```kotlin
    tryCatch(Counter.total == 0)
```

Après un `increment()`, `total` doit être `1`

```kotlin
    Counter.increment()
    tryCatch(Counter.total == 1)
```

Après deux autres appels à `increment()`, `total` doit être `3`

```kotlin
    Counter.increment()
    Counter.increment()
    tryCatch(Counter.total == 3)
```
