# --description--

Les classes peuvent contenir des **méthodes** — des fonctions qui appartiennent à la classe et opèrent sur ses propriétés :

```kotlin
class Counter(var count: Int) {
    fun increment() {
        count++
    }
    fun value(): Int {
        return count
    }
}
```

Les méthodes sont appelées en utilisant la notation pointée sur une instance :

```kotlin
val c = Counter(0)
c.increment()
println(c.value()) // 1
```

# --asserts--

Après un dépôt de `100.0` sur un compte avec un solde de `0.0`, le solde doit être `100.0`

```kotlin
    val account = BankAccount(0.0)
    account.deposit(100.0)
    tryCatch(account.balance == 100.0)
```

Après un retrait de `30.0`, le solde doit être `70.0`

```kotlin
    account.withdraw(30.0)
    tryCatch(account.balance == 70.0)
```

Après un autre dépôt de `50.0`, le solde doit être `120.0`

```kotlin
    account.deposit(50.0)
    tryCatch(account.balance == 120.0)
```
