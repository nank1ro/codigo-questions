# --description--

Voici un exemple d'utilisation de `break` dans une boucle _while_:
```kotlin
var i = 0
while (i < 3) {
  i++
  if (i == 2) break // [1]
  println(i)
}
// prints 1
```

Comme vous pouvez le voir au __[1]__ quand `i` est égal à _2_, nous _romperions_ la boucle. En fait, les nombres 2 et 3 ne s'impriment jamais.
