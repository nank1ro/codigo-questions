# --instructions--

Qu'y a-t-il de mal avec ce code?
```kotlin
while (isLoading) {
    println("Loading...")
    isLoading = false
}
```

# --answers--

- isLoading n'est pas un nom de variable valide
- La boucle while a une instruction d'impression à l'intérieur du bloc de code
- Le bloc de code n'a pas la bonne indentation
