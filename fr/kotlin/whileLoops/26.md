Qu'est-ce qui cause le crash de ce programme?
```kotlin

    println("repeat")
}
```

# --answers--

- Le code a une boucle infinie
- Rien ne cause l'arrêt de ce code
- La boucle while n'a pas la bonne indentation
- Le bloc de code n'a pas la bonne indentation
