# --instructions--

Comment savons-nous que cette boucle s'arrête?
```kotlin
var isLoading = true

while (isLoading) {
    println("Loading...")
    isLoading = false
}
```

# --answers--

- La variable dans la boucle se met à jour à false
- La boucle ne s'arrêtera jamais
