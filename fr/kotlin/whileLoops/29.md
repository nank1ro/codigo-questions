# --instructions--

Que fait `counter += 1`?
```kotlin
var counter = 1

while (counter < 5) {
    println(counter)
    counter += 1
}
```

# --answers--

- Il augmente le compteur d'un à chaque exécution du bloc de code de la boucle
- Il crée la variable de compteur
- Il augmente le compteur de deux à chaque fois
- Il augmente le compteur de trois à chaque fois
