"__Hello, World!__" est le programme traditionnel premier pour commencer la programmation dans un nouveau langage.
Nous utilisons la fonction `println()` pour afficher les données sur le périphérique de sortie standard (écran).
Pour imprimer `Hello, World!` sur l'écran avec Kotlin, nous pouvons écrire
```kotlin
println("Hello, World!")
```

---

`println()` déplace le curseur vers une nouvelle ligne après l'affichage de son paramètre, de sorte que la sortie suivante apparaît sur la ligne suivante.
Vous pouvez utiliser `print()` à la place, qui laisse le curseur sur la même ligne.
