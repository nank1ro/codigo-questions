# --description--

Commençons par l'opérateur de comparaison **égal** `==`.
Il retourne un **boolean** (`true` ou `false`) indiquant si deux expressions sont égales, par exemple :
```kotlin
println(2 == 2) // true
println(2 == 3) // false
```
