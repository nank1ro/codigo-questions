# --description--

Continuons avec l'opérateur de comparaison **pas égal** `!=`.
Il retourne un **boolean** (`true` ou `false`) indiquant si deux expressions ne sont **PAS** égales, par exemple :
```kotlin
println(2 != 2) // false
println(2 != 3) // true
```
C'est exactement l'opposé de l'opérateur *égal*
