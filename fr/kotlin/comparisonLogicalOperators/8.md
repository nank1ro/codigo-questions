# --description--

Continuons avec l'opérateur logique **ou** `||`.
Il retourne le premier opérande qui s'évalue à *true* ou le dernier s'ils sont tous *false*.
```kotlin
println(2 == 2 || 2 == 3) // true
println(1 == 2 || 1 == 3) // false
```
