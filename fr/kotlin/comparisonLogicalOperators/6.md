# --description--

Continuons avec l'opérateur de comparaison **inférieur ou égal à** `<=`.
Il retourne un **boolean** (`true` ou `false`) indiquant si une expression est inférieure ou égale à l'autre, par exemple :
```kotlin
println(2 <= 2) // true
println(3 <= 2) // false
println(3 <= 4) // true
```
