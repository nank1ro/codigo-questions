# --description--

L'ordre des éléments dans un `Set` n'est pas important.
En fait, si vous essayez de comparer deux `Set`s avec le même élément mais dans un ordre différent, vous obtenez qu'ils sont égaux.

# --instructions--

- Vérifiez si le premier et le deuxième `Set`s sont égaux
- Vérifiez si le premier et le troisième `Set`s sont égaux

# --seed--

```kotlin
val names1 = setOf("Charlotte", "Mia")
val names2 = setOf("Mia", "Charlotte")
val names3 = setOf("Luke", "Ariana")
println([/][/][/])
```
