# --description--

We can use `..<` to loop until the next number excluded, or `...` to loop until the next number included

# --instructions--

Complétez la boucle `for` dans le code suivant in order to execute the code block `3` times

# --seed--

```swift
for i in 1[/]3 {
    print(i)
}
```
