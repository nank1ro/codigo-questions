# --description--

La fonction `stride()` retourne une séquence de nombres.
Elle nécessite les paramètres _from_, _to_ et _by_.
Voici la syntaxe de la fonction :
```swift
stride(from:to:by:)
```
Gardez à l'esprit que la valeur `to` est exclue

# --instructions--

Complétez la fonction `stride()` dans le code suivant pour afficher les nombres suivants `(3, 6, 9, 12)`

# --seed--

```swift
for i in stride(from: [/], to: [/], by: [/]) {
    print(i)
}
```
