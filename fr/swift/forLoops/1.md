# --description--

Nous savons comment répéter du code en utilisant une boucle `while`.
Comme ce programme qui répète des instructions pour afficher `hello`
```swift
var counter = 0

while counter < 5 {
    print("hello")
    counter += 1;
}
```
Mais nous pouvons faire la même chose avec les boucles `for` :
```swift
for i in 0..<5 {
    print("hello")
}
```
