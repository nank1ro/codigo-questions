# --description--

Un tableau ne doit pas nécessairement avoir une longueur fixe.
Vous pouvez ajouter des éléments à la fin d'un tableau à tout moment !
Pour ajouter un élément à un tableau, nous utilisons la fonction `append` :
```swift
var letters = ["a", "b"]
letters.append("c")
print(letters)
// Affiche ["a", "b", "c"]
```
