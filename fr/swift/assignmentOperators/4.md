# --description--

Regardons l'opérateur **multiplication assignment** `*=`.
Il multiplie la variable par une valeur et assigne le résultat à cette variable.
Les éléments suivants sont exactement les mêmes
```swift
var num = num * 5
// est égal à
num *= 5
```
