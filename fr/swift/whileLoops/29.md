# --instructions--

Quel est le problème avec ce code ?
```swift
while isLoading {
    print("Loading...")
    isLoading = false
}
```

# --answers--

- isLoading n'est pas un nom de variable valide
- La boucle while a une instruction print à l'intérieur du bloc de code
- Le bloc de code n'a pas la bonne indentation
