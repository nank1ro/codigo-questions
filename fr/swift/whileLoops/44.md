# --instructions--

Ce bloc de code sera-t-il un jour exécuté ?
```swift
var shouldRepeat = false

repeat {
    print("run code")
} while shouldRepeat
```

# --answers--

- Oui car le repeat-while s'exécute une fois avant de considérer la condition de la boucle
- Non car shouldRepeat est false
- Ce bloc de code sera exécuté deux fois
- Ce bloc de code sera exécuté trois fois
