# --instructions--

Que fait `counter += 1` ?
```swift
var counter = 1

while counter < 5 {
    print(counter)
    counter += 1
}
```

# --answers--

- Il augmente counter de un à chaque fois que la boucle exécute son bloc de code
- Il crée la variable counter
- Il augmente counter de deux à chaque fois
- Il augmente counter de trois à chaque fois
