# --description--

Les variables sont appelées de cette façon parce que la valeur qu'elles stockent peut changer.
Nous pouvons mettre à jour `x` en utilisant `=` et en lui donnant une nouvelle valeur.
```swift
var x = 1
print(x) // prints 1
x = 2
print(x) // prints 2
```
