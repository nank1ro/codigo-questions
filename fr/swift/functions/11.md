# --instructions--

Quel est le __problème__ avec cet extrait de code ?
```swift
print(greet(name: "Mathew"))
/// This function greets to the person passed in as argument
func greet(name: String) -> String {
    return "Hello, \(name). Good morning!"
}
print(greet(name: "Alex"))
```

# --answers--

- On ne peut pas utiliser greet() avant de la définir
- On ne peut pas utiliser greet() deux fois
- greet() ne retourne rien
- On ne peut pas utiliser name comme paramètre
