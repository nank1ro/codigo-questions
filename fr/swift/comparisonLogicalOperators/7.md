# --description--

Maintenant, voyons les opérateurs **logiques**, commençons par le premier appelé **ET** `&&`.
Il retourne le premier opérande qui s'évalue à *false* ou le dernier si tous sont *true*.
```swift
print(2 == 2 && 2 == 3) // false
print(1 == 1 && 1 == 1.0) // true
```
