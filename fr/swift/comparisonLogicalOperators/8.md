# --description--

Continuons avec l'opérateur logique **ou** `||`.
Il retourne le premier opérande qui s'évalue à *true* ou le dernier si tous sont *false*.
```swift
print(2 == 2 || 2 == 3) // true
print(1 == 2 || 1 == 3) // false
```
