# --description--

Continuons avec l'opérateur de comparaison **supérieur ou égal à** `>=`.
Il retourne une **valeur booléenne** (`true` ou `false`) indiquant si une expression est supérieure ou égale à l'autre, par exemple :
```swift
print(2 >= 2) // true
print(3 >= 2) // true
print(3 >= 4) // false
```
