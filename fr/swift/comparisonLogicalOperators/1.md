# --description--

Commençons par l'opérateur de comparaison **égal** `==`.
Il retourne une **valeur booléenne** (`true` ou `false`) indiquant si deux expressions sont égales, par exemple :
```swift
print(2 == 2) // true
print(2 == 3) // false
```
