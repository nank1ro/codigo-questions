# --instructions--

Utilisez `\()` pour insérer le nombre `500` dans l'espace vide de l'interpolation de chaîne

# --seed--

```swift
print("There are  items")
```

# --solutions--

```swift
print("There are \(500) items")
```

# --output--

There are 500 items
