# --description--

Ensuite, nous ajoutons un type de valeur différent entre accolades pour qu'il s'affiche comme une seule instruction print. Comme ici, avec `\(5)`

# --instructions--

Remplissez l'espace vide pour obtenir cette sortie
```swift
5 friends
```

# --seed--

```swift
print(" friends")
```

# --solutions--

```swift
print("\(5) friends")
```

# --output--

5 friends
