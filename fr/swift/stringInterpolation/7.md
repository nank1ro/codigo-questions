# --description--

Nous pouvons utiliser les parenthèses pour insérer des valeurs autant de fois que nous le souhaitons dans l'interpolation de chaîne

# --instructions--

Voyez ici avec `2` puis avec `8`

# --seed--

```swift
print("\([/]) new friends and \([/]) old friends")
```

# --answers--

- 8

# --solutions--

```swift
print("\(2) new friends and \(8) old friends")
```
