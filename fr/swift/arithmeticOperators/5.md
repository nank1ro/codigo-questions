# --description--

Regardons l'opérateur **reste** `%`.
Il est utilisé pour trouver le reste d'une division entre deux nombres, comme :
```swift
print(5 % 2) // 1
```
Cela s'évalue à 1 car 5 divisé par 2 a un quotient de 2 et un reste de 1
```swift
print(9 % 3) // 0
```
Cela s'évalue à 0 car 9 divisé par 3 a un quotient de 3 et laisse un reste de 0
