# --description--

Voyons l'opérateur **division assignment** `/=`.
Il divise la variable par une valeur et assigne le résultat à cette variable.
Donc les éléments suivants sont exactement les mêmes
```c
num = num / 5;
// is equal to
num /= 5;
```
