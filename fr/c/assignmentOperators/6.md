# --description--

Voyons l'opérateur **modulus assignment** `%=`.
Il calcule le modulus de la variable et d'une valeur et assigne le résultat à cette variable.
Donc les éléments suivants sont exactement les mêmes
```c
num = num % 5;
// is equal to
num %= 5;
```
