# --instructions--

Pourquoi `true` n'est-il pas une chaîne de caractères ?
```c
bool a = true;
```

# --answers--

- C'est stocké dans une variable
- Il n'y a pas de guillemets autour et le type n'est pas char
- Cela commence par une majuscule
