# --instructions--

Quel est le __problème__ avec cet extrait de code ?
```c
greet("Daniel");
void greet(char *name) {
    printf("Hello %s!\n", name);
};
```

# --answers--

- Nous ne pouvons pas utiliser greet() avant de la définir
- Nous ne pouvons pas utiliser greet() deux fois
- greet() ne retourne rien
- Nous ne pouvons pas utiliser name comme paramètre
