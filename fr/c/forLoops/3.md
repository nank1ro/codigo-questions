# --description--

Nous pouvons utiliser `<` pour boucler jusqu'au nombre suivant exclus, ou `<=` pour boucler jusqu'au nombre suivant inclus

# --instructions--

Complétez la boucle `for` dans le code suivant afin d'exécuter le bloc de code `3` fois

# --seed--

```c
for (int i = 1; i[/]3; i++) {
        printf("%d\n", i);
}
```
