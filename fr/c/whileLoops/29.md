# --instructions--

Qu'y a-t-il de mal avec ce code?
```c
while (is_loading) {
    printf("Loading...\n");
    is_loading = false;
}
```

# --answers--

- is_loading n'est pas un nom de variable valide
- La boucle while a une instruction d'impression à l'intérieur du bloc de code
- Le bloc de code n'a pas l'indentation correcte
