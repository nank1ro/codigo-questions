# --instructions--

Que fait `counter += 1`?
```c
int counter = 1;

while (counter < 5) {
    printf("%d\n", counter);
    counter += 1;
}
```

# --answers--

- Il augmente le compteur de un à chaque fois que la boucle exécute son bloc de code
- Il crée la variable counter
- Il augmente le compteur de deux à chaque fois
- Il augmente le compteur de trois à chaque fois
