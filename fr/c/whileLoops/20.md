# --instructions--

Comment savons-nous que la boucle `while` ignore son bloc de code?
```c
bool is_loading = false;

while (is_loading) {
    printf("Loading...\n");
}
```

# --answers--

- La variable dans la condition est comparée à true
- La variable dans la condition est comparée à false

# --solutions--
