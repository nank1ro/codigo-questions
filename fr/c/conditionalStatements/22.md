# --instructions--

Comment savons-nous que `Hello World` sera toujours imprimé?
```c
bool online = false;
if (online) {
    printf("User is online\n");
}
printf("Hello World\n");
```

# --answers--

- Parce qu'il ne se trouve pas à l'intérieur de la déclaration conditionnelle
- Parce que la déclaration if ne peut pas avoir plus d'une ligne
- Parce qu'il n'y a pas de déclaration else
- Parce qu'il n'y a pas de déclaration else if
