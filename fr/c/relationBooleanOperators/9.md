# --description--

Terminons avec l'opérateur booléen **non** `!`.
Il retourne un booléen qui est l'inverse de l'état logique d'une expression.
```c
>>> !true
0
>>> !false
1
>>> !(2 == 2)
0
```

# --instructions--

Imprimez un booléen vérifiant si `num` n'est **PAS** égal à `5`
