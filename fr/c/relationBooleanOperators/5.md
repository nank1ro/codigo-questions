# --description--

Continuons avec l'opérateur relationnel **plus grand ou égal** `>=`.
Il retourne un **booléen**, vrai `1` ou faux `0`, indiquant si une expression est plus grande ou égale à l'autre, par exemple:
```c
>>> 2 >= 2
1
>>> 3 >= 2
1
>>> 3 >= 4
0
```
