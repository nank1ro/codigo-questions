# --description--

Continuons avec l'opérateur relationnel **non égal** `!=`.
Il retourne un **booléen**, vrai `1` ou faux `0`, indiquant si deux expressions ne sont **PAS** égales, par exemple:
```c
>>> 2 != 2
0
>>> 2 != 3
1
```
C'est exactement l'opposé de l'opérateur *égal*
