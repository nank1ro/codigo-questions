# --description--

Commençons par l'opérateur relationnel **égal** `==`.
Il retourne un **booléen**, vrai `1` ou faux `0`, indiquant si deux expressions sont égales, par exemple:
```c
>>> 2 == 2
1
>>> 2 == 3
0
```
