# --instructions--

Qu'est-ce qui ne va pas avec ce code ?
```c
int manchester_united = 0;
int barcelona = 1;
printf("The score is %i : %i\n", manchester_united, barcelona);
```

# --answers--

- Rien ne va pas avec le code
- manchester_united et barcelona n'ont pas de valeurs
- Nous ne pouvons pas utiliser de variables dans la chaîne formatée
- Nous ne pouvons pas utiliser %i deux fois dans la même chaîne
