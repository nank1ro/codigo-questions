# --instructions--

Comment afficher les valeurs entières avec les chaînes formatées ?

# --answers--

- Nous les plaçons en utilisant l'expression %i
- Nous ne pouvons pas afficher les valeurs numériques avec les chaînes formatées
- Nous les plaçons entre crochets []
- Nous les plaçons entre parenthèses ()
