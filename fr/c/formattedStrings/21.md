# --instructions--

Qu'est-ce qui ne va pas avec ce code ?
```c
int pieces = 5;
char label[] = "we have %i pieces", 5;
```

# --answers--

- Nous ne pouvons pas enregistrer les chaînes formatées dans des variables de chaîne
- Le caractère f est manquant
- Nous ne pouvons pas utiliser de variables à l'intérieur de %
- Rien ne va pas avec le code
