# --description--

En C, nous utilisons la fonction `printf` avec une expression pour afficher deux ou plusieurs valeurs ensemble, comme :
```c
>>> char c = 'C';
>>> printf("Hello %c!\n", c);
Hello C!
```
`%c` est utilisé pour afficher un seul caractère

# --instructions--

Imprimez le nom en utilisant l'expression `%s` (utilisée pour afficher un _tableau de caractères_ ou une _chaîne_)
