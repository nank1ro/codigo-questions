# --instructions--

Qu'est-ce qui ne va pas avec ce code ?
```c
printf(0, "%i friends");
```

# --answers--

- La valeur de l'expression doit venir après la chaîne
- L'expression n'est pas un nombre
- La chaîne manque ses guillemets " "
- L'expression doit obtenir un nom de variable
