# --description--

Les chaînes formatées nous permettent d'afficher des expressions comme ajouter une chaîne à un nombre, sans aucune erreur.
Pour afficher un entier, nous utilisons l'expression `%i`

# --seed--

```c
printf("[/] friends\n", [/]);
```

# --answers--

- %i
- ?i

- 5

# --solutions--

```c
printf("%i friends\n", 5);
```

# --output--

5 friends
