# --description--

Klassen können **Methoden** enthalten — Funktionen, die zur Klasse gehören und auf ihren Eigenschaften arbeiten:

```kotlin
class Counter(var count: Int) {
    fun increment() {
        count++
    }
    fun value(): Int {
        return count
    }
}
```

Methoden werden mit Punktnotation auf einer Instanz aufgerufen:

```kotlin
val c = Counter(0)
c.increment()
println(c.value()) // 1
```

# --asserts--

Nach einer Einzahlung von `100.0` in ein Konto mit Kontostand `0.0` muss der Kontostand `100.0` betragen

```kotlin
    val account = BankAccount(0.0)
    account.deposit(100.0)
    tryCatch(account.balance == 100.0)
```

Nach einer Abhebung von `30.0` muss der Kontostand `70.0` betragen

```kotlin
    account.withdraw(30.0)
    tryCatch(account.balance == 70.0)
```

Nach einer weiteren Einzahlung von `50.0` muss der Kontostand `120.0` betragen

```kotlin
    account.deposit(50.0)
    tryCatch(account.balance == 120.0)
```

class BankAccount(var balance: Double) {
    fun deposit(amount: Double) {
        balance += amount
    }
    fun withdraw(amount: Double) {
        balance -= amount
    }
}
