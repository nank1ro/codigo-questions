# --instructions--

Ein **Companion Object** ist ein Singleton, das einer Klasse zugeordnet ist. Es ermöglicht die Definition von Eigenschaften und Methoden, die zur Klasse selbst gehören, nicht zu Instanzen — ähnlich wie `static`-Mitglieder in anderen Sprachen:

```kotlin
class MathHelper {
    companion object {
        fun square(n: Int): Int = n * n
    }
}
println(MathHelper.square(4)) // 16
```

Erstelle eine Klasse `Counter` mit einem Companion Object, das eine `var`-Eigenschaft `total` vom Typ `Int` (initialisiert mit `0`) und eine Methode `increment()` enthält, die `total` um `1` erhöht.

# --asserts--

`Counter.total` muss bei `0` beginnen

```kotlin
    tryCatch(Counter.total == 0)
```

Nach einem `increment()`-Aufruf muss `total` gleich `1` sein

```kotlin
    Counter.increment()
    tryCatch(Counter.total == 1)
```

Nach zwei weiteren `increment()`-Aufrufen muss `total` gleich `3` sein

```kotlin
    Counter.increment()
    Counter.increment()
    tryCatch(Counter.total == 3)
```
