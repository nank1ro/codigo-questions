# --instructions--

In Kotlin ermöglicht **Vererbung**, dass eine Klasse eine andere Klasse erweitert. Eine Klasse muss mit `open` markiert sein, damit sie als Basisklasse verwendet werden kann. Methoden und Eigenschaften werden mit dem Schlüsselwort `override` überschrieben.

Erstelle eine `open`-Klasse `Animal` mit einer `val`-Eigenschaft `name` vom Typ `String` und einer `open`-Methode `speak(): String`, die `"..."` zurückgibt. Erstelle dann eine Klasse `Dog`, die `Animal` erweitert und `speak()` überschreibt, um `"Woof"` zurückzugeben.

```kotlin

    tryCatch(d.speak() == "Woof")

open class Animal(val name: String) {
    open fun speak(): String {
        return "..."
    }
}

class Dog(name: String) : Animal(name) {
    override fun speak(): String {
        return "Woof"
    }
}
```
