# --description--

Lassen Sie uns den **Modulo** `%` Operator sehen.
Er wird verwendet, um den Rest nach einer Division zwischen zwei Zahlen zu finden, wie:
```kotlin
println(5 % 2) // 1
```
Dies ergibt 1, weil 5 geteilt durch 2 einen Quotienten von 2 und einen Rest von 1 hat
```kotlin
println(9 % 3) // 0
```
Dies ergibt 0, weil 9 geteilt durch 3 einen Quotienten von 3 hat und keinen Rest übrig lässt
