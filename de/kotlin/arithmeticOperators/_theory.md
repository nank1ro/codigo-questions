Operatoren werden verwendet, um Operationen auf Variablen und Werten auszuführen.
Lassen Sie uns mit den arithmetischen Operatoren beginnen, insbesondere mit dem **Addition** `+` Operator.
Er wird verwendet, um zwei Zahlen zu addieren, wie:
```kotlin
println(5 + 3) // 8
```

---

Lassen Sie uns mit dem **Subtraktion** `-` Operator fortfahren.
Er wird verwendet, um eine Zahl von einer anderen zu subtrahieren, wie:
```kotlin
println(5 - 3) // 2
```

---

Lassen Sie uns den **Multiplikation** `*` Operator sehen.
Er wird verwendet, um zwei Zahlen miteinander zu multiplizieren, wie:
```kotlin
println(5 * 3) // 15
```

---

Lassen Sie uns den **Division** `/` Operator sehen.
Er wird verwendet, um zwei Zahlen miteinander zu dividieren, wie:
```kotlin
println(10 / 5) // 2
```

---

Lassen Sie uns den **Modulo** `%` Operator sehen.
Er wird verwendet, um den Rest nach einer Division zwischen zwei Zahlen zu finden, wie:
```kotlin
println(5 % 2) // 1
```
Dies ergibt 1, weil 5 geteilt durch 2 einen Quotienten von 2 und einen Rest von 1 hat
```kotlin
println(9 % 3) // 0
```
Dies ergibt 0, weil 9 geteilt durch 3 einen Quotienten von 3 hat und keinen Rest übrig lässt
