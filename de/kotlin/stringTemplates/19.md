# --instructions--

Verwende `${}`, um die Zahl `500` in den leeren Platz des String-Templates einzufügen

# --seed--

```kotlin
fun main() {
    println("There are  items")
}
```

# --solutions--

```kotlin
fun main() {
    println("There are ${500} items")
}
```

# --output--

There are 500 items
