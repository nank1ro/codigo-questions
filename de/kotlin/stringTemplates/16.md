# --instructions--

Was ist an diesem Code falsch?
```kotlin
println("$(10} friends")
```

# --answers--

- Der Bezeichner sollte sich in ${} befinden
- Der Wert in ${} ist keine Zahl
- Dem String fehlen seine Anführungszeichen " "
- Der Wert in ${} muss ein Variablenname sein
