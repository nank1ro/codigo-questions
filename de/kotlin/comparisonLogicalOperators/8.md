# --description--

Fahren wir mit dem **oder** `||` logischen Operator fort.
Er gibt den ersten Operanden zurück, der zu *true* ausgewertet wird, oder den letzten, wenn alle *false* sind.
```kotlin
println(2 == 2 || 2 == 3) // true
println(1 == 2 || 1 == 3) // false
```
