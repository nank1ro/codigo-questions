# --description--

Fahren wir mit dem **größer als**-Vergleichsoperator `>` fort.
Er gibt einen **booleschen Wert** (`true` oder `false`) zurück, der angibt, ob ein Ausdruck größer als der andere ist, zum Beispiel:
```kotlin
println(2 > 2) // false
println(3 > 2) // true
```
