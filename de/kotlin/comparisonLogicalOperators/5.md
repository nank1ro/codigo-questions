# --description--

Fahren wir mit dem **größer als oder gleich**-Vergleichsoperator `>=` fort.
Er gibt einen **booleschen Wert** (`true` oder `false`) zurück, der angibt, ob ein Ausdruck größer als oder gleich dem anderen ist, zum Beispiel:
```kotlin
println(2 >= 2) // true
println(3 >= 2) // true
println(3 >= 4) // false
```
