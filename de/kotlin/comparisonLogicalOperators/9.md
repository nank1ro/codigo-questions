# --description--

Abschließend mit dem **nicht** `!` logischen Operator.
Er gibt einen booleschen Wert zurück, der das Gegenteil des logischen Zustands eines Ausdrucks ist.
```kotlin
println(!true) // false
println(!false) // true
println(!(2 == 2)) // false
```
