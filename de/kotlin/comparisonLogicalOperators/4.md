# --description--

Fahren wir mit dem **kleiner als**-Vergleichsoperator `<` fort.
Er gibt einen **booleschen Wert** (`true` oder `false`) zurück, der angibt, ob ein Ausdruck kleiner als der andere ist, zum Beispiel:
```kotlin
println(2 < 2) // false
println(2 < 3) // true
```
