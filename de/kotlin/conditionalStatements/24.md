# --instructions--

Wie wissen wir, dass `Hello World` immer gedruckt wird?
```kotlin
var online = false;
if (online) {
    println("User is online")
}
println("Hello World")
```

# --answers--

- Weil es sich nicht in einem Code-Block befindet
- Weil die if-Anweisung nicht mehr als eine Zeile haben kann
- Weil es keine else-Anweisung gibt
- Weil es keine else-if-Anweisung gibt
