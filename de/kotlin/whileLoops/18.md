# --instructions--

Woher wissen wir, dass die `while`-Schleife ihren Code-Block überspringt?
```kotlin
var isLoading = false

while (isLoading) {
    println("Loading...")
}
```

# --answers--

- Die Variable in der Bedingung wird mit true verglichen
- Die Variable in der Bedingung wird mit false verglichen
