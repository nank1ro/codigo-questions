# --instructions--

Was macht `counter += 1`?
```kotlin
var counter = 1

while (counter < 5) {
    println(counter)
    counter += 1
}
```

# --answers--

- Es erhöht den Zähler jedes Mal um eins, wenn die Schleife ihren Codeblock ausführt
- Es erstellt die Zählervariable
- Es erhöht den Zähler jedes Mal um zwei
- Es erhöht den Zähler jedes Mal um drei
