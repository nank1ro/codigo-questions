# --instructions--

Was sind die Hauptunterschiede zwischen einer `List` und einem `Set`?

# --answers--

- Ein `Set` kann keine Duplikate enthalten
- Eine `List` kann keine Duplikate enthalten
- Es gibt keine Unterschiede
- Ein `Set` kann _Venn-Diagramm_-Operationen durchführen
