"__Hallo, Welt!__" ist das traditionelle erste Programm zum Erlernen einer neuen Programmiersprache.
Wir verwenden die `println()`-Funktion, um Daten auf das Standardausgabegerät (Bildschirm) auszugeben.
Um `Hallo, Welt!` auf dem Bildschirm mit Kotlin auszugeben, können wir schreiben
```kotlin
println("Hello, World!")
```

---

`println()` verschiebt den Cursor nach dem Anzeigen seines Parameters in eine neue Zeile, sodass die nachfolgende Ausgabe in der nächsten Zeile angezeigt wird.
Sie können stattdessen `print()` verwenden, das den Cursor in derselben Zeile belässt.
