# --description--

Falls Sie einfach einen Code-Block `n`-mal wiederholen müssen, können Sie die `repeat(times: Int)`-Funktion verwenden.

```kotlin
repeat(3) {
    println("repeat")
}
// prints repeat 3 times
```

Sie können sogar auf den Index zugreifen mit
```kotlin
repeat(3) { index ->
    println(index)
}
// prints 0, 1, 2
```
