# --description--

In Kotlin können wir auch `for-in` für iterierbare Sammlungen verwenden und die gegebene Schließung auf jedem Element aufrufen:
```kotlin
// this is a list, we'll see about that soon
val numbers = listOf(2, 4, 6, 8, 10) 
for (num in numbers) {
    println(num)
}
// prints (2, 4, 6, 8, 10)
```
