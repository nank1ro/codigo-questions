# --description--

Sie konnen auch Kommentarblocke oder _mehrzeilige_ Kommentare erstellen.
_Mehrzeilige_ Kommentare beginnen mit `/*` und enden mit `*/`.
```dart
/*
This is a multi-line
comment
*/
```

# --instructions--

Vervollstandigen Sie den Code, um einen _mehrzeiligen_ Kommentar hinzuzufugen.
# --seed--

```dart
[/]
Comment line 1
Comment line 2
[/]
```
