> Bevor ich beginne, mochte ich Ihnen sagen, dass ich sehr froh bin, Ihnen Dart beizubringen.
Mit dieser Nachricht mochte ich die Gelegenheit nutzen, um Ihnen mitzuteilen, dass die App, die Sie verwenden, direkt in Dart geschrieben wurde, unter Verwendung des Flutter-Frameworks. So ist es mir eine Ehre, mein Fachwissen in dieser Sprache teilen zu konnen.

Dart erlaubt Ihnen wie viele andere Programmiersprachen, Ihren Code zu dokumentieren.
Dies erlaubt es Ihnen, jeden Text neben dem Code zu schreiben.
Kommentare werden vom Compiler ignoriert.

Dart unterstutzt _einzeilige_ Kommentare, _mehrzeilige_ Kommentare und _Dokumentations-_ Kommentare.

So schreiben Sie einen _einzeiligen_ Kommentar:
```dart
// This is a comment. It is not executed.
```
