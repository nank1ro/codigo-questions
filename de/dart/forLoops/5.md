# --description--
Die `break`-Anweisung beendet eine Schleife sofort, wenn eine Bedingung erfüllt ist:

```dart
for (int i = 0; i < 10; i++) {
  if (i == 5) break; // stops at 5
  print(i);
}
```

Das ist nützlich, wenn man nach einem Wert sucht und stoppen möchte, sobald er gefunden wurde.
