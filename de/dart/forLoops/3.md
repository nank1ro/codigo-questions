# --description--
Eine `for`-Schleife kann **rückwärts** zählen, indem man ein Dekrement (`i--`) und eine `>=`-Bedingung verwendet:

```dart
for (int i = 5; i >= 1; i--) {
  print(i); // 5, 4, 3, 2, 1
}
```

Das ist nützlich für Countdowns oder umgekehrte Iteration.
