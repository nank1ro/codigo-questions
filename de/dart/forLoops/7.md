# --instructions--
Was macht die `for-in`-Schleife in Dart?

# --answers--

- Iteriert über jedes Element einer Sammlung
- Iteriert eine feste Anzahl von Malen mit einem Index
- Wiederholt, bis eine boolesche Bedingung wahr wird
- Führt Code nur aus, wenn die Sammlung nicht leer ist
