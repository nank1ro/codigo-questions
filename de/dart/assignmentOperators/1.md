# --description--

Der **Zuweisungsoperator** `=` speichert einen Wert in einer Variable.

```dart
int x = 5;
String name = 'Alice';
```

Nachdem eine Variable zugewiesen wurde, behält sie diesen Wert bis zur Neuzuweisung.

# --instructions--

Vervollständige den Code, um den Wert `42` der Variable `answer` zuzuweisen.

# --seed--

```dart
void main() {
  int answer [/] 42;
  print(answer);
}
```
