# --description--

The **subtraction assignment operator** `-=` subtrahiert einen Wert von einer Variable und speichert das Ergebnis zurück in dieser Variable.

```dart
int lives = 3;
lives -= 1; // lives is now 2
```

Dies ist eine Kurzform für `lives = lives - 1`.

# --instructions--

Vervollständige den Code, um `5` von der Variable `stock` mithilfe von `-=` zu subtrahieren.
