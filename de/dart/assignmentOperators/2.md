# --description--

Der **Additionszuweisungsoperator** `+=` addiert einen Wert zu einer Variable und speichert das Ergebnis zurück in dieser Variable.

```dart
int score = 10;
score += 5; // score is now 15
```

Dies ist eine Kurzform für `score = score + 5`.
