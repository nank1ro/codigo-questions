# --description--

Zuweisungsoperatoren kombinieren eine arithmetische Operation mit einer Zuweisung in einem Schritt.

Zum Beispiel ist `x += 3` eine Kurzform für `x = x + 3`.

# --instructions--

Was macht `x += 5`, wenn `x` derzeit `10` ist?

# --answers--

- Setzt `x` auf `15`
- Setzt `x` auf `5`
- Setzt `x` auf `10`
- Setzt `x` auf `50`

# --solutions--
