# --description--

Du kannst `print()` innerhalb eines `if`-Blocks verwenden, um eine Nachricht anzuzeigen, wenn eine Bedingung erfüllt ist.

```dart
bool isRaining = true;
if (isRaining) {
  print('Take an umbrella');
}
```

Wenn `isRaining` `false` ist, wird nichts ausgegeben.
