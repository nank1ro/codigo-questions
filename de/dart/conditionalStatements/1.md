# --description--

Eine **`if`-Anweisung** führt einen Codeblock nur aus, wenn eine Bedingung `true` ist.

```dart
int age = 20;
if (age >= 18) {
  print('Adult');
}
```

Der Code innerhalb der geschweiften Klammern wird nur ausgeführt, wenn die Bedingung `age >= 18` zu `true` ausgewertet wird.
