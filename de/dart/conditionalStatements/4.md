# --description--

**`else if`** ermöglicht es, mehrere Bedingungen nacheinander zu testen. Der erste Zweig, dessen Bedingung `true` ist, wird ausgeführt, und der Rest wird übersprungen.

```dart
int score = 75;
if (score >= 90) {
  print('A');
} else if (score >= 70) {
  print('B');
} else {
  print('C');
}
// prints: B
```
