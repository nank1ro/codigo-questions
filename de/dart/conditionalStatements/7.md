# --instructions--

Was macht eine `if`-Anweisung in Dart?

# --answers--

- Sie führt den Code in ihrem Block immer aus
- Sie führt ihren Block nur aus, wenn die Bedingung `true` ist
- Sie wiederholt ihren Block, solange die Bedingung `true` ist
- Sie definiert eine neue Funktion
