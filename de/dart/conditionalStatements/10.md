# --instructions--

Was macht die `default`-Klausel in einer `switch`-Anweisung?

# --answers--

- Sie wird ausgeführt, wenn keiner der `case`-Werte übereinstimmt
- Sie wird immer ausgeführt, unabhängig vom Wert
- Sie beendet das Programm
- Sie definiert den ersten zu prüfenden Fall
