# --description--

Eine **`if-else`-Anweisung** führt den `if`-Block aus, wenn die Bedingung `true` ist, und den `else`-Block, wenn sie `false` ist.

```dart
int temperature = 10;
if (temperature > 20) {
  print('Warm');
} else {
  print('Cold');
}
// prints: Cold
```

Genau einer der beiden Zweige wird immer ausgeführt.

Vervollständige die `if-else`-Anweisung, sodass sie `'Positive'` ausgibt, wenn `number` größer als `0` ist, und `'Non-positive'` andernfalls.
