# --instructions--

Gib die korrekte Möglichkeit an, die Variable `number` in eine Ganzzahl zu konvertieren.
```dart
final number = 3.14;
```

# --answers--

```dart
final integer = number.toInt();
```

```dart
final integer = int(number);
```

```dart
final int integer = number;
```

```dart
final integer = number as int;
```
