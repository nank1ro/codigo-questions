# --description--

Der **Ungleichheitsoperator** `!=` gibt `true` zurück, wenn zwei Werte **unterschiedlich** sind, und `false`, wenn sie gleich sind.

```dart
int x = 3;
int y = 7;
bool result = x != y; // true
```

Er ist das Gegenteil von `==`.
