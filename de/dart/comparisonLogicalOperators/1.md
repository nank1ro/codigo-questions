# --description--

Der **Gleichheitsoperator** `==` vergleicht zwei Werte und gibt `true` zurück, wenn sie gleich sind, oder `false`, wenn sie nicht gleich sind.

```dart
int a = 5;
int b = 5;
bool result = a == b; // true
```

Das Ergebnis eines Vergleichs ist immer ein `bool`-Wert.
