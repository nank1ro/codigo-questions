# --instructions--

Der **NICHT-Operator** `!` kehrt einen booleschen Wert um: `!true` wird `false` und `!false` wird `true`.

Deklariere eine globale `bool`-Variable namens `isOffline` mit dem Wert `false`. Deklariere dann eine globale `bool`-Variable namens `isOnline`, die die Negation von `isOffline` unter Verwendung des `!`-Operators speichert.
