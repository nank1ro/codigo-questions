# --instructions--

Was garantiert eine `do-while`-Schleife im Vergleich zu einer `while`-Schleife?

# --answers--

- Der Rumpf wird mindestens einmal ausgeführt, auch wenn die Bedingung anfangs `false` ist
- Der Rumpf wird höchstens einmal ausgeführt
- Die Bedingung wird geprüft, bevor der Rumpf ausgeführt wird
- Die Schleife läuft ewig, sofern kein `break` verwendet wird
