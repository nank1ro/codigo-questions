# --instructions--

Ist es immer notwendig, den Typ einer Variablen in Dart anzugeben?

# --answers--

- Der Typ einer Variablen muss immer angegeben werden
- Dart unterstützt __Typinferenz__, daher kann es ihn selbst ableiten
- Dart unterstützt keine Typen
