# --instructions--

Was ist an diesem Code falsch?
```dart
print("$(10} friends");
```

# --answers--

- Der Ausdruck sollte sich innerhalb von ${} befinden
- Der Wert innerhalb von ${} ist keine Zahl
- Dem String fehlen die Anführungszeichen " "
- Der Wert innerhalb von ${} muss ein Variablenname sein
