# --description--

Eine String-_Interpolation_ ist eine programmatische Möglichkeit, einen String zu generieren.
In Dart können wir das `+`-Zeichen (Verkettung) verwenden, um zwei oder mehr Strings zusammen anzuzeigen, wie:
```dart
print("Hello " + "Dart!");
// prints "Hello Dart!"
```
