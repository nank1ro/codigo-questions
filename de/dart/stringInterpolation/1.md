# --description--

Eine String-_Interpolation_ ist eine programmatische Methode, um einen String zu erzeugen.
In Dart konnen wir das `+`-Zeichen (Verkettung) verwenden, um zwei oder mehr Strings zusammen anzuzeigen, wie zum Beispiel:
```dart
print("Hello " + "Dart!");
// prints "Hello Dart!"
```
