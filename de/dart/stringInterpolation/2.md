# --description--

String-Interpolation erlaubt uns, Ausdrucke wie das Hinzufugen eines Strings zu einer Zahl anzuzeigen, ohne Fehler.
Wenn Sie einen Ausdruck innerhalb von `${}` platzieren, wird er ausgewertet.
Der Ruckgabewert wird in einen String umgewandelt und in den resultierenden String eingefugt

# --seed--

```dart
print([/]);
```

# --answers--

- "${5} friends"

# --solutions--

```dart
print("${5} friends");
```

# --output--

5 friends
