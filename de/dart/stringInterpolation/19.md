# --instructions--

Verwenden Sie `${}`, um die Zahl `500` mit String-Interpolation in den leeren Platz einzufügen 

# --seed--

```dart
void main() {
    print("There are  items");
}
```

# --solutions--

```dart
void main() {
    print("There are ${500} items");
}
```

# --output--

There are 500 items
