# --description--

Dart folgt der Standard-Mathematik **Reihenfolge der Operationen** (Vorrang): Multiplikation und Division werden vor Addition und Subtraktion bewertet.

```dart
int result = 2 + 3 * 4; // 14, not 20
```

Verwende Klammern, um die Reihenfolge zu ändern:

```dart
int result = (2 + 3) * 4; // 20
```

# --instructions--

Deklariere eine Top-Level-`int`-Variable namens `result`, die `3 + 4 * 2 - 1` wertet aus. Lass Dart seinen Standard-Operator-Vorrang anwenden (füge keine Klammern hinzu).
