# --description--

Der **Multiplikationsoperator** `*` multipliziert zwei Werte miteinander.

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

Die Multiplikation einer `int` mit einer anderen `int` ergibt eine `int`. Die Multiplikation mit einer `double` ergibt eine `double`.
