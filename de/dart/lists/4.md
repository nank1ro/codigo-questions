# --description--

Die Eigenschaft `.length` gibt die Anzahl der Elemente in einer Liste zurück:

```dart
var scores = [95, 87, 72, 100];
print(scores.length); // 4
```

Eine leere Liste hat die Länge `0`:

```dart
var empty = [];
print(empty.length); // 0
```
