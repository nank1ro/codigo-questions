# --instructions--

Was ist eine **Klasse** in Dart?

# --answers--

- Eine Funktion, die einen Wert zurückgibt
- Eine Vorlage zum Erstellen von Objekten, die Daten und Verhalten gruppiert
- Ein eingebauter Datentyp wie `int` oder `String`
- Eine Schleife, die einen Codeblock wiederholt
