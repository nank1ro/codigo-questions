# --instructions--

Was bedeutet der Rückgabetyp `void` bei einer Dart-Funktion?

```dart
void sayHi() {
  print("Hi!");
}
```

# --answers--

- Die Funktion gibt die Zahl null zurück
- Die Funktion gibt keinen Wert zurück
- Die Funktion gibt eine leere Zeichenkette zurück
- Die Funktion kann jeden Typ zurückgeben

# --solutions--

- Die Funktion gibt keinen Wert zurück
