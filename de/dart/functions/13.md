# --instructions--

Schreibe eine Funktion `range`, die einen `int`-Parameter `n` nimmt und eine `List<int>` zurückgibt, die die ganzen Zahlen von `1` bis `n` (einschließlich) enthält.

Beispiel:
```dart
print(range(3)); // [1, 2, 3]
print(range(5)); // [1, 2, 3, 4, 5]
```

# --solutions--

```dart
List<int> range(int n) {
  List<int> result = [];
  for (int i = 1; i <= n; i++) {
    result.add(i);
  }
  return result;
}
```
