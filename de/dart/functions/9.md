# --instructions--

Schreibe eine Funktion `greetUser`, die einen `String`-Parameter `name` mit einem **Standardwert** von `"World"` nimmt und die Zeichenkette `"Hello, <name>!"` zurückgibt.

Beispiel:
```dart
print(greetUser());          // Hello, World!
print(greetUser("Alice"));   // Hello, Alice!
```
