# --instructions--

Schreibe eine **rekursive** Funktion `factorial`, die einen `int`-Parameter `n` nimmt und `n!` (die Fakultät von `n`) zurückgibt.

Die Fakultät von `n` ist definiert als:
- `factorial(0) = 1`
- `factorial(n) = n * factorial(n - 1)` für `n > 0`

Beispiel:
```dart
print(factorial(5)); // 120
print(factorial(3)); // 6
```

# --solutions--

```dart
int factorial(int n) {
  if (n == 0) return 1;
  return n * factorial(n - 1);
}
```
