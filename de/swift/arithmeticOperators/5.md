# --description--

Schauen wir uns den **Rest** `%` Operator an.
Er wird verwendet, um den Rest nach einer Division zwischen zwei Zahlen zu finden, wie:
```swift
print(5 % 2) // 1
```
Dies wird zu 1 ausgewertet, weil 5 geteilt durch 2 einen Quotienten von 2 und einen Rest von 1 hat
```swift
print(9 % 3) // 0
```
Dies wird zu 0 ausgewertet, weil 9 geteilt durch 3 einen Quotienten von 3 hat und einen Rest von 0 hinterlässt
