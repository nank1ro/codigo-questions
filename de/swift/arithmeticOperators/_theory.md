Operatoren werden verwendet, um Operationen auf Variablen und Werten auszuführen.
Beginnen wir mit den arithmetischen Operatoren, insbesondere mit dem **Addition** `+` Operator.
Er wird verwendet, um zwei Zahlen zu addieren, wie:
```swift
print(5 + 3) // 8
```

---

Fahren wir mit dem **Subtraction** `-` Operator fort.
Er wird verwendet, um eine Zahl von einer anderen zu subtrahieren, wie:
```swift
print(5 - 3) // 2
```

---

Schauen wir uns den **Multiplikation** `*` Operator an.
Er wird verwendet, um zwei Zahlen miteinander zu multiplizieren, wie:
```swift
print(5 * 3) // 15
```

---

Schauen wir uns den **Division** `/` Operator an.
Er wird verwendet, um zwei Zahlen miteinander zu dividieren, wie:
```swift
print(10 / 5) // 2
```

---

Schauen wir uns den **Rest** `%` Operator an.
Er wird verwendet, um den Rest nach einer Division zwischen zwei Zahlen zu finden, wie:
```swift
print(5 % 2) // 1
```
Dies wird zu 1 ausgewertet, weil 5 geteilt durch 2 einen Quotienten von 2 und einen Rest von 1 hat
```swift
print(9 % 3) // 0
```
Dies wird zu 0 ausgewertet, weil 9 geteilt durch 3 einen Quotienten von 3 hat und einen Rest von 0 hinterlässt
