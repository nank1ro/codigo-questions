# --instructions--

Woher wissen wir, dass diese Schleife nicht stoppt?
```swift
var isLoading = true

while isLoading {
    print("Loading...")
    isLoading = true
}
```

# --answers--

- Weil die Variable im Code-Block nicht auf false aktualisiert wird
- Weil alle while-Schleifen nicht stoppen
- Weil die Variable auf false aktualisiert wird
