# --instructions--

Wird dieser Code-Block jemals ausgeführt?
```swift
var shouldRepeat = false

repeat {
    print("run code")
} while shouldRepeat
```

# --answers--

- Ja, weil die repeat-while-Schleife einmal läuft, bevor die Bedingung der Schleife berücksichtigt wird
- Nein, weil shouldRepeat false ist
- Dieser Code-Block wird zweimal ausgeführt
- Dieser Code-Block wird dreimal ausgeführt
