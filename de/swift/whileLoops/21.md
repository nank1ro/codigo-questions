# --instructions--

Woher wissen wir, dass diese Schleife stoppt?
```swift
var isLoading = true

while isLoading {
    print("Loading...")
    isLoading = false
}
```

# --answers--

- Die Variable in der Schleife wird auf false aktualisiert
- Die Schleife wird nie stoppen
