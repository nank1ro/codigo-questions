# --instructions--

Was macht `counter += 1`?
```swift
var counter = 1

while counter < 5 {
    print(counter)
    counter += 1
}
```

# --answers--

- Es erhöht den Zähler um eins, jedes Mal wenn die Schleife ihren Code-Block ausführt
- Es erstellt die Counter-Variable
- Es erhöht den Zähler jedes Mal um zwei
- Es erhöht den Zähler jedes Mal um drei
