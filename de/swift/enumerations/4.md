# --description--

Bei einigen Enumerationen ist es nützlich, eine Sammlung aller Fälle dieser Enumeration zu haben.
Du aktivierst dies, indem du `: CaseIterable` nach dem Namen der Enumeration schreibst.
Swift stellt eine Sammlung aller Fälle als `allCases` Eigenschaft des Enumerationstyps bereit:
```swift
enum Colors: CaseIterable {
    case blue, red, green
}
for color in Colors.allCases {
    print(color)
}
// prints blue, red, green
```

# --instructions--

Iteriere über die Enumeration, um jeden Fall auszudrucken.
`$0` ist ein Kurzname für ein Argument, das verwendet werden kann, um sich auf die Werte der Argumente des Closure zu beziehen.
`$0` bezieht sich auf das erste Argument des Closure, `$1` auf das zweite Argument usw.
