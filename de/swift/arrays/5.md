# --description--

Ein Array muss keine feste Länge haben.
Sie können jederzeit Elemente am Ende eines Arrays hinzufügen!
Um ein Element zu einem Array hinzuzufügen, verwenden wir die `append`-Funktion:
```swift
var letters = ["a", "b"]
letters.append("c")
print(letters)
// Gibt ["a", "b", "c"] aus
```
