# --description--

Wir können `..<` verwenden, um bis zur nächsten Nummer ausgeschlossen zu schleifen, oder `...` um bis zur nächsten Nummer eingeschlossen zu schleifen

# --instructions--

Vervollständigen Sie die `for`-Schleife im folgenden Code, um den Codeblock `3` mal auszuführen

# --seed--

```swift
for i in 1[/]3 {
    print(i)
}
```
