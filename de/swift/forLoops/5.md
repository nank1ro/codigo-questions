# --description--

Die Funktion `stride()` gibt eine Sequenz von Zahlen zurück.
Sie benötigt die Parameter _from_, _to_ und _by_.
Dies ist die Syntax der Funktion:
```swift
stride(from:to:by:)
```
Beachten Sie, dass der Wert `to` ausgeschlossen ist

# --instructions--

Vervollständigen Sie die Funktion `stride()` im folgenden Code, um die folgenden Zahlen zu drucken `(3, 6, 9, 12)`

# --seed--

for i in stride(from: [/], to: [/], by: [/]) {
    print(i)
}
