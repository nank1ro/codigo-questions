# --description--

Wir wissen, wie man Code mit einer `while`-Schleife wiederholt.
So wird beispielsweise ein Programm, das Anweisungen wiederholt, um `hello` anzuzeigen:
```swift
var counter = 0

while counter < 5 {
    print("hello")
    counter += 1;
}
```
Das können wir aber auch mit `for`-Schleifen machen:
```swift
for i in 0..<5 {
    print("hello")
}
```
