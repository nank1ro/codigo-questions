# --description--

Schauen wir uns den **Divisionszuweisungs**-Operator `/=` an.
Er teilt die Variable durch einen Wert und weist das Ergebnis dieser Variablen zu.
Die folgenden sind genau gleich
```swift
num = num / 5
// is equal to
num /= 5
```
