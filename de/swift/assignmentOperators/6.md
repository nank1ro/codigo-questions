# --description--

Lassen Sie uns den **Restzuweisungs**-Operator `%=` betrachten.
Er berechnet den Rest der Variablen und eines Werts und weist das Ergebnis dieser Variablen zu.
Die folgenden Anweisungen sind also genau gleich
```swift
num = num % 5
// ist gleich zu
num %= 5
```

# --instructions--

Berechnen Sie den Rest von `result` mit dem Wert `3` unter Verwendung des *Restzuweisungs*-Operators, dann geben Sie ihren Wert aus

result [/][/]
print(result)
