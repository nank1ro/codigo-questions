# --description--

Genau wie bei der Additionszuweisung haben wir die **Subtraktionszuweisung** `-=`.
Die Funktionalität ist gleich, der einzige Unterschied ist, dass sie die Subtraktion durchführt.
Die folgenden sind genau gleich
```swift
var num = num - 5
// is equal to
num -= 5
```
