# --description--

Schauen wir uns den **Multiplikationszuweisungs**-Operator `*=` an.
Er multipliziert die Variable mit einem Wert und weist das Ergebnis dieser Variablen zu.
Die folgenden sind genau gleich
```swift
var num = num * 5
// is equal to
num *= 5
```
