# --description--

Variablen werden so genannt, weil sich der Wert, den sie speichern, ändern kann.
Wir können `x` mit Hilfe von `=` aktualisieren und ihm einen neuen Wert geben.
```swift
var x = 1
print(x) // prints 1
x = 2
print(x) // prints 2
```
