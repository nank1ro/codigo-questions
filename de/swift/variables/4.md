# --description--

Wenn wir eine Variable aktualisieren, vergisst sie ihren vorherigen Wert. Hier können wir die Variable `x` zweimal anzeigen und sehen, wie sich ihr Wert aktualisiert.
```swift
var x = 5
print(x) // prints 5
x = 10
print(x) // prints 10
```
