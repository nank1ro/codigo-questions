# --instructions--

Verwende `\()`, um die Zahl `500` in den leeren Platz der String-Interpolation einzufügen

# --seed--

```swift
print("There are  items")
```

# --solutions--

```swift
print("There are \(500) items")
```

# --output--

There are 500 items
