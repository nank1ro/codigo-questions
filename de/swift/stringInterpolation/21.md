# --instructions--

Was stimmt nicht mit diesem Code?
```swift
let manchesterUnited = 2
let barcelona = 2
print("The final score is \(manchesterUnited) : \(barcelona)")
```

# --answers--

- Nichts stimmt nicht mit dem Code
- manchesterUnited und barcelona haben keine Werte
- Wir können keine Variablen innerhalb der geschweifen Klammern verwenden
- Wir können \() nicht zweimal im gleichen String verwenden
