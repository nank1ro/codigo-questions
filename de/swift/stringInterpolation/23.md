# --instructions--

Was stimmt nicht mit diesem Code?
```swift
let pieces = 5
let label = "we have \(pieces) pieces"
```

# --answers--

- Wir können String-Interpolationen nicht in Variablen speichern
- Die Zeichen \() fehlen
- Wir können keine Variablen innerhalb von \() verwenden
- Nichts stimmt nicht mit dem Code
