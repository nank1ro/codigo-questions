# --description--

Danach fügen wir die verschiedene Art von Wert in geschweifte Klammern ein, damit sie als eine Print-Anweisung angezeigt wird. Wie hier, mit `\(5)`

# --instructions--

Fülle den leeren Platz aus, um diese Ausgabe zu erhalten
```swift
5 friends
```

# --seed--

```swift
print(" friends")
```

# --solutions--

```swift
print("\(5) friends")
```

# --output--

5 friends
