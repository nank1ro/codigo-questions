# --description--

Lassen Sie uns mit dem **größer als oder gleich**-Operator `>=` fortfahren.
Er gibt einen **booleschen Wert** (`true` oder `false`) zurück, der angibt, ob ein Ausdruck größer als oder gleich dem anderen ist, z. B.:
```swift
print(2 >= 2) // true
print(3 >= 2) // true
print(3 >= 4) // false
```
