# --description--

Fangen wir mit dem **gleich**-Operator `==` an.
Er gibt einen **booleschen Wert** (`true` oder `false`) zurück, der angibt, ob zwei Ausdrücke gleich sind, z. B.:
```swift
print(2 == 2) // true
print(2 == 3) // false
```
