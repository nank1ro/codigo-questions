# --description--

Lassen Sie uns mit dem **nicht**-Operator `!` abschließen.
Er gibt einen booleschen Wert zurück, der das Gegenteil des logischen Zustands eines Ausdrucks ist.
```swift
print(!true) // false
print(!false) // true
print(!(2 == 2)) // false
```
