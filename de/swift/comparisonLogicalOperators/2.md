# --description--

Lassen Sie uns mit dem **Ungleich** `!=` Vergleichsoperator fortfahren.
Er gibt einen **boolean** (`true` oder `false`) zurück, der angibt, ob zwei Ausdrücke **NICHT** gleich sind, zum Beispiel:
```swift
print(2 != 2) // false
print(2 != 3) // true
```
Er ist genau das Gegenteil des *Gleich*-Operators
