# --description--

Lassen Sie uns mit dem **Kleiner oder gleich** `<=` Vergleichsoperator fortfahren.
Er gibt einen **boolean** (`true` oder `false`) zurück, der angibt, ob ein Ausdruck kleiner oder gleich dem anderen ist, zum Beispiel:
```swift
print(2 <= 2) // true
print(3 <= 2) // false
print(3 <= 4) // true
```
