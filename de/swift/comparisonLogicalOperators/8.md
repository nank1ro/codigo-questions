# --description--

Lassen Sie uns mit dem **oder**-Operator `||` fortfahren.
Er gibt den ersten Operanden zurück, der zu *true* evaluiert, oder den letzten, wenn alle *false* sind.
```swift
print(2 == 2 || 2 == 3) // true
print(1 == 2 || 1 == 3) // false
```
