# --instructions--

Was ist das __Problem__ mit diesem Code-Ausschnitt?
```javascript
console.log(greet("Mathew"));
// This function greets to the person passed in as argument
const greet = (name) => `Hello, ${name}. Good morning!`;
console.log(greet("Alex"));
```

# --answers--

- Wir können greet() nicht verwenden, bevor wir es definieren
- Wir können greet() nicht zweimal verwenden
- greet() gibt nichts zurück
- Wir können nicht `name` als Parameter verwenden
