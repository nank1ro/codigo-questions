# --description--

Schauen wir uns den **Divisionszuweisungs** Operator `/=` an.
Er teilt die Variable durch einen Wert und weist das Ergebnis dieser Variable zu.
Die folgenden sind also genau gleich
```javascript
num = num / 5;
// ist gleichbedeutend mit
num /= 5;
```
