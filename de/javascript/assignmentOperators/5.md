# --description--

Lassen Sie uns den **Divisionszuweisungs**-Operator `/=` betrachten.
Er dividiert die Variable durch einen Wert und weist das Ergebnis dieser Variablen zu.
Die folgenden sind also genau gleich
```javascript
num = num / 5;
// ist gleich to
num /= 5;
```

# --instructions--

Berechnen Sie den Quotienten von `result`, indem Sie seinen Wert durch `5` mit dem *Divisionszuweisungs*-Operator teilen, und geben Sie dann seinen Wert aus
