# --description--

Wir haben bereits eine initialisierte Variable `total`
```javascript
var total = 5;
```
Nehmen wir an, wir möchten die Zahl `2` zur Variable `total` hinzufügen, wir können schreiben
```javascript
total = total + 2;
```
Okay, es funktioniert! Aber es gibt eine kürzere Version, um dasselbe zu tun:
```javascript
total += 2;
```
Das Zeichen `+=` wird **Additionszuweisung** genannt.
Es fügt einen Wert zum Variablenwert hinzu und weist das Ergebnis dieser Variablen zu.
