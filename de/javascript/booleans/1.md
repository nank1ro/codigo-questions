# --description--

JavaScript hat einen grundlegenden Boolean-Typ, genannt `boolean`.
Boolean-Werte werden als logisch bezeichnet, da sie nur true oder false sein können.
Sie können jeden Ausdruck in JavaScript auswerten und erhalten eine von zwei Antworten: `true` oder `false`.
