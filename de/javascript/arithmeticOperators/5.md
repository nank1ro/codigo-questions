# --description--

Schauen wir uns den **Modulo**-Operator `%` an.
Er wird verwendet, um den Rest nach einer Division zwischen zwei Zahlen zu finden, zum Beispiel:
```javascript
console.log(5 % 2);
// prints 1
```
Dies ergibt 1, weil 5 geteilt durch 2 einen Quotienten von 2 und einen Rest von 1 ergibt
```javascript
console.log(9 % 3);
// prints 0
```
Dies ergibt 0, weil 9 geteilt durch 3 einen Quotienten von 3 ergibt und einen Rest von 0 hinterlässt
