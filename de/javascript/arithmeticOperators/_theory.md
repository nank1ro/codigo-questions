Operatoren werden verwendet, um Operationen mit Variablen und Werten durchzuführen.
Beginnen wir mit den arithmetischen Operatoren, insbesondere mit dem **Additions**-Operator `+`.
Er wird verwendet, um zwei Zahlen zu addieren, zum Beispiel:
```javascript
console.log(5 + 3);
// prints 8
```

---

Fahren wir mit dem **Subtraktions**-Operator `-` fort.
Er wird verwendet, um eine Zahl von einer anderen zu subtrahieren, zum Beispiel:
```javascript
console.log(5 - 3);
// prints 2
```

---

Schauen wir uns den **Multiplikations**-Operator `*` an.
Er wird verwendet, um zwei Zahlen miteinander zu multiplizieren, zum Beispiel:
```javascript
console.log(5 * 3);
// prints 15
```

---

Schauen wir uns den **Divisions**-Operator `/` an.
Er wird verwendet, um zwei Zahlen miteinander zu dividieren, zum Beispiel:
```javascript
console.log(10 / 5);
// prints 2
```

---

Schauen wir uns den **Modulo**-Operator `%` an.
Er wird verwendet, um den Rest nach einer Division zwischen zwei Zahlen zu finden, zum Beispiel:
```javascript
console.log(5 % 2);
// prints 1
```
Dies ergibt 1, weil 5 geteilt durch 2 einen Quotienten von 2 und einen Rest von 1 ergibt
```javascript
console.log(9 % 3);
// prints 0
```
Dies ergibt 0, weil 9 geteilt durch 3 einen Quotienten von 3 ergibt und einen Rest von 0 hinterlässt
