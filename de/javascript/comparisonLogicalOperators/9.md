# --description--

Lass uns mit dem **NICHT** `!` logischen Operator fertig werden.
Es gibt einen booleschen Wert zurück, der das Gegenteil des logischen Zustands eines Ausdrucks ist.
```javascript
console.log(!true);
// gibt false aus
console.log(!false);
// gibt true aus
console.log(!(2 == 2));
// gibt false aus
```
