# --description--

Lassen Sie uns mit dem **Größer oder gleich** `>=` Vergleichsoperator fortfahren.
Er gibt einen **boolean** (`true` oder `false`) zurück, der angibt, ob ein Ausdruck größer oder gleich dem anderen ist, zum Beispiel:
```javascript
console.log(2 >= 2);
// gibt true aus
console.log(3 >= 2);
// gibt true aus
console.log(3 >= 4);
// gibt false aus
```
