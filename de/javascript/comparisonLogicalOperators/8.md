# --description--

Lass uns mit dem **ODER** `||` logischen Operator fortfahren.
Es gibt den ersten Operanden zurück, der zu *true* führt, oder den letzten, wenn alle *false* sind.
```javascript
console.log(2 == 2 || 2 == 3);
// gibt true aus
console.log(1 == 2 || 1 == 3);
// gibt false aus
```
