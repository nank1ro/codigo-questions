# --description--

Lassen Sie uns mit dem **Nicht** `!` logischen Operator abschließen.
Er gibt einen boolean zurück, der das Gegenteil des logischen Zustands eines Ausdrucks ist.
```javascript
console.log(!true);
// gibt false aus
console.log(!false);
// gibt true aus
console.log(!(2 == 2));
// gibt false aus
```
