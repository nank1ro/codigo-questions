# --description--

Fangen wir mit dem **gleich** `==` Vergleichsoperator an.
Er gibt einen **booleschen Wert** (`true` oder `false`) zurück, der angibt, ob zwei Ausdrücke gleich sind, zum Beispiel:
```javascript
console.log(2 == 2);
// gibt true aus
console.log(2 == 3);
// gibt false aus
```
