# --description--

Lassen Sie uns mit dem **Gleich** `==` Vergleichsoperator beginnen.
Er gibt einen **boolean** (`true` oder `false`) zuruck, der angibt, ob zwei Ausdrucke gleich sind, zum Beispiel:
```javascript
console.log(2 == 2);
// prints true
console.log(2 == 3);
// prints false
```
