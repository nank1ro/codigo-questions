"__Hello, World!__" ist das traditionelle erste Programm zum Erlernen einer neuen Programmiersprache.
Wir verwenden die `console.log()`-Funktion, um Daten auf das Standardausgabegerät (Bildschirm) auszugeben.
Um `Hello, World!` auf dem Bildschirm mit JavaScript auszudrucken, können wir schreiben
```javascript
console.log("Hello, World!")
```
