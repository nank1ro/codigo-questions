# --description--

_Klassen_ sind vielseitige, flexible Strukturen, die die Bausteine deines Programmcodes bilden.
Eine grundlegende Klasse besteht nur aus dem `class`-Schlüsselwort und ihrem Namen, zum Beispiel:
```javascript
class ClassName {
    // class definition
}
```
