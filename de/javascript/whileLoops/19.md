Woher wissen wir, dass dies eine Endlosschleife ist?
```javascript
var isLogged = false;

while (true) {
    console.log("Please login");
}
```

# --answers--

- Diese Schleife läuft nicht, da die Bedingung falsch ist
- In der Bedingung gibt es keine Variable
- Diese Schleife wird einmal ausgeführt
- Diese Schleife wird zweimal ausgeführt
