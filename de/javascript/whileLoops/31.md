# --instructions--

Was bewirkt `counter += 1`?
```javascript
var counter = 1;

while (counter < 5) {
    console.log(counter);
    counter += 1;
}
```

# --answers--

- Es erhöht den Zähler um eins, jedes Mal wenn die Schleife ihren Code-Block ausführt
- Es erstellt die Counter-Variable
- Es erhöht den Zähler um zwei, jedes Mal
- Es erhöht den Zähler um drei, jedes Mal
