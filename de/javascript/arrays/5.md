# --description--

In JavaScript können wir ein Array aufteilen, wie wir wollen!
```javascript
// Nimmt die ersten beiden Elemente
listName.slice(0, 2);
// Nimmt das vierte bis letzte Element
listName.slice(3);
```
Wenn Ihr Array-Schnitt das allererste oder letzte Element eines Arrays enthält, muss der Index für dieses Element nicht enthalten sein
