# --description--

Ein Array muss keine feste Länge haben.
Sie können jederzeit Elemente zum Ende eines Arrays hinzufügen!
Um ein Element zu einem Array hinzuzufügen, verwenden wir die `push`-Funktion:
```javascript
var letters = ["a", "b"];
letters.push("c");
console.log(letters);
// Prints ["a", "b", "c"]
```
