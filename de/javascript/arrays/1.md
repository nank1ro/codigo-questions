# --description--

Sie können auf ein einzelnes Element eines Arrays über seinen Index zugreifen.
Ein Index ist wie eine Adresse, die die Position des Elements im Array identifiziert.
Der Index erscheint direkt nach dem Array-Namen, zwischen Klammern, wie folgt:
```javascript
arrayName[index];
```
Array-Indizes beginnen mit `0`, **nicht** `1`! Sie greifen auf das erste Element eines Arrays wie folgt zu: `arrayName[0]`.
Das zweite Element in einem Array befindet sich bei Index 1: `arrayName[1]`.
