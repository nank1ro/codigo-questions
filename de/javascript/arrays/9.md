# --description--

Manchmal müssen Sie nach einem Element in einem Array suchen.
In JavaScript können wir die `indexOf()`-Methode verwenden:
```javascript
var names = ["Trevor", "Zac", "Glenn"];
console.log(names.indexOf('Zac'));
// prints 1
```
Der obige Code gibt den ersten Index aus, der den String `"Zac"` enthält, in diesem Fall `1`.
Wir können auch Elemente an einem bestimmten Index in ein Array einfügen, indem wir die `splice()`-Methode verwenden:
```javascript
names.splice(1, 0, "Ali");
// prints ["Trevor", "Ali", "Zac", "Glenn"]
```
Der obige Code fügt `"Ali"` am Index `1` ein, wodurch alles nach diesem Index um 1 nach unten verschoben wird.
Der zweite Wert `0` bedeutet _deleteCount_, in diesem Fall löschen wir kein Element im Array; aber wenn wir `1` angegeben hätten, würde der Wert `Zac` aus dem Array entfernt werden
