In JavaScript können wir das `+` Zeichen verwenden, um zwei oder mehr Strings zusammen anzuzeigen, wie hier:
```javascript
console.log("Hello " + "JavaScript!") // prints "Hello JavaScript!"
```

---

Template Literals ermöglichen es uns, Ausdrücke wie das Hinzufügen eines Strings zu einer Zahl anzuzeigen, ohne das `+` Zeichen zu verwenden.
Um ein gültiges Template Literal zu erstellen, musst du Backticks \`\` statt Doppel- oder einfachen Anführungszeichen verwenden.
Du kannst Variablen innerhalb von Template Literals mit dem Dollarzeichen und geschweiften Klammern einfügen `${expression}`

---

Jede Template Literal-Anweisung besteht aus zwei Teilen, dem `${}` wo wir die Zahl oder Variable einfügen, und dem normalen String

---

Als nächstes fügen wir die verschiedenen Arten von Werten in geschweifte Klammern ein, damit sie als eine console.log-Anweisung angezeigt werden. Wie hier mit `${5}`

---

Das Einfügen von Variablen wie `friends` in den runden Klammern zeigt auch ihren Wert an

---

Wir können `${}` verwenden, um Werte so oft wie wir möchten in Template Literals einzufügen

---

Template Literals werden am besten in Print-Anweisungen verwendet, aber wir können sie auch wie normale Strings in Variablen speichern.
