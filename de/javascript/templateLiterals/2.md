# --description--

Template Literals ermöglichen es uns, Ausdrücke wie das Hinzufügen eines Strings zu einer Zahl anzuzeigen, ohne das `+` Zeichen zu verwenden.
Um ein gültiges Template Literal zu erstellen, musst du Backticks \`\` statt Doppel- oder einfachen Anführungszeichen verwenden.
Du kannst Variablen innerhalb von Template Literals mit dem Dollarzeichen und geschweiften Klammern einfügen `${expression}`

# --seed--

```javascript
console.log([/]);
```

# --answers--

# --solutions--

console.log(`${5} friends`);
