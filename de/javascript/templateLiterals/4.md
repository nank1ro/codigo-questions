# --description--

Als nächstes fügen wir die verschiedenen Arten von Werten in geschweifte Klammern ein, damit sie als eine console.log-Anweisung angezeigt werden. Wie hier mit `${5}`

# --instructions--

Fülle den leeren Platz aus, um diese Ausgabe zu erhalten
```javascript
5 friends
```

# --seed--

```javascript
console.log(` friends`);
```

# --solutions--

```javascript
console.log(`${5} friends`);
```

# --output--

5 friends
