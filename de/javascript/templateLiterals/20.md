# --instructions--

Was ist an diesem Code falsch?
```javascript
var manchesterUnited = 2;
var barcelona = 2;
console.log(`The final score is ${manchesterUnited} : ${barcelona}`);
```

# --answers--

- Nichts ist an diesem Code falsch
- manchesterUnited und barcelona haben keine Werte
- Wir können keine Variablen in den geschweiften Klammern verwenden
- Wir können ${} nicht zweimal in derselben Zeichenkette verwenden
