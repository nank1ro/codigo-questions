# --instructions--

Was ist an diesem Code falsch?
```javascript
var pieces = 5;
var label = `we have ${pieces} pieces`;
```

# --answers--

- Wir können Template Literals nicht in Variablen speichern
- Die Zeichen ${} fehlen
- Wir können keine Variablen in ${} verwenden
- Nichts ist an diesem Code falsch
