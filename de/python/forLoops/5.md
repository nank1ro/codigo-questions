# --description--

Die `range()`-Funktion gibt eine Zahlenfolge zurück, die standardmäßig bei 0 beginnt und um 1 erhöht wird (standardmäßig) und vor einer angegebenen Zahl stoppt.
Dies ist die Syntax der Funktion:
```python
range(start, stop, step)
```
`start` und `step` sind optional, wahrend `stop` erforderlich ist
