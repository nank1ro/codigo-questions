# --description--

Wir wissen, wie man Code mit einer `while`-Schleife wiederholt.
Wie dieses Programm Anweisungen wiederholt, um `hello` anzuzeigen
```python
counter = 0

while counter < 5:
    print("hello")
    counter += 1
```
Aber wir können das Gleiche mit `for`-Schleifen tun:
```python
for i in range(5):
    print("hello")
```
