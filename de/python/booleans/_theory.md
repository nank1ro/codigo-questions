Booleans stellen einen von zwei Werten dar: `True` oder `False`.
Sie können jeden Ausdruck in Python auswerten und erhalten eine von zwei Antworten, `True` oder `False`.

---

Wir können den booleschen Wert `True` in einer Variablen speichern, genau wie eine Zahl oder eine Zeichenkette.

---

Der Gegenteil von `True` ist `False`
