# --description--

Variablen werden auf diese Weise genannt, weil sich der von ihnen gespeicherte Wert ändern kann.
Wir können `x` aktualisieren, indem wir `=` verwenden und ihr einen neuen Wert geben.
```python
>>> x = 1
>>> print(x)
1
>>> x = 2
>>> print(x)
2
```
