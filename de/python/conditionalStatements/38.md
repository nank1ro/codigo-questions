# --instructions--

Wo schreiben wir die Bedingung der if-Anweisung?

# --answers--

- Between round brackets () (Zwischen runden Klammern ())
- Between square brackets [] (Zwischen eckigen Klammern [])
- Between the if keyword and the colon : (Zwischen dem if-Schlüsselwort und dem Doppelpunkt :)
- Before the if keyword (Vor dem if-Schlüsselwort)
