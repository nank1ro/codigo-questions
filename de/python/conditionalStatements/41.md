# --instructions--

Was ist ein Codeblock einer if-Anweisung?

# --answers--

- Code that runs if the condition is False (Code, der ausgeführt wird, wenn die Bedingung False ist)
- Code that determines if a piece of code should run (Code, der bestimmt, ob ein Stück Code ausgeführt werden soll)
- A comparison keyword (Ein Vergleichsschlüsselwort)
