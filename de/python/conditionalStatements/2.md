# --description--

Lassen Sie uns mit dem vorherigen Beispiel fortfahren.
```python
nice_weather = True
if (nice_weather):
    # draußen spielen
```
Wir haben gesehen, dass die `if`-Anweisung den Codeblock nur ausführt, wenn die Bedingung `True` ist.
Ein weiterer wichtiger Aspekt sind die **Doppelpunkte** `:` und die **Einrückung**, die den Beginn eines Codeblocks anzeigen.
Die Einrückung bezieht sich auf die Leerzeichen am Anfang einer Codezeile.
Während in anderen Programmiersprachen die Einrückung nur der Lesbarkeit dient, ist sie in Python unerlässlich.
Sie können Ihre bevorzugte Anzahl von Leerzeichen verwenden (2, 4, 6, 8), wobei 4 bevorzugt ist.
Hier in der App empfehlen wir, die **TAB**-Taste zu verwenden, um Ihre Codezeilen einzurücken
