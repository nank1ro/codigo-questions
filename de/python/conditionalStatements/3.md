Wir haben gerade gesehen, wie man einen Codeblock ausführt, wenn eine Bedingung eintritt. Nun sehen wir, wie man einen anderen Codeblock ausführt, wenn die erste Bedingung fehlschlägt.
Wir gehen draußen spielen, wenn das Wetter schön ist; andernfalls bleiben wir zu Hause.
In Python können wir die `else`-Anweisung verwenden, wie:
```python
nice_weather = True
if (nice_weather):
    # draußen spielen
else:
    # zu Hause bleiben
```
