# --instructions--

Wie wissen wir, dass `Hello World` immer ausgegeben wird?
```python
online = False;
if (online):
    print("User is online")
print("Hello World")
```

# --answers--

- Because of its indentation (Wegen seiner Einrückung)
- Because the if statement can't have more than one line (Weil die if-Anweisung nicht mehr als eine Zeile haben kann)
- Because there isn't an else statement (Weil es keine else-Anweisung gibt)
- Because there isn't an elif statement (Weil es keine elif-Anweisung gibt)
