# --description--

Wir können auch eine bedingte Anweisung (`if`, `elif` oder `else`) in eine andere bedingte Anweisung verschachteln, um eine komplexere Struktur zu erstellen.
```python
num = 4
if (num < 3):
    print("the number is lower than 3")
else:
    if (num == 3):
        print("the number is 3")
    elif (num == 4):
        print("the number is 4")
    else:
        print("the number is greather than 4")
```
und die Ausgabe dieses Codes ist `the number is 4`.
