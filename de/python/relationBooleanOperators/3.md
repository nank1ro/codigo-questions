# --description--

Fahren wir mit dem relationalen Operator **grer als oder gleich** `>=` fort.
Er gibt einen **Boolean** (`True` oder `False`) zurck, der angibt, ob ein Ausdruck grer als oder gleich dem anderen ist, zum Beispiel:
```python
>>> 2 >= 2
True
>>> 3 >= 2
True
>>> 3 >= 4
False
```
