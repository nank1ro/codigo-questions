# --description--

Fahren wir mit dem booleschen Operator **or** fort.
Er gibt den ersten Operanden zurck, der zu *True* ausgewertet wird, oder den letzten, wenn alle *False* sind.
```python
>>> 2 == 2 or 2 == 3
True
>>> 1 == 2 or 1 == 3
False
```
