Lassen Sie uns mit dem **Größer als oder gleich**-Operator `>=` fortfahren.
Er gibt einen **Boolean** (`True` oder `False`) zurück, der angibt, ob ein Ausdruck größer als oder gleich dem anderen ist, zum Beispiel:
```python
>>> 2 >= 2
True
>>> 3 >= 2
True
>>> 3 >= 4
False
```
