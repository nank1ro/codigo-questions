Fahren wir mit dem relationalen Operator **kleiner als oder gleich** `<=` fort.
Er gibt einen **Boolean** (`True` oder `False`) zurck, der angibt, ob ein Ausdruck kleiner als oder gleich dem anderen ist, zum Beispiel:
```python
>>> 2 <= 2
True
>>> 3 <= 2
False
>>> 3 <= 4
True
```
