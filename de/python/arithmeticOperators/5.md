# --description--

Lassen Sie uns den **Modulo**-Operator `%` betrachten.
Er wird verwendet, um den Rest nach einer Division zweier Zahlen zu finden, wie zum Beispiel:
```python
>>> 5 % 2
1
```
Dies ergibt 1, weil 5 geteilt durch 2 einen Quotienten von 2 und einen Rest von 1 hat
```python
>>> 9 % 3
0
```
Dies ergibt 0, weil 9 geteilt durch 3 einen Quotienten von 3 hat und einen Rest von 0 lasst
