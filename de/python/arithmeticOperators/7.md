# --description--

Lassen Sie uns den **Ganzzahl-Divisions**-Operator `//` betrachten.
Dieser Operator gibt den ganzzahligen Teil des Quotienten zuruck, zum Beispiel:
```python
>>> 5.0 / 2
2.5
>>> 5.0 // 2
2.0
```
Auch als Ganzzahldivision bezeichnet. Der resultierende Wert ist eine ganze Zahl, obwohl der *Typ* des Ergebnisses nicht unbedingt int ist.
