# --description--

**Dictionaries** (Worterburcher) ahneln Listen und Tupeln, aber Sie greifen auf Werte zu, indem Sie einen *Schlussel* anstelle eines Index nachschlagen.
Ein Schlussel kann eine beliebige Zeichenfolge oder Zahl sein.
Dictionaries werden in geschweiften Klammern eingeschlossen, wie hier:
```python
d = {"key1": 1, "key2": 2, "key3": 3}
```
Dies ist ein Dictionary namens `d` mit drei *Schlussel-Wert-Paaren*.
Der Schlussel `key1` zeigt auf den Wert `1`, `key2` auf `2` und so weiter.
