# --description--

Da Dictionaries mutable sind, konnen sie auf viele Weisen geandert werden. Elemente konnen mit dem Befehl `del` aus einem Dictionary entfernt werden:
```python
del dict_name[key_name]
```
Dies entfernt den Schlussel `key_name` und seinen zugehorigen Wert aus dem Dictionary.
