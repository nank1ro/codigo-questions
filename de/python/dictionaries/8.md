# --description--

Wie bei Listen konnen wir mit den Schlusselwortern `for..in` durch Dictionary-Elemente schleifen
Um sowohl den Schlussel als auch den Wert in der Schleife zu erhalten, konnen wir die Methode `items()` verwenden:
```python
for key, value in dict_name:
    print(key, value)
```
