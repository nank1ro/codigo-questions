# --description--

Wie Listen sind auch Dictionaries _mutable_ (veranderlich).
Das bedeutet, dass sie nach ihrer Erstellung geandert werden konnen.
Ein Vorteil davon ist, dass wir nach der Erstellung neue _Schlussel-/Wert-Paare_ zum Dictionary hinzufugen konnen, wie hier:
```python
dict_name[new_key_name] = new_value
```
