# --description--

Lassen Sie uns den **Modulozuweisungs**-Operator `%=` betrachten.
Er berechnet den Modulo der Variablen und eines Werts und weist das Ergebnis dieser Variablen zu.
Die folgenden Anweisungen sind also genau gleich
```python
num = num % 5
# ist gleich zu
num %= 5
```
