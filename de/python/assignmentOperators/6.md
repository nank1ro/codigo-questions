# --description--

Sehen wir uns den Operator für die **Potenzzuweisung** `**=` an.
Er erhoeht die Variable auf eine angegebene Potenz und weist das Ergebnis der Variablen zu.
Die folgenden Anweisungen sind also genau gleich:
```python
num = num ** 5
# ist gleich zu
num **= 5
```
