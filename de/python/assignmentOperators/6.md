# --description--

Sehen Sie sich den **Potenzzuweisungs**-Operator `**=` an.
Er erhöht die Variable zu einer angegebenen Potenz und weist das Ergebnis der Variablen zu.
Die folgenden sind also genau gleich
```python
num = num ** 5
# ist gleich zu
num **= 5
```
