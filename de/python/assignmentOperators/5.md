# --description--

Sehen wir uns den Operator für die **Moduluszuweisung** `%=` an.
Er berechnet den Modulus der Variable und eines Wertes und weist das Ergebnis dieser Variablen zu.
Die folgenden Anweisungen sind also genau gleich:
```python
num = num % 5
# ist gleich zu
num %= 5
```
