# --description--

Sehen wir uns den Operator für die **Divisionszuweisung** `/=` an.
Er teilt die Variable durch einen Wert und weist das Ergebnis dieser Variablen zu.
Die folgenden Anweisungen sind also genau gleich:
```python
num = num / 5
# ist gleich zu
num /= 5
```
