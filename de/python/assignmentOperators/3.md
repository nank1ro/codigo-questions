# --description--

Lassen Sie uns den **Divisionszuweisungs**-Operator `/=` betrachten.
Er dividiert die Variable durch einen Wert und weist das Ergebnis dieser Variablen zu.
Die folgenden Anweisungen sind also genau gleich
```python
num = num / 5
# ist gleich zu
num /= 5
```
