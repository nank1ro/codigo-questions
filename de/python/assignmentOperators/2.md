# --description--

Genau wie bei der Additionszuweisung gibt es die **Subtraktionszuweisung** `-=`.
Die Funktionsweise ist die gleiche, der einzige Unterschied ist, dass sie die Subtraktion durchführt.
Die folgenden Anweisungen sind also genau gleich:
```python
num = num - 5
# ist gleich zu
num -= 5
```
