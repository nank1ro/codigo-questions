# --description--

Lassen Sie uns den **Multiplikationszuweisungs**-Operator `*=` betrachten.
Er multipliziert die Variable mit einem Wert und weist das Ergebnis dieser Variablen zu.
Die folgenden Anweisungen sind also genau gleich
```python
num = num * 5
# ist gleich zu
num *= 5
```
