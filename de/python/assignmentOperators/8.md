# --description--

Sehen Sie sich den **Ganzzahldivisions-Zuweisungs**-Operator `//=` an.
Er dividiert die Variable durch einen Wert mit Ganzzahldivision und weist das Ergebnis dieser Variablen zu.
Die folgenden sind also genau gleich
```python
num = num // 5
# ist gleich zu
num //= 5
```
