# --description--

Lassen Sie uns den **Potenzierungszuweisungs**-Operator `**=` betrachten.
Er potenziert die Variable mit einem angegebenen Exponenten und weist das Ergebnis der Variablen zu.
Die folgenden Anweisungen sind also genau gleich
```python
num = num ** 5
# ist gleich zu
num **= 5
```
