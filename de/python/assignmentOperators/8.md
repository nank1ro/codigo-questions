# --description--

Sehen wir uns den Operator für die **Ganzzahl-Divisionszuweisung** `//=` an.
Er fuehrt eine Ganzzahldivision der Variable durch einen Wert durch und weist das Ergebnis dieser Variablen zu.
Die folgenden Anweisungen sind also genau gleich:
```python
num = num // 5
# ist gleich zu
num //= 5
```

Berechnen Sie die Ganzzahldivision von `result` mit dem Wert `3` mit dem Operator für die *Ganzzahl-Divisionszuweisung* und geben Sie dann ihren Wert aus
