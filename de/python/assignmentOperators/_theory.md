Wir haben bereits gelernt, dass wir zum Zuweisen eines Werts zu einer Variablen das Zeichen `=` verwenden können, wie:
```python
a = 5
```

---

Wir haben bereits eine initialisierte Variable `total`
```python
total = 5
```
Angenommen, wir möchten die Zahl `2` zu der Variablen `total` hinzufügen, wir können schreiben
```python
total = total + 2
```
Okay, es funktioniert! Aber es gibt eine kürzere Version, um das Gleiche zu tun:
```python
total += 2
```
Das Zeichen `+=` wird **Additionszuweisung** genannt.
Es addiert einen Wert zum Variablenwert und weist das Ergebnis dieser Variablen zu.

---

Genauso wie bei der Additionszuweisung haben wir die **Subtraktionszuweisung** `-=`.
Die Funktionalität ist dieselbe, der einzige Unterschied ist, dass sie die Subtraktion durchführt.
Die folgenden sind also genau gleich
```python
num = num - 5
# ist gleich zu
num -= 5
```

---

Sehen Sie sich den **Multiplikationszuweisungs**-Operator `*=` an.
Er multipliziert die Variable mit einem Wert und weist das Ergebnis dieser Variablen zu.
Die folgenden sind also genau gleich
```python
num = num * 5
# ist gleich zu
num *= 5
```

---

Sehen Sie sich den **Divisionszuweisungs**-Operator `/=` an.
Er dividiert die Variable durch einen Wert und weist das Ergebnis dieser Variablen zu.
Die folgenden sind also genau gleich
```python
num = num / 5
# ist gleich zu
num /= 5
```

---

Sehen Sie sich den **Potenzzuweisungs**-Operator `**=` an.
Er erhöht die Variable zu einer angegebenen Potenz und weist das Ergebnis der Variablen zu.
Die folgenden sind also genau gleich
```python
num = num ** 5
# ist gleich zu
num **= 5
```

---

Sehen Sie sich den **Modulo-Zuweisungs**-Operator `%=` an.
Er berechnet den Modulo der Variablen und eines Werts und weist das Ergebnis dieser Variablen zu.
Die folgenden sind also genau gleich
```python
num = num % 5
# ist gleich zu
num %= 5
```

---

Sehen Sie sich den **Ganzzahldivisions-Zuweisungs**-Operator `//=` an.
Er dividiert die Variable durch einen Wert mit Ganzzahldivision und weist das Ergebnis dieser Variablen zu.
Die folgenden sind also genau gleich
```python
num = num // 5
# ist gleich zu
num //= 5
```
