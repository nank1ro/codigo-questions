# --description--

Python ist eine objektorientierte Programmiersprache, was bedeutet, dass sie Programmierkonstrukte manipuliert, die Objekte genannt werden.
Sie konnen ein Objekt als eine einzelne Datenstruktur betrachten, die sowohl Daten als auch Funktionen enthalt; die Funktionen eines Objekts werden seine Methoden genannt.
Wenn Sie beispielsweise Folgendes aufrufen:
```python
dict_name.items()
```
uberpruft Python, ob `my_dict` eine `items()`-Methode hat (die alle Worterbucher haben) und fuhrt diese Methode aus, wenn sie sie findet.
