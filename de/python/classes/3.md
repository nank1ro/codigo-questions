# --description--

Lassen Sie uns `pass` durch etwas anderes ersetzen.
Die Methode `__init__()` ist fur Klassen erforderlich und wird verwendet, um die Objekte zu __initialisieren__, die sie erstellt.
`__init__()` nimmt immer mindestens ein Argument, `self`, das sich auf das erstellte Objekt bezieht.
Sie konnen `__init__()` als die Methode betrachten, die jedes Objekt hochfahrt, das die Klasse erstellt.
