# --description--

Oft muss man beim Programmieren einen Codeblock wiederholen, zum Beispiel:
```python
print("2 seconds")
print("3 seconds")
print("4 seconds")
print("5 seconds")
```
Dies erzeugt die folgende Ausgabe:
```python
2 seconds
3 seconds
4 seconds
5 seconds
```
Offensichtlich wuerden wir fuer lange Anweisungen viel Zeit mit dem Schreiben des Codes verbringen, aber zum Glueck koennen wir Schleifen verwenden.
Lernen wir die `while`-Schleife kennen und erhalten Sie die gleiche Ausgabe wie oben.
```python
count = 2
while (count <= 5):
    print(f"{count} seconds")
    count += 1
```
Also haben wir eine Variable `count` erstellt und `2` als Anfangswert zugewiesen.
Dann haben wir die `while`-Anweisung verwendet, die den Codeblock ausfuehrt, solange die Bedingung `count <= 5` `True` (wahr) ist.
Innerhalb des Codeblocks sollten wir **NICHT** vergessen, die Zeile `count += 1` hinzuzufuegen.
Sie erhoet den `count`-Wert, andernfalls wird unsere Schleife unendlich
