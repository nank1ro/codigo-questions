# --instructions--

Wie wissen wir, dass dies eine Endlosschleife ist?
```python
is_logged = False

while True:
    print("Please login")
```

# --answers--

- Diese Schleife wird nicht ausgefuehrt, da die Bedingung False ist
- Es gibt keine Variable in der Bedingung
- Diese Schleife wird einmal ausgefuehrt
- Diese Schleife wird zweimal ausgefuehrt
