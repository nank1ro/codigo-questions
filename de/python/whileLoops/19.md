# --instructions--

Wie wissen wir, dass die `while`-Schleife ihren Codeblock ueberspringt?
```python
is_loading = False

while is_loading:
    print("Loading...")
```

# --answers--

- Die Variable in der Bedingung wird mit True verglichen
- Die Variable in der Bedingung wird mit False verglichen
