# --instructions--

Wie wissen wir, dass diese Schleife stoppt?
```python
is_loading = True

while is_loading:
    print("Loading...")
    is_loading = False
```

# --answers--

- Die Variable in der Schleife wird auf False aktualisiert
- Die Schleife wird nie stoppen
