# --instructions--

Was macht `counter += 1`?
```python
counter = 1

while counter < 5:
    print(counter)
    counter += 1
```

# --answers--

- Es erhöht den Zähler um eins, jedes Mal wenn die Schleife ihren Codeblock ausführt
- Es erstellt die Variable counter
- Es erhöht den Zähler um zwei jedes Mal
- Es erhöht den Zähler um drei jedes Mal
