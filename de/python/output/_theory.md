"__Hello, World!__" ist das traditionelle erste Programm zum Einstieg in das Programmieren in einer neuen Sprache.
Wir verwenden die Funktion `print()`, um Daten auf dem standardmäßigen Ausgabegerät (Bildschirm) auszugeben.
Um `Hello, World!` auf dem Bildschirm mit Python auszugeben, können wir schreiben:
```python
print("Hello, World!")
```
