# --description--

In Python konnen wir auf sehr einfache Weise durch eine Liste iterieren, indem wir die Schlusselworter `for..in` verwenden:
```python
>>> numbers = [1, 2, 3]
>>> for num in numbers:
>>>     print(num)
1
2
3
```
Auf das Schlusselwort `for` folgt ein Variablenname, dem wiederum der Wert jedes Listenelements zugewiesen wird.
