# --description--

Sie konnen uber einen Index auf ein einzelnes Element der Liste zugreifen.
Ein Index ist wie eine Adresse, die die Position des Elements in der Liste identifiziert.
Der Index erscheint direkt nach dem Listennamen, zwischen Klammern, wie folgt:
```python
listenname[index]
```

Listenindizes beginnen mit `0`, **nicht** mit `1`! Sie greifen auf das erste Element in einer Liste wie folgt zu: `listenname[0]`.
Das zweite Element in einer Liste befindet sich am Index 1: `listenname[1]`.
