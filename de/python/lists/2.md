# --description--

Eine Liste muss keine feste Lange haben.
Sie konnen jederzeit Elemente an das Ende einer Liste anfugen!
Um ein Element zu einer Liste hinzuzufugen, verwenden wir das Schlagwort `append`:
```python
>>> letters = ["a", "b"]
>>> letters.append("c")
>>> print(letters)
['a', 'b', 'c']
```
