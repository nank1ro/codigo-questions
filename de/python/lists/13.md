# --description--

Gleichermaßen können wir eine Liste in ein Tupel konvertieren

# --instructions--

Verwenden Sie die Funktion `tuple()`, um die Liste in ein Tupel zu konvertieren

# --seed--

```python
list1 = ["dog", "bear", "cat"]
tuple1 = [/]
print(tuple1)
```

# --answers--

- tuple[list1]
- tuple(list1)

# --solutions--

```python
list1 = ["dog", "bear", "cat"]
tuple1 = tuple(list1)
print(tuple1)
```

# --output--

('dog', 'bear', 'cat')
