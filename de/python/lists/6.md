# --description--

Sie konnen eine Zeichenfolge genau wie eine Liste zerteilen! Tatsachlich konnen Sie sich Zeichenfolgen als Listen von Zeichen vorstellen: Jedes Zeichen ist ein sequentielles Element in der Liste, beginnend mit dem Index `0`.
```python
listenname[:2]
# Greift auf die ersten beiden Elemente zu
listenname[3:]
# Greift auf das vierte bis zum letzten Element zu
```
Wenn Ihr Listenabschnitt das allererste oder letzte Element einer Liste (oder einer Zeichenfolge) enthalt, muss der Index fur dieses Element nicht eingeschlossen werden.

# --instructions--

- Weisen Sie `first_name` einen Abschnitt von `full_name` vom Index `0` bis zum (aber nicht einschlieCenden) Index `4` zu.
- Weisen Sie `last_name` einen Abschnitt von `full_name` vom Index `4` bis zum Ende der Zeichenfolge zu

# --seed--

```python
full_name = "Sam Smith"
first_name = [/][/]
last_name = [/][/]
print(first_name)
print(last_name)
```
