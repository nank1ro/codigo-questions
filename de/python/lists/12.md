# --description--

Es kann vorkommen, dass wir unser Tupel in eine Liste konvertieren möchten.
Um dies zu tun, können wir die Funktion `list()` verwenden

# --instructions--

Verwenden Sie die Funktion `list()`, um das Tupel in eine Liste zu konvertieren

# --seed--

```python
tuple1 = ("dog", "bear", "cat")
list1 = [/]
list1.append("frog")
print(list1)
```

# --answers--

- list[tuple1]
- list(tuple1)

# --solutions--

```python
tuple1 = ("dog", "bear", "cat")
list1 = list(tuple1)
list1.append("frog")
print(list1)
```

# --output--

['dog', 'bear', 'cat', 'frog']
