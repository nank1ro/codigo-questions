# --description--

Es gibt Zeiten, in denen wir unser Tupel in eine Liste umwandeln mochten.
Dafur konnen wir die Funktion `list()` verwenden

# --instructions--

Verwenden Sie die Funktion `list()`, um das Tupel in eine Liste umzuwandeln

# --seed--

```python
tuple1 = ("dog", "bear", "cat")
list1 = [/]
list1.append("frog")
print(list1)
```

# --answers--

- list[tuple1]
- list(tuple1)

# --solutions--

```python
tuple1 = ("dog", "bear", "cat")
list1 = list(tuple1)
list1.append("frog")
print(list1)
```

# --output--

['dog', 'bear', 'cat', 'frog']
