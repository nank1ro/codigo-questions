# --description--

Ein Listenindex verhalt sich wie jeder andere Variablenname! Er kann verwendet werden, um auf Werte zuzugreifen sowie Werte zuzuweisen.
Sie haben gesehen, wie man auf einen Listenindex wie folgt zugreift:
```python
names = ["Jeremiah", "Barney", "Ivan", "Noel"]
names[0] # Ruft den Wert "Jeremiah" ab
```
So funktioniert eine Zuweisung:
```python
names = ["Jeremiah", "Barney", "Ivan", "Noel"]
names[0] = "Jordan"
names[0] # Ruft den neuen Wert "Jordan" ab
```
