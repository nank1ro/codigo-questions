# --description--

Manchmal mochten Sie nur auf einen Teil einer Liste zugreifen.
Betrachten Sie den folgenden Code:
```python
>>> numbers = [1, 2, 3, 4]
>>> slice = numbers[1:3]
>>> print(slice)
[2, 3]
```
Zuerst erstellen wir eine Liste namens `numbers`.
Dann nehmen wir einen Unterabschnitt der Liste und speichern ihn in der Slice-Liste.
Wir tun dies, indem wir die Indizes, die wir einschlieCen mochten, nach dem Namen der Liste definieren: `numbers[1:3]`.
In Python schlieCen wir bei der Angabe eines Teils einer Liste auf diese Weise das Element mit dem ersten Index, `1`, ein, aber wir schlieCen das Element mit dem zweiten Index, `3`, aus.
