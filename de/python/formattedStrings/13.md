# --instructions--

Was ist an diesem Code falsch?
```python
print("{10} friends"f)
```

# --answers--

- Das Zeichen f sollte vor der Zeichenkette stehen
- Der Wert zwischen den geschweiften Klammern ist keine Zahl
- Der Zeichenkette fehlen ihre Anfuhrungszeichen " "
- Der Wert innerhalb der geschweiften Klammern muss ein Variablenname sein
