# --description--

Als Nachstes fugen wir den anderen Werttyp in geschweiften Klammern hinzu, damit er als eine print-Anweisung angezeigt wird. Wie hier, mit `{5}`

# --instructions--

Fullen Sie den leeren Platz aus, um diese Ausgabe zu erhalten
```python
5 friends
```

# --seed--

```python
print(f" friends")
```

# --solutions--

```python
print(f"{5} friends")
```

# --output--

5 friends
