# --instructions--

Pakete sind nichts anderes als Ordner, die eine `__init__.py`-Datei enthalten.
Was könnte passieren, wenn wir vergessen, eine solche Datei hinzuzufügen?

# --answers--

- Python erkennt den Ordner nicht als Paket
- Wir können Pakete immer noch importieren, aber nicht deren Unterpakete
- Wir können Funktionen immer noch importieren
