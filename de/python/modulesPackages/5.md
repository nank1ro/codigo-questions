# --instructions--

Pakete sind nichts anderes als Ordner, die eine Datei `__init__.py` enthalten.
Was konnte passieren, wenn wir vergessen, eine solche Datei hinzuzufugen?

# --answers--

- Python won't recognize the folder as a package
- We can still import packages but not subpackages
- We can still import functions
