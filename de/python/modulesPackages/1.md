# --description--

Ein __Modul__ ist eine Datei, die beliebigen Python-Code enthalten kann.
Module ermöglichen es uns, Code immer wieder zu verwenden.

# --instructions--

Welche dieser Dinge könnten in einem Modul enthalten sein?

# --answers--

- Dateien
- Variablen
- Skripte
- Funktionen
