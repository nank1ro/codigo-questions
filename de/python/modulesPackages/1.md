# --description--

Ein __Modul__ ist eine Datei, die jede Art von Python-Code enthalten kann.
Module ermoglichen es uns, Code immer wieder zu verwenden.

# --instructions--

Welche dieser Dinge konnten in einem Modul enthalten sein?

# --answers--

- Files
- Variables
- Scripts
- Functions
