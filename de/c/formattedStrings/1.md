# --description--

In C verwenden wir die `printf` Funktion mit einem Ausdruck, um zwei oder mehr Werte zusammen anzuzeigen, wie:
```c
>>> char c = 'C';
>>> printf("Hello %c!\n", c);
Hello C!
```
`%c` wird verwendet, um ein einzelnes Zeichen anzuzeigen

Geben Sie den Namen mit dem `%s` Ausdruck aus (wird verwendet, um ein _Array von Zeichen_ oder einen _String_ anzuzeigen)
