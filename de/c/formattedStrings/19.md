# --instructions--

Was ist falsch an diesem Code?
```c
int manchester_united = 0;
int barcelona = 1;
printf("The score is %i : %i\n", manchester_united, barcelona);
```

# --answers--

- Nichts ist falsch mit dem Code
- manchester_united und barcelona haben keine Werte
- Wir können keine Variablen im formatierten String verwenden
- Wir können %i nicht zweimal im gleichen String verwenden
