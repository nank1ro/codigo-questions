# --description--

Mit formatierten Strings können wir Ausdrücke anzeigen, wie z.B. das Hinzufügen eines Strings zu einer Zahl, ohne Fehler.
Um eine ganze Zahl anzuzeigen, verwenden wir den `%i` Ausdruck

# --seed--

```c
printf("[/] friends\n", [/]);
```

# --answers--

- %i
- ?i
- %0
- 5

# --solutions--

```c
printf("%i friends\n", 5);
```

# --output--

5 friends
