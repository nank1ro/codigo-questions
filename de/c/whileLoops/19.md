# --instructions--

Wie wissen wir, dass dies eine unendliche Schleife ist?
```c
bool is_logged = false;

while (true) {
    printf("Please login\n");
}
```

# --answers--

- Diese Schleife wird nicht ausgeführt, da die Bedingung falsch ist
- Es gibt keine Variable in der Bedingung
- Diese Schleife wird einmal ausgeführt
- Diese Schleife wird zweimal ausgeführt
