# --instructions--

Woher wissen wir, dass die `while`-Schleife ihren Codeblock überspringt?
```c
bool is_loading = false;

while (is_loading) {
    printf("Loading...\n");
}
```

# --answers--

- Die Variable in der Bedingung wird mit true verglichen
- Die Variable in der Bedingung wird mit false verglichen
