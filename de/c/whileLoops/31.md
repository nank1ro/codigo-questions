# --instructions--

Was macht `counter += 1`?
```c
int counter = 1;

while (counter < 5) {
    printf("%d\n", counter);
    counter += 1;
}
```

# --answers--

- Es erhöht counter jedes Mal um eins, wenn die Schleife ihren Code-Block ausführt
- Es erstellt die counter-Variable
- Es erhöht counter jedes Mal um zwei
- Es erhöht counter jedes Mal um drei
