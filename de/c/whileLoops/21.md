# --instructions--

Woher wissen wir, dass diese Schleife stoppt?
```c
bool is_loading = true;

while (is_loading) {
    printf("Loading...\n");
    is_loading = false;
}
```

# --answers--

- Die Variable in der Schleife wird auf false aktualisiert
- Die Schleife wird niemals stoppen
