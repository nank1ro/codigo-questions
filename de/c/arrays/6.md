# --description--

Sie können die Länge in Bytes eines Arrays ermitteln, indem Sie das `sizeof` des Arrays ermitteln und dann durch die Größe eines Elements teilen.
Es funktioniert, weil jedes Element im Array denselben Typ und damit dieselbe Größe hat.
Die resultierende *Länge* ist die Anzahl der enthaltenen Elemente

# --instructions--

Vervollständigen Sie den Code, um die Länge des Arrays `numbers` auszugeben

int numbers[4] = {5, 6, 8, 4};

printf("%d\n", length);
