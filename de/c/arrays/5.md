# --description--

Ein Listenindex verhält sich wie jeder andere Variablenname! Er kann verwendet werden, um auf Werte zuzugreifen sowie sie zuzuweisen.
Sie haben gesehen, wie man auf einen Listenindex wie folgt zugreift:
```c
int numbers[4] = {5, 6, 7, 8};
numbers[0]; // Ruft den Wert 5 ab
```
So funktioniert eine Zuweisung:
```c
int numbers[4] = {5, 6, 7, 8};
numbers[0] = 1;
printf("%d\n", numbers[0]); // gibt den neuen Wert 1 aus
```
