# --description--

Sie können auf ein einzelnes Element des Arrays über seinen Index zugreifen.
Ein Index ist wie eine Adresse, die die Position des Elements im Array identifiziert.
Der Index erscheint direkt nach dem Array-Namen, zwischen Klammern, wie hier:
```
list_name[index];
```

Array-Indizes beginnen mit `0`, **nicht** `1`! Sie greifen auf das erste Element in einem Array wie folgt zu: `list_name[0]`.
Das zweite Element in einem Array hat den Index 1: `list_name[1]`.
