# --description--

Wir wissen, wie man Code mit einer `while`-Schleife wiederholt.
Wie dieses Programm, das Anweisungen wiederholt, um `hello` anzuzeigen
```c
int counter = 0;

while (counter < 5) {
    printf("Hello\n");
    counter++;
}
```
Aber wir können dasselbe mit `for`-Schleifen tun:
```c
for (int i = 0; i < 5; i++) {
    printf("Hello\n");
}
```
