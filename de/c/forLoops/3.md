# --description--

Wir können `<` verwenden, um bis zur nächsten Zahl ausgeschlossen zu schleifen, oder `<=` um bis zur nächsten Zahl eingeschlossen zu schleifen

# --instructions--

Vervollständigen Sie die `for`-Schleife im folgenden Code, um den Codeblock `3`-mal auszuführen

# --seed--

```c
for (int i = 1; i[/]3; i++) {
        printf("%d\n", i);
}
```
