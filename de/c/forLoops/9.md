# --description--

Wir können `<` verwenden, um bis zur nächsten ausgeschlossenen Zahl zu schleifen, oder `<=`, um bis zur nächsten einbezogenen Zahl zu schleifen

# --instructions--

Vervollständigen Sie die `for`-Schleife im folgenden Code, um den Codeblock `3` Mal auszuführen

# --seed--

        printf("%d\n", i);
}
