# --instructions--

Woher wissen wir, dass `Hello World` immer gedruckt wird?
```c
bool online = false;
if (online) {
    printf("User is online\n");
}
printf("Hello World\n");
```

# --answers--

- Weil es nicht innerhalb der bedingten Anweisung liegt
- Weil die if-Anweisung nicht mehr als eine Zeile haben kann
- Weil es keine else-Anweisung gibt
- Weil es keine else if-Anweisung gibt
