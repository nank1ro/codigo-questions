# --description--

Lassen Sie uns die **Ganzzahldivision** mit der `floor()` Funktion betrachten.
Diese Funktion gibt den ganzzahligen Teil des Quotienten zurück, zum Beispiel:
```
>>> 5.0 / 2
2.5
>>> floor(2.5)
2.0
```
Auch als Ganzzahldivision bezeichnet. Der resultierende Wert ist eine ganze Zahl, obwohl der *Typ* des Ergebnisses nicht unbedingt int ist.
