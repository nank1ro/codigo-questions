# --description--

C hat keinen **Potenz**-Operator, daher müssen wir die `pow()`-Funktion verwenden, die in der `math.h`-Bibliothek enthalten ist.
Die Potenzierung entspricht der wiederholten Multiplikation der Basis: Das heißt, **b** mit dem Exponenten *n* ist das Produkt aus der Multiplikation von *n* Basen:
![exponentiation](https://bit.ly/3zcz6Lt)
```
>>> pow(5, 2);
25
```
