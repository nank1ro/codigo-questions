# --description--

C hat keinen **Exponentiation** Operator, also müssen wir die `pow()` Funktion aus der `math.h` Bibliothek verwenden.
Exponentiation entspricht der wiederholten Multiplikation der Basis: das heißt, **b** mit Exponent *n* ist das Produkt der Multiplikation von *n* Basen:
![exponentiation](https://bit.ly/3zcz6Lt)
```
>>> pow(5, 2);
25
```
