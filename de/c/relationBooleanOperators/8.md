# --description--

Lassen Sie uns mit dem **Nicht** `!` Boolean-Operator abschließen.
Er gibt einen Boolean zurück, der das Gegenteil des logischen Zustands eines Ausdrucks ist.
```c
>>> !true
0
>>> !false
1
>>> !(2 == 2)
0
```
