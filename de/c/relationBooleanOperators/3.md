# --description--

Lassen Sie uns mit dem **Ungleich** `!=` Relationaloperator fortfahren.
Er gibt einen **Boolean** zurück, true `1` oder false `0`, der angibt, ob zwei Ausdrücke **NICHT** gleich sind, zum Beispiel:
```c
>>> 2 != 2
0
>>> 2 != 3
1
```
Er ist genau das Gegenteil des *Gleich*-Operators
