# --description--

Fahren wir mit dem **Kleiner-als-oder-gleich**-Operator `<=` fort.
Er gibt einen **booleschen** Wert zurück, wahr `1` oder falsch `0`, der angibt, ob ein Ausdruck kleiner als oder gleich dem anderen ist. Zum Beispiel:
```c
>>> 2 <= 2
1
>>> 3 <= 2
0
>>> 3 <= 4
1
```
