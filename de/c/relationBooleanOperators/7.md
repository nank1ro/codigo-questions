# --description--

Jetzt schauen wir uns die **booleschen** Operatoren an. Beginnen wir mit dem ersten, dem **und**-Operator `&&`.
Er gibt den ersten Operanden zurück, der zu *falsch* evaluiert wird, oder den letzten, wenn alle *wahr* sind.
```c
>>> 2 == 2 && 2 == 3
0
>>> 1 == 1 && 1 == 1.0
1
```
