# --description--

Beginnen wir mit dem **Gleichheits**-Operator `==`.
Er gibt einen **booleschen** Wert zurück, wahr `1` oder falsch `0`, der angibt, ob zwei Ausdrücke gleich sind. Zum Beispiel:
```c
>>> 2 == 2
1
>>> 2 == 3
0
```
