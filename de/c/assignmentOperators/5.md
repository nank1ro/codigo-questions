# --description--

Schauen wir uns den **Divisionszuweisungs**-Operator `/=` an.
Er dividiert die Variable durch einen Wert und weist das Ergebnis der Variablen zu.
Die folgenden sind genau gleich
```c
num = num / 5;
// ist gleich wie
num /= 5;
```
