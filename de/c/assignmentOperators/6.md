# --description--

Schauen wir uns den **Modulo-Zuweisungs**-Operator `%=` an.
Er berechnet den Modulo aus der Variablen und einem Wert und weist das Ergebnis der Variablen zu.
Die folgenden sind genau gleich
```c
num = num % 5;
// ist gleich wie
num %= 5;
```
