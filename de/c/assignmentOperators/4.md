# --description--

Schauen wir uns den **Multiplikationszuweisungs**-Operator `*=` an.
Er multipliziert die Variable mit einem Wert und weist das Ergebnis der Variablen zu.
Die folgenden sind genau gleich
```c
num = num * 5;
// ist gleich wie
num *= 5;
```
