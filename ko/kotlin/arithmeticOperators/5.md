# --description--

이번에는 **나머지** `%` 연산자를 살펴보겠습니다.
이 연산자는 두 숫자의 나눗셈 후 나머지를 구하는 데 사용됩니다. 예를 들면:
```kotlin
println(5 % 2) // 1
```
5를 2로 나누면 몫이 2이고 나머지가 1이므로 결과는 1입니다
```kotlin
println(9 % 3) // 0
```
9를 3으로 나누면 몫이 3이고 나머지가 0이므로 결과는 0입니다
