# --description--

조건이 발생할 때 코드 블록을 실행하는 방법을 살펴보았습니다. 이제 첫 번째 조건이 실패할 경우 다른 코드 블록을 실행하는 방법을 알아봅시다.
날씨가 좋으면 밖에서 놀고, 그렇지 않으면 집에 있습니다.
Kotlin에서는 `else` 문을 사용할 수 있습니다:
```kotlin
var niceWeather = true
if (niceWeather) {
    // play outside
} else {
    // stay home
}
```
