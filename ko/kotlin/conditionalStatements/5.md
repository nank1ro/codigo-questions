# --description--

`else if` 문은 원하는 만큼 추가할 수 있으며, 제한이 없습니다
```kotlin
var num = 4
if (num == 2) {
    println("the number is 2")
} else if (num == 3) {
    println("the number is 3")
} else if (num == 4) {
    println("the number is 4")
} else if (num == 5) {
    println("the number is 5")
} else if (num == 6) {
    println("the number is 6")
}
```
이 코드의 출력은 `the number is 4`입니다.
