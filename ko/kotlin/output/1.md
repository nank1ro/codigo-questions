# --description--

"__Hello, World!__"는 새로운 프로그래밍 언어를 배울 때 전통적으로 작성하는 첫 번째 프로그램입니다.

Kotlin으로 화면에 `Hello, World!`를 출력하려면 다음과 같이 작성할 수 있습니다.
```kotlin
println("Hello, World!")
```
