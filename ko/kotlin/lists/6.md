# --description--

또 다른 유용한 리스트 연산은 주어진 요소가 리스트에 있는지 확인하는 `contains` 메서드입니다.
예를 들어, 이름 리스트가 있다면 `contains` 메서드를 사용하여 특정 이름이 리스트에 있는지 확인할 수 있습니다.
```kotlin
val names = listOf("Thomas", "Donald", "Scarlett")
println(names.contains("Scarlett"))
// prints true
```
