# --description--

이전 예제에서 보았듯이, `add` 함수를 사용하여 항목을 하나씩 추가할 수 있습니다.
하지만 다른 리스트의 모든 요소를 한 번에 추가해야 한다면 `addAll` 함수 또는 `+=` 단축키를 사용하면 됩니다:
```kotlin
val letters = mutableListOf("a", "b")
val newLetters = listOf("c", "d", "e")
letters.addAll(newLetters)
println(letters)
// prints [a, b, c, d, e]
```
