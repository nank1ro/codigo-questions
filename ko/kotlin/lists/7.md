# --description--

가변 리스트는 고정된 길이를 가질 필요가 없습니다.
언제든지 리스트의 끝에 항목을 추가할 수 있습니다!
가변 리스트에 항목을 추가하려면 `add` 함수 또는 `+=` 단축키를 사용합니다:
```kotlin
val letters = mutableListOf("a", "b")
letters.add("c")
println(letters)
// prints [a, b, c]
```
