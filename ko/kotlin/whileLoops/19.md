# --instructions--

이 루프가 멈추는 것을 어떻게 알 수 있습니까?
```kotlin
var isLoading = true

while (isLoading) {
    println("Loading...")
    isLoading = false
}
```

# --answers--

- 루프 내부에서 변수가 false로 업데이트됩니다
- 이 루프는 절대 멈추지 않습니다
