# --instructions--

이 루프가 멈추지 않는 것을 어떻게 알 수 있습니까?
```kotlin
var isLoading = true

while (isLoading) {
    println("Loading...")
    isLoading = true
}
```

# --answers--

- 코드 블록 내부에서 변수가 false로 업데이트되지 않기 때문입니다
- 모든 while 루프는 멈추지 않기 때문입니다
- 변수가 false로 업데이트되기 때문입니다
