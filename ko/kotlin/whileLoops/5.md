# --description--

다음은 _while_ 루프 내에서 `break`를 사용하는 예시입니다:
```kotlin
var i = 0
while (i < 3) {
  i++
  if (i == 2) break // [1]
  println(i)
}
// prints 1
```

보시다시피 __[1]__에서 `i`가 _2_와 같을 때, 루프를 _break_합니다. 실제로 숫자 2와 3은 출력되지 않습니다.
