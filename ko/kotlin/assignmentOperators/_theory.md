대입 연산자는 변수에 값을 대입하는 데 사용됩니다.
가장 기본적인 대입 연산자는 `=`로, 오른쪽의 값을 왼쪽의 변수에 대입합니다:
```kotlin
var x = 10
```
여기서 `10`이 변수 `x`에 대입됩니다.
기존 변수에 새 값을 재대입할 수도 있습니다:
```kotlin
x = 20
```
이제 `x`는 값 `20`을 가집니다.

---

`+=` 연산자는 변수에 값을 더하고 결과를 다시 변수에 대입하는 축약 표현입니다.
다음처럼 쓰는 대신:
```kotlin
x = x + 5
```
이렇게 쓸 수 있습니다:
```kotlin
x += 5
```
두 표현식 모두 같은 일을 합니다: `x`의 값을 `5`만큼 증가시킵니다.

---

`-=` 연산자는 변수에서 값을 빼고 결과를 다시 변수에 대입합니다.
다음처럼 쓰는 대신:
```kotlin
x = x - 3
```
이렇게 쓸 수 있습니다:
```kotlin
x -= 3
```
이렇게 하면 `x`의 값이 `3`만큼 감소합니다.

---

`*=` 연산자는 변수에 값을 곱하고 결과를 다시 변수에 대입합니다.
다음처럼 쓰는 대신:
```kotlin
x = x * 4
```
이렇게 쓸 수 있습니다:
```kotlin
x *= 4
```
이렇게 하면 `x`를 `4`로 곱하고 결과를 다시 `x`에 저장합니다.

---

`/=` 연산자는 변수를 값으로 나누고 결과를 다시 변수에 대입합니다.
다음처럼 쓰는 대신:
```kotlin
x = x / 2
```
이렇게 쓸 수 있습니다:
```kotlin
x /= 2
```
참고: 두 피연산자가 모두 `Int`인 경우 Kotlin은 **정수 나눗셈**을 수행합니다(소수 부분은 버려집니다):
```kotlin
var x = 7
x /= 2
// x는 이제 3입니다 (3.5가 아님)
```
