# --description--

범위를 역순으로 순회할 수 있습니다.

`3..1`이 작동할 것이라고 예상하겠지만, 안타깝게도 Kotlin 팀은 이 기능을 다른 방식으로 가져오기로 결정했습니다.

실제로 다음 코드를 실행하면:
```kotlin
for (i in 3..1) println(i)
```

아무것도 출력되지 않는 것을 확인할 수 있습니다.
이것이 작동하도록 하려면 `downTo` 키워드를 사용해야 합니다:

```kotlin
for (i in 3 downTo 1) println(i)
// prints 3, 2, 1
```

`downTo`는 감소하는 범위를 생성합니다.
