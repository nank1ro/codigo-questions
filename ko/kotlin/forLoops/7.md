# --description--

단순히 코드 블록을 `n`번 반복해야 하는 경우, `repeat(times: Int)` 함수를 사용할 수 있습니다.

```kotlin
repeat(3) {
    println("repeat")
}
// prints repeat 3 times
```

인덱스에도 접근할 수 있습니다
```kotlin
repeat(3) { index ->
    println(index)
}
// prints 0, 1, 2
```
