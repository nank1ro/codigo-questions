# --description--

범위의 기본 _스텝_은 __1__이지만, 다른 값을 명시적으로 설정할 수 있습니다.

`step` 키워드를 사용하여 `for` 루프의 __스텝__을 정의할 수 있습니다.

```kotlin
for (i in 1..10 step 2) {
    println(i)
}
// prints 1, 3, 5, 7, 9
```

보시다시피, 코드 블록이 _1_ 대신 _2_의 스텝으로 실행되어 출력이 완전히 달라집니다.
