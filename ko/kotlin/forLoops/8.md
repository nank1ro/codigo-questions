# --description--

Kotlin에서는 `for-in`을 사용하여 반복 가능한 컬렉션의 각 요소에 주어진 클로저를 호출할 수도 있습니다:
```kotlin
// this is a list, we'll see about that soon
val numbers = listOf(2, 4, 6, 8, 10)
for (num in numbers) {
    println(num)
}
// prints (2, 4, 6, 8, 10)
```
