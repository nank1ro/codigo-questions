# --description--

이제 **논리** 연산자를 살펴보겠습니다. 먼저 __AND__ `&&`부터 시작합니다.
이 연산자는 *false*로 평가되는 첫 번째 피연산자를 반환하거나, 모두 *true*이면 마지막 피연산자를 반환합니다.
```kotlin
println(2 == 2 && 2 == 3) // false
println(1 == 1 && 1.0 == 1.0) // true
```
