# --description--

**버림 나눗셈 대입** `//=` 연산자를 알아봅시다.
변수를 값으로 버림 나눗셈한 후 그 결과를 해당 변수에 할당합니다.
따라서 다음 두 가지는 완전히 동일합니다
```python
num = num // 5
# is equal to
num //= 5
```
