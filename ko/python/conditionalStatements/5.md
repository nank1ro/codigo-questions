# --description--

`elif` 문은 원하는 만큼 추가할 수 있으며, 제한이 없습니다
```python
num = 4
if (num == 2):
    print("the number is 2")
elif (num == 3):
    print("the number is 3")
elif (num == 4):
    print("the number is 4")
elif (num == 5):
    print("the number is 5")
elif (num == 6):
    print("the number is 6")
```
이 코드의 출력은 `the number is 4`입니다.
