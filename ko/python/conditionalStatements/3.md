# --description--

날씨가 좋으면 밖에서 놀고, 그렇지 않으면 집에 있습니다.
Python에서는 `else` 문을 사용할 수 있습니다:
```python
nice_weather = True
if (nice_weather):
    # play outside
else:
    # stay home
```
