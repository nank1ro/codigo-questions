# --description--

딕셔너리의 모든 키를 나열하고 싶다면 어떻게 해야 할까요?
`keys()` 메서드를 사용하면 됩니다.

# --seed--

```python
user = {"first_name": "John", "last_name": "Hood", "age": 30}
print([/][/])
```

# --answers--

- user
- .keys()
- .keys
- .key
- .allKeys()

# --solutions--

```python
user = {"first_name": "John", "last_name": "Hood", "age": 30}
print(user.keys())
```

# --output--

dict_keys(['first_name', 'last_name', 'age'])
