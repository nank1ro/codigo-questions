# --description--

**딕셔너리**는 리스트와 튜플과 비슷하지만, 인덱스 대신 *키*를 사용하여 값에 접근합니다.
키는 어떤 문자열이나 숫자도 될 수 있습니다.
딕셔너리는 중괄호로 감싸며, 다음과 같이 작성합니다:
```python
d = {"key1": 1, "key2": 2, "key3": 3}
```
이것은 세 개의 *키-값 쌍*을 가진 `d`라는 딕셔너리입니다.
키 `key1`은 값 `1`을 가리키고, `key2`는 `2`를 가리키는 식입니다.
