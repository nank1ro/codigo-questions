# --description--

딕셔너리의 모든 값을 나열하고 싶다면 어떻게 해야 할까요?
`values()` 메서드를 사용하면 됩니다.

# --seed--

```python
user = {"first_name": "John", "last_name": "Hood", "age": 30}
print([/][/])
```

# --answers--

- user
- .keys()
- .values()
- .values
- .allValues()

# --solutions--

```python
user = {"first_name": "John", "last_name": "Hood", "age": 30}
print(user.values())
```

# --output--

dict_values(['John', 'Hood', 30])
