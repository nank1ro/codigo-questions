# --description--

딕셔너리는 변경 가능하기 때문에 여러 가지 방법으로 수정할 수 있습니다. `del` 명령으로 딕셔너리에서 항목을 제거할 수 있습니다:
```python
del dict_name[key_name]
```
이 명령은 딕셔너리에서 키 `key_name`과 그에 연결된 값을 제거합니다.
