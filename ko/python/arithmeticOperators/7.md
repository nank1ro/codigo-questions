# --description--

**정수 나눗셈** `//` 연산자를 살펴보겠습니다.
이 연산자는 몫의 정수 부분을 반환합니다. 예를 들면:
```python
>>> 5.0 / 2
2.5
>>> 5.0 // 2
2.0
```
정수 나눗셈이라고도 합니다. 결과값은 정수이지만, 결과의 *타입*이 반드시 int인 것은 아닙니다.
