# --instructions--

이 반복문이 멈추지 않는 것을 어떻게 알 수 있습니까?
```python
is_loading = True

while is_loading:
    print("Loading...")
    is_loading = True
```

# --answers--

- 코드 블록 내에서 변수가 False로 업데이트되지 않기 때문입니다
- 모든 while 반복문은 멈추지 않기 때문입니다
- 변수가 False로 업데이트되기 때문입니다
