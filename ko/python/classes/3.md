# --description--

`pass`를 다른 것으로 바꿔 봅시다.
`__init__()` 메서드는 클래스에 필수이며, 클래스가 생성하는 객체를 __초기화__하는 데 사용됩니다.
`__init__()`는 항상 최소한 하나의 인수 `self`를 받으며, 이는 생성되는 객체를 참조합니다.
`__init__()`는 클래스가 생성하는 각 객체를 부팅하는 메서드라고 생각할 수 있습니다.
