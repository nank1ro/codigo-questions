# --description--

**같지 않음** `!=` 관계 연산자를 알아봅시다.
이 연산자는 두 표현식이 같지 **않은지** 여부를 나타내는 **불리언** (`True` 또는 `False`) 값을 반환합니다. 예를 들어:
```python
>>> 2 != 2
False
>>> 2 != 3
True
```
이것은 *같음* 연산자의 정반대입니다
