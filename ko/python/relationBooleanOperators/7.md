# --description--

이 연산자는 *False*로 평가되는 첫 번째 피연산자를 반환하거나, 모두 *True*이면 마지막 피연산자를 반환합니다.
```python
>>> 2 == 2 and 2 == 3
False
>>> 1 == 1 and 1 == 1.0
True
```
