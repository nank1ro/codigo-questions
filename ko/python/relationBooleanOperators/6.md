# --description--

**작거나 같음** `<=` 관계 연산자를 알아봅시다.
이 연산자는 하나의 표현식이 다른 표현식보다 작거나 같은지 여부를 나타내는 **불리언** (`True` 또는 `False`) 값을 반환합니다. 예를 들어:
```python
>>> 2 <= 2
True
>>> 3 <= 2
False
>>> 3 <= 4
True
```
