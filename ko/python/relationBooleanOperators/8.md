# --description--

**or** 불리언 연산자를 알아봅시다.
이 연산자는 *True*로 평가되는 첫 번째 피연산자를 반환하거나, 모두 *False*이면 마지막 피연산자를 반환합니다.
```python
>>> 2 == 2 or 2 == 3
True
>>> 1 == 2 or 1 == 3
False
```
