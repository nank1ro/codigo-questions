변수는 데이터 값을 저장하는 컨테이너입니다.
Python의 모든 변수는 객체이며, 다른 프로그래밍 언어와 달리 Python에는 변수를 선언하는 명령어가 없습니다.
변수를 만들려면 공백이 포함되지 않도록 **이름**을 지정해야 합니다.
변수는 처음으로 값을 할당하는 순간 생성됩니다.
`x`라는 이름의 변수를 생성하는 예시는 다음과 같습니다:
```python
x = 1
```
이렇게 하면 `x`라는 이름의 변수에 값 `1`을 할당한 것입니다.
변수 `x`를 출력하면 숫자 `1`이 반환됩니다:
```python
>>> print(x)
1
```

---

변수는 저장된 값을 변경할 수 있기 때문에 이렇게 불립니다.
`=`를 사용하여 새로운 값을 지정함으로써 `x`를 업데이트할 수 있습니다.
```python
>>> x = 1
>>> print(x)
1
>>> x = 2
>>> print(x)
2
```

---

다른 변수의 값을 변수에 할당할 수도 있습니다. 여기서 `y` 변수에 `x`의 값을 할당할 수 있습니다
```python
>>> x = 5
>>> y = x
>>> print(y)
5
```

---

변수를 업데이트하면 이전 값을 잊어버립니다. 여기서 `x` 변수를 두 번 출력하여 값이 어떻게 업데이트되는지 확인할 수 있습니다.
```python
>>> x = 5
>>> print(x)
5
>>> x = 10
>>> print(x)
10
```

---

문자열 변수는 작은따옴표 또는 큰따옴표를 사용하여 선언할 수 있습니다:
```python
>>> x = "May"
>>> x = 'May'
```
둘 다 같은 것입니다.

---

여러 단어로 된 변수 이름을 사용하려면 **스네이크 케이스**를 사용합니다.
이는 `_`를 사용하여 추가 단어를 연결하는 것을 의미합니다.
