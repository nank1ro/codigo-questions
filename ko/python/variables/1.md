# --description--

변수는 데이터 값을 저장하는 컨테이너입니다.
Python의 모든 변수는 객체이며, 다른 프로그래밍 언어와 달리 Python에는 변수를 선언하는 명령어가 없습니다.
변수를 만들려면 공백이 포함되지 않도록 **이름**을 지정해야 합니다.
변수는 처음으로 값을 할당하는 순간 생성됩니다.
`x`라는 이름의 변수를 생성하는 예시는 다음과 같습니다:
```python
x = 1
```
이렇게 하면 `x`라는 이름의 변수에 값 `1`을 할당한 것입니다.
변수 `x`를 출력하면 숫자 `1`이 반환됩니다:
```python
>>> print(x)
1
```
