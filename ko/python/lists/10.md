# --description--

Python에서는 `for..in` 키워드를 사용하여 매우 간단하게 리스트를 반복할 수 있습니다:
```python
>>> numbers = [1, 2, 3]
>>> for num in numbers:
>>>     print(num)
1
2
3
```
`for` 키워드 뒤에 변수 이름이 오며, 이 변수에는 리스트의 각 항목 값이 차례로 할당됩니다.
