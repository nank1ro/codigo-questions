# --description--

문자열도 리스트와 정확히 같은 방식으로 슬라이싱할 수 있습니다! 사실, 문자열을 문자의 리스트로 생각할 수 있습니다: 각 문자는 인덱스 `0`부터 시작하는 리스트의 순차적인 항목입니다.
```python
list_name[:2]
# Grabs the first two items
list_name[3:]
# Grabs the fourth through last items
```
리스트(또는 문자열)의 맨 처음이나 맨 마지막 항목이 슬라이스에 포함되는 경우, 해당 항목의 인덱스는 생략할 수 있습니다.

# --instructions--

- `full_name`의 인덱스 `0`부터 인덱스 `4`를 *포함하지 않는* 범위까지의 슬라이스를 `first_name`에 할당하세요.
- `full_name`의 인덱스 `4`부터 문자열 끝까지의 슬라이스를 `last_name`에 할당하세요

# --seed--

```python
full_name = "Sam Smith"
first_name = [/][/]
last_name = [/][/]
print(first_name)
print(last_name)
```
