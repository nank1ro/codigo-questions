# --description--

리스트는 고정된 길이를 가질 필요가 없습니다.
원할 때 언제든지 리스트의 끝에 항목을 추가할 수 있습니다!
리스트에 항목을 추가하려면 `append` 키워드를 사용합니다:
```python
>>> letters = ["a", "b"]
>>> letters.append("c")
>>> print(letters)
['a', 'b', 'c']
```
