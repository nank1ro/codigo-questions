# --description--

`while` 루프를 사용하여 코드를 반복하는 방법을 알고 있습니다.
다음은 `hello`를 출력하기 위해 문장을 반복하는 프로그램입니다:
```python
counter = 0

while counter < 5:
    print("hello")
    counter += 1
```
하지만 `for` 루프로도 동일한 작업을 할 수 있습니다:
```python
for i in range(5):
    print("hello")
```
