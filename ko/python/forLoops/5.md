# --description--

`range()` 함수는 기본적으로 0부터 시작하여 1씩 증가하며 지정된 숫자 전에 멈추는 숫자 시퀀스를 반환합니다.
이 함수의 구문은 다음과 같습니다:
```python
range(start, stop, step)
```
`start`와 `step`은 선택 사항이며, `stop`은 필수입니다
