# --description--

이미 초기화된 변수 `total`이 있습니다
```javascript
var total = 5;
```
변수 `total`에 숫자 `2`를 더하고 싶다면, 다음과 같이 작성할 수 있습니다
```javascript
total = total + 2;
```
네, 잘 동작합니다! 하지만 같은 작업을 더 짧게 작성하는 방법이 있습니다:
```javascript
total += 2;
```
`+=` 기호는 **덧셈 대입 연산자**라고 합니다.
변수의 값에 값을 더하고 그 결과를 해당 변수에 할당합니다.
