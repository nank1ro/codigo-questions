# --description--

덧셈 대입 연산자와 마찬가지로, **뺄셈 대입 연산자** `-=`가 있습니다.
기능은 동일하며, 유일한 차이점은 뺄셈을 수행한다는 것입니다.
따라서 다음 두 표현은 완전히 동일합니다
```javascript
var num = num - 5;
// is equal to
num -= 5;
```
