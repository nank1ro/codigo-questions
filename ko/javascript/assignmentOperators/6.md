# --description--

**나머지 대입 연산자** `%=`를 살펴보겠습니다.
변수와 값의 나머지를 계산하고 그 결과를 해당 변수에 할당합니다.
따라서 다음 두 표현은 완전히 동일합니다
```javascript
num = num % 5;
// is equal to
num %= 5;
```
