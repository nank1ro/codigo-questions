# --description--

배열과 마찬가지로, `Object.entries()` 메서드를 사용하여 딕셔너리 요소를 반복할 수 있습니다.
```javascript
var user = {"first_name": "John", "last_name": "Hood", "age": 30};
for (const [key, value] of Object.entries(user)) {
    console.log(`${key}: ${value}`);
}
```
