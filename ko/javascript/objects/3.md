# --description--

배열과 마찬가지로, 딕셔너리는 _변경 가능(mutable)_ 합니다.
이는 생성된 후에도 변경할 수 있다는 것을 의미합니다 (상수로 선언되지 않은 경우).
이것의 장점 중 하나는 다음과 같이 딕셔너리가 생성된 후에 새로운 _키/값 쌍_을 추가할 수 있다는 것입니다:
```javascript
dictName[newKeyName] = newValue;
```
