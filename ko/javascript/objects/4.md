# --description--

딕셔너리 변수는 변경 가능하기 때문에 다양한 방법으로 변경할 수 있습니다.
'delete' 키워드를 사용하여 딕셔너리에서 항목을 제거할 수 있습니다:
```javascript
delete dictName[keyName];
```
이렇게 하면 딕셔너리에서 키 `keyName`과 해당 값이 제거됩니다.
