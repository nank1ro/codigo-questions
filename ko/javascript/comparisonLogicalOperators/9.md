# --description--

마지막으로 **not** `!` 논리 연산자를 살펴보겠습니다.
이 연산자는 표현식의 논리 상태를 반전시킨 불리언을 반환합니다.
```javascript
console.log(!true);
// prints false
console.log(!false);
// prints true
console.log(!(2 == 2));
// prints false
```
