# --description--

**부등** `!=` 비교 연산자를 계속 살펴보겠습니다.
이 연산자는 두 표현식이 같지 **않은지** 여부를 나타내는 **불리언** (`true` 또는 `false`)을 반환합니다. 예를 들면:
```javascript
console.log(2 != 2);
// prints false
console.log(2 != 3);
// prints true
```
이것은 *동등* 연산자와 정확히 반대입니다
