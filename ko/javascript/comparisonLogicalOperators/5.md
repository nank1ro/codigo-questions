# --description--

**이상** `>=` 비교 연산자를 계속 살펴보겠습니다.
이 연산자는 하나의 표현식이 다른 표현식보다 크거나 같은지 여부를 나타내는 **불리언** (`true` 또는 `false`)을 반환합니다. 예를 들면:
```javascript
console.log(2 >= 2);
// prints true
console.log(3 >= 2);
// prints true
console.log(3 >= 4);
// prints false
```
