# --description--

**동등** `==` 비교 연산자부터 시작하겠습니다.
이 연산자는 두 표현식이 같은지 여부를 나타내는 **불리언** (`true` 또는 `false`)을 반환합니다. 예를 들면:
```javascript
console.log(2 == 2);
// prints true
console.log(2 == 3);
// prints false
```
