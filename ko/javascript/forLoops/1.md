# --description--

`while` 루프를 사용하여 코드를 반복하는 방법을 알고 있습니다.
다음 프로그램처럼 `hello`를 출력하는 명령문을 반복합니다
```javascript
var counter = 0;

while (counter < 5) {
    console.log("hello");
    counter++;
}
```
하지만 `for` 루프로도 같은 작업을 할 수 있습니다:
```javascript
for (let i = 0; i < 5; i++) {
    console.log("hello");
}
```
