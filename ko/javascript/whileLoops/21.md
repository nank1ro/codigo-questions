# --instructions--

이 반복문이 멈추는 것을 어떻게 알 수 있습니까?
```javascript
var isLoading = true;

while (isLoading) {
    console.log("Loading...");
    isLoading = false;
}
```

# --answers--

- 반복문 내의 변수가 false로 업데이트됩니다
- 반복문은 절대 멈추지 않습니다
