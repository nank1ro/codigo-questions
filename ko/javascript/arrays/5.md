# --description--

배열은 고정된 길이를 가질 필요가 없습니다.
언제든지 배열의 끝에 항목을 추가할 수 있습니다!
배열에 항목을 추가하려면 `push` 함수를 사용합니다:
```javascript
var letters = ["a", "b"];
letters.push("c");
console.log(letters);
// Prints ["a", "b", "c"]
```
