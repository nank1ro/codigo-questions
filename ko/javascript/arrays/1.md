# --description--

배열은 여러 가지 정보를 하나의 변수 이름 아래에 순서대로 저장할 수 있는 데이터 타입입니다.
배열은 하나 또는 여러 타입의 값을 저장하며, **인덱스**를 사용하여 이러한 값들을 구분합니다.
다음과 같은 형식의 표현식으로 배열에 항목을 할당할 수 있습니다:
```javascript
var arrayName = [item1, item2];
```
