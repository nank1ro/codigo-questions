# --description--

배열 요소는 어떤 타입이든 될 수 있습니다.
```javascript
var arrayName = ["one", 2, true];
```
실제로 위 코드에는 순서대로 문자열, 정수, 불리언이 있습니다.
하지만 배열 안에 배열을 넣을 수도 있습니다!

# --instructions--

배열에서 값 하나를 출력하세요

var array1 = ["a", "b", "c"];

var array3 = [array1, array2];
