# --description--

템플릿 리터럴을 사용하면 `+` 기호를 사용하지 않고도 문자열에 숫자를 추가하는 것과 같은 표현식을 표시할 수 있습니다.
유효한 템플릿 리터럴을 만들려면 큰따옴표나 작은따옴표 대신 백틱 \`\`을 사용해야 합니다.
달러 기호와 중괄호 `${expression}`을 사용하여 템플릿 리터럴 안에 변수를 삽입할 수 있습니다

# --seed--

```javascript
console.log([/]);
```

# --answers--

- `${5} friends`

# --solutions--

```javascript
console.log(`${5} friends`);
```

# --output--

5 friends
