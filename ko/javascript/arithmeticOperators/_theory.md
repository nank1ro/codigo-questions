연산자는 변수와 값에 대한 연산을 수행하는 데 사용됩니다.
산술 연산자부터 시작하겠습니다. 특히 **덧셈** `+` 연산자입니다.
이 연산자는 두 숫자를 더하는 데 사용됩니다. 예를 들면:
```javascript
console.log(5 + 3);
// prints 8
```

---

이번에는 **뺄셈** `-` 연산자를 알아보겠습니다.
이 연산자는 한 숫자에서 다른 숫자를 빼는 데 사용됩니다. 예를 들면:
```javascript
console.log(5 - 3);
// prints 2
```

---

이번에는 **곱셈** `*` 연산자를 알아보겠습니다.
이 연산자는 두 숫자를 곱하는 데 사용됩니다. 예를 들면:
```javascript
console.log(5 * 3);
// prints 15
```

---

이번에는 **나눗셈** `/` 연산자를 알아보겠습니다.
이 연산자는 두 숫자를 나누는 데 사용됩니다. 예를 들면:
```javascript
console.log(10 / 5);
// prints 2
```

---

이번에는 **나머지** `%` 연산자를 알아보겠습니다.
이 연산자는 두 숫자를 나눈 후 나머지를 구하는 데 사용됩니다. 예를 들면:
```javascript
console.log(5 % 2);
// prints 1
```
5를 2로 나누면 몫이 2이고 나머지가 1이므로 결과는 1입니다
```javascript
console.log(9 % 3);
// prints 0
```
9를 3으로 나누면 몫이 3이고 나머지가 0이므로 결과는 0입니다
