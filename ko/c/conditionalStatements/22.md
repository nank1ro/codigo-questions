# --instructions--

`Hello World`가 항상 출력된다는 것을 어떻게 알 수 있습니까?
```c
bool online = false;
if (online) {
    printf("User is online\n");
}
printf("Hello World\n");
```

# --answers--

- 조건문 안에 있지 않기 때문입니다
- if 문은 한 줄 이상 가질 수 없기 때문입니다
- else 문이 없기 때문입니다
- else if 문이 없기 때문입니다
