# --description--

의사 결정은 특정 조건이 충족될 때만 코드를 실행하고 싶을 때 필요합니다.
날씨가 좋을 때만 밖에서 놀고 싶다고 가정해 봅시다.
프로그래밍에서는 불리언 변수 `nice_weather`를 저장하고, 이 변수가 `true`일 때 `if`를 사용하여 밖에서 노는 동작을 수행할 수 있습니다:
```c
bool nice_weather = true;
if (nice_weather) {
    // play outside
}
```
