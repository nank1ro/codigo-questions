# --description--

이제 **불리언** 연산자를 알아보겠습니다. __and__ `&&`라고 불리는 첫 번째 연산자부터 시작하겠습니다.
이 연산자는 *거짓*으로 평가되는 첫 번째 피연산자를 반환하거나, 모두 *참*이면 마지막 피연산자를 반환합니다.
```c
>>> 2 == 2 && 2 == 3
0
>>> 1 == 1 && 1 == 1.0
1
```
