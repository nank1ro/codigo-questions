# --description--

**부등** `!=` 관계 연산자를 계속 알아보겠습니다.
이 연산자는 두 표현식이 같지 **않은지** 여부를 나타내는 **불리언** 값, 참 `1` 또는 거짓 `0`을 반환합니다. 예를 들어:
```c
>>> 2 != 2
0
>>> 2 != 3
1
```
이것은 *동등* 연산자와 정확히 반대입니다
