# --description--

**or** `||` 불리언 연산자를 계속 알아보겠습니다.
이 연산자는 *참*으로 평가되는 첫 번째 피연산자를 반환하거나, 모두 *거짓*이면 마지막 피연산자를 반환합니다.
```c
>>> 2 == 2 || 2 == 3
1
>>> 1 == 2 || 1 == 3
0
```
