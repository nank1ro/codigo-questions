# --description--

마지막으로 **not** `!` 불리언 연산자를 알아보겠습니다.
이 연산자는 표현식의 논리 상태를 반전시킨 불리언 값을 반환합니다.
```c
>>> !true
0
>>> !false
1
>>> !(2 == 2)
0
```

# --instructions--

`num`이 `5`와 같지 **않은지** 확인하는 불리언 값을 출력하세요

# --seed--

```c
int num = 5;
printf("%d\n", [/](num[/]5));
```
