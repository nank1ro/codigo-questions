# --description--

**곱셈 대입** `*=` 연산자를 살펴보겠습니다.
변수에 값을 곱하고 그 결과를 해당 변수에 할당합니다.
따라서 다음 두 가지는 완전히 동일합니다
```c
num = num * 5;
// is equal to
num *= 5;
```
