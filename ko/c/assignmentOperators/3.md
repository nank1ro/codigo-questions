# --description--

덧셈 대입과 마찬가지로 **뺄셈 대입** `-=`이 있습니다.
기능은 동일하며, 유일한 차이점은 뺄셈을 수행한다는 것입니다.
따라서 다음 두 가지는 완전히 동일합니다
```c
num = num - 5;
// is equal to
num -= 5;
```
