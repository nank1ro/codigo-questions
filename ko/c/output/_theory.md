"__Hello, World!__"는 새로운 프로그래밍 언어를 처음 배울 때 작성하는 전통적인 첫 번째 프로그램입니다.
`printf()` 함수를 사용하여 표준 출력 장치(화면)에 데이터를 출력합니다.
모든 C 프로그램은 라이브러리를 사용하며, 이를 통해 필요한 함수를 실행할 수 있습니다. 예를 들어 화면 출력 함수는 `stdio.h` 헤더 파일에 정의되어 있습니다.
가장 먼저 실행되는 코드는 항상 main 함수에 위치합니다.
C에서 화면에 `Hello, World!`를 출력하려면 다음과 같이 작성할 수 있습니다
```c
printf("Hello, World!");
```
마지막으로 프로그램이 성공적으로 실행되었음을 나타내기 위해 `0`을 반환합니다
