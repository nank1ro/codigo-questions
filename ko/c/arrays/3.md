# --description--

배열 인덱스는 다른 변수 이름과 동일하게 동작합니다! 값을 접근하거나 할당하는 데 사용할 수 있습니다.
배열 인덱스에 접근하는 방법은 다음과 같습니다:
```c
int numbers[4] = {5, 6, 7, 8};
numbers[0]; // Gets the value 5
```
할당은 다음과 같이 합니다:
```c
int numbers[4] = {5, 6, 7, 8};
numbers[0] = 1;
printf("%d\n", numbers[0]); // prints the new value 1
```
