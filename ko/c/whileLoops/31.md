# --instructions--

`counter += 1`은 무엇을 합니까?
```c
int counter = 1;

while (counter < 5) {
    printf("%d\n", counter);
    counter += 1;
}
```

# --answers--

- 반복문이 코드 블록을 실행할 때마다 counter를 1씩 증가시킵니다
- counter 변수를 생성합니다
- 매번 counter를 2씩 증가시킵니다
- 매번 counter를 3씩 증가시킵니다
