변수는 데이터 값을 저장하는 컨테이너입니다.
C에서 모든 변수는 객체이며, 다른 프로그래밍 언어와 마찬가지로 C에는 변수를 선언하는 명령어가 있습니다.
변수를 생성하려면 공백이 포함되지 않도록 주의하면서 **타입**과 **이름**을 지정해야 합니다.
변수는 처음으로 값을 할당하는 순간 생성됩니다.
`x`라는 이름의 변수를 생성하는 예시는 다음과 같습니다:
```c
int x = 1;
```
이렇게 하면 `x`라는 이름의 _정수_ 변수에 값 `1`을 할당한 것입니다.
변수 `x`를 출력하면 숫자 `1`을 돌려받습니다:
```c
>>> printf("%i\n", x);
1
```

---

변수는 저장하는 값이 변할 수 있기 때문에 이렇게 불립니다.
`=`를 사용하여 `x`에 새로운 값을 할당하면 업데이트할 수 있습니다.
```c
>>> x = 1;
>>> printf("%i\n", x);
1
>>> x = 2;
>>> printf("%i\n", x);
2
```

---

다른 변수의 값을 변수에 할당할 수도 있습니다. 여기서 `y` 변수에 `x`의 값을 할당할 수 있습니다
```c
>>> int x = 5;
>>> int y = x;
>>> printf("%i\n", y);
5
```

---

변수를 업데이트하면 이전 값을 잊어버립니다.
여기서 `x` 변수를 두 번 출력하여 값이 어떻게 업데이트되는지 확인할 수 있습니다.
```c
>>> int x = 5;
>>> printf("%i\n", x);
5
>>> x = 10;
>>> printf("%i\n", x);
10
```

---

C에서 문자열 변수는 큰따옴표를 사용해서만 선언할 수 있습니다:
```c
char x[] = "May";
```

---

여러 단어로 된 변수 이름을 사용하려면 **스네이크 케이스**를 사용합니다.
이는 추가 단어를 연결하기 위해 `_`를 사용하는 것을 의미합니다.
