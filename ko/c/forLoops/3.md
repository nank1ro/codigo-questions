# --description--

`<`를 사용하면 다음 숫자를 제외하고 반복하고, `<=`를 사용하면 다음 숫자를 포함하여 반복할 수 있습니다

# --instructions--

코드 블록을 `3`번 실행하도록 다음 코드에서 `for` 반복문을 완성하세요

# --seed--

```c
for (int i = 1; i[/]3; i++) {
        printf("%d\n", i);
}
```
