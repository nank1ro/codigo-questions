# --description--

함수는 여러 개의 입력 매개변수를 가질 수 있으며, 함수의 괄호 안에 쉼표로 구분하여 작성합니다.
```c
void say_hello(char *name, bool new_user) {
  char greet[40] = "Hello ";
  strcat(greet, name);
  if (new_user) {
    strcat(greet, "! Welcome on board :)");
  }
  printf("%s\n", greet);
}

int main() {
    // prints "Hello Tom"
    say_hello("Tom", true);
    return 0;
};
```

# --instructions--

두 번째 매개변수에 `true`를 전달하여 `say_hello`라는 유효한 함수를 만들고 결과를 출력하도록 코드를 완성하세요
