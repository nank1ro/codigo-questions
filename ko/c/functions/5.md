# --description--

함수에서 함수가 하는 일을 설명하는 _선택적 주석_을 추가할 수 있습니다:
```c
/*
 * Function:  hello_world
 * --------------------
 * prints "Hello, World!" to the screen
 */
function hello_world() {
    printf("Hello, World!\n");
}
```
한 줄 주석에는 `//`를, 여러 줄 주석에는 `/* */`를 사용할 수 있습니다
