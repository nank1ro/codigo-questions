# --description--

C에서는 `printf` 함수와 표현식을 사용하여 두 개 이상의 값을 함께 표시할 수 있습니다. 예시:
```c
>>> char c = 'C';
>>> printf("Hello %c!\n", c);
Hello C!
```
`%c`는 단일 문자를 표시하는 데 사용됩니다

# --instructions--

`%s` 표현식(_문자 배열_ 또는 _문자열_을 표시하는 데 사용됨)을 사용하여 이름을 출력하세요
