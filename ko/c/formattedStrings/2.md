# --description--

서식 문자열을 사용하면 오류 없이 문자열에 숫자를 추가하는 것과 같은 표현식을 표시할 수 있습니다.
정수를 표시하려면 `%i` 표현식을 사용합니다

# --seed--

```c
printf("[/] friends\n", [/]);
```

# --answers--

- %i
- ?i
- %0
- 5

# --solutions--

```c
printf("%i friends\n", 5);
```
