# --description--

`floor()` 함수를 사용한 **내림 나눗셈**을 알아보겠습니다.
이 함수는 몫의 정수 부분을 반환합니다. 예를 들면:
```
>>> 5.0 / 2
2.5
>>> floor(2.5)
2.0
```
정수 나눗셈이라고도 합니다. 결과값은 정수이지만, 결과의 *타입*이 반드시 int인 것은 아닙니다.
