# --description--

C에는 **거듭제곱** 연산자가 없으므로 `math.h` 라이브러리에 포함된 `pow()` 함수를 사용해야 합니다.
거듭제곱은 밑의 반복 곱셈에 해당합니다. 즉, 밑 **b**에 지수 *n*을 적용하면 *n*개의 밑을 곱한 결과입니다:
![exponentiation](https://bit.ly/3zcz6Lt)
```
>>> pow(5, 2);
25
```
