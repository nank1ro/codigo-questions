# --description--

_점 문법_을 사용하여 인스턴스의 프로퍼티에 접근할 수 있습니다.
점 문법에서는 인스턴스 이름 바로 뒤에 마침표 `.`로 구분하여 공백 없이 프로퍼티 이름을 작성합니다:
```swift
someInstance.someProperty
```
같은 문법을 사용하여 프로퍼티의 값을 업데이트할 수도 있습니다
