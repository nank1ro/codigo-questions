# --description--

Swift에서는 원하는 대로 배열을 슬라이스할 수 있습니다!
```swift
// 처음 두 항목을 가져옵니다
listName[..<2]
// 네 번째부터 마지막 항목까지 가져옵니다
listName[3...]
```
배열 슬라이스에 맨 처음 또는 맨 마지막 항목이 포함되는 경우, 해당 항목의 인덱스를 포함할 필요가 없습니다
