변수에 값을 할당하기 위해 `=` 기호를 사용할 수 있다는 것을 이미 배웠습니다. 예를 들면 다음과 같습니다:
```swift
let a = 5
```

---

이미 초기화된 변수 `total`이 있습니다
```swift
var total = 5
```
변수 `total`에 숫자 `2`를 더하고 싶다면, 다음과 같이 작성할 수 있습니다
```swift
total = total + 2
```
네, 잘 작동합니다! 하지만 같은 작업을 수행하는 더 짧은 방법이 있습니다:
```swift
total += 2
```
`+=` 기호는 **덧셈 대입**이라고 합니다.
이 연산자는 변수의 값에 값을 더한 후 그 결과를 해당 변수에 할당합니다.

---

덧셈 대입과 마찬가지로, **뺄셈 대입** `-=`이 있습니다.
기능은 동일하며, 유일한 차이점은 뺄셈을 수행한다는 것입니다.
따라서 다음 두 표현은 완전히 동일합니다
```swift
var num = num - 5
// is equal to
num -= 5
```

---

**곱셈 대입** `*=` 연산자를 살펴보겠습니다.
이 연산자는 변수에 값을 곱한 후 그 결과를 해당 변수에 할당합니다.
따라서 다음 두 표현은 완전히 동일합니다
```swift
var num = num * 5
// is equal to
num *= 5
```

---

**나눗셈 대입** `/=` 연산자를 살펴보겠습니다.
이 연산자는 변수를 값으로 나눈 후 그 결과를 해당 변수에 할당합니다.
따라서 다음 두 표현은 완전히 동일합니다
```swift
num = num / 5
// is equal to
num /= 5
```

---

**나머지 대입** `%=` 연산자를 살펴보겠습니다.
이 연산자는 변수와 값의 나머지를 계산한 후 그 결과를 해당 변수에 할당합니다.
따라서 다음 두 표현은 완전히 동일합니다
```swift
num = num % 5
// is equal to
num %= 5
```
