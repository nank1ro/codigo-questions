# --description--

**곱셈 대입** `*=` 연산자를 살펴보겠습니다.
이 연산자는 변수에 값을 곱한 후 그 결과를 해당 변수에 할당합니다.
따라서 다음 두 표현은 완전히 동일합니다
```swift
var num = num * 5
// is equal to
num *= 5
```
