# --description--

이전에 `removeValue()` 메서드를 사용하여 딕셔너리에서 _키-값 쌍_ 을 제거하는 방법을 배웠습니다.
키에 `nil` 값을 할당하여 요소를 제거할 수도 있습니다
```swift
dictName[keyName] = nil
// keyName has been removed from the dictionary dictName
```
