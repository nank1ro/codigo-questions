# --description--

`keys` 속성을 사용하면 됩니다.

# --seed--

```swift
var user: [String: Any] = ["first_name": "John", "last_name": "Hood", "age": 30]
print([/][/])
```

# --answers--

- .keys()
- .keys
- .key
