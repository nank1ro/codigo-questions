# --description--

배열과 마찬가지로, `for..in` 키워드를 사용하여 딕셔너리 요소를 반복할 수 있습니다.
반복문에서 키와 값을 모두 얻으려면 별도의 속성을 지정할 필요가 없습니다:
```swift
for (key, value) in dictName {
    print("\(key): \(value)")
}
```
