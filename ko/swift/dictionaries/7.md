# --description--

딕셔너리의 모든 값을 나열하고 싶다면 어떻게 해야 할까요?
`values` 속성을 사용하면 됩니다.

# --seed--

```swift
var user: [String: Any] = ["first_name": "John", "last_name": "Hood", "age": 30]
print([/][/])
```

# --answers--

- user
- .values()
- .values
- .value
- .allValues()

# --solutions--

```swift
var user: [String: Any] = ["first_name": "John", "last_name": "Hood", "age": 30]
print(user.values)
```

["John", "Hood", 30]
