# --description--

_nil 병합 연산자_ `a ?? b`는 옵셔널 `a`에 값이 있으면 언래핑하고, `a`가 `nil`이면 기본값 `b`를 반환합니다.
표현식 `a`는 항상 옵셔널 타입입니다.
표현식 `b`는 `a` 안에 저장된 타입과 일치해야 합니다.
nil 병합 연산자는 아래 코드의 축약형입니다:
```swift
a != nil ? a! : b
```
