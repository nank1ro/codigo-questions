# --description--

마지막으로 **not** `!` 논리 연산자를 알아보겠습니다.
이 연산자는 표현식의 논리 상태를 반전시킨 불리언을 반환합니다.
```swift
print(!true) // false
print(!false) // true
print(!(2 == 2)) // false
```
