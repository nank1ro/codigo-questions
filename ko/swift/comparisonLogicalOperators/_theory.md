**같음** `==` 비교 연산자부터 시작하겠습니다.
이 연산자는 두 표현식이 같은지 여부를 나타내는 **불리언** (`true` 또는 `false`)을 반환합니다. 예를 들어:
```swift
print(2 == 2) // true
print(2 == 3) // false
```

---

**같지 않음** `!=` 비교 연산자를 계속 알아보겠습니다.
이 연산자는 두 표현식이 같지 **않은지** 여부를 나타내는 **불리언** (`true` 또는 `false`)을 반환합니다. 예를 들어:
```swift
print(2 != 2) // false
print(2 != 3) // true
```
이것은 *같음* 연산자와 정확히 반대입니다

---

**큼** `>` 비교 연산자를 계속 알아보겠습니다.
이 연산자는 하나의 표현식이 다른 표현식보다 큰지 여부를 나타내는 **불리언** (`true` 또는 `false`)을 반환합니다. 예를 들어:
```swift
print(2 > 2) // false
print(3 > 2) // true
```

---

**작음** `<` 비교 연산자를 계속 알아보겠습니다.
이 연산자는 하나의 표현식이 다른 표현식보다 작은지 여부를 나타내는 **불리언** (`true` 또는 `false`)을 반환합니다. 예를 들어:
```swift
print(2 < 2) // false
print(2 < 3) // true
```

---

**크거나 같음** `>=` 비교 연산자를 계속 알아보겠습니다.
이 연산자는 하나의 표현식이 다른 표현식보다 크거나 같은지 여부를 나타내는 **불리언** (`true` 또는 `false`)을 반환합니다. 예를 들어:
```swift
print(2 >= 2) // true
print(3 >= 2) // true
print(3 >= 4) // false
```

---

**작거나 같음** `<=` 비교 연산자를 계속 알아보겠습니다.
이 연산자는 하나의 표현식이 다른 표현식보다 작거나 같은지 여부를 나타내는 **불리언** (`true` 또는 `false`)을 반환합니다. 예를 들어:
```swift
print(2 <= 2) // true
print(3 <= 2) // false
print(3 <= 4) // true
```

---

이제 **논리** 연산자를 살펴보겠습니다. 먼저 __AND__ `&&`부터 시작합니다.
이 연산자는 *false*로 평가되는 첫 번째 피연산자를 반환하거나, 모두 *true*이면 마지막 피연산자를 반환합니다.
```swift
print(2 == 2 && 2 == 3) // false
print(1 == 1 && 1 == 1.0) // true
```

---

**or** `||` 논리 연산자를 계속 알아보겠습니다.
이 연산자는 *true*로 평가되는 첫 번째 피연산자를 반환하거나, 모두 *false*이면 마지막 피연산자를 반환합니다.
```swift
print(2 == 2 || 2 == 3) // true
print(1 == 2 || 1 == 3) // false
```

---

마지막으로 **not** `!` 논리 연산자를 알아보겠습니다.
이 연산자는 표현식의 논리 상태를 반전시킨 불리언을 반환합니다.
```swift
print(!true) // false
print(!false) // true
print(!(2 == 2)) // false
```
