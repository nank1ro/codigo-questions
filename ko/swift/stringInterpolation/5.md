# --description--

다음으로, 다른 종류의 값을 중괄호 안에 넣어 하나의 print 문으로 표시합니다. 여기에서는 `\(5)`를 사용합니다

# --instructions--

다음 출력을 얻기 위해 빈 공간을 채우세요
```swift
5 friends
```

# --seed--

```swift
print(" friends")
```

# --solutions--

```swift
print("\(5) friends")
```

# --output--

5 friends
