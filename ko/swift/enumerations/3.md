# --description--

`switch` 문을 사용하여 개별 열거형 값을 매칭할 수 있습니다:
```swift
let color = Colors.red
switch color {
    case .blue:
        print("Blue")
    case .red:
        print("Red")
    case .green:
        print("Green")
}
// prints "Red"
```
모든 열거형 케이스에 대해 `case`를 제공할 필요가 없는 경우, `default` 케이스를 사용하여 명시적으로 처리되지 않은 케이스를 포괄할 수 있다는 점을 기억하세요
