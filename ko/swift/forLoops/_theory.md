`while` 반복문을 사용하여 코드를 반복하는 방법을 알고 있습니다.
다음 프로그램은 `hello`를 출력하기 위해 구문을 반복합니다
```swift
var counter = 0

while counter < 5 {
    print("hello")
    counter += 1;
}
```
하지만 `for` 반복문으로도 동일한 작업을 할 수 있습니다:
```swift
for i in 0..<5 {
    print("hello")
}
```

---

`for` 반복문에서 반복문이 실행될 횟수를 지정할 수 있습니다

---

`..<`를 사용하면 다음 숫자를 제외하고 반복하고, `...`를 사용하면 다음 숫자를 포함하여 반복할 수 있습니다

---

`i`라는 변수를 카운터 변수라고 합니다.
원하는 이름을 지정할 수 있습니다.
카운터 변수는 현재 반복문이 몇 번째 반복 중인지를 나타냅니다

---

`stride()` 함수는 숫자 시퀀스를 반환합니다.
이 함수는 _from_, _to_, _by_ 매개변수가 필요합니다.
함수의 구문은 다음과 같습니다:
```swift
stride(from:to:by:)
```
`to` 값은 제외된다는 점을 기억하세요

---

`stride()` 함수를 사용하면 닫힌 범위도 사용할 수 있으며, 다음과 같이 작성합니다:
```swift
stride(from:through:by:)
```
이 경우 `through` 값은 포함됩니다

---

Swift에는 `forEach` 반복문도 있습니다.
`forEach`는 `for-in` 반복문과 동일한 순서로 시퀀스의 각 요소에 대해 주어진 클로저를 호출합니다:
```swift
// 이것은 배열입니다. 곧 배울 것입니다
let numbers: [Int] = [1, 3, 5, 7, 9]
numbers.forEach { num in
    print(num)
}
```
`forEach` 메서드를 사용하는 것은 `for-in` 반복문과 두 가지 중요한 차이점이 있습니다:
1. `break` 또는 `continue` 구문을 사용하여 현재 본문 클로저의 호출을 종료하거나 후속 호출을 건너뛸 수 없습니다.
2. 본문 클로저에서 `return` 구문을 사용하면 클로저만 종료되며 외부 범위를 종료하지 않고, 후속 호출도 건너뛰지 않습니다.
