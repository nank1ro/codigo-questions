# --description--

`stride()` 함수는 숫자 시퀀스를 반환합니다.
이 함수는 _from_, _to_, _by_ 매개변수가 필요합니다.
함수의 구문은 다음과 같습니다:
```swift
stride(from:to:by:)
```
`to` 값은 제외된다는 점을 기억하세요
