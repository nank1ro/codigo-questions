# --description--

Swift에는 `forEach` 반복문도 있습니다.
`forEach`는 `for-in` 반복문과 동일한 순서로 시퀀스의 각 요소에 대해 주어진 클로저를 호출합니다:
```swift
// 이것은 배열입니다. 곧 배울 것입니다
let numbers: [Int] = [1, 3, 5, 7, 9]
numbers.forEach { num in
    print(num)
}
```
`forEach` 메서드를 사용하는 것은 `for-in` 반복문과 두 가지 중요한 차이점이 있습니다:
1. `break` 또는 `continue` 구문을 사용하여 현재 본문 클로저의 호출을 종료하거나 후속 호출을 건너뛸 수 없습니다.
2. 본문 클로저에서 `return` 구문을 사용하면 클로저만 종료되며 외부 범위를 종료하지 않고, 후속 호출도 건너뛰지 않습니다.
