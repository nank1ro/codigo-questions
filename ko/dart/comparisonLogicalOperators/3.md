# --description--

**보다 큼 연산자** `>` 는 왼쪽 값이 오른쪽 값보다 엄격히 크면 `true`를 반환합니다.

```dart
int a = 8;
int b = 3;
bool result = a > b; // true
```

왼쪽 값이 오른쪽 값 이하이면 `false`를 반환합니다.
