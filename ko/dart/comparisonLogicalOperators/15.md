# --description--

**OR 연산자** `||` 는 두 개의 부울 식을 결합합니다. **최소한 하나**의 쪽이 `true`일 때 `true`를 반환합니다. **두 쪽 모두** `false`일 때만 `false`를 반환합니다.

```dart
bool a = false;
bool b = true;
bool result = a || b; // true

bool c = false;
bool d = false;
bool result2 = c || d; // false
```
