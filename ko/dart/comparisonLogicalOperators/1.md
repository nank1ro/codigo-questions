# --description--

**같음 연산자** `==` 는 두 값을 비교하며 같으면 `true`, 다르면 `false`를 반환합니다.

```dart
int a = 5;
int b = 5;
bool result = a == b; // true
```

비교의 결과는 항상 `bool` 값입니다.

# --instructions--

코드를 완성하여 `10`이 `10`과 같은지 확인하고 결과를 `isEqual`에 저장합니다.
