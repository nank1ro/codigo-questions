# --description--

**이상 연산자** `>=` 는 왼쪽 값이 오른쪽 값보다 크거나 **같으면** `true`를 반환합니다.

```dart
int score = 50;
bool passed = score >= 50; // true
```

`>` 와는 달리 이 연산자는 두 값이 같을 때도 `true`를 반환합니다.
