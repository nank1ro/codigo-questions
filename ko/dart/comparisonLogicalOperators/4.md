# --description--

**보다 작음 연산자** `<` 는 왼쪽 값이 오른쪽 값보다 엄격히 작으면 `true`를 반환합니다.

```dart
int a = 2;
int b = 9;
bool result = a < b; // true
```

왼쪽 값이 오른쪽 값 이상이면 `false`를 반환합니다.
