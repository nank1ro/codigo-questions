# --instructions--

`int` 매개변수 `n`을 받아 `n!`(`n`의 팩토리얼)을 반환하는 **재귀** 함수 `factorial`을 작성하세요.

`n`의 팩토리얼은 다음과 같이 정의됩니다：
- `factorial(0) = 1`
- `factorial(n) = n * factorial(n - 1)` (`n > 0`인 경우)

예시:
```dart
print(factorial(5)); // 120
print(factorial(3)); // 6
```

# --solutions--

```dart
int factorial(int n) {
  if (n == 0) return 1;
  return n * factorial(n - 1);
}
```
