# --description--

함수는 **매개변수**를 받을 수 있습니다 — 함수를 호출할 때 전달되는 값입니다. 매개변수는 타입과 이름을 함께 괄호 안에 선언합니다：

```dart
int square(int n) {
  return n * n;
}

void main() {
  print(square(4)); // prints 16
}
```

각 매개변수는 타입(`int`)과 이름(`n`)을 가집니다. 여러 매개변수는 쉼표로 구분합니다.
