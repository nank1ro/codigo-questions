# --description--

함수는 호출자에게 **값을 반환**할 수 있습니다. 함수 이름 앞에 반환 타입을 선언하고, `return` 키워드를 사용해 값을 돌려줍니다：

```dart
int getAge() {
  return 25;
}
```

함수 이름 앞의 타입(`int`)은 함수가 반환할 값의 종류를 Dart에 알려줍니다. `String`, `int`, `double`, `bool` 등을 사용할 수 있습니다.
