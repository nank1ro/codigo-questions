# --description--

**`if-else` 문**은 조건이 `true`이면 `if` 블록을 실행하고, `false`이면 `else` 블록을 실행합니다.

```dart
int temperature = 10;
if (temperature > 20) {
  print('Warm');
} else {
  print('Cold');
}
// prints: Cold
```

두 분기 중 항상 정확히 하나만 실행됩니다.
