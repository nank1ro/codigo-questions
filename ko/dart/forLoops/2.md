# --description--
현재 카운트를 추적하기 위해 반복문 본체 안에서 반복 변수를 사용할 수 있습니다. 예를 들어, 합계를 누적하는 경우:

```dart
int total = 0;
for (int i = 1; i <= 5; i++) {
  total += i;
}
```

반복문 후에 `total`은 15가 됩니다.
