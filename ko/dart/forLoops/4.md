# --description--
`for-in` 반복문은 인덱스 없이 컬렉션의 모든 요소를 반복 처리합니다:

```dart
List<String> fruits = ['apple', 'banana', 'cherry'];
for (var fruit in fruits) {
  print(fruit);
}
```

각 반복에서 다음 요소가 반복 변수（여기서는 `fruit`）에 할당됩니다.
