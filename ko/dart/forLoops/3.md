# --description--
`for` 반복문은 감소 연산자(`i--`)와 `>=` 조건을 사용하여 **아래로** 카운트할 수 있습니다:

```dart
for (int i = 5; i >= 1; i--) {
  print(i); // 5, 4, 3, 2, 1
}
```

이것은 카운트다운이나 역순 반복에 유용합니다.
