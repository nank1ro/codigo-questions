# --description--
`break` 문은 조건이 충족되면 즉시 반복문을 종료합니다:

```dart
for (int i = 0; i < 10; i++) {
  if (i == 5) break; // stops at 5
  print(i);
}
```

이것은 값을 검색할 때 찾으면 즉시 멈추고 싶은 경우에 유용합니다.
