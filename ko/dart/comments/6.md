# --description--

이 두 가지 주석 작성 방법 외에도, Dart에는 _문서화 주석_이 있습니다.

_문서화 주석_은 `///` 또는 `/**`로 시작하는 여러 줄 또는 한 줄 주석입니다. 연속된 줄에서 `///`를 사용하면 여러 줄 문서화 주석과 동일한 효과가 있습니다.

_문서화 주석_은 문서를 자동으로 생성할 수 있게 해주므로 매우 유용합니다.
특정 코드 블록이 무엇을 하는지 명확하게 하기 위해 코드에 _문서화 주석_을 추가하는 것이 좋습니다.

_문서화 주석_ 내에서 분석기는 대괄호로 묶이지 않은 모든 텍스트를 무시합니다.
대괄호를 사용하여 _클래스_, _메서드_, _필드_, _변수_, _함수_ 및 _매개변수_를 참조할 수 있습니다.

다음은 예시입니다:
```dart
/// A domesticated South American camelid (Lama glama).
///
/// Just like any other animal, llamas need to eat,
/// so don't forget to [feed] them some [Food].
class Llama {
  String? name;

  /// Feeds your llama [food].
  ///
  /// The typical llama eats one bale of hay per week.
  void feed(Food food) {
    // ...
  }

  /// Exercises your llama with an [activity] for
  /// [timeLimit] minutes.
  void exercise(Activity activity, int timeLimit) {
    // ...
  }
}
```

생성된 문서에서 `[feed]`는 `feed` 메서드의 문서에 대한 링크가 되고, `[Food]`는 `Food` 클래스의 문서에 대한 링크가 됩니다.
`[activity]`와 `[timeLimit]`는 각각 `activity`와 `timeLimit`의 문서에 대한 링크가 됩니다.
