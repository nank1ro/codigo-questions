# --description--

주석 블록, 즉 _여러 줄_ 주석을 만들 수도 있습니다.
_여러 줄_ 주석은 `/*`로 시작하고 `*/`로 끝납니다.
```dart
/*
This is a multi-line
comment
*/
```
