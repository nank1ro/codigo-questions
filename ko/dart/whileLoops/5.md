# --description--

`break` 키워드는 반복문 조건에 관계없이 반복문을 즉시 종료합니다.

```dart
int i = 0;
while (true) {
  if (i == 3) break;
  print(i);
  i++;
}
```

`break` 없이는 `while (true)` 반복문이 영원히 실행됩니다. 여기서는 `i`가 `3`에 도달하면 멈춥니다.

# --instructions--
