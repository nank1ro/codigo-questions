# --description--

카운터 변수는 `while` 반복문이 몇 번 실행될지를 제어합니다.

```dart
int count = 0;
while (count < 5) {
  count++;
}
print(count); // 5
```

카운터는 `0`에서 시작하고, `count < 5`가 `false`가 될 때까지 반복문이 각 반복마다 증가시킵니다.

# --instructions--

`1`부터 `5`까지 세고 각 값을 출력하는 반복문을 완성하세요.
