# --description--

`do-while` 반복문은 `while` 반복문과 비슷하지만, **항상 최소 한 번은 본문을 실행합니다** — 조건은 각 반복 *후에* 확인됩니다.

```dart
int i = 0;
do {
  print(i);
  i++;
} while (i < 3);
```

처음부터 조건이 `false`이더라도 본문은 한 번 실행됩니다.
