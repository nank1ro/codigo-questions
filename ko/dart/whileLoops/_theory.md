`while` 반복문은 조건이 `true`인 동안 코드 블록을 반복합니다.

```dart
int i = 0;
while (i < 3) {
  print(i);
  i++;
}
```

반복문은 각 반복 **전에** 조건을 확인합니다. `i < 3`이 `false`가 되면 반복문이 멈춥니다.

---

카운터 변수는 `while` 반복문이 몇 번 실행될지를 제어합니다.

```dart
int count = 0;
while (count < 5) {
  count++;
}
print(count); // 5
```

카운터는 `0`에서 시작하고, `count < 5`가 `false`가 될 때까지 반복문이 각 반복마다 증가시킵니다.

---

`do-while` 반복문은 `while` 반복문과 비슷하지만, **항상 최소 한 번은 본문을 실행합니다** — 조건은 각 반복 *후에* 확인됩니다.

```dart
int i = 0;
do {
  print(i);
  i++;
} while (i < 3);
```

처음부터 조건이 `false`이더라도 본문은 한 번 실행됩니다.

---

`while` 반복문의 조건에는 어떤 불리언 표현식도 사용할 수 있습니다. 조건이 `false`로 평가되는 순간 반복문이 종료됩니다.

```dart
int n = 100;
while (n > 1) {
  n ~/= 2;
}
print(n); // 0
```

여기서 `n`은 정수 나눗셈(`~/`)을 사용하여 각 반복마다 절반이 됩니다.

---

`break` 키워드는 반복문 조건에 관계없이 반복문을 즉시 종료합니다.

```dart
int i = 0;
while (true) {
  if (i == 3) break;
  print(i);
  i++;
}
```

`break` 없이는 `while (true)` 반복문이 영원히 실행됩니다. 여기서는 `i`가 `3`에 도달하면 멈춥니다.
