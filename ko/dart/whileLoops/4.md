# --description--

`while` 반복문의 조건에는 어떤 불리언 표현식도 사용할 수 있습니다. 조건이 `false`로 평가되는 순간 반복문이 종료됩니다.

```dart
int n = 100;
while (n > 1) {
  n ~/= 2;
}
print(n); // 0
```

여기서 `n`은 정수 나눗셈(`~/`)을 사용하여 각 반복마다 절반이 됩니다.

# --instructions--

`n`이 `1`보다 큰 동안 (`~/= 2`를 사용하여) `n`을 계속 절반으로 나누고, 그 후 결과를 출력하는 `while` 반복문을 완성하세요.
