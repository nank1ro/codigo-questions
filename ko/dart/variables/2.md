# --description--

이전 예제에서 변수를 생성하는 것을 보았습니다:

```dart
int number = 1;
```

`=` 기호에 속지 마세요.
이것은 수학에서의 등호가 아니라 변수에 값을 할당하기 때문에 __할당 연산자__라고 합니다.
반면에 등호는 `==`입니다.
