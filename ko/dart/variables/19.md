__컴파일 타임__ 상수와 __런타임__ 상수의 차이점은 무엇입니까?

# --answers--

- __컴파일 타임__ 상수는 프로그램 실행 전에 알려진 고정 값을 포함하고, __런타임__ 상수는 프로그램 실행 중에만 고정 값을 얻습니다
- __컴파일 타임__ 상수는 프로그램 실행 후에 알려진 고정 값을 포함하고, __런타임__ 상수는 프로그램 실행 전에 고정 값을 얻습니다
- 두 가지 사이에 차이점은 없습니다
