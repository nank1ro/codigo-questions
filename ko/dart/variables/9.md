# --description--

상수 변수를 선언한 후에는 더 이상 값을 변경할 수 없습니다. 예를 들어:

```dart
const number = 2;
number = 3; // Error
```

이 코드는 다음과 같은 오류를 발생시킵니다:
> Constant variables can't be assigned a value.

즉, 상수 변수의 값을 변경하는 것은 불가능합니다.
