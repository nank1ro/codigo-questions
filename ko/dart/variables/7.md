# --description--

Dart는 __타입 추론__을 지원합니다.
변수의 타입을 명시할 필요가 없으며, Dart가 타입을 추론할 수 있습니다.
`var` 키워드는 Dart에게 가장 적절한 타입을 사용하도록 지시합니다.

```dart
var number = 5;
```

숫자 `5`가 `int` 타입이라고 Dart에 알려줄 필요가 없습니다.
Dart가 타입을 추론하여 `number`를 `int` 타입으로 만듭니다.
