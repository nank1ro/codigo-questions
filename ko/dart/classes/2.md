# --description--

**생성자**는 클래스의 인스턴스를 만들 때 실행되는 특별한 메서드입니다. 객체의 필드를 초기화하는 데 사용됩니다.

Dart에서 가장 간단한 생성자는 **축약 문법** `this.fieldName`을 사용하여 매개변수를 필드에 직접 대입합니다:

```dart
class Car {
  String brand;
  Car(this.brand);
}

void main() {
  final car = Car('Toyota');
  print(car.brand); // Toyota
}
```
