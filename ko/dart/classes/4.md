# --description--

**게터**는 접근할 때 프로퍼티처럼 보이는 특별한 메서드입니다. `get` 키워드를 사용하여 정의합니다:

```dart
class Circle {
  double radius;
  Circle(this.radius);

  double get area => 3.14 * radius * radius;
}

void main() {
  final c = Circle(5);
  print(c.area); // 메서드 호출이 아닌 필드처럼 접근
}
```

게터는 필드에서 계산된 값에 유용합니다.
