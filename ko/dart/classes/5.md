# --description--

**상속**을 사용하면 클래스가 다른 클래스의 필드와 메서드를 재사용할 수 있습니다. `extends` 키워드를 사용하여 서브클래스(자식 클래스)를 만듭니다:

```dart
class Animal {
  String name;
  Animal(this.name);

  void speak() {
    print('...');
  }
}

class Cat extends Animal {
  Cat(String name) : super(name);

  @override
  void speak() {
    print('Meow');
  }
}
```

자식 클래스는 `super(...)`로 부모 생성자를 호출합니다. `@override` 어노테이션은 메서드가 부모의 버전을 재정의한다는 것을 나타냅니다.

# --instructions--

`Cat`이 `Animal`을 상속하고, 부모 생성자를 호출하며, `speak()`를 오버라이드하여 `'Meow'`를 출력하도록 코드를 완성하세요.
