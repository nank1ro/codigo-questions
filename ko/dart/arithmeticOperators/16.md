# --description--

Dart는 표준 수학적 **연산 순서**(우선순위)를 따릅니다: 곱셈과 나눗셈은 덧셈과 뺄셈 전에 계산됩니다.

```dart
int result = 2 + 3 * 4; // 14, not 20
```

괄호를 사용하여 순서를 변경합니다:

```dart
int result = (2 + 3) * 4; // 20
```

# --instructions--

최상위 `int` 변수 `result`를 선언하여 `3 + 4 * 2 - 1`을 계산합니다. Dart에 기본 연산자 우선순위를 적용하도록 합니다(괄호를 추가하지 마십시오).
