# --description--

**정수 나눗셈 연산자** `~/`는 두 수를 나누고 결과를 정수로 **잘라냅니다**(모든 소수 부분 폐기).

```dart
int a = 10;
int b = 3;
int result = a ~/ b; // 3  (not 3.333...)
```

나머지 없이 정수 몫이 필요할 때 `~/`을 사용하십시오.

# --instructions--

`17`을 `5`로 정수 나누는 코드를 완성하고 결과를 `result`에 저장합니다.

# --seed--

```dart
void main() {
  int result = 17 [/] 5;
  print(result);
```
