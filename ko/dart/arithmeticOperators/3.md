# --description--

**곱셈 연산자** `*`는 두 값을 곱합니다.

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

`int`를 다른 `int`로 곱하면 `int`가 생성됩니다. `double`로 곱하면 `double`이 생성됩니다.
