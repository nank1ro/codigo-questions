**덧셈 연산자** `+`는 두 값을 더합니다.

```dart
int a = 3;
int b = 4;
int sum = a + b; // 7
```

`int` 또는 `double` 값으로 `+`를 사용할 수 있습니다.

---

**뺄셈 연산자** `-`는 한 값에서 다른 값을 뺍니다.

```dart
int a = 10;
int b = 4;
int diff = a - b; // 6
```

두 번째 피연산자가 첫 번째보다 크면 결과가 음수일 수 있습니다.

---

**곱셈 연산자** `*`는 두 값을 곱합니다.

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

`int`를 다른 `int`로 곱하면 `int`가 생성됩니다. `double`로 곱하면 `double`이 생성됩니다.

---

**나눗셈 연산자** `/`는 한 값을 다른 값으로 나눕니다. Dart에서는 `/`가 결과가 정수인 경우에도 **항상 `double`을 반환합니다**.

```dart
int a = 10;
int b = 4;
double result = a / b; // 2.5

double exact = 10 / 2; // 5.0  (not 5!)
```

`/`의 결과를 저장하려면 항상 `double` 변수를 사용하십시오.

---

**정수 나눗셈 연산자** `~/`는 두 수를 나누고 결과를 정수로 **잘라냅니다**(모든 소수 부분 폐기).

```dart
int a = 10;
int b = 3;
int result = a ~/ b; // 3  (not 3.333...)
```

나머지 없이 정수 몫이 필요할 때 `~/`을 사용하십시오.
