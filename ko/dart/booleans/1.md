# --description--

__불리언__은 `true` 또는 `false` 중 2가지 값만을 가질 수 있는 데이터 타입입니다.
Dart에서 불리언 타입은 `bool` 키워드로 선언합니다.

```dart
bool isRaining = true;
```

변수 `isRaining`은 값 `true`를 저장합니다. 이는 현재 비가 오고 있다는 의미입니다.
불리언 값은 항상 소문자로 작성됩니다: `true`와 `false`.
