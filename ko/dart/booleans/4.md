# --description--

__부정 연산자__ `!`는 ("not"이라고도 함) 불리언 값을 반대로 뒤집습니다.
`true`에 `!`를 적용하면 `false`가 되고, `false`에 `!`를 적용하면 `true`가 됩니다.

```dart
bool isActive = true;
print(!isActive); // false
```

이는 조건의 반대를 확인하고 싶을 때 유용합니다.
