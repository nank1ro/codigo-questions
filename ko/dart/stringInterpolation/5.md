# --description--

`$` 기호 뒤에 오는 것이 프로그램 식별자로 인식되지 않으면 Dart 는 컴파일 오류를 발생시킵니다. 문자열 안의 `$` 뒤에는 항상 식별자 또는 중괄호로 둘러싼 표현식 (`${}`) 이 와야 합니다.

# --instructions--

이 코드를 실행하면 어떻게 됩니까?

```dart
void main() {
    print("$5 friends");
}
```

# --answers--

- `$5` 가 유효하지 않은 문자열 보간이라 컴파일에 실패합니다
- `$5 friends` 를 출력합니다
- `5 friends` 를 출력합니다
- 아무것도 출력하지 않습니다
