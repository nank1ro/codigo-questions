# --description--

타입을 사용하면 코드에서 사용하는 모든 다양한 데이터를 분류할 수 있습니다.
Dart에서 __타입__은 주어진 데이터를 어떻게 사용할 것인지 컴파일러에게 알려주는 방법입니다.
다음은 Dart가 지원하는 타입의 예시입니다:
- int
- String
- double
- dynamic
- num

Dart는 그 밖에도 많은 타입을 지원합니다. 위에 나열된 것들이 주로 사용하게 될 주요 타입입니다.

# --instructions--

타입은 무엇을 위한 것입니까?

# --answers--

- 다양한 데이터 타입을 분류합니다
- 값을 저장합니다
- 무언가가 참인지 거짓인지 나타냅니다
- 데이터를 화면에 출력합니다
