# --description--

`is` 키워드를 사용하면 변수가 지정한 타입인지 확인할 수 있습니다. 하지만 `is!` 키워드를 사용하여 변수가 지정한 타입이 아닌지도 확인할 수 있습니다
```dart
final number = 3.14;
print(number is! int); // true
```
