# --description--

Dart의 리스트는 **0부터 인덱싱**됩니다. 즉, 첫 번째 요소는 인덱스 `0`, 두 번째는 인덱스 `1` 등으로 되어 있습니다.

```dart
var colors = ['red', 'green', 'blue'];
print(colors[0]); // red
print(colors[2]); // blue
```

인덱스로 요소에 접근하는 것은 `list[index]` 구문으로 합니다.
