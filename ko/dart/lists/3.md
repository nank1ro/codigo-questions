# --description--

`.add()` 메서드는 리스트의 **끝**에 하나의 요소를 추가합니다:

```dart
var nums = [1, 2, 3];
nums.add(4);
print(nums); // [1, 2, 3, 4]
```

`.add()`는 리스트를 **제자리에서** 수정하고 `void`를 반환한다는 점에 유의하세요.
