# --description--

`.contains()` 메서드는 리스트에 특정 값이 포함되어 있는지 확인합니다. 발견되면 `true`, 그렇지 않으면 `false`를 반환합니다:

```dart
var fruits = ['apple', 'mango', 'grape'];
print(fruits.contains('mango'));  // true
print(fruits.contains('orange')); // false
```

이것은 반복문 없이 포함 여부를 확인하는 데 유용합니다.

# --instructions--
