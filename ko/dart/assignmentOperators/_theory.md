**대입 연산자** `=`는 값을 변수에 저장합니다.

```dart
int x = 5;
String name = 'Alice';
```

일단 할당되면 변수는 다시 할당될 때까지 그 값을 유지합니다.

---

**더하기 대입 연산자** `+=`는 변수에 값을 더하고 결과를 그 변수에 저장합니다.

```dart
int score = 10;
score += 5; // score는 이제 15
```

이는 `score = score + 5`의 약자입니다.

---

**빼기 대입 연산자** `-=`는 변수에서 값을 빼고 결과를 그 변수에 저장합니다.

```dart
int lives = 3;
lives -= 1; // lives는 이제 2
```

이는 `lives = lives - 1`의 약자입니다.

---

**곱하기 대입 연산자** `*=`는 변수에 값을 곱하고 결과를 그 변수에 저장합니다.

```dart
int side = 4;
side *= 3; // side는 이제 12
```

이는 `side = side * 3`의 약자입니다.
