# --description--

**곱하기 대입 연산자** `*=`는 변수에 값을 곱하고 결과를 그 변수에 저장합니다.

```dart
int side = 4;
side *= 3; // side는 이제 12
```

이는 `side = side * 3`의 약자입니다.
