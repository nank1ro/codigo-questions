# --description--

Un ciclo `while` ripete un blocco di codice finché una condizione è `true`.

```dart
int i = 0;
while (i < 3) {
  print(i);
  i++;
}
```

Il ciclo controlla la condizione **prima** di ogni iterazione. Quando `i < 3` diventa `false`, il ciclo si ferma.
