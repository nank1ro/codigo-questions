# --instructions--

Scrivi una funzione **ricorsiva** `factorial` che prende un parametro `int` `n` e restituisce `n!` (il fattoriale di `n`).

Il fattoriale di `n` è definito come:
- `factorial(0) = 1`
- `factorial(n) = n * factorial(n - 1)` per `n > 0`

Esempio:
```dart
print(factorial(5)); // 120
print(factorial(3)); // 6
```

# --solutions--

```dart
int factorial(int n) {
  if (n == 0) return 1;
  return n * factorial(n - 1);
}
```
