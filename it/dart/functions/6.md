# --instructions--

Cosa significa il tipo di ritorno `void` in una funzione Dart?

```dart
void sayHi() {
  print("Hi!");
}
```

# --answers--

- La funzione restituisce il numero zero
- La funzione non restituisce alcun valore
- La funzione restituisce una stringa vuota
- La funzione può restituire qualsiasi tipo

# --solutions--
