The **assignment operator** `=` memorizza un valore in una variabile.

```dart
int x = 5;
String name = 'Alice';
```

Una volta assegnata, la variabile mantiene quel valore fino a quando non viene riassegnata.

---

The **addition assignment operator** `+=` aggiunge un valore a una variabile e memorizza il risultato nella variabile.

```dart
int score = 10;
score += 5; // score is now 15
```

Questo è un'abbreviazione per `score = score + 5`.

---

The **subtraction assignment operator** `-=` sottrae un valore da una variabile e memorizza il risultato nella variabile.

```dart
int lives = 3;
lives -= 1; // lives is now 2
```

Questo è un'abbreviazione per `lives = lives - 1`.

---

The **multiplication assignment operator** `*=` moltiplica una variabile per un valore e memorizza il risultato nella variabile.

```dart
int side = 4;
side *= 3; // side is now 12
```

Questo è un'abbreviazione per `side = side * 3`.

---

Gli operatori di assegnazione combinano un'operazione aritmetica con un'assegnazione in un unico passaggio.

Ad esempio, `x += 3` è un'abbreviazione per `x = x + 3`.
