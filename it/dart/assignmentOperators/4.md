# --description--

L'**operatore di assegnazione di moltiplicazione** `*=` moltiplica una variabile per un valore e memorizza il risultato nella variabile.

```dart
int side = 4;
side *= 3; // side is now 12
```

Questo è un'abbreviazione per `side = side * 3`.
