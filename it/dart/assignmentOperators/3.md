# --description--

L'**operatore di assegnazione di sottrazione** `-=` sottrae un valore da una variabile e memorizza il risultato nella variabile.

```dart
int lives = 3;
lives -= 1; // lives is now 2
```

Questo è un'abbreviazione per `lives = lives - 1`.

# --instructions--

Completa il codice per sottrarre `5` dalla variabile `stock` usando `-=`.
