# --description--

L'**operatore di assegnazione** `=` memorizza un valore in una variabile.

```dart
int x = 5;
String name = 'Alice';
```

Una volta assegnata, la variabile mantiene quel valore fino a quando non viene riassegnata.

# --instructions--

Completa il codice per assegnare il valore `42` alla variabile `answer`.

# --seed--

```dart
void main() {
  int answer [/] 42;
  print(answer);
}
```
