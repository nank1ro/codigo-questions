# --description--

Una variabile booleana può anche contenere il valore `false`.
Usa `false` quando qualcosa non è il caso.

```dart
bool isLoggedIn = false;
print(isLoggedIn); // false
```

Come `true`, anche `false` deve essere scritto in minuscolo.
