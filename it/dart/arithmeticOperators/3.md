# --description--

L'**operatore di moltiplicazione** `*` moltiplica due valori insieme.

```dart
int a = 5;
int b = 6;
int product = a * b; // 30
```

Moltiplicare un `int` per un altro `int` produce un `int`. Moltiplicare con un `double` produce un `double`.
