# --description--

Dart segue il **ordine standard delle operazioni** (precedenza): la moltiplicazione e la divisione vengono valutate prima dell'addizione e della sottrazione.

```dart
int result = 2 + 3 * 4; // 14, not 20
```

Usa le parentesi per cambiare l'ordine:

```dart
int result = (2 + 3) * 4; // 20
```

# --instructions--

Dichiara una variabile `int` di livello superiore denominata `result` che valuta `3 + 4 * 2 - 1`. Lascia che Dart applichi la sua precedenza di operatore predefinita (non aggiungere parentesi).
