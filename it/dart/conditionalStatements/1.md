# --description--

Un'**istruzione `if`** esegue un blocco di codice solo quando una condizione è `true`.

```dart
int age = 20;
if (age >= 18) {
  print('Adult');
}
```

Il codice all'interno delle parentesi graffe viene eseguito solo se la condizione `age >= 18` viene valutata come `true`.
