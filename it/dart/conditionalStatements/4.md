# --description--

**`else if`** consente di testare più condizioni in sequenza. Viene eseguito il primo ramo la cui condizione è `true`, e il resto viene saltato.

```dart
int score = 75;
if (score >= 90) {
  print('A');
} else if (score >= 70) {
  print('B');
} else {
  print('C');
}
// prints: B
```
