# --description--

Un'**istruzione `if-else`** esegue il blocco `if` quando la condizione è `true`, e il blocco `else` quando è `false`.

```dart
int temperature = 10;
if (temperature > 20) {
  print('Warm');
} else {
  print('Cold');
}
// prints: Cold
```

Esattamente uno dei due rami viene sempre eseguito.
