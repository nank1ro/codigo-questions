# --description--

Puoi usare `print()` all'interno di un blocco `if` per visualizzare un messaggio quando una condizione è soddisfatta.

```dart
bool isRaining = true;
if (isRaining) {
  print('Take an umbrella');
}
```

Se `isRaining` è `false`, non viene stampato nulla.
