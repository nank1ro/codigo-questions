# --instructions--

L'**operatore NOT** `!` inverte un valore booleano: `!true` diventa `false` e `!false` diventa `true`.

Dichiara una variabile `bool` di livello superiore chiamata `isOffline` con il valore `false`. Quindi dichiara una variabile `bool` di livello superiore chiamata `isOnline` che memorizza la negazione di `isOffline` usando l'operatore `!`.
