# --description--

L'**operatore OR** `||` combina due espressioni booleane. Restituisce `true` quando **almeno un** lato è `true`. Restituisce `false` solo quando **entrambi** i lati sono `false`.

```dart
bool a = false;
bool b = true;
bool result = a || b; // true

bool c = false;
bool d = false;
bool result2 = c || d; // false
```
