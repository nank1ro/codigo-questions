# --description--

L'**operatore di uguaglianza** `==` confronta due valori e restituisce `true` se sono uguali, o `false` se non lo sono.

```dart
int a = 5;
int b = 5;
bool result = a == b; // true
```

Il risultato di un confronto è sempre un valore `bool`.
