# --description--

L'**operatore di disuguaglianza** `!=` restituisce `true` quando due valori sono **diversi**, e `false` quando sono uguali.

```dart
int x = 3;
int y = 7;
bool result = x != y; // true
```

È il contrario di `==`.

# --instructions--

Complete the code to check if `4` is not equal to `9` and store the result in `isDifferent`.

# --seed--

```dart
void main() {
  bool isDifferent = 4 [/] 9;
  print(isDifferent);
}
```
