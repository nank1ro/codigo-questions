# --description--

Il metodo `.add()` aggiunge un singolo elemento alla **fine** di una lista:

```dart
var nums = [1, 2, 3];
nums.add(4);
print(nums); // [1, 2, 3, 4]
```

Nota che `.add()` modifica la lista **in-place** e restituisce `void`.
