# --description--
Puoi usare la variabile del ciclo all'interno del corpo per tenere traccia del conteggio corrente. Ad esempio, accumulando una somma:

```dart
int total = 0;
for (int i = 1; i <= 5; i++) {
  total += i;
}
```

Dopo il ciclo, `total` è uguale a 15.
