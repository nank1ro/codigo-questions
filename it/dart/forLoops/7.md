# --instructions--
Cosa fa il ciclo `for-in` in Dart?

# --answers--

- Itera su ogni elemento di una collezione
- Itera un numero fisso di volte usando un indice
- Si ripete finché una condizione booleana diventa vera
- Esegue il codice solo se la collezione non è vuota
