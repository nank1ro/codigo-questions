# --description--
L'istruzione `break` esce immediatamente da un ciclo quando una condizione è soddisfatta:

```dart
for (int i = 0; i < 10; i++) {
  if (i == 5) break; // stops at 5
  print(i);
}
```

Questo è utile quando si cerca un valore e si vuole fermarsi non appena viene trovato.
