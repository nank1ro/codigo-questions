# --description--
Un ciclo `for` può contare **al contrario** usando un decremento (`i--`) e una condizione `>=`:

```dart
for (int i = 5; i >= 1; i--) {
  print(i); // 5, 4, 3, 2, 1
}
```

Questo è utile per i conto alla rovescia o per l'iterazione in senso inverso.
