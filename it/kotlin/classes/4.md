# --description--

Le classi possono contenere **metodi** — funzioni che appartengono alla classe e operano sulle sue proprietà:

```kotlin
class Counter(var count: Int) {
    fun increment() {
        count++
    }
    fun value(): Int {
        return count
    }
}
```

I metodi vengono chiamati usando la notazione a punto su un'istanza:

```kotlin
val c = Counter(0)
c.increment()
println(c.value()) // 1
```

# --asserts--

Dopo aver depositato `100.0` in un conto con saldo `0.0`, il saldo deve essere `100.0`

```kotlin
    val account = BankAccount(0.0)
    account.deposit(100.0)
    tryCatch(account.balance == 100.0)
```

Dopo aver prelevato `30.0`, il saldo deve essere `70.0`

```kotlin
    account.withdraw(30.0)
    tryCatch(account.balance == 70.0)
```

class BankAccount(var balance: Double) {
    fun deposit(amount: Double) {
        balance += amount
    }
    fun withdraw(amount: Double) {
        balance -= amount
    }
}
